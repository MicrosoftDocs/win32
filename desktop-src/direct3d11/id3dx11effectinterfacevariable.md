# ID3DX11EffectInterfaceVariable interface

Accesses an interface variable.

## Members

The **ID3DX11EffectInterfaceVariable** interface inherits from [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectInterfaceVariable** also has these types of members:

-   [Methods](#methods)

### Methods

The **ID3DX11EffectInterfaceVariable** interface has these methods.



| Method                                                                      | Description                       |
|:----------------------------------------------------------------------------|:----------------------------------|
| [**GetClassInstance**](id3dx11effectinterfacevariable-getclassinstance.md) | Get a class instance.<br/>  |
| [**SetClassInstance**](id3dx11effectinterfacevariable-setclassinstance.md) | Sets a class instance.<br/> |



 

## Remarks

> [!Note]  
> The DirectX SDK does not supply any compiled binaries for effects. You must use Effects 11 source to build your effects-type application. For more information about using Effects 11 source, see [Differences Between Effects 10 and Effects 11](d3d11-graphics-programming-guide-effects-differences.md).
