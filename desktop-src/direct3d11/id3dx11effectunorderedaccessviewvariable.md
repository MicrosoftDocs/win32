# ID3DX11EffectUnorderedAccessViewVariable interface

Accesses an unordered access view.

## Members

The **ID3DX11EffectUnorderedAccessViewVariable** interface inherits from [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectUnorderedAccessViewVariable** also has these types of members:

-   [Methods](#methods)

### Methods

The **ID3DX11EffectUnorderedAccessViewVariable** interface has these methods.



| Method                                                                                                      | Description                                        |
|:------------------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| [**GetUnorderedAccessView**](id3dx11effectunorderedaccessviewvariable-getunorderedaccessview.md)           | Get an unordered-access-view.<br/>           |
| [**GetUnorderedAccessViewArray**](id3dx11effectunorderedaccessviewvariable-getunorderedaccessviewarray.md) | Get an array of unordered-access-views.<br/> |
| [**SetUnorderedAccessView**](id3dx11effectunorderedaccessviewvariable-setunorderedaccessview.md)           | Set an unordered-access-view.<br/>           |
| [**SetUnorderedAccessViewArray**](id3dx11effectunorderedaccessviewvariable-setunorderedaccessviewarray.md) | Set an array of unordered-access-views.<br/> |



 

## Remarks

> [!Note]  
> The DirectX SDK does not supply any compiled binaries for effects. You must use Effects 11 source to build your effects-type application. For more information about using Effects 11 source, see [Differences Between Effects 10 and Effects 11](d3d11-graphics-programming-guide-effects-differences.md).
