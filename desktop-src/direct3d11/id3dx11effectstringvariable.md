# ID3DX11EffectStringVariable interface

A string-variable interface accesses a string variable.

## Members

The **ID3DX11EffectStringVariable** interface inherits from [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectStringVariable** also has these types of members:

-   [Methods](#methods)

### Methods

The **ID3DX11EffectStringVariable** interface has these methods.



| Method                                                               | Description                         |
|:---------------------------------------------------------------------|:------------------------------------|
| [**GetString**](id3dx11effectstringvariable-getstring.md)           | Get the string.<br/>          |
| [**GetStringArray**](id3dx11effectstringvariable-getstringarray.md) | Get an array of strings.<br/> |



 

## Remarks

> [!Note]  
> The DirectX SDK does not supply any compiled binaries for effects. You must use Effects 11 source to build your effects-type application. For more information about using Effects 11 source, see [Differences Between Effects 10 and Effects 11](d3d11-graphics-programming-guide-effects-differences.md).
