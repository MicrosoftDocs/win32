# Built-in Keying Module Identifiers

The identifiers for the keying modules that are built in to the Windows Filtering Platform (WFP) are each represented by a GUID.

These identifiers are defined as follows.

<dl> <dt>

<span id="FWPM_KEYING_MODULE_IKE"></span><span id="fwpm_keying_module_ike"></span>**FWPM\_KEYING\_MODULE\_IKE**
</dt> <dd> <dl> <dt>



Internet Key Exchange (IKE) keying module.


</dt> </dl> </dd> <dt>

<span id="FWPM_KEYING_MODULE_IKEV2"></span><span id="fwpm_keying_module_ikev2"></span>**FWPM\_KEYING\_MODULE\_IKEV2**
</dt> <dd> <dl> <dt>



Internet Key Exchange version 2 (IKEv2) keying module.

> [!Note]  
> Available only on Windows Server 2008 R2 and Windows 7.

 


</dt> </dl> </dd> <dt>

<span id="FWPM_KEYING_MODULE_AUTHIP"></span><span id="fwpm_keying_module_authip"></span>**FWPM\_KEYING\_MODULE\_AUTHIP**
</dt> <dd> <dl> <dt>



AuthIP keying module.


</dt> </dl> </dd> </dl>
