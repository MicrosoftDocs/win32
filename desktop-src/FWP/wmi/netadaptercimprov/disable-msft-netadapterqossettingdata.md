# Disable method of the MSFT\_NetAdapterQosSettingData class

Disables QoS on the network adapter.

## Syntax


```mof
uint32 Disable(
  [out] string CmdletOutput
);
```



## Parameters

<dl> <dt>

*CmdletOutput* \[out\]
</dt> <dd>

Returns an embedded instance of the [**MSFT\_NetAdapterQosSettingData**](msft-netadapterqossettingdata.md) class.

</dd> </dl>
