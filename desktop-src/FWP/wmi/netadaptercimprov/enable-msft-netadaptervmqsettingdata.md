# Enable method of the MSFT\_NetAdapterVmqSettingData class

Enables the Virtual Machine Queue (VMQ) on the network adapter.

## Syntax


```mof
uint32 Enable(
  [out] string cmdletOutput
);
```



## Parameters

<dl> <dt>

*cmdletOutput* \[out\]
</dt> <dd>

Returns an embedded instance of a [**MSFT\_NetAdapterVmqSettingData**](msft-netadaptervmqsettingdata.md) class.

</dd> </dl>
