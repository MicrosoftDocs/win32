# TTM\_SETTIPTEXTCOLOR message

Sets the text color in a tooltip window.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

New text color.

</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

The return value for this message is not used.

## Remarks

When visual styles are enabled, this message has no effect.
