# TTM\_GETTOOLCOUNT message

Retrieves a count of the tools maintained by a tooltip control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns a count of tools.
