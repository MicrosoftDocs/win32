# RB\_MINIMIZEBAND message

Resizes a band in a rebar control to its smallest size.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Zero-based index of the band to be minimized.

</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

The return value is not used.
