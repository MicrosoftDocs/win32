# TB\_MOVEBUTTON message

Moves a button from one index to another.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Zero-based index of the button to be moved.

</dd> <dt>

*lParam* 
</dt> <dd>

Zero-based index where the button will be moved.

</dd> </dl>

## Return value

Returns nonzero if successful, or zero otherwise.
