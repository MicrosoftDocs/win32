# TTM\_GETTIPTEXTCOLOR message

Retrieves the text color in a tooltip window.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns a **COLORREF** value that represents the text color.

|-------------------------------------|---------------------------------------------------------------------------------------|
