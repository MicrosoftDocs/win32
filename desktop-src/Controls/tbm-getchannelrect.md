# TBM\_GETCHANNELRECT message

Retrieves the size and position of the bounding rectangle for a trackbar's channel. (The channel is the area over which the slider moves. It contains the highlight when a range is selected.)

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>

Pointer to a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure. The message fills this structure with the channel's bounding rectangle, in client coordinates of the trackbar's window.

</dd> </dl>

## Return value

No return value.
