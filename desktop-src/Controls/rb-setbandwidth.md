# RB\_SETBANDWIDTH message

Sets the width for a docked band.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Index of the band.

</dd> <dt>

*lParam* 
</dt> <dd>

New width in pixels.

</dd> </dl>

## Return value

Returns **TRUE** if the value was set and **FALSE** otherwise.
