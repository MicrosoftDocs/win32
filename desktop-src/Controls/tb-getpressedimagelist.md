# TB\_GETPRESSEDIMAGELIST message

Gets the image list that a toolbar control uses to display buttons in a pressed state.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Must be zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Must be zero.

</dd> </dl>

## Return value

Returns the handle to the image list, or **NULL** if no image list is set.

|-------------------------------------|---------------------------------------------------------------------------------------|
