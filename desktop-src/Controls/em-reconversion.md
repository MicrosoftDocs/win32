# EM\_RECONVERSION message

Invokes the Input Method Editor (IME) reconversion dialog box.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Not used; must be zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Not used; must be zero.

</dd> </dl>

## Return value

This message always returns zero.
