# TDM\_SET\_PROGRESS\_BAR\_POS message

Sets the position of the progress bar in a task dialog.

## Parameters

<dl> <dt>

*wParam* \[in\]
</dt> <dd>

An **int** that specifies the new position.

</dd> <dt>

*lParam* \[in\]
</dt> <dd>

Must be zero.

</dd> </dl>

## Return value

Returns the previous position.
