# CBEM\_GETCOMBOCONTROL message

Gets the handle to the child combo box control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns the handle to the combo box control within the ComboBoxEx control.
