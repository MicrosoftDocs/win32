# LVM\_GETGROUPCOUNT message

Gets the number of groups.

## Parameters

<dl> <dt>

*wParam* \[out\]
</dt> <dd>

Not used. Should be 0.

</dd> <dt>

*lParam* 
</dt> <dd>

Not used. Should be 0.

</dd> </dl>
