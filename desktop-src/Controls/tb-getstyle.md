# TB\_GETSTYLE message

Retrieves the styles currently in use for a toolbar control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns a **DWORD** value that is a combination of [toolbar control styles](toolbar-control-and-button-styles.md).

| Requirement | Value |
|-------------------------------------|---------------------------------------------------------------------------------------|
