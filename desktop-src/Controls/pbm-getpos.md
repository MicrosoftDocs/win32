# PBM\_GETPOS message

Retrieves the current position of the progress bar.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns a **UINT** value that represents the current position of the progress bar.
