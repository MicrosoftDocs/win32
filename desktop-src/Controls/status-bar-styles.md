# Status Bar Styles

This section lists the styles, in addition to standard window styles, supported by *status bar* controls.



| Constant                                                                                                                                                         | Description                                                                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SBARS_SIZEGRIP"></span><span id="sbars_sizegrip"></span><dl> <dt>**SBARS\_SIZEGRIP**</dt> </dl> | The status bar control will include a sizing grip at the right end of the status bar. A sizing grip is similar to a sizing border; it is a rectangular area that the user can click and drag to resize the parent window. <br/> |
| <span id="SBT_TOOLTIPS"></span><span id="sbt_tooltips"></span><dl> <dt>**SBT\_TOOLTIPS**</dt> </dl>       | [Version 4.71.](common-control-versions.md)Use this style to enable tooltips.<br/>                                                                                                                                             |
| <span id="SBARS_TOOLTIPS"></span><span id="sbars_tooltips"></span><dl> <dt>**SBARS\_TOOLTIPS**</dt> </dl> | [Version 5.80.](common-control-versions.md)Identical to SBT\_TOOLTIPS. Use this flag for versions 5.00 or later. <br/>                                                                                                         |
