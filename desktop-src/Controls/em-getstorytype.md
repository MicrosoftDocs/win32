# EM\_GETSTORYTYPE message

Gets the story type.


```C++
#define EM_GETSTORYTYPE       (WM_USER + 290)
```



## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

The story index.

</dd> <dt>

*lParam* 
</dt> <dd>

Reserved; must be 0.

</dd> </dl>
