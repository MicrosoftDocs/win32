# EM\_GETOPTIONS message

Retrieves rich edit control options.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Not used; must be zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Not used; must be zero.

</dd> </dl>

## Return value

This message returns a combination of the current option flag values described in the [**EM\_SETOPTIONS**](em-setoptions.md) message.
