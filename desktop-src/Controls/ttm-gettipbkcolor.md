# TTM\_GETTIPBKCOLOR message

Retrieves the background color in a tooltip window.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns a **COLORREF** value that represents the background color.
