# TB\_SETINDENT message

Sets the indentation for the first button in a toolbar control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Value specifying the indentation, in pixels.

</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns nonzero if successful, or zero otherwise.
