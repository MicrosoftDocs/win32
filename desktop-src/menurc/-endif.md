# \#endif

The **\#endif** directive marks the end of a conditional-compilation block defined by a **\#ifdef** directive. One **\#endif** is required for each **\#if**, **\#ifdef**, or **\#ifndef** directive.

``` syntax
#endif
```

This directive has no parameters.
