# IVMVirtualMachine::HasSSE property

\[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the [Hyper-V WMI provider (V2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Determines whether the processor supports the SSE instruction set.

This property is read-only.

## Syntax


```C++
HRESULT get_HasSSE(
  [out, retval] VARIANT_BOOL *sseEnabled
);
```



## Property value

**TRUE** if the processor supported the SSE instruction set, **FALSE** otherwise.
