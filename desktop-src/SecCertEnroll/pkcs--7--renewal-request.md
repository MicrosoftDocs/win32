# PKCS \#7 Renewal Request

The following example contains a PKCS \#7 renewal request. The example was generated by using the Certreq.exe and Certutil.exe tools. The .inf file used as input to Certreq.exe contains the following configuration.

``` syntax
[NewRequest]
RequestType=pkcs7
RenewalCert=248f7d3c00000000086e
```
