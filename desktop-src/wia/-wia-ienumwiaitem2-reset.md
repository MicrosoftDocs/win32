# IEnumWiaItem2::Reset method

Resets the enumeration reference to the first [**IWiaItem2**](-wia-iwiaitem2.md) object.

## Syntax


```C++
HRESULT Reset();
```



## Parameters

This method has no parameters.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S\_OK**. Otherwise, it returns an **HRESULT** error code.
