# Item.ThumbWidth property

The width, in pixels, of the thumbnail image. This property returns -1 if this item does not support thumbnails. Read-only.

This property is read-only.

## Syntax


```JScript
propVal = Item.ThumbWidth
```



## Property value

Variable that receives the width.
