# Raytracing HLSL Resource Types

The following HLSL resource types support the Direct3D 12 raytracing pipeline.

## In this section



| Topic                                                                                                       | Description                                                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RaytracingAccelerationStructure**](/raytracingaccelerationstructure.md)<br/>                              | A resource type that can be declared in HLSL and passed into **TraceRay** to indicate the top-level acceleration resource built using **BuildRaytracingAccelerationStructure**.<br/>                                                                                                                                                                                                                                              |
