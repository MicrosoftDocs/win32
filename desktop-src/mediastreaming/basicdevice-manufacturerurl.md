# BasicDevice.ManufacturerUrl property

Gets the manufacturer URL of the device.

This property is read-only.

## Syntax


```C++
HRESULT get_ManufacturerUrl(
  [out] HSTRING *value
);
```



## Property value

A pointer to the device’s manufacturer URL.
