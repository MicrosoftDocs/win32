# BasicDevice.PresentationUrl property

Gets the presentation URL of the device.

This property is read-only.

## Syntax


```C++
HRESULT get_PresentationUrl(
  [out] HSTRING *value
);
```



## Property value

A pointer to the device’s presentation URL.
