# BasicDevice.ManufacturerName property

Gets the manufacturer name of the device.

This property is read-only.

## Syntax


```C++
HRESULT get_ManufacturerName(
  [out] HSTRING *value
);
```



## Property value

A pointer to the device’s manufacturer name.
