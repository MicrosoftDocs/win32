# XMFLOAT3PK Operators

The [**XMFLOAT3PK**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmfloat3pk?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                               | Description                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operator uint32\_t**](/windows/win32/DirectXPackedVector/?branch=master)<br/> | Returns an instance of `uint32_t` containing the components of the `XMFLOAT3PK` instance in a packed format.<br/> |
| [**operator =**](xmfloat3pk-operator-eq.md)<br/>               | Assigns vector component data to the current instance of `XMFLOAT3PK`.<br/>                                       |
