# XMUSHORTN4 constructors

Constructors exposed by `XMUSHORTN4`.

Constructs instances of the [**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMUSHORTN4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn4-xmushortn4(constfloat)) | Default constructor for <code>XMUSHORTN4</code>. <br> Default constructor for [**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4). <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORTN4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn4-xmushortn4(constfloat)) | Initializes a new instance of <code>XMUSHORTN4</code> from a four element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) from a from a four element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORTN4 (const uint16_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn4-xmushortn4(constuint16_t)) | Initializes a new instance of <code>XMUSHORTN4</code> from a four element <code>uint16_t</code> array argument. <br> This constructor initializes a new instance of [**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) from a from a four element <code>uint16_t</code> array argument. <br> **Note:** This constructor is only available under C++.<br> |

| [**XMUSHORTN4 (uint16_t,uint16_t,uint16_t,uint16_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn4-xmushortn4(uint16_t_uint16_t_uint16_t_uint16_t)) | Initializes a new instance of <code>XMUSHORTN4</code> from four <code>uint16_t</code> arguments. <br> This constructor initializes a new instance of [**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) from four <code>uint16_t</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
