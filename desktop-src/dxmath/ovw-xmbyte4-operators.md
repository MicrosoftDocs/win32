# XMBYTE4 operators

The [**XMBYTE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyte4) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                              | Description                                                                                                        |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**operator =**](/previous-versions/windows/desktop/legacy/ee419283(v=vs.85))<br/> | Assigns the vector component data from one instance of `XMBYTE4` to the current instance of `XMBYTE4`. <br/> |
