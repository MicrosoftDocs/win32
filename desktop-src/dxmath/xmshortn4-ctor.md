# XMSHORTN4 constructors

Constructors exposed by `XMSHORTN4`.

Constructs instances of the [**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMSHORTN4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn4-xmshortn4(constfloat)) | Default constructor for <code>XMSHORTN4</code>. <br> Default constructor for [**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4). <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORTN4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn4-xmshortn4(constfloat)) | Initializes a new instance of <code>XMSHORTN4</code> from a four element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) from a from a four element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORTN4 (const int16_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn4-xmshortn4(constint16_t)) | Initializes a new instance of <code>XMSHORTN4</code> from a four element <code>int16_t</code> array argument. <br> This constructor initializes a new instance of [**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) from a from a four element <code>int16_t</code> array argument. <br> **Note:** This constructor is only available under C++.<br> |

| [**XMSHORTN4 (int16_t,int16_t,int16_t,int16_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn4-xmshortn4(int16_t_int16_t_int16_t_int16_t)) | Initializes a new instance of <code>XMSHORTN4</code> from four <code>int16_t</code> arguments. <br> This constructor initializes a new instance of [**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) from four <code>int16_t</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
