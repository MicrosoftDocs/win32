# XMSHORTN2 constructors

Constructors exposed by `XMSHORTN2`.

Constructs instances of the [**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMSHORTN2 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn2-xmshortn2(constfloat)) | Default constructor for <code>XMSHORTN2</code>. <br> Default constructor for [**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2). <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORTN2 (float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn2-xmshortn2(float_float)) | Initializes a new instance of <code>XMSHORTN2</code> from two normalized <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) from two normalized <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORTN2 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn2-xmshortn2(constfloat)) | Initializes a new instance of <code>XMSHORTN2</code> from a two element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) from a from a two element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORTN2 (const int16_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn2-xmshortn2(constfloat)) | Initializes a new instance of <code>XMSHORTN2</code> from a two element <code>int16_t</code> array argument. <br> This constructor initializes a new instance of [**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) from a from a two element <code>int16_t</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORTN2 (int16_t,int16_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshortn2-xmshortn2(int16_t_int16_t)) | Initializes a new instance of <code>XMSHORTN2</code> from two <code>int16_t</code> arguments. <br> This constructor initializes a new instance of [**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) from two <code>int16_t</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
