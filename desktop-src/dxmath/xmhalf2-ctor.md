# XMHALF2 constructors

Constructors exposed by `XMHALF2`.

Constructs instances of the [**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMHALF2 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmhalf2-xmhalf2(constfloat)) | Default constructor for <code>XMHALF2</code>. <br> Default constructor for [**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2). <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMHALF2 (HALF,HALF)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmhalf2-xmhalf2(half_half)) | Initializes a new instance of <code>XMHALF2</code> from two <code>HALF</code> arguments.<br> This constructor initializes a new instance of [**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) from two <code>HALF</code> arguments.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMHALF2 (const HALF*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmhalf2-xmhalf2(consthalf)) | Initializes a new instance of <code>XMHALF2</code> from a two element <code>HALF</code> array argument. <br> This constructor initializes a new instance of [**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) from a from a two element <code>HALF</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMHALF2 (float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmhalf2-xmhalf2(float_float)) | Initializes a new instance of <code>XMHALF2</code> from two <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) from two <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMHALF2 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmhalf2-xmhalf2(constfloat)) | Initializes a new instance of <code>XMHALF2</code> from a two element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) from a two element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> |
