# XMUBYTE4 constructors

Constructors exposed by `XMUBYTE4`.

Constructs instances of the [**XMUBYTE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmubyte4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMUBYTE4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmubyte4-xmubyte4(constfloat)) | Default constructor for <code>XMUBYTE4</code>.<br> Default constructor for [**XMUBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmubyte4).<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUBYTE4 (Uuint32_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmubyte4-xmubyte4(uint32_t)) | Initializes a new instance of <code>XMUBYTE4</code> from a <code>Uuint32_t</code> variable containing component data in a packed format.<br> This constructor initializes a new instance of [**XMUBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmubyte4) from a <code>Uuint32_t</code> variable containing component data in a packed format.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUBYTE4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmubyte4-xmubyte4(constuint8_t)) | Initializes a new instance of <code>XMUBYTE4</code> from a four element <code>float</code> array argument.<br> This constructor initializes a new instance of [**XMUBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmubyte4) from a four element <code>float</code> array argument.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUBYTE4 (const uint8_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmubyte4-xmubyte4(constfloat)) | Initializes a new instance of <code>XMUBYTE4</code> from a four element <code>int8_t</code> array argument.<br> This constructor initializes a new instance of [**XMUBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmubyte4) from a four element <code>int8_t</code> array argument.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUBYTE4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmubyte4-xmubyte4(float_float_float_float)) | Initializes a new instance of <code>XMUBYTE4</code> from four <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMUBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmubyte4) from four <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUBYTE4 (uint8_t,uint8_t,uint8_t,uint8_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmubyte4-xmubyte4(uint8_t_uint8_t_uint8_t_uint8_t)) | Initializes a new instance of <code>XMUBYTE4</code> from four <code>int8_t</code> arguments. <br> This constructor initializes a new instance of [**XMUBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmubyte4) from four <code>uint8_t</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
