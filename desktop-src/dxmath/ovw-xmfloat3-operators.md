# XMFLOAT3 Operators

The [**XMFLOAT3**](/windows/win32/DirectXMath/?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                               | Description                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmfloat3-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMFLOAT3` to the current instance of `XMFLOAT3`. <br/> |
