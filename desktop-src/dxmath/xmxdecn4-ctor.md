# XMXDECN4 constructors

Constructors exposed by `XMXDECN4`.

Constructs instances of the [**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMXDECN4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdecn4-xmxdecn4(constfloat)) | Default constructor for <code>XMXDECN4</code>. <br> Default constructor for [**XMXDECN4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdecn4). <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMXDECN4 (uint32_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdecn4-xmxdecn4(uint32_t)) | Initializes a new instance of <code>XMXDECN4</code> from a <code>uint32_t</code> variable containing component data in a packed format. <br> This constructor initializes a new instance of [**XMXDECN4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) from a <code>uint32_t</code> variable containing component data in a packed format. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMXDECN4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdecn4-xmxdecn4(constfloat)) | Initializes a new instance of <code>XMXDECN4</code> from a four element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMXDECN4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) from a from a four element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMXDECN4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdecn4-xmxdecn4(float_float_float_float)) | Initializes a new instance of <code>XMXDECN4</code> from four normalized <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMXDECN4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) from four normalized <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
