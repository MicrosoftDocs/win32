# XMMATRIX Extensions

[**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) describes a 4\*4 matrix aligned on a 16-byte boundary that maps to four hardware vector registers.

> [!Note]  
> The features listed here are only available under C++.
