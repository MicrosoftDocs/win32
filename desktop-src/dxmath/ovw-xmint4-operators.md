# XMINT4 Operators

The [**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                             | Description                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmint4-operator-assign)<br/> | Assigns the vector component data from one instance of `XMINT4` to the current instance of `XMINT4`.<br/> |
