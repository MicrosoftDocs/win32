# XMFLOAT3PK constructors

Constructors exposed by `XMFLOAT3PK`.

Constructs instances of the [**XMFLOAT3PK**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMFLOAT3PK ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-xmfloat3pk(constfloat)) | Default constructor for <code>XMFLOAT3PK</code>.<br> Default constructor for [**XMFLOAT3PK**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk).<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMFLOAT3PK (uint32_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-xmfloat3pk(uint32_t)) | Initializes a new instance of <code>XMFLOAT3PK</code> from a <code>uint32_t</code> variable containing component data in a packed format.<br> This constructor initializes a new instance of [**XMFLOAT3PK**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) from a <code>uint32_t</code> variable containing component data in a packed format.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMFLOAT3PK (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-xmfloat3pk(constfloat)) | Initializes a new instance of <code>XMFLOAT3PK</code> from a three element <code>float</code> array argument.<br> This constructor initializes a new instance of [**XMFLOAT3PK**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) from a three element <code>float</code> array argument.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMFLOAT3PK (float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-xmfloat3pk(float_float_float)) | Initializes a new instance of <code>XMFLOAT3PK</code> from three <code>float</code> arguments.<br> This constructor initializes a new instance of [**XMFLOAT3PK**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) from three <code>float</code> arguments.<br> **Note:** This constructor is only available under C++.<br> |
