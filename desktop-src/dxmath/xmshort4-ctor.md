# XMSHORT4 constructors

Constructors exposed by `XMSHORT4`.

Constructs instances of the [**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMSHORT4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshort4-xmshort4(constfloat)) | Default constructor for <code>XMSHORT4</code>. <br> Default constructor for [**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4). <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORT4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshort4-xmshort4(constfloat)) | Initializes a new instance of <code>XMSHORT4</code> from a four element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) from a four element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORT4 (const int16_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshort4-xmshort4(constfloat)) | Initializes a new instance of <code>XMSHORT4</code> from a four element <code>int16_t</code> array argument. <br> This constructor initializes a new instance of [**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) from a from a four element <code>int16_t</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORT4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshort4-xmshort4(float_float_float_float)) | Initializes a new instance of <code>XMSHORT4</code> from four <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) from four <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMSHORT4 (int16_t,int16_t,int16_t,int16_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmshort4-xmshort4(int16_t_int16_t_int16_t_int16_t)) | Initializes a new instance of <code>XMSHORT4</code> from four <code>int16_t</code> arguments. <br> This constructor initializes a new instance of [**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) from four <code>int16_t</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
