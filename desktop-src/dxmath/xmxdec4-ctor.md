# XMXDEC4 constructors

Constructors exposed by `XMXDEC4`.

Constructs instances of the [**XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMXDEC4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdec4-xmxdec4(constfloat)) | Default constructor for <code>XMXDEC4</code>. <br> Default constructor for [**XMXDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdec4) . <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMXDEC4 (uint32_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdec4-xmxdec4(uint32_t)) | Initializes a new instance of <code>XMXDEC4</code> from a <code>uint32_t6</code> variable containing component data in a packed format. <br> This constructor initializes a new instance of [**XMXDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdec4) from a <code>uint32_t</code> variable containing component data in a packed format. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMXDEC4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdec4-xmxdec4(constfloat)) | Initializes a new instance of <code>XMXDEC4</code> from a four element <code>float</code> array argument.<br> This constructor initializes a new instance of [**XMXDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdec4) from a four element <code>float</code> array argument.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMXDEC4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmxdec4-xmxdec4(float_float_float_float)) | Initializes a new instance of <code>XMXDEC4</code> from four <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMXDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdec4) from four <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
