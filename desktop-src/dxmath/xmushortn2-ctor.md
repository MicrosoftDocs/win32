# XMUSHORTN2 constructors

Constructors exposed by `XMUSHORTN2`.

Constructs instances of the [**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMUSHORTN2 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn2-xmushortn2(constfloat)) | Default constructor for <code>XMUSHORTN2</code>.<br> Default constructor for [**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2).<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORTN2 (float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn2-xmushortn2(float_float)) | Initializes a new instance of <code>XMUSHORTN2</code> from two normalized <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) from two normalized <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORTN2 (const uint16_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn2-xmushortn2(constuint16_t)) | Initializes a new instance of <code>XMUSHORTN2</code> from a two element <code>uint16_t</code> array argument. <br> This constructor initializes a new instance of [**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) from a from a two element <code>uint16_t</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORTN2 (uint16_t,uint16_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushortn2-xmushortn2(uint16_t_uint16_t)) | Initializes a new instance of <code>XMUSHORTN2</code> from two <code>uint16_t</code> arguments. <br> This constructor initializes a new instance of [**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) from two <code>uint16_t</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
