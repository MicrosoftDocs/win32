# XMFLOAT4X3 constructors

Constructors exposed by `XMFLOAT4X3`.

Constructs instances of the [**XMFLOAT4X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x3) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMFLOAT4X3 ()**](/windows/desktop/api/directxmath/nf-directxmath-xmfloat4x3-xmfloat4x3) | Default constructor for <code>XMFLOAT4X3</code><br> Default constructor for [**XMFLOAT4X3**](/windows/desktop/api/directxmath/ns-directxmath-xmfloat4x3)<br> **Note:** This is only available for C++ based development.<br> | 
| [**XMFLOAT4X3 (const float*)**](/windows/desktop/api/directxmath/nf-directxmath-xmfloat4x3-xmfloat4x3(constfloat)) | Initializes a new instance of the <code>XMFLOAT4X3</code> structure from a twelve element <code>float</code> array. <br> Initializes a new instance of the [**XMFLOAT4X3**](/windows/desktop/api/directxmath/ns-directxmath-xmfloat4x3) structure from a twelve element <code>float</code> array. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMFLOAT4X3 (float,float,float,float,float,float,float,float,float,float,float,float)**](/windows/desktop/api/directxmath/nf-directxmath-xmfloat4x3-xmfloat4x3(float_float_float_float_float_float_float_float_float_float_float_float)) | Initializes a new instance of the <code>XMFLOAT4X3</code> structure from twelve scalar <code>float</code> values. <br> Initializes a new instance of the [**XMFLOAT4X3**](/windows/desktop/api/directxmath/ns-directxmath-xmfloat4x3) structure from twelve scalar <code>float</code> values. <br> **Note:** This constructor is only available under C++.<br> |
