# DirectXMath Library 2D vector comparison functions

Lists the 2D vector comparison functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equal)<br/>                     | Tests whether two 2D vectors are equal.<br/>                                                                                                                                                                                                        |
| [**XMVector2EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalint)<br/>               | Tests whether two 2D vectors are equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
| [**XMVector2EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalintr)<br/>             | Tests whether two 2D vectors are equal, treating each component as an unsigned integer. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector2EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalr)<br/>                   | Tests whether two 2D vectors are equal. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector2Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greater)<br/>                 | Tests whether one 2D vector is greater than another 2D vector.<br/>                                                                                                                                                                                 |
| [**XMVector2GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequal)<br/>   | Tests whether one 2D vector is greater-than-or-equal-to another 2D vector.<br/>                                                                                                                                                                     |
| [**XMVector2GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequalr)<br/> | Tests whether one 2D vector is greater-than-or-equal-to another 2D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector2GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterr)<br/>               | Tests whether one 2D vector is greater than another 2D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector2IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isinfinite)<br/>           | Tests whether any component of a 2D vector is positive or negative infinity.<br/>                                                                                                                                                                   |
| [**XMVector2IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isnan)<br/>                     | Tests whether any component of a 2D vector is a NaN.<br/>                                                                                                                                                                                           |
| [**XMVector2Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector2less)<br/>                       | Tests whether one 2D vector is less than another 2D vector.<br/>                                                                                                                                                                                    |
| [**XMVector2LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lessorequal)<br/>         | Tests whether one 2D vector is less-than-or-equal-to another 2D vector.<br/>                                                                                                                                                                        |
| [**XMVector2NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2nearequal)<br/>             | Tests whether one 2D vector is near another 2D vector.<br/>                                                                                                                                                                                         |
| [**XMVector2NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequal)<br/>               | Tests whether two 2D vectors are not equal.<br/>                                                                                                                                                                                                    |
| [**XMVector2NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequalint)<br/>         | Test whether two vectors are not equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
