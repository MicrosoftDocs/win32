# XMINT3 Operators

The [**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                             | Description                                                                                                      |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmint3-operator-assign)<br/> | Assigns the vector component data from one instance of `XMINT3` to the current instance of `XMINT3`. <br/> |
