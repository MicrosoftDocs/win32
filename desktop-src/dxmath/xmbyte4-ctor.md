# XMBYTE4 constructors

Constructors exposed by `XMBYTE4`.

Constructs instances of the [**XMBYTE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyte4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMBYTE4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmbyte4-xmbyte4(constfloat)) | Default constructor for <code>XMBYTE4</code>.<br> Default constructor for [**XMBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte4).<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMBYTE4 (uint32_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmbyte4-xmbyte4(uint32_t)) | Initializes a new instance of <code>XMBYTE4</code> from a <code>uint32_t</code> variable containing component data in a packed format.<br> This constructor initializes a new instance of [**XMBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte4) from a <code>uint32_t</code> variable containing component data in a packed format.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMBYTE4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmbyte4-xmbyte4(constfloat)) | Initializes a new instance of <code>XMBYTE4</code> from a four element <code>float</code> array argument.<br> This constructor initializes a new instance of [**XMBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte4) from a four element <code>float</code> array argument.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMBYTE4 (const int8_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmbyte4-xmbyte4(constint8_t)) | Initializes a new instance of <code>XMBYTE4</code> from a four element <code>int8_t</code> array argument. <br> This constructor initializes a new instance of [**XMBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte4) from a from a four element <code>int8_t</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMBYTE4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmbyte4-xmbyte4(float_float_float_float)) | Initializes a new instance of <code>XMBYTE4</code> from four <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte4) from four <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMBYTE4 (int8_t,int8_t,int8_t,int8_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmbyte4-xmbyte4(int8_t_int8_t_int8_t_int8_t)) | Initializes a new instance of <code>XMBYTE4</code> from four <code>int8_t</code> arguments.<br> This constructor initializes a new instance of [**XMBYTE4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte4) from four <code>int8_t</code> arguments.<br> **Note:** This constructor is only available under C++.<br> |
