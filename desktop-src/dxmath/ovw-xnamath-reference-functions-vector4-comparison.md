# DirectXMath Library 4D vector comparison functions

Lists the 4D vector comparison functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector4Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equal)<br/>                     | Tests whether two 4D vectors are equal.<br/>                                                                                                                                                                                                        |
| [**XMVector4EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalint)<br/>               | Tests whether two 4D vectors are equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
| [**XMVector4EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalintr)<br/>             | Tests whether two 4D vectors are equal, treating each component as an unsigned integer. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector4EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalr)<br/>                   | Tests whether two 4D vectors are equal. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector4Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greater)<br/>                 | Tests whether one 4D vector is greater than another 4D vector.<br/>                                                                                                                                                                                 |
| [**XMVector4GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequal)<br/>   | Tests whether one 4D vector is greater-than-or-equal-to another 4D vector.<br/>                                                                                                                                                                     |
| [**XMVector4GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequalr)<br/> | Tests whether one 4D vector is greater-than-or-equal-to another 4D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector4GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterr)<br/>               | Tests whether one 4D vector is greater than another 4D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector4IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isinfinite)<br/>           | Tests whether any component of a 4D vector is positive or negative infinity.<br/>                                                                                                                                                                   |
| [**XMVector4IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isnan)<br/>                     | Tests whether any component of a 4D vector is a NaN.<br/>                                                                                                                                                                                           |
| [**XMVector4Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector4less)<br/>                       | Tests whether one 4D vector is less than another 4D vector.<br/>                                                                                                                                                                                    |
| [**XMVector4LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lessorequal)<br/>         | Tests whether one 4D vector is less-than-or-equal-to another 4D vector.<br/>                                                                                                                                                                        |
| [**XMVector4NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4nearequal)<br/>             | Tests whether one 4D vector is near another 4D vector.<br/>                                                                                                                                                                                         |
| [**XMVector4NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequal)<br/>               | Tests whether two 4D vectors are not equal.<br/>                                                                                                                                                                                                    |
| [**XMVector4NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequalint)<br/>         | Test whether two 4D vectors are not equal, treating each component as an unsigned integer.<br/>                                                                                                                                                     |
