# XMDEC4 constructors

Constructors exposed by `XMDEC4`.

Constructs instances of the [**XMDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdec4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMDEC4 (uint32_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmdec4-xmdec4(uint32_t)) | Initializes a new instance of <code>XMDEC4</code> from a <code>uint32_t</code> variable containing component data in a packed format. <br> This constructor initializes a new instance of [**XMDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmdec4) from a <code>uint32_t</code> variable containing component data in a packed format. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMDEC4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmdec4-xmdec4(constfloat)) | Initializes a new instance of <code>XMDEC4</code> from a four element <code>float</code> array argument.<br> This constructor initializes a new instance of [**XMDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmdec4) from a four element <code>float</code> array argument.<br> **Note:** This constructor is only available under C++.<br> | 
| [**XMDEC4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmdec4-xmdec4(float_float_float_float)) | Initializes a new instance of <code>XMDEC4</code> from four <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmdec4) from four <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMDEC4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmdec4-xmdec4(constfloat)) | Default constructor for <code>XMDEC4</code>. <br> Default constructor for [**XMDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmdec4) . <br> **Note:** This constructor is only available under C++.<br> |
