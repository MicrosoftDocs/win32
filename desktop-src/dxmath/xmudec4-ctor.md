# XMUDEC4 constructors

Constructors exposed by `XMUDEC4`.

Constructs instances of the [**XMUDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudec4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMUDEC4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmudec4-xmudec4(constfloat)) | Default constructor for <code>XMUDEC4</code>. <br> Default constructor for [**XMUDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmudec4) . <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUDEC4 (uint32_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmudec4-xmudec4(uint32_t)) | Initializes a new instance of <code>XMUDEC4</code> from a <code>uint32_t</code> variable containing component data in a packed format. <br> This constructor initializes a new instance of [**XMUDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmudec4) from a <code>uint32_t</code> variable containing component data in a packed format. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUDEC4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmudec4-xmudec4(constfloat)) | Initializes a new instance of <code>XMUDEC4</code> from a four element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMUDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmudec4) from a from a four element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUDEC4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmudec4-xmudec4(float_float_float_float)) | Initializes a new instance of <code>XMUDEC4</code> from four <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMUDEC4**](/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmudec4) from four <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
