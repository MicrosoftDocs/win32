# XMUSHORT4 constructors

Constructors exposed by `XMUSHORT4`.

Constructs instances of the [**XMUSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort4) structure.

> [!Note]  
> The constructors listed here are only available under C++.

 

### Overload list




| Constructor | Description | 
|-------------|-------------|
| [**XMUSHORT4 ()**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushort4-xmushort4(constfloat)) | Default constructor for <code>XMUSHORT4</code>. <br> Default constructor for [**XMUSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort4). <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORT4 (const float*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushort4-xmushort4(constfloat)) | Initializes a new instance of <code>XMUSHORT4</code> from a four element <code>float</code> array argument. <br> This constructor initializes a new instance of [**XMUSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort4) from a four element <code>float</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORT4 (const uint16_t*)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushort4-xmushort4(constuint16_t)) | Initializes a new instance of <code>XMUSHORT4</code> from a four element <code>uint16_t</code> array argument. <br> This constructor initializes a new instance of [**XMUSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort4) from a from a four element <code>uint16_t</code> array argument. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORT4 (float,float,float,float)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushort4-xmushort4(float_float_float_float)) | Initializes a new instance of <code>XMUSHORT4</code> from four <code>float</code> arguments. <br> This constructor initializes a new instance of [**XMUSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort4) from four <code>float</code> arguments. <br> **Note:** This constructor is only available under C++.<br> | 
| [**XMUSHORT4 (uint16_t,uint16_t,uint16_t,uint16_t)**](/windows/desktop/api/directxpackedvector/nf-directxpackedvector-xmushort4-xmushort4(uint16_t_uint16_t_uint16_t_uint16_t)) | Initializes a new instance of <code>XMUSHORT4</code> from four <code>uint16_t</code> arguments. <br> This constructor initializes a new instance of [**XMUSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort4) from a four <code>uint16_t</code> arguments. <br> **Note:** This constructor is only available under C++.<br> |
