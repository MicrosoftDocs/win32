# ITsSbProvisioningPluginNotifySink Methods

The [**ITsSbProvisioningPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioningpluginnotifysink) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**LockVirtualMachine method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-lockvirtualmachine)
</dt> <dd>

Notifies Remote Desktop Connection Broker (RD Connection Broker) that the virtual machine is locked.

</dd> <dt>

[**OnJobCancelled method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcancelled)
</dt> <dd>

Notifies RD Connection Broker that the job is cancelled.

</dd> <dt>

[**OnJobCompleted method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcompleted)
</dt> <dd>

Notifies RD Connection Broker that the job is complete.

</dd> <dt>

[**OnJobCreated method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcreated)
</dt> <dd>

Notifies RD Connection Broker that a provisioning job is created.

</dd> <dt>

[**OnVirtualMachineHostStatusChanged method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged)
</dt> <dd>

Notifies RD Connection Broker that the status of the host of a virtual machine is changed.

</dd> <dt>

[**OnVirtualMachineStatusChanged method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged)
</dt> <dd>

Notifies RD Connection Broker that the status of a virtual machine is changed.

</dd> </dl>
