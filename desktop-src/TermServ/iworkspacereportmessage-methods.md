# IWorkspaceReportMessage Methods

The [**IWorkspaceReportMessage**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage) interface supports the following methods.

## In this section

<dl> <dt>

[**RegisterErrorLogMessage method**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorlogmessage)
</dt> <dd>

Registers the specified error message to use in the UI.

</dd> <dt>

[**IsErrorMessageRegistered method**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-iserrormessageregistered)
</dt> <dd>

Determines whether a specified error message is registered in a specified workspace.

</dd> <dt>

[**RegisterErrorEvent method**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorevent)
</dt> <dd>

Registers the specified error event message to use in the UI.

</dd> </dl>
