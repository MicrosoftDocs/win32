# Disconnect method of the Win32\_VirtualDesktopSession class

Disconnects the virtual desktop session.

## Syntax


```mof
uint32 Disconnect();
```



## Parameters

This method has no parameters.

## Return value

Returns 0 on success, otherwise returns a WMI error code.
