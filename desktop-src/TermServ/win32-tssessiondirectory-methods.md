# Win32\_TSSessionDirectory Methods

The [**Win32\_TSSessionDirectory**](win32-tssessiondirectory.md) class exposes the following methods.

## In this section

<dl> <dt>

[**CreateUserDiskTemplate method**](createuserdisktemplate-win32-tssessiondirectory.md)
</dt> <dd>

Creates a user disk template.

</dd> <dt>

[**DisableUserVhd method**](disableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Disables a user profile VHD.

</dd> <dt>

[**EnableUserVhd method**](enableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Enables a user profile VHD on an RDSH server.

</dd> <dt>

[**GetCurrentRedirectableAddresses method**](getcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Obtains the currently configured list of DNS eligible addresses that can be used for redirection.

</dd> <dt>

[**GetRedirectableAddresses method**](getredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Obtains the entire list of DNS eligible addresses.

</dd> <dt>

[**PingSessionDirectory method**](pingsessiondirectory-win32-tssessiondirectory.md)
</dt> <dd>

Checks whether the Remote Desktop Connection Broker (RD Connection Broker) server is available.

</dd> <dt>

[**SetCurrentRedirectableAddresses method**](setcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Sets the configured list of DNS eligible addresses that can be used for redirection.

</dd> <dt>

[**SetLoadBalancingState method**](setloadbalancingstate-win32-tssessiondirectory.md)
</dt> <dd>

Sets the value to indicate whether the server will participate in RD Connection Broker load balancing.

</dd> <dt>

[**SetServerWeight method**](setserverweight-win32-tssessiondirectory.md)
</dt> <dd>

Sets the server weight value for RD Connection Broker load balancing.

</dd> <dt>

[**SetSessionDirectoryActive method**](win32-tssessiondirectory-setsessiondirectoryactive.md)
</dt> <dd>

The **SetSessionDirectoryActive** method sets the **SessionDirectoryActive** property.

</dd> <dt>

[**SetSessionDirectoryExposeServerIP method**](win32-tssessiondirectory-setsessiondirectoryexposeserverip.md)
</dt> <dd>

Sets the **SessionDirectoryExposeServerIP** property.

</dd> <dt>

[**SetSessionDirectoryProperty method**](win32-tssessiondirectory-setsessiondirectoryproperty.md)
</dt> <dd>

Sets the **SessionDirectoryLocation** property or the **SessionDirectoryClusterName** property for the class.

</dd> <dt>

[**SetTSRedirectorMode method**](settsredirectormode-win32-tssessiondirectory.md)
</dt> <dd>

Sets the value to indicate whether the server will act as a Remote Desktop Services redirector.

</dd> </dl>
