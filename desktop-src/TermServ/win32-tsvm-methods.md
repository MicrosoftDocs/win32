# Win32\_TSVm Methods

The [**Win32\_TSVm Methods**](win32-tsvm.md) class exposes the following methods.

## In this section

<dl> <dt>

[**QueryOfflineInformation method**](queryofflineinformation-win32-tsvm.md)
</dt> <dd>

Retrieves the properties of the current virtual desktop server (VDS), but only when the server is offline.

</dd> <dt>

[**DumpVmInfo method**](dumpvminfo-win32-tsvm.md)
</dt> <dd>

This member is for internal testing purposes, and should not be used in your code.

</dd> <dt>

[**Export method**](export-win32-tsvm.md)
</dt> <dd>

Retrieves the virtual machine session monitoring information.

</dd> </dl>
