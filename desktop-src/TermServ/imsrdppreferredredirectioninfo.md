# IMsRdpPreferredRedirectionInfo interface

Provides a property to control using a redirection server.

## Members

The **IMsRdpPreferredRedirectionInfo** interface inherits from the [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface. **IMsRdpPreferredRedirectionInfo** also has these types of members:

-   [Properties](#properties)

### Properties

The **IMsRdpPreferredRedirectionInfo** interface has these properties.



| Property                                                                                               | Access type           | Description                                                          |
|:-------------------------------------------------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------|
| [**UseRedirectionServerName**](imsrdppreferredredirectioninfo-useredirectionservername.md)<br/> | Read/write<br/> | Gets and sets whether to use the redirection server name.<br/> |
