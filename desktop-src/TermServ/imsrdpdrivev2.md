# IMsRdpDriveV2 interface

Contains information about a drive object. This is an enhancement of the [**IMsRdpDrive**](imsrdpdrive.md) interface.

## Members

The **IMsRdpDriveV2** interface inherits from [**IMsRdpDrive**](imsrdpdrive.md). **IMsRdpDriveV2** also has these types of members:

-   [Properties](#properties)

### Properties

The **IMsRdpDriveV2** interface has these properties.



| Property                                                              | Access type          | Description                                                |
|:----------------------------------------------------------------------|:---------------------|:-----------------------------------------------------------|
| [**DriveLetterIndex**](imsrdpdrivev2-driveletterindex.md)<br/> | Read-only<br/> | Contains the index of the letter for the drive.<br/> |
