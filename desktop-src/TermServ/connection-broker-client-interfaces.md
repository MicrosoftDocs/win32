# Remote Desktop Connection Broker client interfaces

The Remote Desktop Connection Broker client API defines the following interfaces.

## In this section

<dl> <dt>

[**IConnectionBrokerClient**](iconnectionbrokerclient.md)
</dt> <dd>

Provides the methods needed to use the Remote Desktop Connection Broker client.

</dd> <dt>

[**IConnectionBrokerRequest**](iconnectionbrokerrequest.md)
</dt> <dd>

Provides the methods needed to obtain the results of the asynchronous [**IConnectionBrokerClient::GetTargetInfo**](iconnectionbrokerclient-gettargetinfo.md) method.

</dd> </dl>
