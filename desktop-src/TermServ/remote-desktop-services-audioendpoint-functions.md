# Remote Desktop Services AudioEndpoint Functions

The following functions are used with the AudioEndpoint API.

## In this section

<dl> <dt>

[*GetTSAudioEndpointEnumeratorForSession*](gettsaudioendpointenumeratorforsession.md)
</dt> <dd>

Returns a reference to an audio endpoint enumerator for the specified session ID.

</dd> </dl>

## Remarks

The Remote Desktop Services AudioEndpoint API is for use in Remote Desktop scenarios; it is not for client applications.
