# ItsPubPlugin2 Methods

The [**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2) interface exposes the following methods.

## In this section

<dl> <dt>

[**DeletePersonalDesktopAssignment method**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

Called to delete a mapping between the specified user and a virtual machine in a personal virtual desktop collection.

</dd> <dt>

[**GetResource2 method**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

This method is reserved and should always return **E\_NOTIMPL**.

</dd> <dt>

[**GetResource2List method**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

Retrieves a list of resources assigned to the specified user.

</dd> <dt>

[**ResolvePersonalDesktop method**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

Called to resolve a mapping between the specified user and a virtual machine in a personal virtual desktop collection.

</dd> </dl>
