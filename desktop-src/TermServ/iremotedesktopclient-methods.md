# IRemoteDesktopClient methods

The [**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) interface exposes the following methods.

## In this section

<dl> <dt>

[**attachEvent method**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-attachevent)
</dt> <dd>

Attaches an event handler to an event.

</dd> <dt>

[**Connect method**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-connect)
</dt> <dd>

Initiates a connection by using the properties currently set on the Remote Desktop Protocol (RDP) app container client control.

</dd> <dt>

[**DeleteSavedCredentials method**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-deletesavedcredentials)
</dt> <dd>

Deletes saved credentials for the specified remote computer.

</dd> <dt>

[**detachEvent method**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-detachevent)
</dt> <dd>

Detaches an event handler from an event.

</dd> <dt>

[**Disconnect method**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-disconnect)
</dt> <dd>

Disconnects the active connection.

</dd> <dt>

[**Reconnect method**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-reconnect)
</dt> <dd>

Initiates an automatic reconnection of the Remote Desktop Protocol (RDP) app container client control to fit the session to the new width and height.

</dd> <dt>

[**UpdateSessionDisplaySettings method**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-updatesessiondisplaysettings)
</dt> <dd>

Updates the width and height settings for the Remote Desktop Protocol (RDP) app container client control.

</dd> </dl>
