# ITSRemoteProgram2::RemoteApplicationProgram property

The path and file name of the RemoteApp program.

This property is write-only.

## Syntax


```C++
HRESULT put_RemoteApplicationProgram(
  [in] BSTR bstrRemoteApplicationProgram
);
```



## Property value

The path and file name of the RemoteApp program.
