# SynchronizeAllPublishingData method of the Win32\_RDMSManagementData class

Synchronizes all publishing data for Remote Desktop Management Services (RDMS).

## Syntax


```mof
uint32 SynchronizeAllPublishingData();
```



## Parameters

This method has no parameters.

## Return value

Returns 0 on success, otherwise returns a WMI error code.
