# IMsRdpClientAdvancedSettings::ScaleBitmapCachesByBPP property

This property is not supported.

This property is read/write.

## Syntax


```C++
HRESULT put_ScaleBitmapCachesByBPP(
  [in]  LONG bScale
);

HRESULT get_ScaleBitmapCachesByBPP(
  [out] LONG *pbScale
);
```



## Property value

Set this parameter to 0 to disable the feature or a nonzero value to enable the feature.

## Error codes

Returns **S\_FALSE**.
