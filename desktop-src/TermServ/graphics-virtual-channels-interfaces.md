# Graphics virtual channels interfaces

The graphics virtual channels API supports the following interfaces.

## In this section

<dl> <dt>

[**IWRdsGraphicsChannel**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannel)
</dt> <dd>

This interface is used by the RemoteFX graphics services to send and receive data to a virtual graphics channel.

</dd> <dt>

[**IWRdsGraphicsChannelEvents**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelevents)
</dt> <dd>

This interface receives notifications that relate to a graphics virtual channel.

</dd> <dt>

[**IWRdsGraphicsChannelManager**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelmanager)
</dt> <dd>

This interface is used by the RemoteFX graphics services API to create the graphics virtual channels necessary for remoting graphics data.

</dd> </dl>
