# ITsSbTaskPluginNotifySink Methods

The [**ITsSbTaskPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**OnDeleteTaskTime method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime)
</dt> <dd>

Notifies Remote Desktop Connection Broker (RD Connection Broker) that a task has been removed from the queue.

</dd> <dt>

[**OnReportTasks method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks)
</dt> <dd>

Notifies RD Connection Broker of a new task report.

</dd> <dt>

[**OnSetTaskTime method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime)
</dt> <dd>

Notifies RD Connection Broker that a task has been scheduled.

</dd> <dt>

[**OnUpdateTaskStatus method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus)
</dt> <dd>

Notifies RD Connection Broker that the status of a task has changed.

</dd> </dl>
