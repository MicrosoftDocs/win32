# IMsRdpClientAdvancedSettings8 interface

Exposes methods and properties that manage advanced settings of the Remote Desktop ActiveX control.

To obtain an instance of this interface, use the [**IMsRdpClient8::AdvancedSettings9**](imsrdpclient8-advancedsettings9.md) property.

## Members

The **IMsRdpClientAdvancedSettings8** interface inherits from [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md). **IMsRdpClientAdvancedSettings8** also has these types of members:

-   [Properties](#properties)

### Properties

The **IMsRdpClientAdvancedSettings8** interface has these properties.



| Property                                                                                  | Access type           | Description                                                           |
|:------------------------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------|
| [**BandwidthDetection**](imsrdpclientadvancedsettings8-bandwidthdetection.md)<br/> | Read/write<br/> | Specifies if bandwidth changes are automatically detected.<br/> |
