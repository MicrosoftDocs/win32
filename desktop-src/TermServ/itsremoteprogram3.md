# ITSRemoteProgram3 interface

Defines properties and methods for use with a RemoteApp.

## Members

The **ITSRemoteProgram3** interface inherits from **ITSRemoteProgram2**. **ITSRemoteProgram3** also has these types of members:

-   [Methods](#methods)

### Methods

The **ITSRemoteProgram3** interface has these methods.



| Method                                                     | Description                                                  |
|:-----------------------------------------------------------|:-------------------------------------------------------------|
| [**ServerStartApp**](itsremoteprogram3-serverstartapp.md) | Specifies an App to start in the remote session. <br/> |
