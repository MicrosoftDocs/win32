# IMsRdpCameraRedirConfigCollection::RedirectByDefault property

Specifies whether or not any new camera gets redirected by default.

This property is read/write.

## Syntax

```C++
HRESULT put_RedirectByDefault(
    [in] VARIANT_BOOL fRedirect
);

HRESULT get_RedirectByDefault(
    [out, retval] VARIANT_BOOL* pfRedirect
);
```

## Property value

A value that indicates whether or not any new camera gets redirected by default.
