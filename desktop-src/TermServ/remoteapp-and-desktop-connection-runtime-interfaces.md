# RemoteApp and Desktop Connection Runtime interfaces

The RemoteApp and Desktop Connection Runtime API supports interfaces that allow the development of custom clients in RemoteApp and Desktop Connection.

## In this section

<dl> <dt>

[**IWorkspace**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace)
</dt> <dd>

Exposes methods that provide information about a connection in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspace2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace2)
</dt> <dd>

Exposes additional methods that provide information about a connection in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspace3**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace3)
</dt> <dd>

Exposes methods that provide information about a connection in RemoteApp and Desktop Connection, and adds the ability to retrieve or set a claims token.

</dd> <dt>

[**IWorkspaceClientExt**](/windows/desktop/api/Workspaceruntimeclientext/nn-workspaceruntimeclientext-iworkspaceclientext)
</dt> <dd>

Exposes methods that allow the runtime to disconnect a custom client in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspaceRegistration**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspaceregistration)
</dt> <dd>

Exposes methods that add and remove references to custom clients in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspaceRegistration2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspaceregistration2)
</dt> <dd>

Exposes methods that add and remove references to custom clients in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspaceReportMessage**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage)
</dt> <dd>

Exposes methods that support error message handling for remote workspaces.

</dd> <dt>

[**IWorkspaceResTypeRegistry**](/windows/desktop/api/Workspaceax/nn-workspaceax-iworkspacerestyperegistry)
</dt> <dd>

Exposes methods that allow a plug-in to manage third-party file name extensions in RemoteApp and Desktop Connection runtime.

</dd> <dt>

[**IWorkspaceScriptable**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable)
</dt> <dd>

Exposes methods that manage RemoteApp and Desktop Connection credentials and connections.

</dd> <dt>

[**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2)
</dt> <dd>

Exposes methods that manage RemoteApp and Desktop Connection credentials and connections.

</dd> <dt>

[**IWorkspaceScriptable3**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable3)
</dt> <dd>

Exposes methods that manage RemoteApp and Desktop Connection credentials and connections.

</dd> </dl>
