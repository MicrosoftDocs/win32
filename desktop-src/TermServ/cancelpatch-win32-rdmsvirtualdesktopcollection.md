# CancelPatch method of the Win32\_RDMSVirtualDesktopCollection class

Cancels a software update provisioning job for the virtual machines in a virtual desktop collection.

## Syntax


```mof
uint32 CancelPatch();
```



## Parameters

This method has no parameters.

## Return value

Returns 0 on success, otherwise returns a WMI error code.
