# ITsSbTaskPlugin Methods

The [**ITsSbTaskPlugin**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskplugin) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**InitializeTaskPlugin method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-initializetaskplugin)
</dt> <dd>

Initializes a task that is in the queue of a Remote Desktop Connection Broker plugin.

</dd> <dt>

[**SetTaskQueue method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-settaskqueue)
</dt> <dd>

Updates a task in the queue of a Remote Desktop Connection Broker plugin.

</dd> </dl>
