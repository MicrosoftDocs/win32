# IMsRdpClientSecuredSettings2 interface

Defines additional properties of the Remote Desktop ActiveX control that are restricted to specific Internet Explorer URL security zones.

## Members

The **IMsRdpClientSecuredSettings2** interface inherits from [**IMsRdpClientSecuredSettings**](imsrdpclientsecuredsettings-interface.md). **IMsRdpClientSecuredSettings2** also has these types of members:

-   [Properties](#properties)

### Properties

The **IMsRdpClientSecuredSettings2** interface has these properties.



| Property                                                   | Access type           | Description                                                                                                          |
|:-----------------------------------------------------------|:----------------------|:---------------------------------------------------------------------------------------------------------------------|
| [**PCB**](imsrdpclientsecuredsettings2-pcb.md)<br/> | Read/write<br/> | Specifies the preconnection BLOB (PCB) setting to use prior to connecting for transmission to the server.<br/> |
