# Agent Sessions

Agent sessions is a session type similar to [Child sessions](child-sessions.md), used for performing background tasks.

This is set using the "ConnectToAgentSession" and [**IMsRdpExtendedSettings**](imsrdpextendedsettings.md) [put_Property](imsrdpextendedsettings-property.md).
