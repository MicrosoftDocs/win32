# ITsSbServiceNotification Methods

The [**ITsSbServiceNotification**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbservicenotification) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**NotifyServiceFailure method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure)
</dt> <dd>

Notifies registered plug-ins that the Remote Desktop Connection Broker (RD Connection Broker) service has stopped.

</dd> <dt>

[**NotifyServiceSuccess method**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess)
</dt> <dd>

Notifies registered plug-ins that the RD Connection Broker service has started.

</dd> </dl>
