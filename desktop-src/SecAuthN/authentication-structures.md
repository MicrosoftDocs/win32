## SSPI Structures

The following structures, defined in Sspi.h, are used in SSPI functions.



| Structure                                                                   | Description                                                                                                                                                                                                                                            |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CREDSSP\_CRED**](credssp-cred.md)                                       | Specifies authentication data for both Schannel and Negotiate [*security packages*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly)                                                                                      |
| [**SEC\_WINNT\_AUTH\_IDENTITY**](sec-winnt-auth-identity.md)               | Used to pass a particular user name and password to the run-time library for the purpose of authentication.                                                                                                                                            |
| [**SEC\_WINNT\_AUTH\_IDENTITY\_EX**](sec-winnt-auth-identity-ex.md)        | Contains information about a user. Both an ANSI and [*Unicode*](https://msdn.microsoft.com/library/windows/desktop/ms721629#-security-unicode-gly) form of this structure are provided.                                                                                       |
| [**SecBuffer**](secbuffer.md)                                              | Buffer allocated by a transport application to pass to a [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly).                                                                                           |
| [**SecBufferDesc**](secbufferdesc.md)                                      | Array of [**SecBuffer**](secbuffer.md) structures to pass from a transport application to a security package.                                                                                                                                         |
| [**SecPkgContext\_AccessToken**](secpkgcontext-accesstoken.md)             | Contains a handle to the access token of the [*security context*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-context-gly).                                                                                                       |
| [**SecPkgContext\_ClientCreds**](secpkgcontext-clientcreds.md)             | Specifies client credentials when calling the [**QueryContextAttributes (CredSSP)**](querycontextattributes--credssp-.md) function.                                                                                                                   |
| [**SecPkgContext\_ConnectionInfo**](secpkgcontext-connectioninfo.md)       | Contains protocol and cipher information. This structure is used by the [**QueryContextAttributes (General)**](querycontextattributes--general-.md) function.                                                                                         |
| [**SecPkgContext\_CredentialName**](secpkgcontext-credentialname.md)       | Specifies the credential name.                                                                                                                                                                                                                         |
| [**SecPkgContext\_DceInfo**](secpkgcontext-dceinfo.md)                     | Contains authorization data used by DCE services.                                                                                                                                                                                                      |
| [**SecPkgContext\_EapKeyBlock**](secpkgcontext-eapkeyblock.md)             | Contains key data used by the EAP TLS Authentication Protocol.                                                                                                                                                                                         |
| [**SecPkgContext\_Flags**](secpkgcontext-flags.md)                         | Contains information about the flags in the security context.                                                                                                                                                                                          |
| [**SecPkgContext\_IssuerListInfoEx**](secpkgcontext-issuerlistinfoex.md)   | Contains a list of trusted [*certification authorities*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certification-authority-gly) (CAs).                                                                                            |
| [**SecPkgContext\_Lifespan**](secpkgcontext-lifespan.md)                   | Indicates the life span of a security context.                                                                                                                                                                                                         |
| [**SecPkgContext\_Names**](secpkgcontext-names.md)                         | Contains the name of the user associated with a security context.                                                                                                                                                                                      |
| [**SecPkgContext\_NativeNames**](secpkgcontext-nativenames.md)             | Contains the client and server principal names from the outbound ticket.                                                                                                                                                                               |
| [**SecPkgContext\_NegotiationInfo**](secpkgcontext-negotiationinfo.md)     | Contains information about the [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly) that is being set up or has been set up. It also gives the status on the negotiation to set up the security package. |
| [**SecPkgContext\_PackageInfo**](secpkgcontext-packageinfo.md)             | Contains the name of a [*security support provider*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-support-provider-gly) (SSP).                                                                                            |
| [**SecPkgContext\_PasswordExpiry**](secpkgcontext-passwordexpiry.md)       | Contains information about the expiration of a password or other credential.                                                                                                                                                                           |
| [**SecPkgContext\_SessionKey**](secpkgcontext-sessionkey.md)               | Contains information about the session key.                                                                                                                                                                                                            |
| [**SecPkgContext\_Sizes**](secpkgcontext-sizes.md)                         | Contains the sizes of important structures used in the message support functions.                                                                                                                                                                      |
| [**SecPkgContext\_StreamSizes**](secpkgcontext-streamsizes.md)             | Contains the sizes of the various stream attributes for use with the message support functions.                                                                                                                                                        |
| [**SecPkgContext\_TargetInformation**](secpkgcontext-targetinformation.md) | Contains information about the credential used for the security context.                                                                                                                                                                               |
| [**SecPkgCredentials\_Names**](secpkgcredentials-names.md)                 | Holds the name of the user associated with a [*context*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-context-gly).                                                                                                                                  |
| [**SecPkgInfo**](secpkginfo.md)                                            | Provides general information about a [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly), such as its name and capabilities.                                                                            |
| [**SECURITY\_INTEGER**](security-integer.md)                               | Structure to hold a numeric value. It is used in defining other types.                                                                                                                                                                                 |
| [**SecurityFunctionTable**](securityfunctiontable.md)                      | Dispatch table that contains pointers to the functions defined in SSPI.                                                                                                                                                                                |



 

## Schannel Structures

The following structures are defined for use with Schannel.



| Structure                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                        |
|---------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCH\_CRED\_PUBLIC\_CERTCHAIN**](sch-cred-public-certchain.md)         | Contains a single certificate. A certification chain can be built from this certificate.                                                                                                                                                                                                                                                                                           |
| [**SCH\_CRED\_SECRET\_PRIVKEY**](sch-cred-secret-privkey.md)             | Contains [*private key*](https://msdn.microsoft.com/library/windows/desktop/ms721603#-security-private-key-gly) information needed to authenticate a client or server.                                                                                                                                                                                                                                |
| [**SCHANNEL\_CERT\_HASH**](schannel-cert-hash.md)                        | Contains the hash store data for the certificate that Schannel uses.                                                                                                                                                                                                                                                                                                               |
| [**SCHANNEL\_CERT\_HASH\_STORE**](schannel-cert-hash-store.md)           | Contains the hash store data for the certificate that Schannel uses in kernel-mode.                                                                                                                                                                                                                                                                                                |
| [**SCHANNEL\_ALERT\_TOKEN**](schannel-alert-token.md)                    | Generates a [Secure Sockets Layer Protocol](secure-sockets-layer-protocol.md) (SSL) or Transport Layer Security Protocol (TSL) alert to be sent to the target of a call to either the [**InitializeSecurityContext (Schannel)**](initializesecuritycontext--schannel-.md) function or the [**AcceptSecurityContext (Schannel)**](acceptsecuritycontext--schannel-.md) function. |
| [**SCHANNEL\_CLIENT\_SIGNATURE**](schannel-client-signature.md)          | Specifies a client signature when a call to the [**InitializeSecurityContext (Schannel)**](initializesecuritycontext--schannel-.md) function cannot access the private key for a client certificate (in this case, the function returns **SEC\_I\_SIGNATURE\_NEEDED**).                                                                                                           |
| [**SCHANNEL\_CRED**](schannel-cred.md)                                   | Contains the data for an Schannel credential.                                                                                                                                                                                                                                                                                                                                      |
| [**SCHANNEL\_SESSION\_TOKEN**](schannel-session-token.md)                | Specifies whether reconnections are enabled for an authentication session created by calling either the [**InitializeSecurityContext (Schannel)**](initializesecuritycontext--schannel-.md) function or the [**AcceptSecurityContext (Schannel)**](acceptsecuritycontext--schannel-.md) function.                                                                                |
| [**SecPkgContext\_Authority**](secpkgcontext-authority.md)               | Contains the name of the authenticating authority if one is available. It can be a [*certification authority*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certification-authority-gly) (CA) or the name of a server or domain that authenticated the connection.                                                                                               |
| [**SecPkgContext\_ConnectionInfo**](secpkgcontext-connectioninfo.md)     | Contains protocol and cipher information. This structure is used by the [**QueryContextAttributes (General)**](querycontextattributes--general-.md) function.                                                                                                                                                                                                                     |
| [**SecPkgContext\_IssuerListInfoEx**](secpkgcontext-issuerlistinfoex.md) | Contains a list of trusted [*certification authorities*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certification-authority-gly).                                                                                                                                                                                                                              |
| [**SecPkgContext\_KeyInfo**](secpkgcontext-keyinfo.md)                   | Contains information about the [*session keys*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-session-key-gly) used in a [*security context*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-context-gly). This structure has been superseded by the [**SecPkgContext\_ConnectionInfo**](secpkgcontext-connectioninfo.md) structure.                       |
| [**SecPkgContext\_ProtoInfo**](secpkgcontext-protoinfo.md)               | Holds information about the protocol in use.                                                                                                                                                                                                                                                                                                                                       |
| [**SecPkgContext\_SessionAppData**](secpkgcontext-sessionappdata.md)     | Stores application data for a session context.                                                                                                                                                                                                                                                                                                                                     |
| [**SecPkgCred\_CipherStrengths**](secpkgcred-cipherstrengths.md)         | Holds the minimum and maximum strength permitted for the cipher used by the specified Schannel credential.                                                                                                                                                                                                                                                                         |
| [**SecPkgCred\_SupportedAlgs**](secpkgcred-supportedalgs.md)             | Contains identifiers for algorithms permitted with a specified Schannel credential.                                                                                                                                                                                                                                                                                                |
| [**SecPkgCred\_SupportedProtocols**](secpkgcred-supportedprotocols.md)   | Indicates the protocols permitted with a specified Schannel credential.                                                                                                                                                                                                                                                                                                            |
| [**X509Certificate**](x509certificate.md)                                | Represents an [*X.509*](https://msdn.microsoft.com/library/windows/desktop/ms721636#-security-x-509-gly) certificate.                                                                                                                                                                                                                                                                                       |



 

## Custom Security Package Structures

Custom [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly) use the following structures.



| Structure                                                                   | Description                                                                                                                                                                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LSA\_SECPKG\_FUNCTION\_TABLE**](lsa-secpkg-function-table.md)           | A table of pointers to the [*Local Security Authority*](https://msdn.microsoft.com/library/windows/desktop/ms721592#-security-local-security-authority-gly) (LSA) functions that custom security packages can call. |
| [**SECPKG\_CALL\_INFO**](secpkg-call-info.md)                              | Contains information about an executing function call.                                                                                                                                                        |
| [**SECPKG\_CLIENT\_INFO**](secpkg-client-info.md)                          | Contains information about the user of a security package.                                                                                                                                                    |
| [**SECPK\_CONTEXT\_THUNKS**](secpkg-context-thunks.md)                     | Contains information about calls to the security package that will be executed in-process with the LSA.                                                                                                       |
| [**SECPKG\_DLL\_FUNCTIONS**](secpkg-dll-functions.md)                      | Contains the functions available to custom security packages executing in-process with a client/server application.                                                                                           |
| [**SECPKG\_EVENT\_DOMAIN\_CHANGE**](secpkg-parameters.md)                  | Contains [*session*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-session-gly) and computer information. This structure name is an alias for the [**SECPKG\_PARAMETERS**](secpkg-parameters.md) structure. |
| [**SECPKG\_EVENT\_NOTIFY**](secpkg-event-notify.md)                        | Contains information about a security-related event.                                                                                                                                                          |
| [**SECPKG\_EVENT\_PACKAGE\_CHANGE**](secpkg-event-package-change.md)       | Contains information about security package availability and use.                                                                                                                                             |
| [**SECPKG\_EXTENDED\_INFORMATION**](secpkg-extended-information.md)        | Contains extended information about the security package.                                                                                                                                                     |
| [**SECPKG\_FUNCTION\_TABLE**](secpkg-function-table.md)                    | Contains pointers to the functions implemented by security packages.                                                                                                                                          |
| [**SECPKG\_GSS\_INFO**](secpkg-gss-info.md)                                | Contains information on the GSS OID used to identify a security package.                                                                                                                                      |
| [**SECPKG\_MUTUAL\_AUTH\_LEVEL**](secpkg-mutual-auth-level.md)             | Contains information about the mutual authentication level used by a security package.                                                                                                                        |
| [**SECPKG\_PARAMETERS**](secpkg-parameters.md)                             | Contains [*session*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-session-gly) and machine information.                                                                                                     |
| [**SECPKG\_PRIMARY\_CRED**](secpkg-primary-cred.md)                        | Contains [*primary credentials*](https://msdn.microsoft.com/library/windows/desktop/ms721603#-security-primary-credentials-gly) information.                                                                             |
| [**SECPKG\_SUPPLEMENTAL\_CRED**](secpkg-supplemental-cred.md)              | Contains [*supplemental credentials*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-supplemental-credentials-gly) information.                                                              |
| [**SECPKG\_SUPPLEMENTAL\_CRED\_ARRAY**](secpkg-supplemental-cred-array.md) | Contains supplemental credentials information.                                                                                                                                                                |
| [**SECPKG\_USER\_FUNCTION\_TABLE**](secpkg-user-function-table.md)         | Contains the functions implemented by a security package loaded in-process with client/server applications.                                                                                                   |
| [**SecurityUserData**](securityuserdata.md)                                | Contains information about the logged on user.                                                                                                                                                                |
