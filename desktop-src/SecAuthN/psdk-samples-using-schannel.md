# PSDK Samples Using Schannel

This article describes the Schannel SSL/TLS samples included with the Windows Server 2003 Service Pack 1 Platform Software Development Kit (SDK). These samples demonstrate how to implement secure web communication in your Windows applications using the Schannel security package, helping developers build encrypted client-server connections.

## Download and install the Schannel samples

The C++ SSL sample that shipped with the Windows Server 2003 Service Pack 1 (SP1) [Platform SDK](https://www.microsoft.com/download/details.aspx?id=15656) contains a sample web client and web server that use the Schannel [security package](../secgloss/s-gly.md).

After installing the SDK, you can find the **WebServer** and **WebClient** examples in the `Microsoft Platform SDK\Samples\Security\SSPI\SSL` folder.
