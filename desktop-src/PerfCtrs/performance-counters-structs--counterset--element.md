# structs (counterSet) Element

A list of structures that contain the values of the counters in the parent counter set.

``` syntax
<xs:element name="structs"
    type="man:structs"
 />
```

The **structs** element is defined by the [**counterSet**](performance-counters-counterset-complex-type.md) complex type.

## Remarks

Used only by kernel-mode providers.
