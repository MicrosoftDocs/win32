# Disk Management Structures

The following list identifies the structures that are used in disk management.

## In this section



| Structure                                                                                                 | Description                                                                                                                                                                                                                                                                          |
|-----------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CREATE\_DISK**](create-disk-str.md)<br/>                                                        | Contains information that the [**IOCTL\_DISK\_CREATE\_DISK**](ioctl-disk-create-disk.md) control code uses to initialize GUID partition table (GPT), master boot record (MBR), or raw disks.<br/>                                                                             |
| [**CREATE\_DISK\_GPT**](create-disk-gpt-str.md)<br/>                                               | Contains information used by the [**IOCTL\_DISK\_CREATE\_DISK**](ioctl-disk-create-disk.md) control code to initialize GUID partition table (GPT) disks.<br/>                                                                                                                 |
| [**CREATE\_DISK\_MBR**](create-disk-mbr-str.md)<br/>                                               | Contains information that the [**IOCTL\_DISK\_CREATE\_DISK**](ioctl-disk-create-disk.md) control code uses to initialize master boot record (MBR) disks.<br/>                                                                                                                 |
| [**DEVICE\_COPY\_OFFLOAD\_DESCRIPTOR**](device-copy-offload-descriptor.md)<br/>                    | Contains the copy offload capabilities for a storage device.<br/>                                                                                                                                                                                                              |
| [**DEVICE\_LB\_PROVISIONING\_DESCRIPTOR**](device-lb-provisioning-descriptor.md)<br/>              | Contains the thin provisioning capabilities for a storage device.<br/>                                                                                                                                                                                                         |
| [**DEVICE\_SEEK\_PENALTY\_DESCRIPTOR**](device-seek-penalty-descriptor.md)<br/>                    | Used in conjunction with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request to retrieve the seek penalty descriptor data for a device.<br/>                                                                                                   |
| [**DEVICE\_TRIM\_DESCRIPTOR**](device-trim-descriptor.md)<br/>                                     | Used in conjunction with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request to retrieve the trim descriptor data for a device.<br/>                                                                                                           |
| [**DEVICE\_WRITE\_AGGREGATION\_DESCRIPTOR**](device-write-aggregation-descriptor.md)<br/>          | Reserved for system use.<br/>                                                                                                                                                                                                                                                  |
| [**DEVICE\_POWER\_DESCRIPTOR**](device-power-descriptor.md)<br/>                                   | The [**DEVICE\_POWER\_DESCRIPTOR**](device-power-descriptor.md) structure describes the power capabilities of a storage device.<br/>                                                                                                                                          |
| [**DISK\_CACHE\_INFORMATION**](disk-cache-information-str.md)<br/>                                 | Provides information about the disk cache.<br/>                                                                                                                                                                                                                                |
| [**DISK\_CLUSTER\_INFO**](disk-cluster-info.md)<br/>                                               | Represents information maintained on the partition manager about a disk that is part of a cluster.<br/>                                                                                                                                                                        |
| [**DISK\_DETECTION\_INFO**](disk-detection-info-str.md)<br/>                                       | Contains detected drive parameters.<br/>                                                                                                                                                                                                                                       |
| [**DISK\_EX\_INT13\_INFO**](disk-ex-int13-info-str.md)<br/>                                        | Contains extended Int13 drive parameters.<br/>                                                                                                                                                                                                                                 |
| [**DISK\_EXTENT**](disk-extent-str.md)<br/>                                                        | Represents a disk extent.<br/>                                                                                                                                                                                                                                                 |
| [**DISK\_GEOMETRY**](disk-geometry-str.md)<br/>                                                    | Describes the geometry of disk devices and media.<br/>                                                                                                                                                                                                                         |
| [**DISK\_GEOMETRY\_EX**](disk-geometry-ex-str.md)<br/>                                             | Describes the extended geometry of disk devices and media.<br/>                                                                                                                                                                                                                |
| [**DISK\_GROW\_PARTITION**](disk-grow-partition-str.md)<br/>                                       | Contains information used to increase the size of a partition.<br/>                                                                                                                                                                                                            |
| [**DISK\_INT13\_INFO**](disk-int13-info-str.md)<br/>                                               | Contains standard Int13 drive geometry parameters.<br/>                                                                                                                                                                                                                        |
| [**DISK\_PARTITION\_INFO**](disk-partition-info-str.md)<br/>                                       | Contains the disk partition information.<br/>                                                                                                                                                                                                                                  |
| [**DISK\_PERFORMANCE**](disk-performance-str.md)<br/>                                              | Provides disk performance information.<br/>                                                                                                                                                                                                                                    |
| [**DISKQUOTA\_USER\_INFORMATION**](diskquota-user-information-str.md)<br/>                         | Represents the per-user quota information.<br/>                                                                                                                                                                                                                                |
| [**DRIVE\_LAYOUT\_INFORMATION**](drive-layout-information-str.md)<br/>                             | Contains information about the partitions of a drive.<br/>                                                                                                                                                                                                                     |
| [**DRIVE\_LAYOUT\_INFORMATION\_EX**](drive-layout-information-ex-str.md)<br/>                      | Contains extended information about a drive's partitions.<br/>                                                                                                                                                                                                                 |
| [**DRIVE\_LAYOUT\_INFORMATION\_GPT**](drive-layout-information-gpt-str.md)<br/>                    | Contains information about a drive's GUID partition table (GPT) partitions.<br/>                                                                                                                                                                                               |
| [**DRIVE\_LAYOUT\_INFORMATION\_MBR**](drive-layout-information-mbr-str.md)<br/>                    | Provides information about a drive's master boot record (MBR) partitions.<br/>                                                                                                                                                                                                 |
| [**FORMAT\_PARAMETERS**](format-parameters-str.md)<br/>                                            | Contains information used in formatting a contiguous set of disk tracks.<br/>                                                                                                                                                                                                  |
| [**FORMAT\_EX\_PARAMETERS**](format-ex-parameters-str.md)<br/>                                     | Contains information used in formatting a contiguous set of disk tracks. It is used by the [**IOCTL\_DISK\_FORMAT\_TRACKS\_EX**](ioctl-disk-format-tracks-ex.md) control code.<br/>                                                                                           |
| [**GET\_DISK\_ATTRIBUTES**](get-disk-attributes.md)<br/>                                           | Contains the attributes of a disk device.<br/>                                                                                                                                                                                                                                 |
| [**GET\_LENGTH\_INFORMATION**](get-length-information-str.md)<br/>                                 | Contains disk, volume, or partition length information used by the [**IOCTL\_DISK\_GET\_LENGTH\_INFO**](ioctl-disk-get-length-info.md) control code.<br/>                                                                                                                     |
| [**PARTITION\_INFORMATION**](partition-information-str.md)<br/>                                    | Contains information about a disk partition.<br/>                                                                                                                                                                                                                              |
| [**PARTITION\_INFORMATION\_EX**](partition-information-ex-str.md)<br/>                             | Contains partition information for standard *AT-style* master boot record (MBR) and Extensible Firmware Interface (EFI) disks.<br/>                                                                                                                                            |
| [**PARTITION\_INFORMATION\_GPT**](partition-information-gpt-str.md)<br/>                           | Contains **GUID** partition table (GPT) partition information.<br/>                                                                                                                                                                                                            |
| [**PARTITION\_INFORMATION\_MBR**](partition-information-mbr-str.md)<br/>                           | Contains partition information specific to master boot record (MBR) disks.<br/>                                                                                                                                                                                                |
| [**REASSIGN\_BLOCKS**](reassign-blocks-str.md)<br/>                                                | Contains disk block reassignment data.<br/>                                                                                                                                                                                                                                    |
| [**REASSIGN\_BLOCKS\_EX**](reassign-blocks-ex.md)<br/>                                             | Contains disk block reassignment data.<br/>                                                                                                                                                                                                                                    |
| [**SET\_DISK\_ATTRIBUTES**](set-disk-attributes.md)<br/>                                           | Specifies the attributes to be set on a disk device.<br/>                                                                                                                                                                                                                      |
| [**SET\_PARTITION\_INFORMATION**](set-partition-information-str.md)<br/>                           | Contains information used to set a disk partition's type.<br/>                                                                                                                                                                                                                 |
| [**STORAGE\_ACCESS\_ALIGNMENT\_DESCRIPTOR**](storage-access-alignment-descriptor.md)<br/>          | Used in conjunction with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code to retrieve the storage access alignment descriptor data for a device.<br/>                                                                                  |
| [**STORAGE\_ADAPTER\_DESCRIPTOR**](storage-adapter-descriptor.md)<br/>                             | Used with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code to retrieve the storage adapter descriptor data for a device.<br/>                                                                                                          |
| [**STORAGE\_DESCRIPTOR\_HEADER**](storage-descriptor-header.md)<br/>                               | Used in conjunction with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code to retrieve the properties of a storage device or adapter.<br/>                                                                                              |
| [**STORAGE\_DEVICE\_ATTRIBUTES\_DESCRIPTOR**](storage-device-attributes-descriptor.md)<br/>        | Reserved for future use.<br/>                                                                                                                                                                                                                                                  |
| [**STORAGE\_DEVICE\_DESCRIPTOR**](storage-device-descriptor.md)<br/>                               | Used in conjunction with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code to retrieve the storage device descriptor data for a device.<br/>                                                                                            |
| [**STORAGE\_DEVICE\_ID\_DESCRIPTOR**](storage-device-id-descriptor.md)<br/>                        | Used with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code request to retrieve the device ID descriptor data for a device.<br/>                                                                                                        |
| [**STORAGE\_DEVICE\_IO\_CAPABILITY\_DESCRIPTOR**](storage-device-io-capability-descriptor.md)<br/> | The output buffer for the StorageDeviceIoCapabilityProperty as defined in [**STORAGE\_PROPERTY\_ID**](storage-property-id.md).<br/>                                                                                                                                           |
| [**STORAGE\_DEVICE\_POWER\_CAP**](storage-device-power-cap.md)<br/>                                | This structure is used as an input and output buffer for the [**IOCTL\_STORAGE\_DEVICE\_POWER\_CAP**](ioctl-storage-device-power-cap.md).<br/>                                                                                                                                |
| [**STORAGE\_DEVICE\_RESILIENCY\_DESCRIPTOR**](storage-device-resiliency-descriptor.md)<br/>        | Reserved for system use.<br/>                                                                                                                                                                                                                                                  |
| [**STORAGE\_HW\_FIRMWARE\_ACTIVATE**](storage-hw-firmware-activate.md)<br/>                        | This structure contains information about the downloaded firmware to activate.<br/>                                                                                                                                                                                            |
| [**STORAGE\_HW\_FIRMWARE\_DOWNLOAD**](storage-hw-firmware-download.md)<br/>                        | This structure contains a firmware image payload to be downloaded to the target.<br/>                                                                                                                                                                                          |
| [**STORAGE\_HW\_FIRMWARE\_INFO**](storage-hw-firmware-info.md)<br/>                                | This structure contains information about the device firmware.<br/>                                                                                                                                                                                                            |
| [**STORAGE\_HW\_FIRMWARE\_INFO\_QUERY**](storage-hw-firmware-info-query.md)<br/>                   | This structure contains information about the device firmware.<br/>                                                                                                                                                                                                            |
| [**STORAGE\_HW\_FIRMWARE\_SLOT\_INFO**](storage-hw-firmware-slot-info.md)<br/>                     | This structure contains information about a slot on a device.<br/>                                                                                                                                                                                                             |
| [**STORAGE\_MEDIUM\_PRODUCT\_TYPE\_DESCRIPTOR**](storage-medium-product-type-descriptor.md)<br/>   | Used in conjunction with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request to describe the product type of a storage device.<br/>                                                                                                            |
| [**STORAGE\_MINIPORT\_DESCRIPTOR**](storage-miniport-descriptor.md)<br/>                           | Reserved for system use.<br/>                                                                                                                                                                                                                                                  |
| [**STORAGE\_PHYSICAL\_ADAPTER\_DATA**](storage-physical-adapter-data.md)<br/>                      | Describes a physical storage adapter.<br/>                                                                                                                                                                                                                                     |
| [**STORAGE\_PHYSICAL\_DEVICE\_DATA**](storage-physical-device-data.md)<br/>                        | Describes a physical storage device.<br/>                                                                                                                                                                                                                                      |
| [**STORAGE\_PHYSICAL\_NODE\_DATA**](storage-physical-node-data.md)<br/>                            | Specifies the physical device data of a storage node.<br/>                                                                                                                                                                                                                     |
| [**STORAGE\_PHYSICAL\_TOPOLOGY\_DESCRIPTOR**](storage-physical-topology-descriptor.md)<br/>        | The [**STORAGE\_PHYSICAL\_TOPOLOGY\_DESCRIPTOR**](storage-physical-topology-descriptor.md) structure is one of the query result structures returned from an [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request.<br/>                             |
| [**STORAGE\_PROPERTY\_QUERY**](storage-property-query.md)<br/>                                     | Indicates the properties of a storage device or adapter to retrieve as the input buffer passed to the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code.<br/>                                                                               |
| [**STORAGE\_PROTOCOL\_ATA\_DATA\_TYPE**](storage-protocol-ata-data-type.md)<br/>                   | The ATA protocol data type.<br/>                                                                                                                                                                                                                                               |
| [**STORAGE\_PROTOCOL\_COMMAND**](storage-protocol-command.md)<br/>                                 | This structure is used as an input buffer when using the pass-through mechanism to issue a vendor-specific command to a storage device (via [**IOCTL\_STORAGE\_PROTOCOL\_COMMAND**](ioctl-storage-protocol-command.md)).<br/>                                                 |
| [**STORAGE\_PROTOCOL\_DATA\_DESCRIPTOR**](storage-protocol-data-descriptor.md)<br/>                | This structure is used in conjunction with [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) to return protocol-specific data from a storage device or adapter. <br/>                                                                                    |
| [**STORAGE\_PROTOCOL\_NVME\_DATA\_TYPE**](storage-protocol-nvme-data-type.md)<br/>                 | Describes the type of NVMe protocol-specific data that's to be queried during an [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request.<br/>                                                                                                         |
| [**STORAGE\_PROTOCOL\_SPECIFIC\_DATA**](storage-protocol-specific-data.md)<br/>                    | Describes protocol-specific device data, provided in the input and output buffer of an [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request.<br/>                                                                                                   |
| [**STORAGE\_SPEC\_VERSION**](storage-spec-version.md)<br/>                                         | Storage specification version.<br/>                                                                                                                                                                                                                                            |
| [**STORAGE\_TEMPERATURE\_DATA\_DESCRIPTOR**](storage-temperature-data-descriptor.md)<br/>          | This structure is used in conjunction with [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) to return temperature data from a storage device or adapter. <br/>                                                                                          |
| [**STORAGE\_TEMPERATURE\_INFO**](storage-temperature-info.md)<br/>                                 | Describes device temperature data. Returned as part of [**STORAGE\_TEMPERATURE\_DATA\_DESCRIPTOR**](storage-temperature-data-descriptor.md) when querying for temperature data with an [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request. <br/> |
| [**STORAGE\_TEMPERATURE\_THRESHOLD**](storage-temperature-threshold.md)<br/>                       | This structure is used to set the over or under temperature threshold of a storage device (via [**IOCTL\_STORAGE\_SET\_TEMPERATURE\_THRESHOLD**](ioctl-storage-set-temperature-threshold.md)).<br/>                                                                           |
| [**STORAGE\_WRITE\_CACHE\_PROPERTY**](storage-write-cache-property.md)<br/>                        | Used with the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code to retrieve information about a device's write cache property.<br/>                                                                                                         |
| [**VERIFY\_INFORMATION**](verify-information-str.md)<br/>                                          | Contains information used to verify a disk extent.<br/>                                                                                                                                                                                                                        |
