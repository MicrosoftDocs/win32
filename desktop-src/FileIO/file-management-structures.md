# File Management Structures

The following structures are used in file management.

## In this section



| Structure                                                                                         | Description                                                                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BY\_HANDLE\_FILE\_INFORMATION**](by-handle-file-information-str.md)<br/>                | Contains information that the [**GetFileInformationByHandle**](getfileinformationbyhandle.md) function retrieves.<br/>                                                                                                        |
| [**COPYFILE2\_EXTENDED\_PARAMETERS**](copyfile2-extended-parameters.md)<br/>               | Contains extended parameters for the [**CopyFile2**](copyfile2.md) function.<br/>                                                                                                                                             |
| [**COPYFILE2\_MESSAGE**](copyfile2-message.md)<br/>                                        | Passed to the [*CopyFile2ProgressRoutine*](copyfile2progressroutine.md) callback function with information about a pending copy operation.<br/>                                                                               |
| [**CREATEFILE2\_EXTENDED\_PARAMETERS**](createfile2-extended-parameters.md)<br/>           | Contains optional extended parameters for [**CreateFile2**](createfile2.md).<br/>                                                                                                                                             |
| [**CSV\_CONTROL\_PARAM**](csv-control-param.md)<br/>                                       | Represents a type of CSV control operation.<br/>                                                                                                                                                                               |
| [**CSV\_QUERY\_FILE\_REVISION**](csv-query-file-revision.md)<br/>                          | Contains information about whether files in a stream have been modified.<br/>                                                                                                                                                  |
| [**CSV\_QUERY\_MDS\_PATH**](csv-query-mds-path.md)<br/>                                    | Contains the path that is used by CSV to communicate to the MDS.<br/>                                                                                                                                                          |
| [**CSV\_QUERY\_REDIRECT\_STATE**](csv-query-redirect-state.md)<br/>                        | Contains information about whether files in a stream have been redirected.<br/>                                                                                                                                                |
| [**DUPLICATE\_EXTENTS\_DATA**](duplicate-extents-data.md)<br/>                             | Contains parameters for the [**FSCTL\_DUPLICATE\_EXTENTS**](fsctl-duplicate-extents-to-file.md) control code that performs the [Block Cloning](block-cloning.md) operation.<br/>                                             |
| [**EFS\_CERTIFICATE\_BLOB**](efs-certificate-blob-str.md)<br/>                             | Contains a certificate.<br/>                                                                                                                                                                                                   |
| [**EFS\_HASH\_BLOB**](efs-hash-blob-str.md)<br/>                                           | Contains a certificate hash.<br/>                                                                                                                                                                                              |
| [**ENCRYPTION\_CERTIFICATE**](encryption-certificate-str.md)<br/>                          | Contains a certificate and the SID of its owner.<br/>                                                                                                                                                                          |
| [**ENCRYPTION\_CERTIFICATE\_HASH**](encryption-certificate-hash-str.md)<br/>               | Contains a certificate hash and display information for the certificate.<br/>                                                                                                                                                  |
| [**ENCRYPTION\_CERTIFICATE\_HASH\_LIST**](encryption-certificate-hash-list-str.md)<br/>    | Contains a list of certificate hashes.<br/>                                                                                                                                                                                    |
| [**ENCRYPTION\_CERTIFICATE\_LIST**](encryption-certificate-list-str.md)<br/>               | Contains a list of certificates.<br/>                                                                                                                                                                                          |
| [**EXFAT\_STATISTICS**](exfat-statistics.md)<br/>                                          | Contains statistical information from the exFAT file system.<br/>                                                                                                                                                              |
| [**FAT\_STATISTICS**](fat-statistics-str.md)<br/>                                          | Contains statistical information from the FAT file system.<br/>                                                                                                                                                                |
| [**FILE\_ALIGNMENT\_INFO**](file-alignment-info.md)<br/>                                   | Contains alignment information for a file.<br/>                                                                                                                                                                                |
| [**FILE\_ALLOCATED\_RANGE\_BUFFER**](file-allocated-range-buffer-str.md)<br/>              | Indicates a range of bytes in a file.<br/>                                                                                                                                                                                     |
| [**FILE\_ALLOCATION\_INFO**](file-allocation-info.md)<br/>                                 | Contains the total number of bytes that should be allocated for a file.<br/>                                                                                                                                                   |
| [**FILE\_ATTRIBUTE\_TAG\_INFO**](file-attribute-tag-info.md)<br/>                          | Receives the requested file attribute information. Used for any handles.<br/>                                                                                                                                                  |
| [**FILE\_BASIC\_INFO**](file-basic-info.md)<br/>                                           | Contains the basic information for a file. Used for file handles.<br/>                                                                                                                                                         |
| [**FILE\_COMPRESSION\_INFO**](file-compression-info.md)<br/>                               | Receives file compression information.<br/>                                                                                                                                                                                    |
| [**FILE\_DISPOSITION\_INFO**](file-disposition-info.md)<br/>                               | Indicates whether a file should be deleted. Used for any handles.<br/>                                                                                                                                                         |
| [**FILE\_END\_OF\_FILE\_INFO**](file-end-of-file-info.md)<br/>                             | Contains the specified value to which the end of the file should be set. <br/>                                                                                                                                                 |
| [**FILE\_STORAGE\_INFO**](file-storage-info.md)<br/>                                       | Contains directory information for a file.<br/>                                                                                                                                                                                |
| [**FILE\_FULL\_DIR\_INFO**](file-full-dir-info.md)<br/>                                    | Contains directory information for a file.<br/>                                                                                                                                                                                |
| [**FILE\_ID\_128**](ext-file-id-128.md)<br/>                                               | Defines a 128-bit file identifier.<br/>                                                                                                                                                                                        |
| [**FILE\_ID\_BOTH\_DIR\_INFO**](file-id-both-dir-info.md)<br/>                             | Contains information about files in the specified directory.<br/>                                                                                                                                                              |
| [**FILE\_ID\_DESCRIPTOR**](file-id-descriptor.md)<br/>                                     | Specifies the type of ID that is being used.<br/>                                                                                                                                                                              |
| [**FILE\_ID\_EXTD\_DIR\_INFO**](file-id-extd-dir-info.md)<br/>                             | Contains identification information for a file.<br/>                                                                                                                                                                           |
| [**FILE\_ID\_INFO**](file-id-info.md)<br/>                                                 | Contains identification information for a file.<br/>                                                                                                                                                                           |
| [**FILE\_IO\_PRIORITY\_HINT\_INFO**](file-io-priority-hint-info.md)<br/>                   | Specifies the priority hint for a file I/O operation.<br/>                                                                                                                                                                     |
| [**FILE\_LEVEL\_TRIM**](file-level-trim.md)<br/>                                           | Used as input to the [**FSCTL\_FILE\_LEVEL\_TRIM**](fsctl-file-level-trim.md) control code.<br/>                                                                                                                              |
| [**FILE\_LEVEL\_TRIM\_OUTPUT**](file-level-trim-output.md)<br/>                            | Used as output to the [**FSCTL\_FILE\_LEVEL\_TRIM**](fsctl-file-level-trim.md) control code.<br/>                                                                                                                             |
| [**FILE\_LEVEL\_TRIM\_RANGE**](file-level-trim-range.md)<br/>                              | Specifies a range of a file that is to be trimmed.<br/>                                                                                                                                                                        |
| [**FILE\_MAKE\_COMPATIBLE\_BUFFER**](file-make-compatible-buffer.md)<br/>                  | Specifies the disc to close the current session for. This control code is used for UDF file systems. This structure is used for input when calling [**FSCTL\_MAKE\_MEDIA\_COMPATIBLE**](fsctl-make-media-compatible.md).<br/> |
| [**FILE\_NAME\_INFO**](file-name-info.md)<br/>                                             | Receives the file name.<br/>                                                                                                                                                                                                   |
| [**FILE\_OBJECTID\_BUFFER**](file-objectid-buffer-str.md)<br/>                             | Contains an object identifier and user-defined metadata associated with the object identifier.<br/>                                                                                                                            |
| [**FILE\_QUERY\_ON\_DISK\_VOL\_INFO\_BUFFER**](file-query-on-disk-vol-info-buffer.md)<br/> | Receives the volume information from a call to [**FSCTL\_QUERY\_ON\_DISK\_VOLUME\_INFO**](fsctl-query-on-disk-volume-info.md).<br/>                                                                                           |
| [**FILE\_QUERY\_SPARING\_BUFFER**](file-query-sparing-buffer.md)<br/>                      | Contains defect management properties.<br/>                                                                                                                                                                                    |
| [**FILE\_REMOTE\_PROTOCOL\_INFO**](file-remote-protocol-information.md)<br/>               | Contains file remote protocol information.<br/>                                                                                                                                                                                |
| [**FILE\_RENAME\_INFO**](file-rename-info.md)<br/>                                         | Contains the name to which the file should be renamed.<br/>                                                                                                                                                                    |
| [**FILE\_SEGMENT\_ELEMENT**](file-segment-element.md)<br/>                                 | Union that contains a 64-bit value that points to a page of data.<br/>                                                                                                                                                         |
| [**FILE\_SET\_DEFECT\_MGMT\_BUFFER**](file-set-defect-mgmt-buffer.md)<br/>                 | Specifies the defect management state to be set.<br/>                                                                                                                                                                          |
| [**FILE\_SET\_SPARSE\_BUFFER**](file-set-sparse-buffer.md)<br/>                            | Specifies the sparse state to be set.<br/>                                                                                                                                                                                     |
| [**FILE\_STANDARD\_INFO**](file-standard-info.md)<br/>                                     | Receives extended information for the file.<br/>                                                                                                                                                                               |
| [**FILE\_STREAM\_INFO**](file-stream-info.md)<br/>                                         | Receives file stream information for the specified file.<br/>                                                                                                                                                                  |
| [**FILE\_ZERO\_DATA\_INFORMATION**](file-zero-data-information-str.md)<br/>                | Contains a range of a file to set to zeros. <br/>                                                                                                                                                                              |
| [**FILESYSTEM\_STATISTICS**](filesystem-statistics.md)<br/>                                | Contains statistical information from the file system.<br/>                                                                                                                                                                    |
| [**FILESYSTEM\_STATISTICS\_EX**](filesystem-statistics-ex.md)<br/>                         | Contains statistical information from the file system.<br/> Support for this structure started with Windows 10.<br/>                                                                                                     |
| [**FIND\_BY\_SID\_DATA**](find-by-sid-data.md)<br/>                                        | Contains data for the [**FSCTL\_FIND\_FILES\_BY\_SID**](fsctl-find-files-by-sid.md) control code.<br/>                                                                                                                        |
| [**FIND\_BY\_SID\_OUTPUT**](find-by-sid-output.md)<br/>                                    | Represents a file name.<br/>                                                                                                                                                                                                   |
| [**NTFS\_FILE\_RECORD\_INPUT\_BUFFER**](ntfs-file-record-input-buffer-str.md)<br/>         | Contains data for the [**FSCTL\_GET\_NTFS\_FILE\_RECORD**](fsctl-get-ntfs-file-record.md) control code.<br/>                                                                                                                  |
| [**NTFS\_FILE\_RECORD\_OUTPUT\_BUFFER**](ntfs-file-record-output-buffer-str.md)<br/>       | Receives output data from the [**FSCTL\_GET\_NTFS\_FILE\_RECORD**](fsctl-get-ntfs-file-record.md) control code.<br/>                                                                                                          |
| [**NTFS\_STATISTICS**](ntfs-statistics-str.md)<br/>                                        | Contains statistical information from the NTFS file system.<br/>                                                                                                                                                               |
| [**NTFS\_STATISTICS\_EX**](ntfs-statistics-ex.md)<br/>                                     | Contains statistical information from the NTFS file system.<br/> Support for this structure started with Windows 10.<br/>                                                                                                |
| [**OFSTRUCT**](ofstruct-str.md)<br/>                                                       | Contains information about a file that the [**OpenFile**](openfile.md) function opened or attempted to open.<br/>                                                                                                             |
| [**OVERLAPPED\_ENTRY**](overlapped-entry.md)<br/>                                          | Contains the information returned by a call to the [**GetQueuedCompletionStatusEx**](getqueuedcompletionstatusex-func.md) function.<br/>                                                                                      |
| [**REQUEST\_OPLOCK\_INPUT\_BUFFER**](request-oplock-input-buffer.md)<br/>                  | Contains the information to request an opportunistic lock (oplock) or to acknowledge an oplock break with the [**FSCTL\_REQUEST\_OPLOCK**](fsctl-request-oplock.md) control code.<br/>                                        |
| [**REQUEST\_OPLOCK\_OUTPUT\_BUFFER**](request-oplock-output-buffer.md)<br/>                | Contains the opportunistic lock (oplock) information returned by the [**FSCTL\_REQUEST\_OPLOCK**](fsctl-request-oplock.md) control code.<br/>                                                                                 |
| [**WIM\_ENTRY\_INFO**](wim-entry-info.md)<br/>                                             | Defines metadata specific to each WIM data source hosted on a volume.<br/>                                                                                                                                                     |
| [**WIM\_EXTERNAL\_FILE\_INFO**](wim-external-file-info.md)<br/>                            | Defines metadata specific to files provided by WOF\_PROVIDER\_WIM.<br/>                                                                                                                                                        |
| [**WIN32\_FILE\_ATTRIBUTE\_DATA**](win32-file-attribute-data-str.md)<br/>                  | Contains attribute information for a file or directory.<br/>                                                                                                                                                                   |
| [**WIN32\_FIND\_DATA**](win32-find-data-str.md)<br/>                                       | Contains information about the file that is found by the [**FindFirstFile**](findfirstfile.md), [**FindFirstFileEx**](findfirstfileex.md), or [**FindNextFile**](findnextfile.md) function.<br/>                            |
| [**WIN32\_FIND\_STREAM\_DATA**](win32-find-stream-data-str.md)<br/>                        | Contains information about the stream found by the [**FindFirstStreamW**](findfirststreamw.md) or [**FindNextStreamW**](findnextstreamw.md) function.<br/>                                                                   |
| [**WOF\_FILE\_COMPRESSION\_INFO\_V1**](wof-file-compression-info-v1.md)<br/>               | Defines metadata specific to files provided by WOF\_PROVIDER\_FILE.<br/>                                                                                                                                                       |
