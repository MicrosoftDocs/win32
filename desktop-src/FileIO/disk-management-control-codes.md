# Disk Management Control Codes

The following table identifies the control codes that are used in disk management.

## In this section



| Control Code                                                                                              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IOCTL\_DISK\_ARE\_VOLUMES\_READY**](ioctl-disk-are-volumes-ready.md)<br/>                       | Waits for all volumes on the specified disk to be ready for use.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IOCTL\_DISK\_CREATE\_DISK**](ioctl-disk-create-disk.md)<br/>                                    | Initializes the specified disk and disk partition table using the information in the [**CREATE\_DISK**](create-disk-str.md) structure.<br/>                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IOCTL\_DISK\_DELETE\_DRIVE\_LAYOUT**](ioctl-disk-delete-drive-layout.md)<br/>                   | Removes the boot signature from the master boot record, so that the disk will be formatted from sector zero to the end of the disk.<br/>                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IOCTL\_DISK\_FORMAT\_TRACKS**](ioctl-disk-format-tracks.md)<br/>                                | Formats a specified, contiguous set of tracks on a floppy disk. To provide additional parameters, use [**IOCTL\_DISK\_FORMAT\_TRACKS\_EX**](ioctl-disk-format-tracks-ex.md) instead.<br/>                                                                                                                                                                                                                                                                                                                                          |
| [**IOCTL\_DISK\_FORMAT\_TRACKS\_EX**](ioctl-disk-format-tracks-ex.md)<br/>                         | Formats a specified, contiguous set of tracks on a floppy disk.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IOCTL\_DISK\_GET\_CACHE\_INFORMATION**](ioctl-disk-get-cache-information.md)<br/>               | Retrieves the disk cache configuration data.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IOCTL\_DISK\_GET\_CLUSTER\_INFO**](ioctl-disk-get-cluster-info.md)<br/>                         | Retrieves the attributes of the specified disk device.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IOCTL\_DISK\_GET\_DISK\_ATTRIBUTES**](ioctl-disk-get-disk-attributes.md)<br/>                   | Retrieves the attributes of the specified disk device.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IOCTL\_DISK\_GET\_DRIVE\_GEOMETRY**](ioctl-disk-get-drive-geometry.md)<br/>                     | Retrieves information about the physical disk's geometry: type, number of cylinders, tracks per cylinder, sectors per track, and bytes per sector.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [**IOCTL\_DISK\_GET\_DRIVE\_GEOMETRY\_EX**](ioctl-disk-get-drive-geometry-ex.md)<br/>              | Retrieves extended information about the physical disk's geometry: type, number of cylinders, tracks per cylinder, sectors per track, and bytes per sector.<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**IOCTL\_DISK\_GET\_DRIVE\_LAYOUT**](ioctl-disk-get-drive-layout.md)<br/>                         | Retrieves information for each entry in the partition tables for a disk.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**IOCTL\_DISK\_GET\_DRIVE\_LAYOUT\_EX**](ioctl-disk-get-drive-layout-ex.md)<br/>                  | Retrieves extended information for each entry in the partition tables for a disk.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IOCTL\_DISK\_GET\_LENGTH\_INFO**](ioctl-disk-get-length-info.md)<br/>                           | Retrieves the length of the specified disk, volume, or partition.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IOCTL\_DISK\_GET\_PARTITION\_INFO**](ioctl-disk-get-partition-info.md)<br/>                     | Retrieves information about the type, size, and nature of a disk partition.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**IOCTL\_DISK\_GET\_PARTITION\_INFO\_EX**](ioctl-disk-get-partition-info-ex.md)<br/>              | Retrieves extended information about the type, size, and nature of a disk partition.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IOCTL\_DISK\_GROW\_PARTITION**](ioctl-disk-grow-partition.md)<br/>                              | Enlarges the specified partition.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IOCTL\_DISK\_IS\_WRITABLE**](ioctl-disk-is-writable.md)<br/>                                    | Determines whether the specified disk is writable.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IOCTL\_DISK\_PERFORMANCE**](ioctl-disk-performance.md)<br/>                                     | Enables performance counters that provide disk performance information.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IOCTL\_DISK\_PERFORMANCE\_OFF**](ioctl-disk-performance-off.md)<br/>                            | Disables the performance counters that provide disk performance information.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IOCTL\_DISK\_REASSIGN\_BLOCKS**](ioctl-disk-reassign-blocks.md)<br/>                            | Directs the disk device to map one or more blocks to its spare-block pool.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IOCTL\_DISK\_REASSIGN\_BLOCKS\_EX**](ioctl-disk-reassign-blocks-ex.md)<br/>                     | Directs the disk device to map one or more blocks to its spare-block pool.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IOCTL\_DISK\_RESET\_SNAPSHOT\_INFO**](ioctl-disk-reset-snapshot-info.md)<br/>                   | Clears all Volume Shadow Copy Service (VSS) hardware-based shadow copy (also called "snapshot") information from the disk.<br/> To perform this operation, call the [**DeviceIoControl**](https://msdn.microsoft.com/library/windows/desktop/aa363216) function with the following parameters.<br/>                                                                                                                                                                                                                                                                  |
| [**IOCTL\_DISK\_SET\_CACHE\_INFORMATION**](ioctl-disk-set-cache-information.md)<br/>               | Sets the disk configuration data.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IOCTL\_DISK\_SET\_CLUSTER\_INFO**](ioctl-disk-set-cluster-info.md)<br/>                         | Sets the cluster information on a disk.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IOCTL\_DISK\_SET\_DISK\_ATTRIBUTES**](ioctl-disk-set-disk-attributes.md)<br/>                   | Sets the attributes of the specified disk device.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IOCTL\_DISK\_SET\_DRIVE\_LAYOUT**](ioctl-disk-set-drive-layout.md)<br/>                         | Partitions a disk as specified by drive layout and partition information data.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IOCTL\_DISK\_SET\_DRIVE\_LAYOUT\_EX**](ioctl-disk-set-drive-layout-ex.md)<br/>                  | Partitions a disk according to the specified drive layout and partition information data.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IOCTL\_DISK\_SET\_PARTITION\_INFO**](ioctl-disk-set-partition-info.md)<br/>                     | Sets partition information for the specified disk partition.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IOCTL\_DISK\_SET\_PARTITION\_INFO\_EX**](ioctl-disk-set-partition-info-ex.md)<br/>              | Sets partition information for the specified disk partition, including layout information for AT and EFI (Extensible Firmware Interface) partitions.<br/>                                                                                                                                                                                                                                                                                                                                                                           |
| [**IOCTL\_DISK\_UPDATE\_PROPERTIES**](ioctl-disk-update-properties.md)<br/>                        | Invalidates the cached partition table and re-enumerates the device.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IOCTL\_DISK\_VERIFY**](ioctl-disk-verify.md)<br/>                                               | Verifies the specified extent on a fixed disk.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IOCTL\_STORAGE\_DEVICE\_POWER\_CAP**](ioctl-storage-device-power-cap.md)<br/>                   | Windows applications can use this control code to specify a maximum operational power consumption level for a storage device. The OS will do it's best to transition the device to a power state that will not exceed the given maximum. However, this depends on what the device supports. The actual maximum may be less than or greater than the desired maximum.<br/> To perform this operation, call the [**DeviceIoControl**](https://msdn.microsoft.com/library/windows/desktop/aa363216) function with the following parameters.<br/>                        |
| [**IOCTL\_STORAGE\_FIRMWARE\_ACTIVATE**](ioctl-storage-firmware-activate.md)<br/>                  | Windows applications can use this control code to activate a firmware image on a specified device.<br/> To perform this operation, call the [**DeviceIoControl**](https://msdn.microsoft.com/library/windows/desktop/aa363216) function with the following parameters.<br/>                                                                                                                                                                                                                                                                                          |
| [**IOCTL\_STORAGE\_FIRMWARE\_DOWNLOAD**](ioctl-storage-firmware-download.md)<br/>                  | Windows applications can use this control code to download a firmware image to the target device, but not activate it. If the image to be downloaded is larger than the controller’s maximum data transfer size, this IOCTL will have to be called multiple times until the entire image is downloaded.<br/> To perform this operation, call the [**DeviceIoControl**](https://msdn.microsoft.com/library/windows/desktop/aa363216) function with the following parameters.<br/>                                                                                     |
| [**IOCTL\_STORAGE\_FIRMWARE\_GET\_INFO**](ioctl-storage-firmware-get-info.md)<br/>                 | Windows applications can use this control code to query the storage device for detailed firmware information. A successful call will return information about firmware revisions, activity status, as well as read/write attributes for each slot. The amount of data returned will vary based on storage protocol.<br/> To perform this operation, call the [**DeviceIoControl**](https://msdn.microsoft.com/library/windows/desktop/aa363216) function with the following parameters.<br/>                                                                         |
| [**IOCTL\_STORAGE\_PROTOCOL\_COMMAND**](ioctl-storage-protocol-command.md)<br/>                    | Windows applications can use this control code to return properties of a storage device or adapter. The request indicates the kind of information to retrieve, such as inquiry data for a device or capabilities and limitations of an adapter. <br/>                                                                                                                                                                                                                                                                               |
| [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md)<br/>                        | Windows applications can use this control code to return the properties of a storage device or adapter. The request indicates the kind of information to retrieve, such as the inquiry data for a device or the capabilities and limitations of an adapter. [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) can also be used to determine whether the port driver supports a particular property or which fields in the property descriptor can be modified with a subsequent change-property request.<br/> |
| [**IOCTL\_STORAGE\_SET\_TEMPERATURE\_THRESHOLD**](ioctl-storage-set-temperature-threshold.md)<br/> | Windows applications can use this control code to set the temperature threshold of a device (when it's supported by the device). <br/>                                                                                                                                                                                                                                                                                                                                                                                              |
