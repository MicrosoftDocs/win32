# PageBlackGenerationProcessingGrayComponentReplacementExtent

This topic is not current. For the most current information, see the [Print Schema Specification](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Describes the extented beyond neutrals (into chromatic colors) that GCR applies. 0% = Uniform component replacement, 100% = Gray component replacement.

-   [Element Information](#element-information)
-   [Structure Content](#structure-content)

## Element Information



| Name | Value |
|----------------------------|------------------------------------------------------------|
| Element Type <br/>   | ParameterDef<br/>                                    |
| Scoping Prefix <br/> | Page<br/>                                            |
| Notes <br/>          | Linked to PageBlackGenerationProcessing element<br/> |
