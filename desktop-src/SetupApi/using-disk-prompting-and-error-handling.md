# Using Disk Prompting and Error Handling

The dialog boxes created by the setup functions, [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska), [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora), [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora), and [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora) can be used by a setup application, or by callback routines. The following sections describe using the functions in an application, and their corresponding notifications:

-   [Prompting For a Disk](prompting-for-a-disk.md)
-   [Error Handling](error-handling.md)
