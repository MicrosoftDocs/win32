# TBS Data Types

TBS defines the following data types.



| Data type                                                                                                | Description                                                                                                                                                            |
|----------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TBS_HCONTEXT"></span><span id="tbs_hcontext"></span>**TBS\_HCONTEXT**                          | Context object handle.<br/>                                                                                                                                      |
| <span id="TBS_COMMAND_PRIORITY"></span><span id="tbs_command_priority"></span>**TBS\_COMMAND\_PRIORITY** | Priority associated with commands.<br/>                                                                                                                          |
| <span id="TBS_COMMAND_LOCALITY"></span><span id="tbs_command_locality"></span>**TBS\_COMMAND\_LOCALITY** | The locality that the command is submitted from.<br/>                                                                                                            |
| <span id="TBS_OWNERAUTH_TYPE"></span><span id="tbs_ownerauth_type"></span>**TBS\_OWNERAUTH\_TYPE**       | The type of owner authentication.<br/> **Windows Server 2008 R2, Windows 7, Windows Server 2008 and Windows Vista:** This data type is not available.<br/> |
