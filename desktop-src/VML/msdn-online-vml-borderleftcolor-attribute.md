# VML BorderLeftColor Attribute

This topic describes VML, a feature that is deprecated as of Windows Internet Explorer 9. Webpages and applications that rely on VML should be migrated to SVG or other widely supported standards.

> [!Note]  
> As of December 2011, this topic has been archived. As a result, it is no longer actively maintained. For more information, see [Archived Content](/previous-versions/windows/internet-explorer/ie-developer/). For information, recommendations, and guidance regarding the current version of Windows Internet Explorer, see [Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Left border color of an inline shape. Read/write. **VgColor**.

**Applies To**

[Shape](shape-element--vml.md)

**Tag Syntax**

<v: *element* borderleftcolor=" *expression* ">

**Remarks**

Used by Microsoft Word.

Values include:



| Value | Description                                                                         |
|-------|-------------------------------------------------------------------------------------|
| Null  | No border color. Default.                                                           |
| color | Color used for border.                                                              |
| This  | Indicates that **Automatic** was selected in the user interface of the application. |
