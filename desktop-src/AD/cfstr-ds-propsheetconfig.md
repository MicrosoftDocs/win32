# CFSTR\_DS\_PROPSHEETCONFIG

<dl> <dt>

<span id="CFSTR_DS_PROPSHEETCONFIG"></span><span id="cfstr_ds_propsheetconfig"></span>**CFSTR\_DS\_PROPSHEETCONFIG**
</dt> <dd> <dl> <dt>

"DsPropSheetCfgClipFormat"
</dt> <dt>



The **CFSTR\_DS\_PROPSHEETCONFIG** clipboard format provides a global memory handle (**HGLOBAL**) that contains a [**PROPSHEETCFG**](propsheetcfg.md) structure.

> [!Note]  
> This clipboard format value is not defined in a published header file. To use this clipboard format value, you must define it yourself in the exact format shown.
