# CFSTR\_DS\_MULTISELECTPROPPAGE

<dl> <dt>

<span id="CFSTR_DS_MULTISELECTPROPPAGE"></span><span id="cfstr_ds_multiselectproppage"></span>**CFSTR\_DS\_MULTISELECTPROPPAGE**
</dt> <dd> <dl> <dt>

"DsAdminMultiSelectClipFormat"
</dt> <dt>



The **CFSTR\_DS\_MULTISELECTPROPPAGE** clipboard format provides a global memory handle (**HGLOBAL**) that contains a null-terminated Unicode string that contains a unique name that is used to create the notification object for a multi-selection property sheet extension.

> [!Note]  
> This clipboard format value is not defined in a published header file. To use this clipboard format value, you must define it yourself in the exact format shown.

 


</dt> </dl> </dd> </dl>
