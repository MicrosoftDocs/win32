# RTLRO Dialog Style Bit

If this style bit is set the text in the dialog box is displayed in right-to-left-reading order.

## Value



| Decimal | Hexadecimal | Constant                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **msidbDialogAttributesRTLRO** |
