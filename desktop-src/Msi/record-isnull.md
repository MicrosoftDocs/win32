# Record.IsNull property

The **IsNull** property of the [**Record**](record-object.md) object is a read-only property that returns True if the indicated field is Null and False if the field contains data.

This property is read-only.

## Syntax


```JScript
propVal = Record.IsNull
```



## Property value

Required field number of the value within the record, 1-based.
