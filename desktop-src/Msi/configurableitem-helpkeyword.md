# ConfigurableItem.HelpKeyword property

The **HelpKeyword** property returns the value from the HelpKeyword column of the [ModuleConfiguration table](moduleconfiguration-table.md).

This property is read-only.

## Syntax


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## Property value

## C++

See [**get\_HelpKeyword**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helpkeyword) function.
