# LeftScroll Dialog Style Bit

If this style bit is set, the scroll bar is located on the left side of the dialog box.

## Value



| Decimal | Hexadecimal | Constant                            |
|---------|-------------|-------------------------------------|
| 512     | 0x00000200  | **msidbDialogAttributesLeftScroll** |
