# R (Windows Installer)

[A](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) [F](f-gly.md) [G](g-gly.md) H [I](i-gly.md) J K L [M](m-gly.md) N [O](o-gly.md) [P](p-gly.md) [Q](q-gly.md) R [S](s-gly.md) [T](t-gly.md) [U](u-gly.md) [V](v-gly.md) W X Y Z

<dl> <dt>

<span id="_msi_reduced_ui_gly"></span><span id="_MSI_REDUCED_UI_GLY"></span>**reduced UI**
</dt> <dd>

Level of the installer's [*internal user interface*](i-gly.md) capabilities. Displays only authored modeless dialog boxes. For more information, see [User Interface Levels](user-interface-levels.md).

</dd> <dt>

<span id="_msi_reinstall_gly"></span><span id="_MSI_REINSTALL_GLY"></span>**reinstall**
</dt> <dd>

Partial or complete reinstallation of an installed application. Commonly done to repair an application if any files or registry entries have become corrupted or are missing. For more information, see [Reinstalling a Feature or Application](reinstalling-a-feature-or-application.md).

</dd> <dt>

<span id="_msi_resiliency_gly"></span><span id="_MSI_RESILIENCY_GLY"></span>**resiliency**
</dt> <dd>

Application capability of graceful recovery from error without generating error messages. For more information, see [Resiliency](resiliency.md).

</dd> <dt>

<span id="_msi_rollback_gly"></span><span id="_MSI_ROLLBACK_GLY"></span>**rollback**
</dt> <dd>

Automatic restoration of the original state of the computer should the installation fail. For more information, see [Rollback Installation (Windows Installer)](rollback-installation.md).

</dd> </dl>
