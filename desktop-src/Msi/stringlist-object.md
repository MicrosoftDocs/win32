# StringList object

The **StringList** object is a collection of strings. The client must verify that the **StringList** object exists and is not empty before referencing its properties.

## Members

The **StringList** object has these types of members:

-   [Properties](#properties)

### Properties

The **StringList** object has these properties.



| Property                                     | Description                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**Count**](stringlist-count.md)<br/> | Returns the number of items in the **StringList** object.<br/> |
| [**Item**](stringlist-item.md)<br/>   | Returns a string in the **StringList** object collection.<br/> |
