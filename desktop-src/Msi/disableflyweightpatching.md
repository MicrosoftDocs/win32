# DisableFlyWeightPatching

If this per-machine system policy is set to 1 (one), all [Patch Optimization](patch-optimization.md) options are turned off during the installation.

The [patch optimization](patch-optimization.md) options and DisableFlyWeightPatching policy are available beginning with Windows Installer version 3.0.

## Registry Key

**HKEY\_LOCAL\_MACHINE**\\**Software**\\**Policies**\\**Microsoft**\\**Windows**\\**Installer**

## Data Type

**REG\_DWORD**
