# AdminToolsFolder property

The **AdminToolsFolder** property contains the full path to the file system directory that stores administrative tools. Microsoft Management Console (MMC) saves customized consoles to this directory and roams with the user. If the [**ALLUSERS**](allusers.md) Property is set, this property points to the file system directory that contains the administrative tools for all users of the computer.

## Requirements



| Requirement | Value |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Version<br/> | Windows Installer 5.0 on Windows Server 2012, Windows 8, Windows Server 2008 R2 or Windows 7. Windows Installer 4.0 or Windows Installer 4.5 on Windows Server 2008 or Windows Vista. Windows Installer on Windows Server 2003 or Windows XP. See the [Windows Installer Run-Time Requirements](windows-installer-portal.md) for information about the minimum Windows service pack that is required by a Windows Installer version.<br/> |
