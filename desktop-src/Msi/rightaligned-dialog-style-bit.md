# RightAligned Dialog Style Bit

If this style bit is set, the text is aligned on the right side of the dialog box.

## Value



| Decimal | Hexadecimal | Constant                              |
|---------|-------------|---------------------------------------|
| 256     | 0x00000100  | **msidbDialogAttributesRightAligned** |
