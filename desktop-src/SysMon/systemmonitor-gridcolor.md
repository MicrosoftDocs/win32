# SystemMonitor.GridColor property

Retrieves or sets the color of the grid lines used in the graph.

This property is read-only.

## Syntax


```VB
Property GridColor As stdole.OLE_COLOR
```



## Property value

Color of the horizontal and vertical grid lines used in a graph if the [**ShowHorizontalGrid**](systemmonitor-showhorizontalgrid.md) or [**ShowVerticalGrid**](systemmonitor-showverticalgrid.md) properties are selected.
