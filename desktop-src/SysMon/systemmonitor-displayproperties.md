# SystemMonitor::DisplayProperties method

Displays the **Graph Properties** dialog box.

## Syntax


```VB
Sub DisplayProperties()
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.

## Remarks

This dialog box lets the user modify the properties of the System Monitor control.
