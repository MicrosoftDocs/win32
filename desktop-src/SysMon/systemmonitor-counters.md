# SystemMonitor.Counters property

Retrieves the **Counters** which contains a collection of [**CounterItem**](counteritem.md) objects.

This property is read-only.

## Syntax


```VB
Property Counters As ICounters
```



## Property value

A **Counters** object that contains a collection of [**CounterItem**](counteritem.md) objects.

## Remarks

This is the default property of the [**SystemMonitor**](systemmonitor.md) object.
