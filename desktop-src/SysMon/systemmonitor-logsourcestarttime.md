# SystemMonitor.LogSourceStartTime property

Retrieves the time stamp of the earliest counter value from a counter in your counter collection that was logged in the log files.

## Syntax


```VB
Property LogSourceStartTime As Date
```



## Property value

Time stamp of the earliest counter value from a counter in your counter collection that was logged in the log files.
