# SystemMonitor.LogSourceStopTime property

Retrieves the time stamp of the latest counter value from a counter in your counter collection that was logged in the log files.

## Syntax


```VB
Property LogSourceStopTime As Date
```



## Property value

Time stamp of the latest counter value from a counter in your counter collection that was logged in the log files.
