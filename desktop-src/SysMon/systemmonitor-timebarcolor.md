# SystemMonitor.TimeBarColor property

Retrieves or sets the color of the time bar. The time bar is the vertical bar that moves across the graph window to indicate the passage of each sampling interval in the line graph view.

This property is read-only.

## Syntax


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## Property value

Color of the time bar.
