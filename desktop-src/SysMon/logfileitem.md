# LogFileItem object

Use this class to retrieve the path of a log file accessed by the System Monitor control.

To retrieve this object, call [**LogFiles.Item**](systemmonitor-logfiles-item.md).

## Members

The **LogFileItem** object has these types of members:

-   [Properties](#properties)

### Properties

The **LogFileItem** object has these properties.



| Property                                    | Description                                    |
|:--------------------------------------------|:-----------------------------------------------|
| [**Path**](logfileitem-path.md)<br/> | Retrieves the path of the log file.<br/> |
