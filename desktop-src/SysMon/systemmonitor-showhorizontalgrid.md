# SystemMonitor.ShowHorizontalGrid property

Retrieves or sets a value that determines whether horizontal grid lines are displayed in the graph.

This property is read-only.

## Syntax


```VB
Property ShowHorizontalGrid As Boolean
```



## Property value

True indicates that horizontal grid lines are displayed in the graph; otherwise, false. The default value for this property is false.
