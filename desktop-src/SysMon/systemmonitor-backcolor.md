# SystemMonitor.BackColor property

Retrieves or sets the background color of the graph and report views.

This property is read-only.

## Syntax


```VB
Property BackColor As stdole.OLE_COLOR
```



## Property value

Background color of the graph and report views.

## Remarks

This is an ambient property. The value of this property is determined by the container. Setting the value of this property could affect the illusion of the control and container being a single application.
