# SystemMonitor.ShowScaleLabels property

Retrieves or sets a value that determines whether the scale labels are displayed on the vertical axis of the graph.

This property is read-only.

## Syntax


```VB
Property ShowScaleLabels As Boolean
```



## Property value

True indicates that the scale labels are displayed on the vertical axis of the graph; otherwise, false. The default value for this property is true.
