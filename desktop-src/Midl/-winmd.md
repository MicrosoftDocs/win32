# /winmd switch

The [**/winmd**](-metadata-dir.md) switch specifies the name of the output metadata (.winmd) file.

``` syntax
midlrt /winmd metadata_filename
```

## Switch Options

<dl> <dt>

*metadata\_filename* 
</dt> <dd>

Specifies the name of the metadata file.

</dd> </dl>

## Remarks

## Examples

**midlrt /winmd file.winmd**
