# /nomd switch

The **/nomd** switch specifies that no metadata (.winmd) file is generated.

``` syntax
midlrt /nomd
```

## Switch Options

This switch has no parameters.

## Remarks

## Examples

**midlrt /nomd**

## Requirements



| Requirement | Value |
|-------------------|--------------------------------|
| Client<br/> | Windows 8<br/>           |
| Server<br/> | Windows Server 2012<br/> |
