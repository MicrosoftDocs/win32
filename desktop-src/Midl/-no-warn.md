# /no\_warn switch

The **/no\_warn** switch directs the MIDL compiler to suppress warning messages.

``` syntax
midl /no_warn
```

## Switch Options

This switch has no parameters.

## Remarks

The use of the **/no\_warn** switch is equivalent to **/W0**.

## Examples

**midl /no\_warn filename.idl**

**midl /W0 filename.idl**
