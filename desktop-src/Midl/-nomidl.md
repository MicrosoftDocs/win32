# /nomidl switch

The **/nomidl** switch suppresses running MIDL.EXE after processing the Windows Runtime IDL file.

``` syntax
midlrt /nomidl 
```

## Switch Options

This switch has no parameters.

## Remarks

## Examples

**midlrt /nomidl**

## Requirements



| Requirement | Value |
|-------------------|--------------------------------|
| Client<br/> | Windows 8<br/>           |
| Server<br/> | Windows Server 2012<br/> |
