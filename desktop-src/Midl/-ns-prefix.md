# /ns\_prefix switch

The **/ns\_prefix** switch prepends the "ABI" prefix to all types.

``` syntax
midlrt /ns_prefix 
```

## Switch Options

This switch has no parameters.

## Remarks

## Examples

**midlrt /ns\_prefix**
