# /winrt switch

The [**/winrt**](-winmd.md) switch enables Windows Runtime semantics.

``` syntax
midlrt /winrt 
```

## Switch Options

This switch has no parameters.

## Remarks

## Examples

**midlrt /winrt**

## Requirements



| Requirement | Value |
|-------------------|--------------------------------|
| Client<br/> | Windows 8<br/>           |
| Server<br/> | Windows Server 2012<br/> |
