# /partial switch

The **/partial** switch specifies that unresolved types are resolved against metadata files that are listed by the [**/metadata\_dir**](-mdmerge-metadata-dir.md) switch.

``` syntax
mdmerge /partial 
```

## Switch Options

This switch has no parameters.

## Remarks

## Examples

**mdmerge /partial**

| Requirement | Value |
|-------------------|--------------------------------|
