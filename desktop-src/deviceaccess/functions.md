# Device Access API functions

Functions in the Device Access API.

## In this section

| Topic | Description |
|---|---|
| [**CreateDeviceAccessInstance**](/windows/win32/api/deviceaccess/nf-deviceaccess-createdeviceaccessinstance)<br/> | Creates the object that's used to access a device. The instantiated object implements the [**IDeviceIoControl**](/windows/win32/api/Deviceaccess/nn-deviceaccess-ideviceiocontrol) and [**ICreateDeviceAccessAsync**](/windows/win32/api/Deviceaccess/nn-deviceaccess-icreatedeviceaccessasync) interfaces.<br/> |
