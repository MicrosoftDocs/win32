# SapCreateInstance Method

Creates a communication channel with the secure audio path driver.

## Syntax


```C++
MSSAPDLLObject* SapCreateInstance();
```



## Parameters

This method has no parameters.

## Return value

Pointer to a **MSSAPDLLObject**.
