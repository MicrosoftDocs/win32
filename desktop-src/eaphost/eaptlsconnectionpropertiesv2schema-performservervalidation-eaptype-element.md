# TLSExtensions (v2 schema)

The **TLSExtensions (TLSExtensionsType)** element enables future enhancements to the schema.

``` syntax
<xs:element name="TLSExtensions"
    type="TLSExtensionsType"
 />
```

## Remarks

The **TLSExtensions** element is optional.
