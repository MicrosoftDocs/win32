# PerformServerValidation (PeapExtensionsType) element (v1 schema)

The **PerformServerValidation (PeapExtensionsType)** element indicates whether server validation is performed.

``` syntax
<xs:element
    minOccurs="0"
    ref="extendedPeap:PerformServerValidation"
 />
```

The element is defined by the [**PeapExtensionsType**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md) element.

## Remarks

The **PerformServerValidation** element is optional.

| Role | Minimum supported OS version |
|------|------------------------------|

| Server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
