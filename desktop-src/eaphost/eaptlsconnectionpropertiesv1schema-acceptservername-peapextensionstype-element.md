# TLSExtensions (v1 schema)

The **TLSExtensions (EapType)** element enables future enhancements to the schema.

``` syntax
<xs:element
    minOccurs="0"
    maxOccurs="1"
    ref="extendedTLS: TLSExtensions"
 />
```

The element is defined by the [**EapType**](eaptlsconnectionpropertiesv1schema-eaptype-element.md) element.

## Remarks

The **TLSExtensions** element is optional.
