# Obsolete ADSI Functions

The following functions were exposed in the earlier versions of ADSI SDK but are no longer used in the SDK.

-   **ADsDecodeBinaryData**
-   **AdsFreeAdsValues**
-   **ADsFreeAllErrorRecords**
-   **AdsTypeToPropVariant**
-   **AssertADsMemLeaks**
-   **DumpMemoryTracker**
-   **InitADsMem**
-   **PropVariantToAdsType**
