# IEnumTfRenderingMarkup::Reset method

The **IEnumTfRenderingMarkup::Reset** method resets the enumerator object by moving the current position to the beginning of the enumeration sequence.

## Syntax


```C++
HRESULT Reset();
```



## Parameters

This method has no parameters.

<dl></dl>

## Return value

This method can return one of these values.



| Value                                                                                | Description                           |
|--------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**S\_OK**</dt> </dl> | The method was successful.<br/> |



 

## Remarks

> [!Note]  
> This method is not currently in the public header files. To use this API, you must MIDL-compile the [prototype](prototypes.md).
