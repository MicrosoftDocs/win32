# Text Service Return Values

A text service produces a return value of 0x*HHHH*0600. The following table lists the text service return value.



| Return code/value                                                                                                                                                                                                                          | Description                                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <span id="TF_E_NOCONVERSION"></span><span id="tf_e_noconversion"></span><dl> <dt>**TF\_E\_NOCONVERSION**</dt> <dt>0x80040600</dt> </dl> | No conversion is available for the selected range of text.<br/> |
