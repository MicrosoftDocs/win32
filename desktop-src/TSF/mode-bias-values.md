# Mode Bias Values

The following predefined values are possible values for the [GUID\_PROP\_MODEBIAS](predefined-properties.md) property.

<dl> <dt>

<span id="GUID_MODEBIAS_NONE_____________"></span><span id="guid_modebias_none_____________"></span>GUID\_MODEBIAS\_NONE 
</dt> <dd>

There is no mode bias.

</dd> <dt>

<span id="GUID_MODEBIAS_FILENAME_____________"></span><span id="guid_modebias_filename_____________"></span>GUID\_MODEBIAS\_FILENAME 
</dt> <dd>

The bias is to a filename.

</dd> <dt>

<span id="GUID_MODEBIAS_NAME_____________"></span><span id="guid_modebias_name_____________"></span>GUID\_MODEBIAS\_NAME 
</dt> <dd>

The bias is to a name.

</dd> <dt>

<span id="GUID_MODEBIAS_READING_____________"></span><span id="guid_modebias_reading_____________"></span>GUID\_MODEBIAS\_READING 
</dt> <dd>

The bias is to the reading.

</dd> <dt>

<span id="GUID_MODEBIAS_DATETIME_____________"></span><span id="guid_modebias_datetime_____________"></span>GUID\_MODEBIAS\_DATETIME 
</dt> <dd>

The bias is to a date or time.

</dd> <dt>

<span id="GUID_MODEBIAS_CONVERSATION_____________"></span><span id="guid_modebias_conversation_____________"></span>GUID\_MODEBIAS\_CONVERSATION 
</dt> <dd>

The bias is to a conversation.

</dd> <dt>

<span id="GUID_MODEBIAS_NUMERIC_____________"></span><span id="guid_modebias_numeric_____________"></span>GUID\_MODEBIAS\_NUMERIC 
</dt> <dd>

The bias is to a number.

</dd> <dt>

<span id="GUID_MODEBIAS_HIRAGANA_____________"></span><span id="guid_modebias_hiragana_____________"></span>GUID\_MODEBIAS\_HIRAGANA 
</dt> <dd>

The bias is to hiragana strings.

</dd> <dt>

<span id="GUID_MODEBIAS_KATAKANA_____________"></span><span id="guid_modebias_katakana_____________"></span>GUID\_MODEBIAS\_KATAKANA 
</dt> <dd>

The bias is to katakana strings.

</dd> <dt>

<span id="GUID_MODEBIAS_CHINESE_____________"></span><span id="guid_modebias_chinese_____________"></span>GUID\_MODEBIAS\_CHINESE 
</dt> <dd>

The bias is to Chinese characters.

</dd> <dt>

<span id="GUID_MODEBIAS_HANGUL_____________"></span><span id="guid_modebias_hangul_____________"></span>GUID\_MODEBIAS\_HANGUL 
</dt> <dd>

The bias is to Hangul characters.

</dd> <dt>

<span id="GUID_MODEBIAS_HALFWIDTHKATAKANA_____________"></span><span id="guid_modebias_halfwidthkatakana_____________"></span>GUID\_MODEBIAS\_HALFWIDTHKATAKANA 
</dt> <dd>

The bias is to half-width katakana strings.

</dd> <dt>

<span id="GUID_MODEBIAS_FULLWIDTHALPHANUMERIC_____________"></span><span id="guid_modebias_fullwidthalphanumeric_____________"></span>GUID\_MODEBIAS\_FULLWIDTHALPHANUMERIC 
</dt> <dd>

The bias is to full-width alphanumeric characters.

</dd> <dt>

<span id="GUID_MODEBIAS_HALFWIDTHALPHANUMERIC_____________"></span><span id="guid_modebias_halfwidthalphanumeric_____________"></span>GUID\_MODEBIAS\_HALFWIDTHALPHANUMERIC 
</dt> <dd>

The bias is to half-width alphanumeric characters.

</dd> <dt>

<span id="GUID_MODEBIAS_FULLWIDTHHANGUL_____________"></span><span id="guid_modebias_fullwidthhangul_____________"></span>GUID\_MODEBIAS\_FULLWIDTHHANGUL 
</dt> <dd>

The bias is to full-width Hangul characters.

</dd> </dl>
