# Values for GUID\_COMPARTMENT\_TRANSITORYEXTENSION

The following values of the GUID\_COMPARTMENT\_TRANSITORYEXTENSION compartment are used to control the behavior of transitory extension.

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF\_TRANSITORYEXTENSION\_NONE**
</dt> <dd>

This value stops the transitory extension.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**TF\_TRANSITORYEXTENSION\_FLOATING**
</dt> <dd>

This value starts the transitory extension with the floating UI.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF\_TRANSITORYEXTENSION\_ATSELECTION**
</dt> <dd>

This value starts the transitory extension with the popup UI at the IP or the selection of the parent document manager.

</dd> </dl>
