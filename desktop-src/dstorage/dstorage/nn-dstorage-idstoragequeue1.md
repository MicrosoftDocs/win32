# IDStorageQueue1 interface (dstorage.h)

Represents a DirectStorage queue to perform read operations, allowing enqueuing an operation to set a specific event object to a signaled state.

## Inheritance

The **IDStorageQueue1** interface inherits from the [**IDStorageQueue**](nn-dstorage-idstoragequeue.md) interface.

## Methods

The **IDStorageQueue1** interface has these methods.

| &nbsp; |
| ---- |
| [IDStorageQueue1::EnqeueSetEvent](../dstorage/nf-dstorage-idstoragequeue1-enqueuesetevent.md) <br><br> Enqueues an operation to the queue to set the specified event object to a signaled state.|
