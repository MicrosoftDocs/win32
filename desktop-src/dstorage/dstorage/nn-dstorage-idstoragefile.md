# IDStorageFile interface (dstorage.h)

Represents a file to be accessed by DirectStorage.

## Inheritance

The **IDStorageFile** interface inherits from the **IUnknown** interface.

## Methods

The **IDStorageFile** interface has these methods.

| &nbsp; |
| ---- |
| [IDStorageFile::Close](../dstorage/nf-dstorage-idstoragefile-close.md) <br><br> Closes the file, regardless of the reference count on this object.|
| [IDStorageFile::GetFileInformation](../dstorage/nf-dstorage-idstoragefile-getfileinformation.md) <br><br> Retrieves file information for an opened file.|
