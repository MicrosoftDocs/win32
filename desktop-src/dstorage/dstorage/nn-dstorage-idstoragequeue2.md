# IDStorageQueue2 interface (dstorage.h)

Represents a DirectStorage queue to perform read operations, allowing enqueuing an operation to set a specific event object to a signaled state. Also allows you to retrieve info related to a specified compression format.

## Inheritance

The **IDStorageQueue2** interface inherits from the [**IDStorageQueue1**](nn-dstorage-idstoragequeue1.md) interface.

## Methods

The **IDStorageQueue2** interface has these methods.

| &nbsp; |
| ---- |
| [IDStorageQueue2::GetCompressionSupport](../dstorage/nf-dstorage-idstoragequeue2-getcompressionsupport.md) <br><br> Retrieves support information about the queue for a specified compression format.|
