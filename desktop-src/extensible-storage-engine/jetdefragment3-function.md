# JetDefragment3 Function


_**Applies to:** Windows | Windows Server_

## JetDefragment3 Function

The **JetDefragment3** function is reserved. Do not use the **JetDefragment3** function.

**Windows XP:**  **JetDefragment3** is introduced in Windows XP.
