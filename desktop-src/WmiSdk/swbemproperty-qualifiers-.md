# SWbemProperty.Qualifiers\_ property

The **Qualifiers\_** property of the [**SWbemProperty**](swbemproperty.md) object returns an [**SWbemQualifierSet**](swbemqualifierset.md) object that is a collection of qualifiers for this WMI property. This property is read-only.

This property is read-only.

## Syntax


```VB
SWbemProperty.Qualifiers_ As Object
```
