# SWbemLastError.Properties\_ property

The **Properties\_** property of the [**SWbemLastError**](swbemlasterror.md) object returns an [**SWbemPropertySet**](swbempropertyset.md) object that is a collection of the WMI properties for the current class or instance. This property is read-only.

For an explanation of this syntax, see [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

This property is read-only.

## Syntax


```VB
SWbemLastError.Properties_ As Object
```
