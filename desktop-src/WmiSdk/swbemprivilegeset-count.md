# SWbemPrivilegeSet.Count property

The **Count** property of the [**SWbemPrivilege**](swbemprivilege.md) object contains the number of items in an [**SWbemPrivilegeSet**](swbemprivilegeset.md) collection. This property is read-only.

For an explanation of this syntax, see [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

This property is read-only.

## Syntax


```VB
SWbemPrivilegeSet.Count As Integer
```
