# SWbemMethod.Qualifiers\_ property

The **Qualifiers\_** property of the [**SWbemMethod**](swbemmethod.md) object returns an [**SWbemQualifierSet**](swbemqualifierset.md) object that is the collection of qualifiers for this method. This property is read-only.

This property is read-only.

## Syntax


```VB
SWbemMethod.Qualifiers_ As Object
```
