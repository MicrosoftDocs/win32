# SWbemQualifier.Value property

The **Value** property of the [**SWbemQualifier**](swbemqualifier.md) object contains the variant value of this qualifier. This is the default property of this object.

For an explanation of this syntax, see [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

This property is read/write.

## Syntax


```VB
SWbemQualifier.Value As Variant
```
