# SWbemObjectPath.DisplayName property

The **DisplayName** property of the [**SWbemObjectPath**](swbemobjectpath.md) object is a string that contains the path in a form that can be used as a moniker display name.

For an explanation of this syntax, see [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

This property is read/write.

## Syntax


```VB
SWbemObjectPath.DisplayName As String
```
