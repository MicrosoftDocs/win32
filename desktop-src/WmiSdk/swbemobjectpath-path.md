# SWbemObjectPath.Path property

The **Path** property of the [**SWbemObjectPath**](swbemobjectpath.md) object contains the absolute path. This is the same as the [\_\_Path](wmi-system-properties.md) property in the COM API. This is the default property of this object.

For an explanation of this syntax, see [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

This property is read/write.

## Syntax


```VB
SWbemObjectPath.Path As String
```
