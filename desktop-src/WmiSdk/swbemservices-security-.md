# SWbemServices.Security\_ property

The **Security\_** property of the [**SWbemServices**](swbemservices.md) object is used to get or set the security settings for an **SWbemServices** object. This property is an [**SWbemSecurity**](swbemsecurity.md) object.

For an explanation of this syntax, see [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

This property is read-only.

## Syntax


```VB
SWbemServices.Security_ As Object
```
