<dl> <dt>

<span id="wmi.gloss_temporary_consumer"></span><span id="WMI.GLOSS_TEMPORARY_CONSUMER"></span>**temporary consumer**
</dt> <dd>

A consumer that receives event notifications while the consumer is active. See also, [*permanent consumers*](gloss-p.md).

</dd> <dt>

<span id="wmi.gloss_timer_event"></span><span id="WMI.GLOSS_TIMER_EVENT"></span>**timer event**
</dt> <dd>

An event that either occurs one time at a specific time of day (absolute timer event), or repeatedly at regular intervals (interval timer event). A timer event differs from other events, because it requires explicit action by the consumer for it to occur. Notifications of timer events are sent by the [*CIM Object Manager*](gloss-c.md). A timer event is represented by an instance of the system class [**\_\_TimerEvent**](--timerevent.md).

</dd> </dl>
