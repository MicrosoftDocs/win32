# WM/MediaIsFinale

The **WM/MediaIsFinale** attribute indicates whether the content is the last of a series.

## Global Constant

g\_wszWMMediaIsFinale

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
