# WM/Provider

The **WM/Provider** attribute contains the name of the metadata content provider. Metadata is often obtained through a service.

## Global Constant

g\_wszWMProvider

## Data Type

**WMT\_TYPE\_STRING**

## Example



| File type | Example value |
|-----------|---------------|
| All       | "AMG"         |
