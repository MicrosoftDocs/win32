# WM/Lyrics

The **WM/Lyrics** attribute contains the lyrics as a simple string.

## Global Constant

g\_wszWMLyrics

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

For more structured storage of lyrics, use the [**WM/Lyrics\_Synchronised**](wm-lyrics-synchronised.md) attribute.
