# WM/MediaIsPremiere

The **WM/MediaIsPremiere** attribute indicates whether the content is the first of a series.

## Global Constant

g\_wszWMMediaIsPremiere

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
