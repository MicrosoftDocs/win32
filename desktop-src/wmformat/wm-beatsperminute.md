# WM/BeatsPerMinute

The **WM/BeatsPerMinute** attribute contains the beats per minute of the content.

## Global Constant

g\_wszWMBeatsPerMinute

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

The value of a **BeatsPerMinute** attribute is an integer stored as a string.

### Example



| File type | Example value |
|-----------|---------------|
| Audio     | "120"         |
