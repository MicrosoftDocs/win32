# WM/OriginalReleaseYear

The **WM/OriginalReleaseYear** attribute contains the year during which the content was first released.

## Global Constant

g\_wszWMOriginalReleaseYear

## Data Type

**WMT\_TYPE\_STRING**

## Example



| File type | Example value |
|-----------|---------------|
| All       | "2002"        |
