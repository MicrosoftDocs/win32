# WM/MediaIsTape

The **WM/MediaIsTape** attribute indicates whether the content was broadcast from a previously recorded tape.

## Global Constant

g\_wszWMMediaIsTape

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
