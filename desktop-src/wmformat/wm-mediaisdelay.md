# WM/MediaIsDelay

The **WM/MediaIsDelay** attribute indicates whether the content was broadcast as a tape delay.

## Global Constant

g\_wszWMMediaIsDelay

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
