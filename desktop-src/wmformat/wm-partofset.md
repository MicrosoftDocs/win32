# WM/PartOfSet

The **WM/PartOfSet** attribute contains the part number and the total number of parts in the set to which the file belongs. The two values are separated by a slash.

## Global Constant

g\_wszWMPartOfSet

## Data Type

**WMT\_TYPE\_STRING**

## Example



| File type | Example value |
|-----------|---------------|
| All       | "2/3"         |
