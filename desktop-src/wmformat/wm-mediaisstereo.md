# WM/MediaIsStereo

The **WM/MediaIsStereo** attribute indicates whether the content is broadcast with stereo sound.

## Global Constant

g\_wszWMMediaIsStereo

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
