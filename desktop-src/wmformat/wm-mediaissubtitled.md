# WM/MediaIsSubtitled

The **WM/MediaIsSubtitled** attribute indicates whether the content includes subtitles.

## Global Constant

g\_wszWMMediaIsSubtitled

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
