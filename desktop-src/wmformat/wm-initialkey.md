# WM/InitialKey

The **WM/InitialKey** attribute contains the initial key of the music in the file.

## Global Constant

g\_wszWMInitialKey

## Data Type

**WMT\_TYPE\_STRING**

## Example



| File type | Example value |
|-----------|---------------|
| Audio     | "A Minor"     |
