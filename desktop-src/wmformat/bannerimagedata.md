# BannerImageData

The **BannerImageData** attribute contains a banner image for branding purposes. The type of image this attribute contains is described by the [**BannerImageType**](bannerimagetype.md) attribute.

## Global Constant

g\_wszWMBannerImageData

## Data Type

**WMT\_TYPE\_BINARY**

## Remarks

The image stored in this attribute can be in bitmap, JPEG, or GIF format.
