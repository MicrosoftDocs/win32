# WM/ToolName

The **WM/ToolName** attribute contains the name of the application used to create the file. You can also specify the version of the application using [**WM/ToolVersion**](wm-toolversion.md).

## Global Constant

g\_wszWMToolName

## Data Type

**WMT\_TYPE\_STRING**
