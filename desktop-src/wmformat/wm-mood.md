# WM/Mood

The **WM/Mood** attribute contains a category name for the mood of the content.

## Global Constant

g\_wszWMMood

## Data Type

**WMT\_TYPE\_STRING**

## Example



| File type | Example value |
|-----------|---------------|
| All       | "Joyous"      |
