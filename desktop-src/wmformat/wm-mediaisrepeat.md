# WM/MediaIsRepeat

The **WM/MediaIsRepeat** attribute indicates whether the content is a repeated episode of a series.

## Global Constant

g\_wszWMMediaIsRepeat

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
