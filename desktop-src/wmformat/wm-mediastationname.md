# WM/MediaStationName

The **WM/MediaStationName** attribute contains the name of the station that broadcast the content.

## Global Constant

g\_wszWMMediaStationName

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

This attribute is used for broadcast television.
