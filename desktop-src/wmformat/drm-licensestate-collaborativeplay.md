# DRM\_LicenseState\_CollaborativePlay

The **DRM\_LicenseState\_CollaborativePlay** property contains a [**WM\_LICENSE\_STATE\_DATA**](/windows/desktop/api/Wmsdkidl/) structure that contains details about how this right has been applied to the content.

## Global Constant

g\_wszWMDRM\_LicenseState\_CollaborativePlay

## Data Type

**WMT\_TYPE\_BINARY**

## Remarks

This is a read-only property that is retrieved using [**IWMDRMReader::GetDRMProperty**](/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
