# WM/MediaCredits

The **WM/MediaCredits** attribute contains the credits for broadcast media.

## Global Constant

g\_wszWMMediaCredits

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

This attribute is used for broadcast television or radio.
