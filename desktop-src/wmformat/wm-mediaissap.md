# WM/MediaIsSAP

The **WM/MediaIsSAP** attribute indicates whether the content includes a second audio program (SAP).

## Global Constant

g\_wszWMMediaIsSAP

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.

SAP usually identifies television programs that include an alternate soundtrack in a different language.
