# WM/MediaOriginalChannel

The **WM/MediaOriginalChannel** attribute contains the channel on which the content was broadcast.

## Global Constant

g\_wszWMMediaOriginalChannel

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

This attribute is used for broadcast television.
