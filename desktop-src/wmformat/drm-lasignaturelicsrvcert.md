# DRM\_LASignatureLicSrvCert

The **DRM\_LASignatureLicSrvCert** attribute contains the certificate that verifies the certificate contained in [**DRM\_LASignatureCert**](drm-lasignaturecert.md).

## Global Constant

g\_wszWMDRM\_LASignatureLicSrvCert

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

This property can be set with the [**IWMDRMWriter::SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) method. It is not accessible to the reader object.
