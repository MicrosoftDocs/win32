# WMDRMNET\_Revocation

The **WMDRMNET\_Revocation** property retrieves the revocation list for Windows Media DRM 10 for Network Devices.

## Global Constant

g\_wszWMDRMNET\_Revocation

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

The revocation list contains the certificates of devices that have been revoked.
