# WM/MediaIsLive

The **WM/MediaIsLive** attribute indicates whether the content is a live broadcast.

## Global Constant

g\_wszWMMediaIsLive

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This attribute is used for broadcast television.
