# WM/PlaylistDelay

The **WM/PlaylistDelay** attribute contains the number of milliseconds delay that should precede playback of the file in a playlist.

## Global Constant

g\_wszWMPlaylistDelay

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

Not all media players honor the playlist delay.
