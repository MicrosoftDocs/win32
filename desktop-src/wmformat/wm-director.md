# WM/Director

The **WM/Director** attribute contains the name of the director.

## Global Constant

g\_wszWMDirector

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

Typically video files have directors, audio files are more likely to use [**WM/Producer**](wm-producer.md).
