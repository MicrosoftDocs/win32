# WM/ParentalRating

The **WM/ParentalRating** attribute contains the parental rating of the content.

## Global Constant

g\_wszWMParentalRating

## Data Type

**WMT\_TYPE\_STRING**

## Example



| File type | Example value     |
|-----------|-------------------|
| Audio     | "Explicit Lyrics" |
| Video     | "PG-13"           |
