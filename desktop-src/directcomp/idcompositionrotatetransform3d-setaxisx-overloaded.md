# IDCompositionRotateTransform3D::SetAxisX methods

Changes or animates the value of the AxisX property of a 3D rotation transform. The AxisX property specifies the x-coordinate for the axis vector of rotation. The default value is zero.

### Overload list



| Method                                                                                                       | Description                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | Changes the value of the AxisX property.<br/>  |
| [**SetAxisX(IDCompositionAnimation\*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | Animates the value of the AxisX property.<br/> |
