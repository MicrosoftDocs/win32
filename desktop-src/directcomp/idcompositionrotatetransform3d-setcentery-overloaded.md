# IDCompositionRotateTransform3D::SetCenterY methods

Changes or animates the value of the CenterY property of a 3D rotation transform. The CenterY property specifies the y-coordinate of the point about which the rotation is performed. The default value is zero.

### Overload list



| Method                                                                                                           | Description                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY(float)**](https://msdn.microsoft.com/en-us/library/Hh448959(v=VS.85).aspx)                                     | Changes the value of the CenterY property.<br/>  |
| [**SetCenterX(IDCompositionAnimation\*)**](https://msdn.microsoft.com/en-us/library/Hh448961(v=VS.85).aspx) | Animates the value of the CenterY property.<br/> |
