# Bluetooth and getsockname

Bluetooth uses the [**getsockname**](/windows/desktop/api/winsock/nf-winsock-getsockname) function to retrieve the server device address and port number allocated to a socket through a previous call to the [**bind**](/windows/desktop/api/winsock/nf-winsock-bind) function.

For more information, see [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
