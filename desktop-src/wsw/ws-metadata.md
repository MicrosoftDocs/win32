# WS\_METADATA

An opaque type used to reference a set of [metadata](metadata-import.md) documents.


```C++
typedef struct _WS_METADATA WS_METADATA;
```



## Remarks

This object is not thread safe. For more information, see [thread safety](thread-safety.md).
