# WS\_XML\_BUFFER

The opaque type used for a reference to an [xml buffer](xml-buffer.md).


```C++
typedef struct _WS_XML_BUFFER WS_XML_BUFFER;
```



## Remarks

This object is not thread safe. For more information, see [thread safety](thread-safety.md).
