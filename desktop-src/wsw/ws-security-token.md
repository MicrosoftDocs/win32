# WS\_SECURITY\_TOKEN

The opaque handle representing a [security token](security-processing-results.md).


```C++
typedef struct _WS_SECURITY_TOKEN WS_SECURITY_TOKEN;
```



## Remarks

This object is not thread safe. For more information, see [thread safety](thread-safety.md).
