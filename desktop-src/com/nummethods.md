# NumMethods

Contains the number of methods in the associated interface, including methods from derived interfaces.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      NumMethods = value
```

## Remarks

This is a **REG\_SZ** value.
