# ProgID

Associates a ProgID with a CLSID.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      ProgID = value
```

## Remarks

Every insertable object class has a ProgID. For information on creating a ProgID, see the [&lt;ProgID&gt; key](-progid--key.md).
