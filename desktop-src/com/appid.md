# AppID

Identifies the AppID GUID that corresponds to the named executable.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\AppID
   <Executable_name>
      AppID = {AppID_GUID}
```

## Remarks

This is a **REG\_SZ** value.
