# BaseInterface

Identifies the interface from which the current interface is derived.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      BaseInterface = name
```

## Remarks

This is a **REG\_SZ** value.
