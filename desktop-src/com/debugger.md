# Debugger

Stores the name of the preferred debugger to use with remote COM debugging over RPC.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DebugObjectRPCEnabled\AeDebug
   Debugger = value
```

## Remarks

This is a **REG\_SZ** value.
