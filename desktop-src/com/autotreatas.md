# AutoTreatAs

Automatically sets the CLSID for the [**TreatAs**](treatas.md) key to the specified value.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      AutoTreatAs = value
```

## Remarks

This is a **REG\_SZ** value that specifies the class identifier.
