# Access Flags

These values indicate whether an entry in the access list describes rights that are allowed or denied.



| Constant/value                                                                                                                                                                                                                                      | Description                                   |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------|
| <span id="ACTRL_ACCESS_ALLOWED"></span><span id="actrl_access_allowed"></span><dl> <dt>**ACTRL\_ACCESS\_ALLOWED**</dt> <dt>0x00000000</dt> </dl> | Indicates an access-allowed entry.<br/> |
| <span id="ACTRL_ACCESS_DENIED"></span><span id="actrl_access_denied"></span><dl> <dt>**ACTRL\_ACCESS\_DENIED**</dt> <dt>0x10000000</dt> </dl>    | Indicates an access-denied entry.<br/>  |
