# Version (COM)

Specifies the version number of the control.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      Version = value
```

## Remarks

This is a **REG\_SZ** value.

The version number should match the version of the type library associated with the control.
