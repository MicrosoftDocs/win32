# Insertable (ProgID Key)

Indicates that this class is insertable in OLE 2 containers.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes
   {ProgID}
      Insertable
```

## Remarks

This is a required entry for objects that are insertable in OLE 2 containers.
