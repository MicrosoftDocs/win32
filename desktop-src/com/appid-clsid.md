# AppID (CLSID Key)

Associates an AppID with a CLSID.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      AppID = {AppID_GUID}
```

## Remarks

This is a **REG\_SZ** value.
