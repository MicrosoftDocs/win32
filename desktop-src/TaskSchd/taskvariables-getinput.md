# TaskVariables.GetInput method

For scripting, gets the input variables for a task. This method is not implemented.

## Syntax


```VB
TaskVariables.GetInput( _
  ByRef input _
)
```



## Parameters

<dl> <dt>

*input* \[out\]
</dt> <dd>

The input variables for a task.

</dd> </dl>

## Return value

This method does not return a value.
