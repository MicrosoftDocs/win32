# SessionStateChangeTrigger.UserId property

For scripting, gets or sets the user for the Terminal Server session. When a session state change is detected for this user, a task is started.

## Syntax


```VB
SessionStateChangeTrigger.UserId As String
```



## Property value

The user for the Terminal Server session.
