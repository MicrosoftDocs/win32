# TaskService.ConnectedDomain property

For scripting, gets the name of the domain to which the [**TargetServer**](taskservice-targetserver.md) computer is connected.

## Syntax


```VB
TaskService.ConnectedDomain As String
```



## Property value

The domain to which the [**TargetServer**](taskservice-targetserver.md) computer is connected.
