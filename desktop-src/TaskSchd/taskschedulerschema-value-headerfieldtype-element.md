# Value (headerFieldType) Element

Contains the value of a header field in an email message.

``` syntax
<xs:element name="Value"
    type="string"
 />
```

The **Value** element is defined by the [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) complex type.

## Parent element



| Element                                                                         | Derived from                                                               | Description                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contains a field for a header in an email message. <br/> |
