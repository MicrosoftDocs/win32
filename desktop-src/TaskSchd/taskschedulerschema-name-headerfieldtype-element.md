# Name (headerFieldType) Element

Contains the name of a header field in an email message.

``` syntax
<xs:element name="Name"
    type="nonEmptyString"
 />
```

The **Name** element is defined by the [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) complex type.

## Parent element



| Element                                                                         | Derived from                                                               | Description                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contains a field for a header in an email message. <br/> |
