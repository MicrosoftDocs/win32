# TaskFolderCollection.Item property

For scripting, gets the specified folder from the collection.

## Syntax


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## Property value

A [**TaskFolder**](taskfolder.md) object that represents the requested folder.
