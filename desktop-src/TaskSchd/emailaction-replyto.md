# EmailAction.ReplyTo property

\[This object is no longer supported. Please use IExecAction with the powershell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) cmdlet as a workaround.\]

For scripting, gets or sets the email address that you want to reply to.

This property is read/write.

## Syntax


```VB
EmailAction.ReplyTo As String
```



## Property value

The email address that you want to reply to.
