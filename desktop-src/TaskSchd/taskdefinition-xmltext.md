# TaskDefinition.XmlText property

For scripting, gets or sets the XML-formatted definition of the task.

## Syntax


```VB
TaskDefinition.XmlText As String
```



## Property value

The XML-formatted definition of the task.

## Remarks

The XML for a task is defined by the [Task Scheduler Schema](task-scheduler-schema.md). For an example of task XML, see [Daily Trigger Example (XML)](daily-trigger-example--xml-.md).
