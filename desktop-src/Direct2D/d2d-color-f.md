# D2D\_COLOR\_F

Describes the red, green, blue, and alpha components of a color.


```C++
typedef D3DCOLORVALUE D2D_COLOR_F;
```



## Remarks

For more information, see [**D3DCOLORVALUE**](/previous-versions/windows/desktop/legacy/dd368193(v=vs.85)).
