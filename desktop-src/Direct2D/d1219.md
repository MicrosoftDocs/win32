# D1219: No Overlap Cannot Be Used With Depth Buffer

D2D1\_VERTEX\_OPTIONS\_ASSUME\_NO\_OVERLAP and D2D1\_VERTEX\_OPTIONS\_USE\_DEPTH\_BUFFER may not be used together.
