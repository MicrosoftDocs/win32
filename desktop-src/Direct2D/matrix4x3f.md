# Matrix4x3F class

The **Matrix4x3F** class represents a 4-by-3 matrix and provides convenience methods for creating matrices.

## Members

The **Matrix4x3F** class inherits from [**D2D1\_MATRIX\_4X3\_F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f). **Matrix4x3F** also has these types of members:

-   [Constructors](#constructors)

### Constructors

The **Matrix4x3F** class has these constructors.



| Constructor                                                                                                                                                                                           | Description                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix4x3F()**](matrix4x3f-matrix4x3f--.md)                                                                                                                                                       | Instantiates a new instance of an uninitialized **Matrix4x3F** class.<br/>                                                   |
| [**Matrix4x3F(FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT)(FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT,FLOAT)**](matrix4x3f-matrix4x3f-floats-.md) | Instantiates a new instance of a **Matrix4x3F** class that is initialized with all of the floating point matrix values.<br/> |
