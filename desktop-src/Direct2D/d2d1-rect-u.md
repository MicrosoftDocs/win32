# D2D1\_RECT\_U

Represents a rectangle defined by the coordinates of the upper-left corner (left, top) and the coordinates of the lower-right corner (right, bottom).


```C++
typedef D2D_RECT_U D2D1_RECT_U;
```



## Remarks

**D2D1\_RECT\_U** is a new name for the already defined [**D2D\_RECT\_U**](/windows/desktop/api/dcommon/ns-dcommon-d2d_rect_u) structure.
