# D1197: Wrong D3D Device

The DXGI surface passed to this API is from the wrong device.





 

## Possible Causes

The DXGI surface passed to this API is from the wrong device.

## Possible Fixes

Verify that any DXGI surfaces have the same D3D device as the D2D device context.
