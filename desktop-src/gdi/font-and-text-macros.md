# Font and Text Macros

The following macros are used with fonts and text.



| Macro                            | Description                                            |
|----------------------------------|--------------------------------------------------------|
| [**DeleteFont**](deletefont.md) | Deletes a specified font.                              |
| [**SelectFont**](selectfont.md) | Selects a font into the specified device context (DC). |
