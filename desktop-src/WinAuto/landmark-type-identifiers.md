# Landmark Type Identifiers

This topic describes the named constants used to identify Microsoft UI Automation landmark types.

<dl> <dt>

<span id="UIA_CustomLandmarkTypeId"></span><span id="uia_customlandmarktypeid"></span><span id="UIA_CUSTOMLANDMARKTYPEID"></span>**UIA\_CustomLandmarkTypeId**
</dt> <dd> <dl> <dt>

80000
</dt> <dt>



Indicates a custom string should be used to describe the landmark using the value from [**UIA\_LocalizedLandmarkTypePropertyId**](uiauto-automation-element-propids.md).


</dt> </dl> </dd> <dt>

<span id="UIA_FormLandmarkTypeId"></span><span id="uia_formlandmarktypeid"></span><span id="UIA_FORMLANDMARKTYPEID"></span>**UIA\_FormLandmarkTypeId**
</dt> <dd> <dl> <dt>

80001
</dt> <dt>



Indicates that the landmark is related to form type elements typically used for data entry.


</dt> </dl> </dd> <dt>

<span id="UIA_MainLandmarkTypeId"></span><span id="uia_mainlandmarktypeid"></span><span id="UIA_MAINLANDMARKTYPEID"></span>**UIA\_MainLandmarkTypeId**
</dt> <dd> <dl> <dt>

80002
</dt> <dt>



Indicates that the landmark is for the primary content and not secondary or a lesser priority.


</dt> </dl> </dd> <dt>

<span id="UIA_NavigationLandmarkTypeId"></span><span id="uia_navigationlandmarktypeid"></span><span id="UIA_NAVIGATIONLANDMARKTYPEID"></span>**UIA\_NavigationLandmarkTypeId**
</dt> <dd> <dl> <dt>

80003
</dt> <dt>



Indicates that the landmark is related to navigation type elements.


</dt> </dl> </dd> <dt>

<span id="UIA_SearchLandmarkTypeId"></span><span id="uia_searchlandmarktypeid"></span><span id="UIA_SEARCHLANDMARKTYPEID"></span>**UIA\_SearchLandmarkTypeId**
</dt> <dd> <dl> <dt>

80004
</dt> <dt>



Indicates that the landmark is related to search type elements.


</dt> </dl> </dd> </dl>
