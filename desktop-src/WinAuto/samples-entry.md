# Samples

The following sample applications demonstrate how to use the unmanaged Microsoft UI Automation provider and client APIs. 

| Topic                                                                                                        | Description                                                                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [UI Automation Simple Provider Sample](http://go.microsoft.com/fwlink/p/?linkid=241454)<br/>           | Creates a simple custom control that implements UI Automation provider interfaces<br/>                                                                                      |
| [UI Automation Clean Shutdown Sample](http://go.microsoft.com/fwlink/p/?linkid=241455)<br/>            | Shows the correct way to clean up and shut down a UI Automation control that is hosted in a DLL.<br/>                                                                       |
| [UI Automation Fragment Provider Sample](http://go.microsoft.com/fwlink/p/?linkid=241456)<br/>         | Shows how to implement a UI Automation provider for the elements of a fragment that is hosted in a window (HWND) object.<br/>                                               |
| [UI Automation Document Content Client Sample](http://go.microsoft.com/fwlink/p/?linkid=241457)<br/>   | Demonstrates how an application can use UI Automation control patterns to access document content that is being displayed in another application's window.<br/>             |
| [UI Automation Document Content Provider Sample](http://go.microsoft.com/fwlink/p/?linkid=241458)<br/> | Demonstrates how to use UI Automation to expose an application's textual content so that it is accessible to assistive technology applications such as screen readers.<br/> |
| [UI Automation Core Window Provider Sample](http://go.microsoft.com/fwlink/p/?linkid=231583)<br/>      | Demonstrates how to use UI Automation to make a Direct2D application accessible.<br/>                                                                                       |
