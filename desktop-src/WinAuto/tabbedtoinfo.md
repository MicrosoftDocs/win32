# TabbedToInfo

## Text

**Message 1**

Throwing away extra focus event

**Message 2**

Back to the beginning

## Type

Informational

## Description

**Message 1**

Multiple focus events were fired by a single element when using standard keyboard navigation (Tab or Shift+Tab). The spurious events are discarded.

**Message 2**

Focus shifted back to the starting element in the tab order. Tabbing was terminated at this element because the tab order appears circular.
