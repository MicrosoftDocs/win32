# UI Automation Enumerations

This section describes the enumerated types used by UI Automation client applications and UI Automation provider for Microsoft Win32.

## In this section



| Enumeration                                                                        | Description                                                                                                                                                                                                                                                          |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActiveEnd**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-activeend)<br/>                               | Contains possible values for the [**SelectionActiveEnd**](uiauto-textattribute-ids.md) text attribute, which indicates the location of the caret relative to a text range that represents the currently selected text.<br/> |
| [**AnimationStyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-animationstyle)<br/>                     | Contains values for the AnimationStyle text attribute.<br/>                                                                                                                                                                                                    |
| [**AsyncContentLoadedState**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-asynccontentloadedstate)<br/>   | Contains values that describe the progress of asynchronous loading of content.<br/>                                                                                                                                                                            |
| [**AutomationElementMode**](/windows/desktop/api/UIAutomationClient/ne-uiautomationclient-automationelementmode)<br/>       | Contains values that specify the type of reference to use when returning UI Automation elements.<br/>                                                                                                                                                          |
| [**AutomationIdentifierType**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-automationidentifiertype)<br/>         | Contains values used in the [**UiaLookupId**](/windows/desktop/api/UIAutomationCoreApi/nf-uiautomationcoreapi-uialookupid) function.<br/>                                                                                                                                                                   |
| [**BulletStyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-bulletstyle)<br/>                           | Contains values for the BulletStyle text attribute.<br/>                                                                                                                                                                                                       |
| [**CapStyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-capstyle)<br/>                                 | Contains values that specify the value of the CapStyle text attribute.<br/>                                                                                                                                                                                    |
| [**CaretBidiMode**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-caretbidimode)<br/>                       | Contains possible values for the [**CaretBidiMode**](uiauto-textattribute-ids.md) text attribute, which indicates whether the caret is in text that flows from left to right, or from right to left.<br/>                        |
| [**CaretPosition**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-caretposition)<br/>                       | Contains possible values for the [**CaretPosition**](uiauto-textattribute-ids.md) text attribute, which indicates the location of the caret relative to a line of text in a text range.<br/>                                     |
| [**ConditionType**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-conditiontype)<br/>                       | Contains values that specify a type of [**UiaCondition**](/windows/desktop/api/UIAutomationCoreApi/ns-uiautomationcoreapi-uiacondition).<br/>                                                                                                                                                                 |
| [**DockPosition**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-dockposition)<br/>                            | Contains values that specify the location of a docking window represented by the [Dock](uiauto-implementingdock.md) *control pattern*.<br/>                                                                                                                   |
| [**EventArgsType**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-eventargstype)<br/>                       | Contains values that specify the event type described by a [**UiaEventArgs**](/windows/desktop/api/UIAutomationCoreApi/ns-uiautomationcoreapi-uiaeventargs) structure.<br/>                                                                                                                                   |
| [**ExpandCollapseState**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-expandcollapsestate)<br/>           | Contains values that specify the state of a UI element that can be expanded and collapsed.<br/>                                                                                                                                                                |
| [**FlowDirections**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-flowdirections)<br/>                     | Contains values for the TextFlowDirections text attribute. <br/>                                                                                                                                                                                               |
| [**FillType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-filltype)<br/>                                 | Contains values for the FillType attribute.<br/>                                                                                                                                                                                                               |
| [**HorizontalTextAlignmentEnum**](/previous-versions/windows/desktop/legacy/ee671233(v=vs.85))<br/>   | Contains values for the HorizontalTextAlignment text attribute.<br/>                                                                                                                                                                                           |
| [**LiveSetting**](/windows/desktop/api/uiautomationcore/ne-uiautomationcore-livesetting)<br/>                           | Contains possible values for the LiveSetting property. This property is implemented by provider elements that are part of a live region.<br/>                                                                                                                  |
| [**NavigateDirection**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-navigatedirection)<br/>                          | Contains values used to specify the direction of navigation within the Microsoft UI Automation tree.<br/>                                                                                                                                                      |
| [**NormalizeState**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-normalizestate)<br/>                     | Contains values that specify the behavior of [**UiaGetUpdatedCache**](/windows/desktop/api/UIAutomationCoreApi/nf-uiautomationcoreapi-uiagetupdatedcache).<br/>                                                                                                                                             |
| [**NotificationKind**](/windows/win32/api/uiautomationcore/ne-uiautomationcore-notificationkind)<br/>                     | Defines values that indicate the type of a notification event, and a hint to the listener about the processing of the event.<br/>                                                                                                                              |
| [**NotificationProcessing**](/windows/win32/api/uiautomationcore/ne-uiautomationcore-notificationprocessing)<br/>         | Defines values that indicate how a notification should be processed.<br/>                                                                                                                                                                                      |
| [**OrientationType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-orientationtype)<br/>                       | Contains values that specify the orientation of a control.<br/>                                                                                                                                                                                                |
| [**OutlineStyles**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-outlinestyles)<br/>                       | Contains values for the OutlineStyle text attribute.<br/>                                                                                                                                                                                                      |
| [**PropertyConditionFlags**](/windows/desktop/api/UIAutomationClient/ne-uiautomationclient-propertyconditionflags)<br/>     | Contains values used in creating property conditions.<br/>                                                                                                                                                                                                     |
| [**ProviderOptions**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-provideroptions)<br/>                       | Contains values that specify the type of UI Automation provider. The [**IRawElementProviderSimple::ProviderOptions**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-irawelementprovidersimple-get_provideroptions) property uses this enumeration.<br/>                                               |
| [**ProviderType**](/windows/desktop/api/UIAutomationCoreApi/ne-uiautomationcoreapi-providertype)<br/>                         | Contains values that specify the type of a client-side (proxy) UI Automation provider.<br/>                                                                                                                                                                    |
| [**RowOrColumnMajor**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-roworcolumnmajor)<br/>                 | Contains values that specify whether data in a table should be read primarily by row or by column.<br/>                                                                                                                                                        |
| [**SayAsInterpretAs**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-sayasinterpretas)<br/>                     | Defines the values that indicate how a text-to-speech engine should interpret specific data.<br/>                                                                                                                                                              |
| [**ScrollAmount**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-scrollamount)<br/>                             | Contains values that specify the direction and distance to scroll.<br/>                                                                                                                                                                                        |
| [**StructureChangeType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-structurechangetype)<br/>           | Contains values that specify the type of change in the UI Automation tree structure.<br/>                                                                                                                                                                      |
| [**SupportedTextSelection**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-supportedtextselection)<br/>     | Contains values that specify the supported text selection attribute.<br/>                                                                                                                                                                                      |
| [**SynchronizedInputType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-synchronizedinputtype)<br/>       | Contains values that specify the type of synchronized input.<br/>                                                                                                                                                                                              |
| [**TextDecorationLineStyle**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-textdecorationlinestyle)<br/>              | Contains values that specify the OverlineStyle, StrikethroughStyle, and UnderlineStyle text attributes.<br/>                                                                                                                                                   |
| [**TextEditChangeType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-texteditchangetype)<br/>             | Describes the text editing change being performed by controls when text-edit events are raised or handled.<br/>                                                                                                                                                |
| [**TextPatternRangeEndpoint**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-textpatternrangeendpoint)<br/> | Contains values that specify the endpoints of a text range.<br/>                                                                                                                                                                                               |
| [**TextUnit**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-textunit)<br/>                                 | Contains values that specify units of text for the purposes of navigation.<br/>                                                                                                                                                                                |
| [**ToggleState**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-togglestate)<br/>                               | Contains values that specify the toggle state of a UI Automation element that implements the [Toggle](uiauto-implementingtoggle.md) *control pattern*.<br/>                                                                                                   |
| [**TreeScope**](/windows/desktop/api/UIAutomationClient/ne-uiautomationclient-treescope)<br/>                               | Contains values that specify the scope of various operations in the UI Automation tree.<br/>                                                                                                                                                                   |
| [**TreeTraversalOptions**](/windows/win32/api/uiautomationcoreapi/ne-uiautomationcoreapi-treetraversaloptions)<br/>             | Defines values that can be used to customize tree navigation order.<br/>                                                                                                                                                                                       |
| [**UIAutomationType**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-uiautomationtype)<br/>                 | Contains values used to indicate UI Automation data types.<br/>                                                                                                                                                                                                |
| [**VisualEffects**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-visualeffects)<br/>                       | Contains values for the VisualEffects attribute.<br/>                                                                                                                                                                                                          |
| [**WindowInteractionState**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-windowinteractionstate)<br/>     | Contains values that specify the current state of the window for purposes of user interaction.<br/>                                                                                                                                                            |
| [**WindowVisualState**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-windowvisualstate)<br/>               | Contains values that specify the visual state of a window.<br/>                                                                                                                                                                                                |
| [**ZoomUnit**](/windows/desktop/api/UIAutomationCore/ne-uiautomationcore-zoomunit)<br/>                                     | Contains possible values for the [**IUIAutomationTransformPattern2::ZoomByUnit**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationtransformpattern2-zoombyunit) method, which zooms the viewport of a control by the specified unit.<br/>                                              |
