# UI Automation Enumerations

This section describes the enumerated types used by UI Automation client applications and UI Automation provider for Microsoft Win32.

## In this section



| Enumeration                                                                        | Description                                                                                                                                                                                                                                                          |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActiveEnd**](https://msdn.microsoft.com/library/windows/desktop/hh448742)<br/>                               | Contains possible values for the [**SelectionActiveEnd**](uiauto-textattribute-ids.md#uia-selectionactiveendattributeid) text attribute, which indicates the location of the caret relative to a text range that represents the currently selected text.<br/> |
| [**AnimationStyle**](uiauto-animationstyleenum.md)<br/>                     | Contains values for the AnimationStyle text attribute.<br/>                                                                                                                                                                                                    |
| [**AsyncContentLoadedState**](uiauto-asynccontentloadedstateenum.md)<br/>   | Contains values that describe the progress of asynchronous loading of content.<br/>                                                                                                                                                                            |
| [**AutomationElementMode**](uiauto-automationelementmodeenum.md)<br/>       | Contains values that specify the type of reference to use when returning UI Automation elements.<br/>                                                                                                                                                          |
| [**AutomationIdentifierType**](uiauto-automationidtypeenum.md)<br/>         | Contains values used in the [**UiaLookupId**](uiauto-uialookupidautometh.md) function.<br/>                                                                                                                                                                   |
| [**BulletStyle**](uiauto-bulletstyleenum.md)<br/>                           | Contains values for the BulletStyle text attribute.<br/>                                                                                                                                                                                                       |
| [**CapStyle**](uiauto-capstyleenum.md)<br/>                                 | Contains values that specify the value of the CapStyle text attribute.<br/>                                                                                                                                                                                    |
| [**CaretBidiMode**](https://msdn.microsoft.com/library/windows/desktop/hh448744)<br/>                       | Contains possible values for the [**CaretBidiMode**](uiauto-textattribute-ids.md#uia-caretbidimodeattributeid) text attribute, which indicates whether the caret is in text that flows from left to right, or from right to left.<br/>                        |
| [**CaretPosition**](https://msdn.microsoft.com/library/windows/desktop/hh448745)<br/>                       | Contains possible values for the [**CaretPosition**](uiauto-textattribute-ids.md#uia-caretpositionattributeid) text attribute, which indicates the location of the caret relative to a line of text in a text range.<br/>                                     |
| [**ConditionType**](uiauto-conditiontypeenum.md)<br/>                       | Contains values that specify a type of [**UiaCondition**](uiauto-uiaconditionstruct.md).<br/>                                                                                                                                                                 |
| [**DockPosition**](uiauto-dock-pos-enum.md)<br/>                            | Contains values that specify the location of a docking window represented by the [Dock](uiauto-implementingdock.md) *control pattern*.<br/>                                                                                                                   |
| [**EventArgsType**](uiauto-eventargstypeenum.md)<br/>                       | Contains values that specify the event type described by a [**UiaEventArgs**](uiauto-uiaeventargsstruct.md) structure.<br/>                                                                                                                                   |
| [**ExpandCollapseState**](uiauto-expandcollapsestateenum.md)<br/>           | Contains values that specify the state of a UI element that can be expanded and collapsed.<br/>                                                                                                                                                                |
| [**FlowDirections**](uiauto-flowdirectionsenum.md)<br/>                     | Contains values for the TextFlowDirections text attribute. <br/>                                                                                                                                                                                               |
| [**FillType**](uiauto-filltypeenum.md)<br/>                                 | Contains values for the FillType attribute.<br/>                                                                                                                                                                                                               |
| [**HorizontalTextAlignmentEnum**](uiauto-horizontaltextalignenum.md)<br/>   | Contains values for the HorizontalTextAlignment text attribute.<br/>                                                                                                                                                                                           |
| [**LiveSetting**](https://msdn.microsoft.com/library/windows/desktop/hh437308)<br/>                           | Contains possible values for the LiveSetting property. This property is implemented by provider elements that are part of a live region.<br/>                                                                                                                  |
| [**NavigateDirection**](uiauto-navdirenum.md)<br/>                          | Contains values used to specify the direction of navigation within the Microsoft UI Automation tree.<br/>                                                                                                                                                      |
| [**NormalizeState**](uiauto-normalizestateenum.md)<br/>                     | Contains values that specify the behavior of [**UiaGetUpdatedCache**](uiauto-uiagetupdatedcacheautometh.md).<br/>                                                                                                                                             |
| [**NotificationKind**](uiauto-notificationkind.md)<br/>                     | Defines values that indicate the type of a notification event, and a hint to the listener about the processing of the event.<br/>                                                                                                                              |
| [**NotificationProcessing**](uiauto-notificationprocessing.md)<br/>         | Defines values that indicate how a notification should be processed.<br/>                                                                                                                                                                                      |
| [**OrientationType**](uiauto-orientationtype.md)<br/>                       | Contains values that specify the orientation of a control.<br/>                                                                                                                                                                                                |
| [**OutlineStyles**](uiauto-outlinestylesenum.md)<br/>                       | Contains values for the OutlineStyle text attribute.<br/>                                                                                                                                                                                                      |
| [**PropertyConditionFlags**](uiauto-propertyconditionflagsenum.md)<br/>     | Contains values used in creating property conditions.<br/>                                                                                                                                                                                                     |
| [**ProviderOptions**](uiauto-provoptionsenum.md)<br/>                       | Contains values that specify the type of UI Automation provider. The [**IRawElementProviderSimple::ProviderOptions**](uiauto-irawelementprovidersimple-provideroptions.md) property uses this enumeration.<br/>                                               |
| [**ProviderType**](uiauto-providertypeenum.md)<br/>                         | Contains values that specify the type of a client-side (proxy) UI Automation provider.<br/>                                                                                                                                                                    |
| [**RowOrColumnMajor**](uiauto-roworcolumnmajorenum.md)<br/>                 | Contains values that specify whether data in a table should be read primarily by row or by column.<br/>                                                                                                                                                        |
| [**SayAsInterpretAs**](uiauto-sayasinterpretas.md)<br/>                     | Defines the values that indicate how a text-to-speech engine should interpret specific data.<br/>                                                                                                                                                              |
| [**ScrollAmount**](uiauto-scrollamount.md)<br/>                             | Contains values that specify the direction and distance to scroll.<br/>                                                                                                                                                                                        |
| [**StructureChangeType**](uiauto-structurechangetypeenum.md)<br/>           | Contains values that specify the type of change in the UI Automation tree structure.<br/>                                                                                                                                                                      |
| [**SupportedTextSelection**](uiauto-supportedtextselectionenum.md)<br/>     | Contains values that specify the supported text selection attribute.<br/>                                                                                                                                                                                      |
| [**SynchronizedInputType**](uiauto-synchronizedinputtypeenum.md)<br/>       | Contains values that specify the type of synchronized input.<br/>                                                                                                                                                                                              |
| [**TextDecorationLineStyle**](uiauto-textdeclinestyle.md)<br/>              | Contains values that specify the OverlineStyle, StrikethroughStyle, and UnderlineStyle text attributes.<br/>                                                                                                                                                   |
| [**TextEditChangeType**](uiauto-texteditchangetypeenum.md)<br/>             | Describes the text editing change being performed by controls when text-edit events are raised or handled.<br/>                                                                                                                                                |
| [**TextPatternRangeEndpoint**](uiauto-textpatternrangeendpointenum.md)<br/> | Contains values that specify the endpoints of a text range.<br/>                                                                                                                                                                                               |
| [**TextUnit**](uiauto-textunitenum.md)<br/>                                 | Contains values that specify units of text for the purposes of navigation.<br/>                                                                                                                                                                                |
| [**ToggleState**](uiauto-togglestate.md)<br/>                               | Contains values that specify the toggle state of a UI Automation element that implements the [Toggle](uiauto-implementingtoggle.md) *control pattern*.<br/>                                                                                                   |
| [**TreeScope**](uiauto-treescopeenum.md)<br/>                               | Contains values that specify the scope of various operations in the UI Automation tree.<br/>                                                                                                                                                                   |
| [**TreeTraversalOptions**](uiauto-treetraversaloptions.md)<br/>             | Defines values that can be used to customize tree navigation order.<br/>                                                                                                                                                                                       |
| [**UIAutomationType**](uiauto-uiautomationtypeenum.md)<br/>                 | Contains values used to indicate UI Automation data types.<br/>                                                                                                                                                                                                |
| [**VisualEffects**](uiauto-visualeffectsenum.md)<br/>                       | Contains values for the VisualEffects attribute.<br/>                                                                                                                                                                                                          |
| [**WindowInteractionState**](uiauto-windowinteractionstateenum.md)<br/>     | Contains values that specify the current state of the window for purposes of user interaction.<br/>                                                                                                                                                            |
| [**WindowVisualState**](uiauto-windowvisualstateenum.md)<br/>               | Contains values that specify the visual state of a window.<br/>                                                                                                                                                                                                |
| [**ZoomUnit**](uiauto-zoomunit.md)<br/>                                     | Contains possible values for the [**IUIAutomationTransformPattern2::ZoomByUnit**](uiauto-iuiautomationtransformpattern2-zoombyunit.md) method, which zooms the viewport of a control by the specified unit.<br/>                                              |
