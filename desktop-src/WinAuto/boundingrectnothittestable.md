# BoundingRectNotHitTestable

## Text

The reported rectangle does not appear to contain a point where this element can be retrieved with hit testing.

## Type

Warning

## Description

The bounding rectangle of the element is not reachable by hit testing. For example, it lies outside the physical screen coordinates (or viewable region) of the desktop.
