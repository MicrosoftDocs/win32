# Control Pattern Interfaces for Clients

This section describes control pattern interfaces for unmanaged UI Automation client applications.

## In this section



| Interface                                                                                                | Description                                                                                                                                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAutomationAnnotationPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437209)<br/>           | Provides access to the properties of an annotation in a document.<br/>                                                                                                                                                                                                                                                                            |
| [**IUIAutomationCustomNavigationPattern**](uiauto-iuiautomationcustomnavigationpattern.md)<br/>   | Exposes a method to support access by a Microsoft UI Automation client to controls that support a custom navigation order.<br/>                                                                                                                                                                                                                   |
| [**IUIAutomationDockPattern**](uiauto-iuiautomationdockpattern.md)<br/>                           | Provides access to a control that enables child elements to be arranged horizontally and vertically, relative to each other.<br/>                                                                                                                                                                                                                 |
| [**IUIAutomationDragPattern**](uiauto-iuiautomationdragpattern.md)<br/>                           | Provides access to information exposed by a UI Automation provider for an element that can be dragged as part of a drag-and-drop operation.<br/>                                                                                                                                                                                                  |
| [**IUIAutomationDropTargetPattern**](uiauto-iuiautomationdroptargetpattern.md)<br/>               | Provides access to drag-and-drop information exposed by a UI Automation provider for an element that can be the drop target of a drag-and-drop operation.<br/>                                                                                                                                                                                    |
| [**IUIAutomationExpandCollapsePattern**](uiauto-iuiautomationexpandcollapsepattern.md)<br/>       | Provides access to a control that can visually expand to display content, and collapse to hide content.<br/>                                                                                                                                                                                                                                      |
| [**IUIAutomationGridItemPattern**](uiauto-iuiautomationgriditempattern.md)<br/>                   | Provides access to a child control in a grid-style container that supports the [**IUIAutomationGridPattern**](uiauto-iuiautomationgridpattern.md) interface.<br/>                                                                                                                                                                                |
| [**IUIAutomationGridPattern**](uiauto-iuiautomationgridpattern.md)<br/>                           | Provides access to a control that acts as a container for a collection of child controls that are organized in a two-dimensional logical coordinate system that can be traversed by row and column.<br/>                                                                                                                                          |
| [**IUIAutomationInvokePattern**](uiauto-iuiautomationinvokepattern.md)<br/>                       | Exposes a method that enables a client application to invoke the action of a control (typically a button).<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationItemContainerPattern**](uiauto-iuiautomationitemcontainerpattern.md)<br/>         | Exposes a method that retrieves an item from a container, such as a virtual list.<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationLegacyIAccessiblePattern**](uiauto-iuiautomationlegacyiaccessiblepattern.md)<br/> | Exposes methods and properties that enable UI Automation clients to retrieve UI information from Microsoft Active Accessibility (MSAA) servers.<br/>                                                                                                                                                                                              |
| [**IUIAutomationMultipleViewPattern**](uiauto-iuiautomationmultipleviewpattern.md)<br/>           | Provides access to a control that can switch between multiple representations of the same information or set of child controls.<br/>                                                                                                                                                                                                              |
| [**IUIAutomationObjectModelPattern**](uiauto-iuiautomationobjectmodelpattern.md)<br/>             | Provides access to the underlying object model implemented by a control or application. <br/>                                                                                                                                                                                                                                                     |
| [**IUIAutomationPatternHandler**](uiauto-iuiautomationpatternhandler.md)<br/>                     | Returns a client API wrapper object and to unmarshal property and method requests to an actual provider instance.<br/>                                                                                                                                                                                                                            |
| [**IUIAutomationPatternInstance**](uiauto-iuiautomationpatterninstance.md)<br/>                   | Represents a control pattern object. The client API wrapper uses this interface to implement all property and method calls in terms of the [**GetProperty**](uiauto-iuiautomationpatterninstance-getproperty.md) and [**CallMethod**](uiauto-iuiautomationpatterninstance-callmethod.md) methods.<br/>                                          |
| [**IUIAutomationRangeValuePattern**](uiauto-iuiautomationrangevaluepattern.md)<br/>               | Provides access to a control that presents a range of values.<br/>                                                                                                                                                                                                                                                                                |
| [**IUIAutomationScrollItemPattern**](uiauto-iuiautomationscrollitempattern.md)<br/>               | Exposes a method that enables an item in a scrollable view to be placed in a visible portion of the view.<br/>                                                                                                                                                                                                                                    |
| [**IUIAutomationScrollPattern**](uiauto-iuiautomationscrollpattern.md)<br/>                       | Provides access to a control that acts as a scrollable container for a collection of child elements.<br/>                                                                                                                                                                                                                                         |
| [**IUIAutomationSelectionItemPattern**](uiauto-iuiautomationselectionitempattern.md)<br/>         | Provides access to the selectable child items of a container control that supports [**IUIAutomationSelectionPattern**](uiauto-iuiautomationselectionpattern.md).<br/>                                                                                                                                                                            |
| [**IUIAutomationSelectionPattern**](uiauto-iuiautomationselectionpattern.md)<br/>                 | Provides access to a control that contains selectable child items. The children of this element support [**IUIAutomationSelectionItemPattern**](uiauto-iuiautomationselectionitempattern.md).<br/>                                                                                                                                               |
| [**IUIAutomationSelectionPattern2**](uiauto-iuiautomationselectionpattern2.md)<br/>               | Extends the [**IUIAutomationSelectionPattern**](uiauto-iuiautomationselectionpattern.md) interface to provide information about selected items.<br/>                                                                                                                                                                                             |
| [**IUIAutomationSpreadsheetItemPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437267)<br/> | Enables a client application to retrieve information about an item (cell) in a spreadsheet.<br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationSpreadsheetPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437280)<br/>         | Enables a client application to access the items (cells) in a spreadsheet.<br/>                                                                                                                                                                                                                                                                   |
| [**IUIAutomationStylesPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437282)<br/>                   | Enables UI Automation clients to retrieve the visual styles associated with an element in a document.<br/>                                                                                                                                                                                                                                        |
| [**IUIAutomationSynchronizedInputPattern**](uiauto-iuiautomationsynchronizedinputpattern.md)<br/> | Provides access to the keyboard or mouse input of a control.<br/>                                                                                                                                                                                                                                                                                 |
| [**IUIAutomationTableItemPattern**](uiauto-iuiautomationtableitempattern.md)<br/>                 | Provides access to a child element in a container that supports [**IUIAutomationTablePattern**](uiauto-iuiautomationtablepattern.md). <br/>                                                                                                                                                                                                      |
| [**IUIAutomationTablePattern**](uiauto-iuiautomationtablepattern.md)<br/>                         | Provides access to a control that acts as a container for a collection of child elements. The children of this element support [**IUIAutomationTableItemPattern**](uiauto-iuiautomationtableitempattern.md) and are organized in a two-dimensional logical coordinate system that can be traversed by row and column.<br/>                       |
| [**IUIAutomationTextChildPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437296)<br/>             | Provides access a text-based control (or an object embedded in text) that is a child or descendant of another text-based control. <br/>                                                                                                                                                                                                           |
| [**IUIAutomationTextEditPattern**](uiauto-iuiautomationtexteditpattern.md)<br/>                   | Provides access to a control that modifies text, for example a control that performs auto-correction or enables input composition through an Input Method Editor (IME).<br/>                                                                                                                                                                      |
| [**IUIAutomationTextPattern**](uiauto-iuiautomationtextpattern.md)<br/>                           | Provides access to a control that contains text.<br/>                                                                                                                                                                                                                                                                                             |
| [**IUIAutomationTextPattern2**](uiauto-iuiautomationtextpattern2.md)<br/>                         | Extends the [**IUIAutomationTextPattern**](uiauto-iuiautomationtextpattern.md) interface. <br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationTextRange**](uiauto-iuiautomationtextrange.md)<br/>                               | Provides access to a span of continuous text in a container that supports the [**IUIAutomationTextPattern**](uiauto-iuiautomationtextpattern.md) interface. Client applications can use the [**IUIAutomationTextRange**](uiauto-iuiautomationtextrange.md) interface to select, compare, and retrieve embedded objects from the text span.<br/> |
| [**IUIAutomationTextRange2**](uiauto-iuiautomationtextrange2.md)<br/>                             | Extends the [**IUIAutomationTextRange**](uiauto-iuiautomationtextrange.md) interface to enable UI Automation clients to programmatically invoke context menus.<br/>                                                                                                                                                                              |
| [**IUIAutomationTextRange3**](uiauto-iuiautomationtextrange3.md)<br/>                             | Extends the [**IUIAutomationTextRange2**](uiauto-iuiautomationtextrange2.md) interface to support faster access to the underlying rich text data on a text range.<br/>                                                                                                                                                                           |
| [**IUIAutomationTextRangeArray**](uiauto-iuiautomationtextrangearray.md)<br/>                     | Represents a collection of [**IUIAutomationTextRange**](uiauto-iuiautomationtextrange.md) objects.<br/>                                                                                                                                                                                                                                          |
| [**IUIAutomationTogglePattern**](uiauto-iuiautomationtogglepattern.md)<br/>                       | Provides access to a control that can cycle through a set of states, and maintain a state after it is set.<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationTransformPattern**](uiauto-iuiautomationtransformpattern.md)<br/>                 | Provides access to a control that can be moved, resized, or rotated.<br/>                                                                                                                                                                                                                                                                         |
| [**IUIAutomationTransformPattern2**](https://msdn.microsoft.com/library/windows/desktop/hh437304)<br/>           | Extends the [**IUIAutomationTransformPattern**](uiauto-iuiautomationtransformpattern.md) interface to enable UI Automation clients to programmatically access the viewport zooming functionality of a control.<br/>                                                                                                                              |
| [**IUIAutomationValuePattern**](uiauto-iuiautomationvaluepattern.md)<br/>                         | Provides access to a control that contains a value that does not span a range and that can be represented as a string.<br/>                                                                                                                                                                                                                       |
| [**IUIAutomationVirtualizedItemPattern**](uiauto-iuiautomationvirtualizeditempattern.md)<br/>     | Represents an virtualized item, which is an item that is represented by a placeholder automation element in the UI Automation tree.<br/>                                                                                                                                                                                                          |
| [**IUIAutomationWindowPattern**](uiauto-iuiautomationwindowpattern.md)<br/>                       | Provides access to the fundamental functionality of a window.<br/>                                                                                                                                                                                                                                                                                |
