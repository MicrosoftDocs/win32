# Control Pattern Interfaces for Clients

This section describes control pattern interfaces for unmanaged UI Automation client applications.

## In this section



| Interface                                                                                                | Description                                                                                                                                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAutomationAnnotationPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationannotationpattern)<br/>           | Provides access to the properties of an annotation in a document.<br/>                                                                                                                                                                                                                                                                            |
| [**IUIAutomationCustomNavigationPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationcustomnavigationpattern)<br/>   | Exposes a method to support access by a Microsoft UI Automation client to controls that support a custom navigation order.<br/>                                                                                                                                                                                                                   |
| [**IUIAutomationDockPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdockpattern)<br/>                           | Provides access to a control that enables child elements to be arranged horizontally and vertically, relative to each other.<br/>                                                                                                                                                                                                                 |
| [**IUIAutomationDragPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdragpattern)<br/>                           | Provides access to information exposed by a UI Automation provider for an element that can be dragged as part of a drag-and-drop operation.<br/>                                                                                                                                                                                                  |
| [**IUIAutomationDropTargetPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdroptargetpattern)<br/>               | Provides access to drag-and-drop information exposed by a UI Automation provider for an element that can be the drop target of a drag-and-drop operation.<br/>                                                                                                                                                                                    |
| [**IUIAutomationExpandCollapsePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationexpandcollapsepattern)<br/>       | Provides access to a control that can visually expand to display content, and collapse to hide content.<br/>                                                                                                                                                                                                                                      |
| [**IUIAutomationGridItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgriditempattern)<br/>                   | Provides access to a child control in a grid-style container that supports the [**IUIAutomationGridPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern) interface.<br/>                                                                                                                                                                                |
| [**IUIAutomationGridPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern)<br/>                           | Provides access to a control that acts as a container for a collection of child controls that are organized in a two-dimensional logical coordinate system that can be traversed by row and column.<br/>                                                                                                                                          |
| [**IUIAutomationInvokePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationinvokepattern)<br/>                       | Exposes a method that enables a client application to invoke the action of a control (typically a button).<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationItemContainerPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationitemcontainerpattern)<br/>         | Exposes a method that retrieves an item from a container, such as a virtual list.<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationLegacyIAccessiblePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationlegacyiaccessiblepattern)<br/> | Exposes methods and properties that enable UI Automation clients to retrieve UI information from Microsoft Active Accessibility (MSAA) servers.<br/>                                                                                                                                                                                              |
| [**IUIAutomationMultipleViewPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationmultipleviewpattern)<br/>           | Provides access to a control that can switch between multiple representations of the same information or set of child controls.<br/>                                                                                                                                                                                                              |
| [**IUIAutomationObjectModelPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationobjectmodelpattern)<br/>             | Provides access to the underlying object model implemented by a control or application. <br/>                                                                                                                                                                                                                                                     |
| [**IUIAutomationPatternHandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iuiautomationpatternhandler)<br/>                     | Returns a client API wrapper object and to unmarshal property and method requests to an actual provider instance.<br/>                                                                                                                                                                                                                            |
| [**IUIAutomationPatternInstance**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iuiautomationpatterninstance)<br/>                   | Represents a control pattern object. The client API wrapper uses this interface to implement all property and method calls in terms of the [**GetProperty**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-getproperty) and [**CallMethod**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-callmethod) methods.<br/>                                          |
| [**IUIAutomationRangeValuePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationrangevaluepattern)<br/>               | Provides access to a control that presents a range of values.<br/>                                                                                                                                                                                                                                                                                |
| [**IUIAutomationScrollItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollitempattern)<br/>               | Exposes a method that enables an item in a scrollable view to be placed in a visible portion of the view.<br/>                                                                                                                                                                                                                                    |
| [**IUIAutomationScrollPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollpattern)<br/>                       | Provides access to a control that acts as a scrollable container for a collection of child elements.<br/>                                                                                                                                                                                                                                         |
| [**IUIAutomationSelectionItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern)<br/>         | Provides access to the selectable child items of a container control that supports [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern).<br/>                                                                                                                                                                            |
| [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern)<br/>                 | Provides access to a control that contains selectable child items. The children of this element support [**IUIAutomationSelectionItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern).<br/>                                                                                                                                               |
| [**IUIAutomationSelectionPattern2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern2)<br/>               | Extends the [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern) interface to provide information about selected items.<br/>                                                                                                                                                                                             |
| [**IUIAutomationSpreadsheetItemPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationspreadsheetitempattern)<br/> | Enables a client application to retrieve information about an item (cell) in a spreadsheet.<br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationSpreadsheetPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationspreadsheetpattern)<br/>         | Enables a client application to access the items (cells) in a spreadsheet.<br/>                                                                                                                                                                                                                                                                   |
| [**IUIAutomationStylesPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationstylespattern)<br/>                   | Enables UI Automation clients to retrieve the visual styles associated with an element in a document.<br/>                                                                                                                                                                                                                                        |
| [**IUIAutomationSynchronizedInputPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationsynchronizedinputpattern)<br/> | Provides access to the keyboard or mouse input of a control.<br/>                                                                                                                                                                                                                                                                                 |
| [**IUIAutomationTableItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern)<br/>                 | Provides access to a child element in a container that supports [**IUIAutomationTablePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern). <br/>                                                                                                                                                                                                      |
| [**IUIAutomationTablePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern)<br/>                         | Provides access to a control that acts as a container for a collection of child elements. The children of this element support [**IUIAutomationTableItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern) and are organized in a two-dimensional logical coordinate system that can be traversed by row and column.<br/>                       |
| [**IUIAutomationTextChildPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtextchildpattern)<br/>             | Provides access a text-based control (or an object embedded in text) that is a child or descendant of another text-based control. <br/>                                                                                                                                                                                                           |
| [**IUIAutomationTextEditPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtexteditpattern)<br/>                   | Provides access to a control that modifies text, for example a control that performs auto-correction or enables input composition through an Input Method Editor (IME).<br/>                                                                                                                                                                      |
| [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern)<br/>                           | Provides access to a control that contains text.<br/>                                                                                                                                                                                                                                                                                             |
| [**IUIAutomationTextPattern2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern2)<br/>                         | Extends the [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern) interface. <br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange)<br/>                               | Provides access to a span of continuous text in a container that supports the [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern) interface. Client applications can use the [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) interface to select, compare, and retrieve embedded objects from the text span.<br/> |
| [**IUIAutomationTextRange2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2)<br/>                             | Extends the [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) interface to enable UI Automation clients to programmatically invoke context menus.<br/>                                                                                                                                                                              |
| [**IUIAutomationTextRange3**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange3)<br/>                             | Extends the [**IUIAutomationTextRange2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2) interface to support faster access to the underlying rich text data on a text range.<br/>                                                                                                                                                                           |
| [**IUIAutomationTextRangeArray**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrangearray)<br/>                     | Represents a collection of [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) objects.<br/>                                                                                                                                                                                                                                          |
| [**IUIAutomationTogglePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtogglepattern)<br/>                       | Provides access to a control that can cycle through a set of states, and maintain a state after it is set.<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationTransformPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern)<br/>                 | Provides access to a control that can be moved, resized, or rotated.<br/>                                                                                                                                                                                                                                                                         |
| [**IUIAutomationTransformPattern2**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtransformpattern2)<br/>           | Extends the [**IUIAutomationTransformPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern) interface to enable UI Automation clients to programmatically access the viewport zooming functionality of a control.<br/>                                                                                                                              |
| [**IUIAutomationValuePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationvaluepattern)<br/>                         | Provides access to a control that contains a value that does not span a range and that can be represented as a string.<br/>                                                                                                                                                                                                                       |
| [**IUIAutomationVirtualizedItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationvirtualizeditempattern)<br/>     | Represents an virtualized item, which is an item that is represented by a placeholder automation element in the UI Automation tree.<br/>                                                                                                                                                                                                          |
| [**IUIAutomationWindowPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationwindowpattern)<br/>                       | Provides access to the fundamental functionality of a window.<br/>                                                                                                                                                                                                                                                                                |
