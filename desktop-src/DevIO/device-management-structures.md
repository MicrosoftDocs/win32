# Device Management Structures

The following structures are used with device management:

## In this section



| Topic                                                                                                        | Description                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CHANGER\_ELEMENT**](changer-element-str.md)<br/>                                                   | Represents a changer element.<br/>                                                                                                                                                                                                                       |
| [**CHANGER\_ELEMENT\_LIST**](changer-element-list-str.md)<br/>                                        | Represents a range of elements of a single type, typically for an operation such as getting or initializing the status of multiple elements.<br/>                                                                                                        |
| [**CHANGER\_ELEMENT\_STATUS**](changer-element-status-str.md)<br/>                                    | Represents the status of the specified element.<br/>                                                                                                                                                                                                     |
| [**CHANGER\_ELEMENT\_STATUS\_EX**](changer-element-status-ex-str.md)<br/>                             | Represents the status of the specified element.<br/>                                                                                                                                                                                                     |
| [**CHANGER\_EXCHANGE\_MEDIUM**](changer-exchange-medium-str.md)<br/>                                  | Contains information the [**IOCTL\_CHANGER\_EXCHANGE\_MEDIUM**](ioctl-changer-exchange-medium.md) control code uses to move a piece of media to a destination, and the piece of media originally in the first destination to a second destination.<br/> |
| [**CHANGER\_INITIALIZE\_ELEMENT\_STATUS**](changer-initialize-element-status-str.md)<br/>             | Represents the status of all media changer elements or the specified elements of a particular type.<br/>                                                                                                                                                 |
| [**CHANGER\_MOVE\_MEDIUM**](changer-move-medium-str.md)<br/>                                          | Contains information that the [**IOCTL\_CHANGER\_MOVE\_MEDIUM**](ioctl-changer-move-medium.md) control code uses to move a piece of media to a destination.<br/>                                                                                        |
| [**CHANGER\_PRODUCT\_DATA**](changer-product-data-str.md)<br/>                                        | Represents product data for a changer device. It is used by the [**IOCTL\_CHANGER\_GET\_PRODUCT\_DATA**](ioctl-changer-get-product-data.md) control code.<br/>                                                                                          |
| [**CHANGER\_READ\_ELEMENT\_STATUS**](changer-read-element-status-str.md)<br/>                         | Contains information that the [**IOCTL\_CHANGER\_GET\_ELEMENT\_STATUS**](ioctl-changer-get-element-status.md) control code needs to determine the elements whose status is to be retrieved.<br/>                                                        |
| [**CHANGER\_SEND\_VOLUME\_TAG\_INFORMATION**](changer-send-volume-tag-information-str.md)<br/>        | Contains information that the [**IOCTL\_CHANGER\_QUERY\_VOLUME\_TAGS**](ioctl-changer-query-volume-tags.md) control code uses to determine the volume information to be retrieved.<br/>                                                                 |
| [**CHANGER\_SET\_ACCESS**](changer-set-access-str.md)<br/>                                            | Contains information that the [**IOCTL\_CHANGER\_SET\_ACCESS**](ioctl-changer-set-access.md) control code needs to set the state of the device's insert/eject port, door, or keypad.<br/>                                                               |
| [**CHANGER\_SET\_POSITION**](changer-set-position-str.md)<br/>                                        | Contains information needed by the [**IOCTL\_CHANGER\_SET\_POSITION**](ioctl-changer-set-position.md) control code to set the changer's robotic transport mechanism to the specified element address.<br/>                                              |
| [**CLASS\_MEDIA\_CHANGE\_CONTEXT**](class-media-change-context-str.md)<br/>                           | Contains information associated with a media change event.<br/>                                                                                                                                                                                          |
| [**\_DEV\_BROADCAST\_USERDEFINED**](-dev-broadcast-userdefined-str.md)<br/>                           | Contains the user-defined event and optional data associated with the [DBT\_USERDEFINED](dbt-userdefined.md) device event.<br/>                                                                                                                         |
| [**DEV\_BROADCAST\_DEVICEINTERFACE**](dev-broadcast-deviceinterface-str.md)<br/>                      | Contains information about a class of devices.<br/>                                                                                                                                                                                                      |
| [**DEV\_BROADCAST\_HANDLE**](dev-broadcast-handle-str.md)<br/>                                        | Contains information about a file system handle.<br/>                                                                                                                                                                                                    |
| [**DEV\_BROADCAST\_HDR**](dev-broadcast-hdr-str.md)<br/>                                              | Serves as a standard header for information related to a device event reported through the [**WM\_DEVICECHANGE**](wm-devicechange.md) message.<br/>                                                                                                     |
| [**DEV\_BROADCAST\_OEM**](dev-broadcast-oem-str.md)<br/>                                              | Contains information about a OEM-defined device type.<br/>                                                                                                                                                                                               |
| [**DEV\_BROADCAST\_PORT**](dev-broadcast-port-str.md)<br/>                                            | Contains information about a modem, serial, or parallel port.<br/>                                                                                                                                                                                       |
| [**DEV\_BROADCAST\_VOLUME**](dev-broadcast-volume-str.md)<br/>                                        | Contains information about a logical volume.<br/>                                                                                                                                                                                                        |
| [**DEVICE\_DATA\_SET\_LB\_PROVISIONING\_STATE**](device-data-set-lb-provisioning-state.md)<br/>       | Output structure for the **DeviceDsmAction\_Allocation** action of the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                              |
| [**DEVICE\_DATA\_SET\_RANGE**](device-data-set-range.md)<br/>                                         | Provides data set range information for use with the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                                                |
| [**DEVICE\_DATA\_SET\_REPAIR\_PARAMETERS**](device-data-set-repair-parameters.md)<br/>                | Specifies parameters for the repair operation.<br/>                                                                                                                                                                                                      |
| [**DEVICE\_DSM\_NOTIFICATION\_PARAMETERS**](device-dsm-notification-parameters.md)<br/>               | Contains parameters for the **DeviceDsmAction\_Notification** action for the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                        |
| [**DEVICE\_DSM\_OFFLOAD\_READ\_PARAMETERS**](device-dsm-offload-read-parameters.md)<br/>              | Contains parameters for the **DeviceDsmAction\_OffloadRead** action for the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                         |
| [**DEVICE\_DSM\_OFFLOAD\_WRITE\_PARAMETERS**](device-dsm-offload-write-parameters.md)<br/>            | Specifies parameters for the offload write operation.<br/>                                                                                                                                                                                               |
| [**DEVICE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](device-manage-data-set-attributes.md)<br/>                | Input structure for the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                                                                             |
| [**DEVICE\_MANAGE\_DATA\_SET\_ATTRIBUTES\_OUTPUT**](device-manage-data-set-attributes-output.md)<br/> | Output structure for the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                                                                            |
| [**DEVICE\_MEDIA\_INFO**](device-media-info-str.md)<br/>                                              | Provides information about the media supported by a device.<br/>                                                                                                                                                                                         |
| [**GET\_CHANGER\_PARAMETERS**](get-changer-parameters-str.md)<br/>                                    | Represents the parameters of a changer.<br/>                                                                                                                                                                                                             |
| [**GET\_MEDIA\_TYPES**](get-media-types-str.md)<br/>                                                  | Contains information about the media types supported by a device.<br/>                                                                                                                                                                                   |
| [**MEDIA\_SERIAL\_NUMBER\_DATA**](media-serial-number-data-str.md)<br/>                               | Contains the serial number of a USB device. It is used by the [**IOCTL\_STORAGE\_GET\_MEDIA\_SERIAL\_NUMBER**](ioctl-storage-get-media-serial-number.md) control code.<br/>                                                                             |
| [**PREVENT\_MEDIA\_REMOVAL**](prevent-media-removal-str.md)<br/>                                      | Provides removable media locking data. It is used by the [**IOCTL\_STORAGE\_MEDIA\_REMOVAL**](ioctl-storage-media-removal.md) control code.<br/>                                                                                                        |
| [**READ\_ELEMENT\_ADDRESS\_INFO**](read-element-address-info-str.md)<br/>                             | Represents the volume tag information. It is used by the [**IOCTL\_CHANGER\_QUERY\_VOLUME\_TAGS**](ioctl-changer-query-volume-tags.md) control code.<br/>                                                                                               |
| [**STORAGE\_DEVICE\_NUMBER**](storage-device-number.md)<br/>                                          | Contains information about a device. This structure is used by the [IOCTL\_STORAGE\_GET\_DEVICE\_NUMBER](ioctl-storage-get-device-number.md) control code.<br/>                                                                                         |
| [**STORAGE\_HOTPLUG\_INFO**](storage-hotplug-info-str.md)<br/>                                        | Provides information about the hotplug information of a device.<br/>                                                                                                                                                                                     |
| [**STORAGE\_OFFLOAD\_READ\_OUTPUT**](storage-offload-read-output.md)<br/>                             | Output structure for the **DeviceDsmAction\_OffloadRead** action of the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                             |
| [**STORAGE\_OFFLOAD\_TOKEN**](storage-offload-token.md)<br/>                                          | The token used to represent a portion of a file used in by offload read and write operations.<br/>                                                                                                                                                       |
| [**STORAGE\_OFFLOAD\_WRITE\_OUTPUT**](storage-offload-write-output.md)<br/>                           | Output structure for the **DeviceDsmAction\_OffloadWrite** action of the [**IOCTL\_STORAGE\_MANAGE\_DATA\_SET\_ATTRIBUTES**](ioctl-storage-manage-data-set-attributes.md) control code.<br/>                                                            |
| [**STORAGE\_READ\_CAPACITY**](storage-read-capacity.md)<br/>                                          | Contains information about the size of a device. This is returned from the [**IOCTL\_STORAGE\_READ\_CAPACITY**](ioctl-storage-read-capacity.md) control code.<br/>                                                                                      |
