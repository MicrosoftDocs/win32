| Topic                                            | Description                                                                                                  |

| [**PWM\_POLARITY**](/windows/desktop/api/Pwm/ne-pwm-pwm_polarity)<br/> | Represents the Pulse Width Modulation (PWM) signal level polarity with respect to the duty cycle.<br/> |
