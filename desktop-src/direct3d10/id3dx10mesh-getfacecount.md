# ID3DX10Mesh::GetFaceCount method

Retrieves the number of faces in the mesh.

## Syntax


```C++
UINT GetFaceCount();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**UINT**](../winprog/windows-data-types.md)**

Returns the number of faces in the mesh.
