# ID3DX10Font::GetDC method

Return a handle to a display device context (DC) that has the font set onto it.

## Syntax


```C++
HDC GetDC();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**HDC**](../winprog/windows-data-types.md)**

Handle to a display DC.
