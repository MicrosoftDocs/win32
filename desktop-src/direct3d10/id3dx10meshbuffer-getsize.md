# ID3DX10MeshBuffer::GetSize method

Get the size of the mesh buffer, in bytes.

## Syntax


```C++
SIZE_T GetSize();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**SIZE\_T**](../winprog/windows-data-types.md)**

The size of the mesh buffer, in bytes.
