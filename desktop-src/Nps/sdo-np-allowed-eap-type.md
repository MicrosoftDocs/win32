# NP-Allowed-EAP-Type

EAP encryption modes permitted for a connection.

**LDAP Name:** msNPAllowedEapType

**ID:** 4106 = [**IAS\_ATTRIBUTE\_NP\_ALLOWED\_EAP\_TYPE**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntax:** Integer

**Values:  **

Defined by the Internet Assigned Numbers Authority (IANA) in the [Extensible Authentication Protocol (EAP) Registry](https://www.iana.org/assignments/eap-numbers) document based on [RFC 3748](https://www.ietf.org/rfc/rfc3748.txt).

-   **MD5-Challenge** (4)
-   **EAP\_TLS** (13)
-   **PEAP** (25)
-   **MS-EAP-Authentication** (26)
