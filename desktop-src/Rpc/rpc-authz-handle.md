# RPC\_AUTHZ\_HANDLE

The **RPC\_AUTHZ\_HANDLE** data type declares an authorization handle. This handle points to the privileges information for the client application that made the remote procedure call.


```C++
typedef void* RPC_AUTHZ_HANDLE;
```
