# Policy-Replication-Flags attribute

Determines which LSA properties are replicated to clients.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Policy-Replication-Flags             |
| Ldap-Display-Name | policyReplicationFlags               |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.633               |
| System-Id-Guid    | 19405b96-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
