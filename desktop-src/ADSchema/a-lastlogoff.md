# Last-Logoff attribute

This attribute is not used.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Last-Logoff                          |
| Ldap-Display-Name | lastLogoff                           |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | Each time the user logs off.         |
| Attribute-Id      | 1.2.840.113556.1.4.51                |
| System-Id-Guid    | bf967996-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
