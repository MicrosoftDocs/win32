# Authentication-Options attribute

The authentication options used in ADSI to bind to directory services objects.



| Entry | Value |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | Authentication-Options                                                                                                                                                                                                                                                       |
| Ldap-Display-Name | authenticationOptions                                                                                                                                                                                                                                                        |
| Size              | 4 bytes. Values defined in IADS.h: ADS\_SECURE\_AUTHENTICATION 0x1, ADS\_USE\_ENCRYPTION 0x2, ADS\_USE\_SSL 0x2, ADS\_READONLY\_SERVER 0x4, ADS\_PROMPT\_CREDENTIALS 0x8, ADS\_NO\_AUTHENTICATION 0x10, ADS\_FAST\_BIND 0x20, ADS\_USE\_SIGNING 0x40, ADS\_USE\_SEALING 0x80 |
| Update Privilege  | \-                                                                                                                                                                                                                                                                           |
| Update Frequency  | \-                                                                                                                                                                                                                                                                           |
| Attribute-Id      | 1.2.840.113556.1.4.11                                                                                                                                                                                                                                                        |
| System-Id-Guid    | bf967928-0de6-11d0-a285-00aa003049e2                                                                                                                                                                                                                                         |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                                                                                                                                                                                                         |
