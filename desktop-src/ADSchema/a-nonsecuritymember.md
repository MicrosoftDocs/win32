# Non-Security-Member attribute

Nonsecurity members of a group. Used for Exchange distribution lists.



| Entry | Value |
|-------------------|--------------------------------------------------|
| CN                | Non-Security-Member                              |
| Ldap-Display-Name | nonSecurityMember                                |
| Size              | \-                                               |
| Update Privilege  | Domain administrator                             |
| Update Frequency  | Whenever a user is added or removed from the DL. |
| Attribute-Id      | 1.2.840.113556.1.4.530                           |
| System-Id-Guid    | 52458018-ca6a-11d0-afff-0000f80367c1             |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
