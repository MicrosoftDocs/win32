# Object-Version attribute

This can be used to store a version number for the object.



| Entry | Value |
|-------------------|----------------------------------------------------------------|
| CN                | Object-Version                                                 |
| Ldap-Display-Name | objectVersion                                                  |
| Size              | 4 bytes                                                        |
| Update Privilege  | The designer of the object would set this value.               |
| Update Frequency  | This value should be incremented each time the object changes. |
| Attribute-Id      | 1.2.840.113556.1.2.76                                          |
| System-Id-Guid    | 16775848-47f3-11d1-a9c3-0000f80367c1                           |
| Syntax            | [**Enumeration**](s-enumeration.md)                           |
