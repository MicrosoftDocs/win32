# System-Poss-Superiors attribute

The list of classes that can contain this class. This list can only be modified by the system.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------|
| CN                | System-Poss-Superiors                                                       |
| Ldap-Display-Name | systemPossSuperiors                                                         |
| Size              | \-                                                                          |
| Update Privilege  | Schema administrator                                                        |
| Update Frequency  | When the class is created or a new possible superior is added to the class. |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-Id-Guid    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)             |
