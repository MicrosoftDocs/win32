# ms-WMI-int8Max attribute

The maximum value for a WMI 64-bit integer parameter object.



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | ms-WMI-int8Max                       |
| Ldap-Display-Name | msWMI-Int8Max                        |
| Size              | 8 bytes                              |
| Update Privilege  | Group Policy Administrator           |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1633              |
| System-Id-Guid    | e3d8b547-003d-4946-a32b-dc7cedc96b74 |
| Syntax            | [**Interval**](s-interval.md)       |
