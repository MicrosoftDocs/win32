# Proxy-Addresses attribute

A proxy address is the address by which a Microsoft Exchange Server recipient object is recognized in a foreign mail system. Proxy addresses are required for all recipient objects, such as custom recipients and distribution lists.



| Entry | Value |
|-------------------|------------------------------------------------------------------------------------------------------|
| CN                | Proxy-Addresses                                                                                      |
| Ldap-Display-Name | proxyAddresses                                                                                       |
| Size              | \-                                                                                                   |
| Update Privilege  | This value is set by the system.                                                                     |
| Update Frequency  | Created by a DLL provided with the Address-Type directory object when the address type is installed. |
| Attribute-Id      | 1.2.840.113556.1.2.210                                                                               |
| System-Id-Guid    | bf967a06-0de6-11d0-a285-00aa003049e2                                                                 |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                          |
