# ACS-RSVP-Account-Files-Location attribute

The directory location of RSVP account files. Defaults to the "system32" subdirectory located in the default system directory.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ACS-RSVP-Account-Files-Location             |
| Ldap-Display-Name | aCSRSVPAccountFilesLocation                 |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.900                      |
| System-Id-Guid    | f072230f-aef5-11d1-bdcf-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
