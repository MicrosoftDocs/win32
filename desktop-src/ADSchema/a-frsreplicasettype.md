# FRS-Replica-Set-Type attribute

This is a code that indicates whether this is a sysvol replica set, a DFS replica set, or other replica set.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | FRS-Replica-Set-Type                 |
| Ldap-Display-Name | fRSReplicaSetType                    |
| Size              | \-                                   |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.31                |
| System-Id-Guid    | 26d9736b-6070-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
