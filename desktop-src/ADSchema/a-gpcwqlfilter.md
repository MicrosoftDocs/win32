# GPC-WQL-Filter attribute

Used to store a string that contains a GUID for the filter and a WMI namespace path.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-WQL-Filter                                                                  |
| Ldap-Display-Name | gPCWQLFilter                                                                    |
| Size              | \-                                                                              |
| Update Privilege  | Domain administrator                                                            |
| Update Frequency  | Only when the administrator changes the filter property in the Group Policy UI. |
| Attribute-Id      | 1.2.840.113556.1.4.1694                                                         |
| System-Id-Guid    | 7bd4c7a6-1add-4436-8c04-3999a880154c                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                     |
