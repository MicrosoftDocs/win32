# ms-DS-Source-Object-DN attribute

Contains the string representation of the distinguished name of the object in another forest that corresponds to this object.

| CN                | ms-DS-Source-Object-DN                      |
| Ldap-Display-Name | msDS-SourceObjectDN                         |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |

| System-Id-Guid    | 773e93af-d3b4-48d4-b3f9-06457602d3d0        |
