# Domain-Replica attribute

Unicode String Attribute, gives the list of Windows NT 4.0 Replication Domain Controllers.



|                   |                                             |
|-------------------|---------------------------------------------|
| CN                | Domain-Replica                              |
| Ldap-Display-Name | domainReplica                               |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.158                      |
| System-Id-Guid    | bf96795e-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
