# Extension-Name attribute

The name of a property page used to extend the UI of a directory object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Extension-Name                              |
| Ldap-Display-Name | extensionName                               |
| Size              | \-                                          |
| Update Privilege  | The creator of the extension object.        |
| Update Frequency  | Whenever a new UI extension is created.     |
| Attribute-Id      | 1.2.840.113556.1.2.227                      |
| System-Id-Guid    | bf967972-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
