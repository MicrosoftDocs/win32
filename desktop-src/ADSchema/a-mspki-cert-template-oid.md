# ms-PKI-Cert-Template-OID attribute

Specifies the object identifier for a certificate template.



|                   |                                                                                                   |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Cert-Template-OID                                                                          |
| Ldap-Display-Name | msPKI-Cert-Template-OID                                                                           |
| Size              | 64 bytes                                                                                          |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1436                                                                           |
| System-Id-Guid    | 3164c36a-ba26-468c-8bda-c1e5cc256728                                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                       |
