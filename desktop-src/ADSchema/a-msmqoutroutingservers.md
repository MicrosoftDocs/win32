# MSMQ-Out-Routing-Servers attribute

DN links to MSMQ routing servers through which all outgoing traffic for this computer should be routed.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Out-Routing-Servers                |
| Ldap-Display-Name | mSMQOutRoutingServers                   |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.928                  |
| System-Id-Guid    | 9a0dc32b-c100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
