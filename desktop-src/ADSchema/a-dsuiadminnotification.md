# DS-UI-Admin-Notification attribute

This is a list of the GUIDs of COM objects that support a callback interface that DSAdmin calls when an action has occurred on an object through the UI.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | DS-UI-Admin-Notification                    |
| Ldap-Display-Name | dSUIAdminNotification                       |
| Size              | \-                                          |
| Update Privilege  | Domain Administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1343                     |
| System-Id-Guid    | f6ea0a94-6f91-11d2-9905-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
