# FRS-Level-Limit attribute

Limit depth of directory tree to replicate for file replication.

| CN                | FRS-Level-Limit                      |
| Ldap-Display-Name | fRSLevelLimit                        |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
