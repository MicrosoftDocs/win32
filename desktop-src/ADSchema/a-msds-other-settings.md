# ms-DS-Other-Settings attribute

This multiple-valued attribute is used to store any configurable setting for the DS stored in the NAME=VALUE format.

|-------------------|---------------------------------------------|
| CN                | ms-DS-Other-Settings                        |
| Ldap-Display-Name | msDS-Other-Settings                         |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1621                     |
| System-Id-Guid    | 79d2f34c-9d7d-42bb-838f-866b3e4400e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
