# Print-Memory attribute

The amount of memory installed in a printer.

|-------------------|--------------------------------------|
| CN                | Print-Memory                         |
| Ldap-Display-Name | printMemory                          |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
