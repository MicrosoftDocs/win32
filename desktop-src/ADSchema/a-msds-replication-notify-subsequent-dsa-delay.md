# ms-DS-Replication-Notify-Subsequent-DSA-Delay attribute

This attribute controls the delay in time between notification of each subsequent replica partner for an NC.



| Entry | Value |
|-------------------|-----------------------------------------------|
| CN                | ms-DS-Replication-Notify-Subsequent-DSA-Delay |
| Ldap-Display-Name | msDS-Replication-Notify-Subsequent-DSA-Delay  |
| Size              | \-                                            |
| Update Privilege  | This value is set by the system.              |
