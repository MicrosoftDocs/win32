# ms-WMI-int8Min attribute

The minimum value for a WMI 64-bit integer parameter object.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ms-WMI-int8Min                       |
| Ldap-Display-Name | msWMI-Int8Min                        |
| Size              | 8 bytes                              |
| Update Privilege  | Group Policy Administrator           |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1634              |
| System-Id-Guid    | ed1489d1-54cc-4066-b368-a00daa2664f1 |
| Syntax            | [**Interval**](s-interval.md)       |
