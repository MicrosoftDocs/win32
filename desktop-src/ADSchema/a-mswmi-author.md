# ms-WMI-Author attribute

The author of an instance of a class.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-Author                               |
| Ldap-Display-Name | msWMI-Author                                |
| Size              | Less that 25 characters.                    |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1623                     |
| System-Id-Guid    | 6366c0c1-6972-4e66-b3a5-1d52ad0c0547        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
