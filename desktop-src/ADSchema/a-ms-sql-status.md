# MS-SQL-Status attribute

The status for the SQL server. The status is **TRUE** if the server is running.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Status                        |
| Ldap-Display-Name | mS-SQL-Status                        |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | At system startup.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1380              |
| System-Id-Guid    | 9a7d4770-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Interval**](s-interval.md)       |
