# Print-Color attribute

**TRUE** if a printer can print in color.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Print-Color                          |
| Ldap-Display-Name | printColor                           |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.243               |
| System-Id-Guid    | 281416d3-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
