# IndexedScopes attribute

List of directory scopes (for example, C:\\ or D:\) that are indexed.

| CN                | IndexedScopes                               |
| Ldap-Display-Name | indexedScopes                               |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
