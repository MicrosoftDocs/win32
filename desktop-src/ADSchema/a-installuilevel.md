# Install-Ui-Level attribute

This attribute contains information for the type (level) of installation that is used for the user interface. Possible installation levels are as follows: 2 INSTALLUILEVEL\_NONE (silent installation) 3 INSTALLUILEVEL\_BASIC (simple installation with error handling) 4 INSTALLUILEVEL\_REDUCED (authored UI, wizard dialog boxes suppressed) 5 INSTALLUILEVEL\_FULL (authored UI with wizards, progress, errors)

| CN                | Install-Ui-Level                     |
| Ldap-Display-Name | installUiLevel                       |
