# Phone-Mobile-Other attribute

A list of alternate mobile phone numbers.



|                   |                                                                                  |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Phone-Mobile-Other                                                               |
| Ldap-Display-Name | otherMobile                                                                      |
| Size              | \-                                                                               |
| Update Privilege  | Domain administrator or account owner.                                           |
| Update Frequency  | When the user's record is created and whenever the phone number needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.647                                                           |
| System-Id-Guid    | 0296c11e-40da-11d1-a9c0-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
