# ms-ieee-80211-Data attribute

Stores a list of preferred network configurations in Group Policy for wireless networks.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | ms-ieee-80211-Data                                                               |
| Ldap-Display-Name | msieee80211-Data                                                                 |
| Size              | \-                                                                               |
| Update Privilege  | Domain administrator                                                             |
| Update Frequency  | Each time a Domain Admin changes the wireless network policy for a domain or OU. |
| Attribute-Id      | 1.2.840.113556.1.4.1821                                                          |
| System-Id-Guid    | 0e0d0938-2658-4580-a9f6-7a0ac7b566cb                                             |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                            |
