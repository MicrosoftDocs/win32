# Enable-Per-User-Reversibly-Encrypted-Password extended right

Extended control access right that allows users to enable or disable the "reversible encrypted password" setting for user and computer objects.



| Entry | Value |
|--------------|-----------------------------------------------|
| CN           | Enable-Per-User-Reversibly-Encrypted-Password |
| Display-Name | Enable Per User Reversibly Encrypted Password |
| Rights-GUID  | 05c74c5e-4deb-43b4-bd9f-86664c2a7fd5          |
