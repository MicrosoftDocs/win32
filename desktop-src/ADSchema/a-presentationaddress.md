# Presentation-Address attribute

Specifies a presentation address associated with an object that represents an OSI application entity.

|-------------------|-----------------------------------------------------------------------|
| CN                | Presentation-Address                                                  |
| Ldap-Display-Name | presentationAddress                                                   |
| Size              | \-                                                                    |
| Update Privilege  | \-                                                                    |
| Update Frequency  | \-                                                                    |
| Attribute-Id      | 2.5.4.29                                                              |
| System-Id-Guid    | a8df744b-c5ea-11d1-bbcb-0080c76670c0                                  |
| Syntax            | [**Object(Presentation-Address)**](s-object-presentation-address.md) |
