# Generate-RSoP-Logging extended right

The user who has the rights on an OU or Domain will be able to generate logging mode RSoP data for the users or computers within the OU.



| Entry | Value |
|--------------|--------------------------------------------|
| CN           | Generate-RSoP-Logging                      |
| Display-Name | Generate Resultant Set of Policy (Logging) |
| Rights-GUID  | b7b1b3de-ab09-4242-9e30-9980e5d322f7       |
