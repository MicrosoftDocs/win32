# Alt-Security-Identities attribute

Contains mappings for X.509 certificates or external Kerberos user accounts to this user for the purpose of authentication.



| Entry | Value |
|-------------------|-----------------------------------------------------|
| CN                | Alt-Security-Identities                             |
| Ldap-Display-Name | altSecurityIdentities                               |
| Size              | \-                                                  |
| Update Privilege  | Domain administrator                                |
| Update Frequency  | Each time a new authentication mechanism is needed. |
| Attribute-Id      | 1.2.840.113556.1.4.867                              |
| System-Id-Guid    | 00fbf30c-91fe-11d1-aebc-0000f80367c1                |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)         |
