# Unexpire-Password extended right

Extended control access right that allows a user to restore an expired password for a user object.



| Entry | Value |
|--------------|--------------------------------------|
| CN           | Unexpire-Password                    |
| Display-Name | Unexpire Password                    |
| Rights-GUID  | ccc2dc7d-a6ad-4a7a-8846-c04e3cc53501 |
