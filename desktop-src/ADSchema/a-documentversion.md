# documentVersion attribute

The documentVersion attribute type specifies the version number of a document.



|                   |                                             |
|-------------------|---------------------------------------------|
| CN                | documentVersion                             |
| Ldap-Display-Name | documentVersion                             |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.13                  |
| System-Id-Guid    | 94b3a8a9-d613-4cec-9aad-5fbcc1046b43        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
