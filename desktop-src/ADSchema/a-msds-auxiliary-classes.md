# ms-DS-Auxiliary-Classes attribute

This attribute lists the auxiliary classes that have been dynamically attached to an object. This attribute is not associated with a class. It is automatically populated by the system.



| Entry | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-DS-Auxiliary-Classes                                         |
| Ldap-Display-Name | msDS-Auxiliary-Classes                                          |
| Size              | \-                                                              |
| Update Privilege  | This value is set by the system.                                |
| Update Frequency  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1458                                         |
| System-Id-Guid    | c4af1073-ee50-4be0-b8c0-89a41fe99abe                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
