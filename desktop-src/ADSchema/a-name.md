# RDN attribute

The Relative Distinguished Name (RDN) of an object. An RDN is the relative portion of a distinguished name (DN), which uniquely identifies an LDAP object.



| Entry | Value |
|-------------------|------------------------------------------------|
| CN                | RDN                                            |
| Ldap-Display-Name | name                                           |
| Size              | \-                                             |
| Update Privilege  | This value is set by the schema administrator. |
| Update Frequency  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1                           |
| System-Id-Guid    | bf967a0e-0de6-11d0-a285-00aa003049e2           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
