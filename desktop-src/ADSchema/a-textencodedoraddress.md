# Text-Encoded-OR-Address attribute

This attribute is used to support X.400 addresses in a text format.

| CN                | Text-Encoded-OR-Address                     |
| Ldap-Display-Name | textEncodedORAddress                        |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
