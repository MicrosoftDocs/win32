# MS-SQL-ServiceAccount attribute

A user defined string. The default is set to ServiceAccount.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-ServiceAccount                       |
| Ldap-Display-Name | mS-SQL-ServiceAccount                       |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | At system setup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1369                     |
| System-Id-Guid    | 64933a3e-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
