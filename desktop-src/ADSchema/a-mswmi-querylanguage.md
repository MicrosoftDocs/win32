# ms-WMI-QueryLanguage attribute

Identifies a WQL query language.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-QueryLanguage                        |
| Ldap-Display-Name | msWMI-QueryLanguage                         |
| Size              | Less than twenty characters.                |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1643                     |
| System-Id-Guid    | 7d3cfa98-c17b-4254-8bd7-4de9b932a345        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
