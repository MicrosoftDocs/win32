# ms-DS-Revealed-Users attribute

Used with RODCs to identify the user objects whose secrets have been disclosed to that RODC.



| Entry | Value |
|-------------------|-------------------------------------------------|
| CN                | ms-DS-Revealed-Users                            |
| Ldap-Display-Name | msDS-RevealedUsers                              |
| Size              | \-                                              |
| Update Privilege  | \-                                              |
| Update Frequency  | \-                                              |

| System-Id-Guid    | 185c7821-3749-443a-bd6a-288899071adb            |
| Syntax            | [**Object(DN-Binary)**](s-object-dn-binary.md) |
