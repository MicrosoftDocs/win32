# ipHostNumber attribute

Contains the IP address of the host in dotted decimal notation, omitting the leading zeros.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ipHostNumber                         |
| Ldap-Display-Name | ipHostNumber                         |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.19                     |
| System-Id-Guid    | de8bb721-85dc-4fde-b687-9657688e667e |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
