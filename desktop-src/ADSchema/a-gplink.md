# GP-Link attribute

A sorted list of Group Policy options. Each option is a **DWORD**. Use of the UNICODE string is a convenience.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | GP-Link                                     |
| Ldap-Display-Name | gPLink                                      |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.891                      |
| System-Id-Guid    | f30e3bbe-9ff0-11d1-b603-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
