# msmq-Peek extended right

Allows peeking at messages in the queue.



| Entry | Value |
|--------------|--------------------------------------|
| CN           | msmq-Peek                            |
| Display-Name | Peek Message                         |
| Rights-GUID  | 06bd3201-df3e-11d1-9c86-006008764d0e |
