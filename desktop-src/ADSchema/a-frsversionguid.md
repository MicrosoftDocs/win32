# FRS-Version-GUID attribute

If present, changing this value indicates a configuration change has been made on this replica set.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Version-GUID                                      |
| Ldap-Display-Name | fRSVersionGUID                                        |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.43                                 |
| System-Id-Guid    | 26d9736c-6070-11d1-a9c6-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
