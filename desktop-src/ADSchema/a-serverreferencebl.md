# Server-Reference-BL attribute

Found in the domain naming context. The distinguished name of a computer under the sites folder.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Server-Reference-BL                     |
| Ldap-Display-Name | serverReferenceBL                       |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.516                  |
| System-Id-Guid    | 26d9736e-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
