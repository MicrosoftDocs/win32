# ms-DS-Top-Quota-Usage attribute

The list of top quota users currently in the directory database, ordered by decreasing quota usage.

|-------------------|---------------------------------------------|
| CN                | ms-DS-Top-Quota-Usage                       |
| Ldap-Display-Name | msDS-TopQuotaUsage                          |
| Size              | \-                                          |
| Update Privilege  | \-                                          |

| System-Id-Guid    | 7b7cce4f-f1f5-4bb6-b7eb-23504af19e75        |
