# Help-Data16 attribute

This attribute was used for the Win16 help file format for Exchange 4.0. It is not used for any other versions of Exchange.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Help-Data16                                           |
| Ldap-Display-Name | helpData16                                            |
| Size              | \-                                                    |
| Update Privilege  | This is used by the system.                           |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.402                                |
| System-Id-Guid    | 5fd424a7-1262-11d0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
