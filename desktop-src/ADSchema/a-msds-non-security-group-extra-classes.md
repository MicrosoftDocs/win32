# ms-DS-Non-Security-Group-Extra-Classes attribute

The common names of the nonstandard classes that can be added to a nonsecurity group through the Active Directory Users and Computers snap-in.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Non-Security-Group-Extra-Classes      |
| Ldap-Display-Name | msDS-Non-Security-Group-Extra-Classes       |
| Size              | \-                                          |
| Update Privilege  | Domain Administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1689                     |
| System-Id-Guid    | 2de144fc-1f52-486f-bdf4-16fcc3084e54        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
