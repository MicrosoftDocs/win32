# ipNetworkNumber attribute

Contains an IP network number in dotted decimal notation, omitting the leading zeros.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ipNetworkNumber                      |
| Ldap-Display-Name | ipNetworkNumber                      |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| System-Id-Guid    | 4e3854f4-3087-42a4-a813-bb0c528958d3 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
