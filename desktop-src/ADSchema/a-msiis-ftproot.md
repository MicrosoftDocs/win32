# ms-IIS-FTP-Root attribute

This attribute determines the file server share. It is used in conjunction with ms-IIS-FTP-Dir to determine the FTP user home directory.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------|
| CN                | ms-IIS-FTP-Root                                                                                                                 |
| Ldap-Display-Name | msIIS-FTPRoot                                                                                                                   |
| Size              | 30-50 characters (15-25 Unicode characters for each property)                                                                   |
| Update Privilege  | Domain administrator & local system                                                                                             |
| Update Frequency  | This property is set when the administrator creates the website and sets FTP isolation. It's rarely going to change after that. |
| Attribute-Id      | 1.2.840.113556.1.4.1785                                                                                                         |
| System-Id-Guid    | 2a7827a4-1483-49a5-9d84-52e3812156b4                                                                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                                                     |
