# Repl-Topology-Stay-Of-Execution attribute

The delay between deleting a server object and it being permanently removed from the replication topology.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Repl-Topology-Stay-Of-Execution      |
| Ldap-Display-Name | replTopologyStayOfExecution          |
| Size              | 4 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | Whenever a server object is deleted. |
| Attribute-Id      | 1.2.840.113556.1.4.677               |
| System-Id-Guid    | 7bfdcb83-4807-11d1-a9c3-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
