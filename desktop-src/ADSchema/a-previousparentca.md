# Previous-Parent-CA attribute

Reference to the certification authorities that issued the last expired certificate for a certification authority.

|-------------------|-----------------------------------------|
| CN                | Previous-Parent-CA                      |
| Ldap-Display-Name | previousParentCA                        |
| Size              | \-                                      |
| Update Privilege  | \-                                      |

| Attribute-Id      | 1.2.840.113556.1.4.694                  |
| System-Id-Guid    | 963d273d-48be-11d1-a9c3-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
