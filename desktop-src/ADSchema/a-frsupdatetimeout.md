# FRS-Update-Timeout attribute

The maximum time, in minutes, to wait to complete an update before giving up.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | FRS-Update-Timeout                   |
| Ldap-Display-Name | fRSUpdateTimeout                     |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.485               |
| System-Id-Guid    | 1be8f172-a9ff-11d0-afe2-00c04fd930c9 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
