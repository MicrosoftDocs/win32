# Address-Type attribute

A character string that describes the format of the user's address. Address types map to address formats. That is, by looking at a recipient's address type, client applications can determine how to format an address that is appropriate for the recipient.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------------------------------|
| CN                | Address-Type                                                                                                              |
| Ldap-Display-Name | addressType                                                                                                               |
| Size              | Address types: 3COM, ATT, CCMAIL, COMPUSERVE, EX, FAX, MSFAX, MCI, MHS, MS, MSA, MSN,PROFS,SMTP, SNADS, TELEX, X400, X500 |
| Update Privilege  | This value is set by the system.                                                                                          |
| Update Frequency  | Set when the object is created.                                                                                           |
| Attribute-Id      | 1.2.840.113556.1.2.350                                                                                                    |
| System-Id-Guid    | 5fd42464-1262-11d0-a060-00aa006c33ed                                                                                      |
| Syntax            | [**String(Teletex)**](s-string-teletex.md)                                                                               |
