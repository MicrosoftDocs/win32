# ms-WMI-TargetPath attribute

List of key/value pairs for uniquely identifying a WMI object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-TargetPath                           |
| Ldap-Display-Name | msWMI-TargetPath                            |
| Size              | Less than 100 characters.                   |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1648                     |
| System-Id-Guid    | 5006a79a-6bfe-4561-9f52-13cf4dd3e560        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
