# Create-Time-Stamp attribute

The date when this object was created. This value is replicated.



| Entry | Value |
|-------------------|---------------------------------------------------------------|
| CN                | Create-Time-Stamp                                             |
| Ldap-Display-Name | createTimeStamp                                               |
| Size              | 8 bytes                                                       |
| Update Privilege  | This value is set by the system.                              |
| Update Frequency  | When the object is created.                                   |
| Attribute-Id      | 2.5.18.1                                                      |
| System-Id-Guid    | 2df90d73-009f-11d2-aa4c-00c04fd7d83a                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
