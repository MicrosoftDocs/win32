# Attribute-ID attribute

The unique X.500 OID for identifying an attribute.



| Entry | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | Attribute-ID                                                    |
| Ldap-Display-Name | attributeID                                                     |
| Size              | \-                                                              |
| Update Privilege  | Schema administrator                                            |
| Update Frequency  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.30                                           |
| System-Id-Guid    | bf967922-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
