# Token-Groups-No-GC-Acceptable attribute

This attribute contains the list of SIDs due to a transitive group membership expansion operation on a given user or computer. Token groups cannot be retrieved if a Global Catalog is not present to retrieve the transitive reverse memberships.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Token-Groups-No-GC-Acceptable        |
| Ldap-Display-Name | tokenGroupsNoGCAcceptable            |
| Size              | \-                                   |
| Update Privilege  | The system sets this value.          |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1303              |
| System-Id-Guid    | 040fc392-33df-11d2-98b2-0000f87a57d4 |
| Syntax            | [**String(Sid)**](s-string-sid.md)  |
