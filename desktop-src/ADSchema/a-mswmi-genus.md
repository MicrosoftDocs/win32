# ms-WMI-Genus attribute

Identifies the object type of an encoding.



|                   |                                                                 |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-WMI-Genus                                                    |
| Ldap-Display-Name | msWMI-Genus                                                     |
| Size              | 4 bytes. 1 indicates a class. 2 indicates an instance.          |
| Update Privilege  | Domain administrator                                            |
| Update Frequency  | Whenever a class containing the attribute is added or modified. |
| Attribute-Id      | 1.2.840.113556.1.4.1677                                         |
| System-Id-Guid    | 50c8673a-8f56-4614-9308-9e1340fb9af3                            |
| Syntax            | [**Enumeration**](s-enumeration.md)                            |
