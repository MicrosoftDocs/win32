# Postal-Address attribute

The mailing address for the object.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------|
| CN                | Postal-Address                                                              |
| Ldap-Display-Name | postalAddress                                                               |
| Size              | \-                                                                          |
| Update Privilege  | Domain administrator or account owner.                                      |
| Update Frequency  | When the user's record is created and whenever the address needs to change. |
| Attribute-Id      | 2.5.4.16                                                                    |
| System-Id-Guid    | bf9679fc-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
