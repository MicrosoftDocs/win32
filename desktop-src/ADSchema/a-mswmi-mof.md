# ms-WMI-Mof attribute

Holds a MOF definition of some WMI object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-Mof                                  |
| Ldap-Display-Name | msWMI-Mof                                   |
| Size              | \-                                          |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1638                     |
| System-Id-Guid    | 6736809f-2064-443e-a145-81262b1f1366        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
