# Comment attribute (AD Schema)

The user's comments. This string can be a null string.



| Entry | Value |
|-------------------|--------------------------------------------------------------------------|
| CN                | Comment                                                                  |
| Ldap-Display-Name | info                                                                     |
| Size              | \-                                                                       |
| Update Privilege  | Domain administrator or account owner.                                   |
| Update Frequency  | Whenever the user or administrator needs to add comments on the account. |
| Attribute-Id      | 1.2.840.113556.1.2.81                                                    |
| System-Id-Guid    | bf96793e-0de6-11d0-a285-00aa003049e2                                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                              |
