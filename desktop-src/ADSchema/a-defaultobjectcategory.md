# Default-Object-Category attribute

The object category to use for the object if one is not specified.

|-------------------|-----------------------------------------|
| CN                | Default-Object-Category                 |
| Ldap-Display-Name | defaultObjectCategory                   |
| Size              | \-                                      |
| Update Privilege  | Schema administrator                    |
