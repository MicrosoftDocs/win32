# User-Shared-Folder attribute

Specifies a UNC path to the user's shared documents folder. The path must be a network UNC path of the form **\\\\***Server***\\***Share***\\***Directory*. This value can be a null string.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | User-Shared-Folder                                                                |
| Ldap-Display-Name | userSharedFolder                                                                  |
| Size              | \-                                                                                |
| Update Privilege  | Domain administrator or account owner.                                            |
| Update Frequency  | When the user's record is created and whenever the shared folder needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.751                                                            |
| System-Id-Guid    | 9a9a021f-4a5b-11d1-a9c3-0000f80367c1                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                       |
