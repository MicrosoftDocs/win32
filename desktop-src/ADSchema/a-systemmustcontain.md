# System-Must-Contain attribute

The list of mandatory attributes for a class. These attributes must be specified when an instance of the class is created. The list of attributes can only be modified by the system.



|                   |                                                                               |
|-------------------|-------------------------------------------------------------------------------|
| CN                | System-Must-Contain                                                           |
| Ldap-Display-Name | systemMustContain                                                             |
| Size              | \-                                                                            |
| Update Privilege  | Schema administrator                                                          |
| Update Frequency  | When the class is created or a new mandatory attribute is added to the class. |
| Attribute-Id      | 1.2.840.113556.1.4.197                                                        |
| System-Id-Guid    | bf967a45-0de6-11d0-a285-00aa003049e2                                          |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)               |
