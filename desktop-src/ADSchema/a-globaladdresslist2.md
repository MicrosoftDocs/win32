# Global-Address-List2 attribute

This attribute is used on a Microsoft Exchange container to store the distinguished name of a newly created global address list (GAL)



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Global-Address-List2                    |
| Ldap-Display-Name | globalAddressList2                      |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2047                 |
| System-Id-Guid    | 4898f63d-4112-477c-8826-3ca00bd8277d    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
