# Phone-Pager-Other attribute

A list of alternate pager numbers.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Phone-Pager-Other                                                                |
| Ldap-Display-Name | otherPager                                                                       |
| Size              | \-                                                                               |
| Update Privilege  | Domain administrator or account owner.                                           |
| Update Frequency  | When the user's record is created and whenever the phone number needs to change. |
| Attribute-Id      | 1.2.840.113556.1.2.118                                                           |
| System-Id-Guid    | f0f8ffa4-1191-11d0-a060-00aa006c33ed                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
