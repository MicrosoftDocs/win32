# Inter-Site-Topology-Generator attribute

This attribute will be used to support failover for the computer designated as the one that runs Knowledge Consistency Checker inter-site topology generation in a given site.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Inter-Site-Topology-Generator           |
| Ldap-Display-Name | interSiteTopologyGenerator              |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1246                 |
| System-Id-Guid    | b7c69e5e-2cc7-11d2-854e-00a0c983f608    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
