# Site-Server attribute

Licensing main server for a given Site.



| Entry | Value |
|-------------------|----------------------------------------------|
| CN                | Site-Server                                  |
| Ldap-Display-Name | siteServer                                   |
| Size              | \-                                           |
| Update Privilege  | Domain administrator                         |
| Update Frequency  | Whenever the licensing site needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.494                       |
| System-Id-Guid    | 1be8f17c-a9ff-11d0-afe2-00c04fd930c9         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
