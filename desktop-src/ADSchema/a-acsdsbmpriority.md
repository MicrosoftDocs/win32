# ACS-DSBM-Priority attribute

This attributes contains the priority for this Subnet Bandwidth Manager (SBM). When a new Designated Subnet Bandwidth Manager (DSBM) needs to be elected, this value is sent to other SBMs in the domain as part of a DSBM\_willing message. The SBM with the highest priority is elected as the new DSBM.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-Priority                    |
| Ldap-Display-Name | aCSDSBMPriority                      |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.776               |
| System-Id-Guid    | 1cb3559e-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
