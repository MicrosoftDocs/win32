# Transport-Type attribute

The distinguished name for a type of transport being used to connect sites together. This value can point to an IP or SMTP transport.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Transport-Type                          |
| Ldap-Display-Name | transportType                           |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | When connecting sites.                  |
| Attribute-Id      | 1.2.840.113556.1.4.791                  |
| System-Id-Guid    | 26d97374-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
