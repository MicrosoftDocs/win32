# Print-Duplex-Supported attribute

Indicates the type of duplex support a printer has.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Print-Duplex-Supported                                |
| Ldap-Display-Name | printDuplexSupported                                  |
| Size              | 4 bytes. Duplex supported = 2. Simplex only = 1 or 0. |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1311                               |
| System-Id-Guid    | 281416cc-1968-11d0-a28f-00aa003049e2                  |
| Syntax            | [**Boolean**](s-boolean.md)                          |
