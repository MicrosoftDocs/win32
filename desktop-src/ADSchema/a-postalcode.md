# Postal-Code attribute

The postal or zip code for mail delivery.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------|
| CN                | Postal-Code                                                                 |
| Ldap-Display-Name | postalCode                                                                  |
| Size              | \-                                                                          |
| Update Privilege  | Domain administrator or account owner.                                      |
| Update Frequency  | When the user's record is created and whenever the address needs to change. |
| Attribute-Id      | 2.5.4.17                                                                    |
| System-Id-Guid    | bf9679fd-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
