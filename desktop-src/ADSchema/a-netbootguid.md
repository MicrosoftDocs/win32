# Netboot-GUID attribute

Diskless boot: A computer's on-board GUID. Corresponds to the computer's network card MAC address.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Netboot-GUID                                          |
| Ldap-Display-Name | netbootGUID                                           |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.359                                |
| System-Id-Guid    | 3e978921-8c01-11d0-afda-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
