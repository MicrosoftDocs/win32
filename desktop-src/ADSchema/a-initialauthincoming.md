# Initial-Auth-Incoming attribute

Contains information about an initial incoming authentication request by a client to this server. This request is then sent by this server to the authentication server for the domain.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Initial-Auth-Incoming                       |
| Ldap-Display-Name | initialAuthIncoming                         |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.539                      |
| System-Id-Guid    | 52458023-ca6a-11d0-afff-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
