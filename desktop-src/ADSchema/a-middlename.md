# Other-Name attribute

Additional names for a user. For example, middle name, patronymic, matronymic, or others.

|-------------------|---------------------------------------------|
| CN                | Other-Name                                  |
| Ldap-Display-Name | middleName                                  |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
