# ms-PKI-Minimal-Key-Size attribute

Indicates the minimum private key size.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Minimal-Key-Size                                                                           |
| Ldap-Display-Name | msPKI-Minimal-Key-Size                                                                            |
| Size              | 4 bytes                                                                                           |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1433                                                                           |
| System-Id-Guid    | e96a63f5-417f-46d3-be52-db7703c503df                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
