# Invocation-Id attribute

Used to uniquely identify each Microsoft Exchange Server directory in the organization.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Invocation-Id                                         |
| Ldap-Display-Name | invocationId                                          |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | When the Exchange Server is installed.                |
| Attribute-Id      | 1.2.840.113556.1.2.115                                |
| System-Id-Guid    | bf96798e-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
