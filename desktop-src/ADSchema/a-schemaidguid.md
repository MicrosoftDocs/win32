# Schema-ID-GUID attribute

The unique identifier for an attribute.



| Entry | Value |
|-------------------|---------------------------------------------------------------------|
| CN                | Schema-ID-GUID                                                      |
| Ldap-Display-Name | schemaIDGUID                                                        |
| Size              | 16 bytes                                                            |
| Update Privilege  | This value is set by the system.                                    |
| Update Frequency  | This value is set when the object is created and cannot be changed. |
| Attribute-Id      | 1.2.840.113556.1.4.148                                              |
| System-Id-Guid    | bf967923-0de6-11d0-a285-00aa003049e2                                |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)               |
