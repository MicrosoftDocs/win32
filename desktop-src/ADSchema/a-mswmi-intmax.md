# ms-WMI-intMax attribute

The maximum value for a WMI 32-bit integer parameter object.

| CN                | ms-WMI-intMax                        |
| Ldap-Display-Name | msWMI-IntMax                         |
| Size              | 4 bytes                              |
| Update Privilege  | Group Policy Administrator           |
