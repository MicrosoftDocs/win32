# organizationalStatus attribute

The organizationalStatus attribute type specifies a category by which a person is often referred to in an organization.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | organizationalStatus                        |
| Ldap-Display-Name | organizationalStatus                        |
| Size              | \-                                          |
| Update Privilege  | \-                                          |

| Attribute-Id      | 0.9.2342.19200300.100.1.45                  |
