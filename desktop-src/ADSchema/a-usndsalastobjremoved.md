# USN-DSA-Last-Obj-Removed attribute

Contains the update sequence number (USN) for the last system object that was removed from a server.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | USN-DSA-Last-Obj-Removed             |
| Ldap-Display-Name | uSNDSALastObjRemoved                 |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | Whenever a directory object changes. |
| Attribute-Id      | 1.2.840.113556.1.2.267               |
| System-Id-Guid    | bf967a71-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
