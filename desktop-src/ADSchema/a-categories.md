# Categories attribute

List of category IDs (GUIDs) for categories that apply to this application.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Categories                                  |
| Ldap-Display-Name | categories                                  |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.672                      |
| System-Id-Guid    | 7bfdcb7e-4807-11d1-a9c3-0000f80367c1        |
