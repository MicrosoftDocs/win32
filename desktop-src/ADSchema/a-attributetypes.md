# Attribute-Types attribute

A multi-valued property that contains strings that represent each attribute in the schema.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Attribute-Types                             |
| Ldap-Display-Name | attributeTypes                              |
| Size              | \-                                          |
| Update Privilege  | Schema administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 2.5.21.5                                    |
| System-Id-Guid    | 9a7ad944-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
