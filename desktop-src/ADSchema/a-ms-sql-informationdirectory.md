# MS-SQL-InformationDirectory attribute

The informational directory for the current instance of SQL Server.



| Entry | Value |
|-------------------|------------------------------------------|
| CN                | MS-SQL-InformationDirectory              |
| Ldap-Display-Name | mS-SQL-InformationDirectory              |
| Size              | \-                                       |
| Update Privilege  | Domain administrator                     |
| Update Frequency  | At system startup or when AD is updated. |
| Attribute-Id      | 1.2.840.113556.1.4.1392                  |
| System-Id-Guid    | d0aedb2e-ccee-11d2-9993-0000f87a57d4     |
| Syntax            | [**Boolean**](s-boolean.md)             |
