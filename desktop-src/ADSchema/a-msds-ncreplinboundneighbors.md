# ms-DS-NC-Repl-Inbound-Neighbors attribute

Replication partners for this partition. This server obtains replication data from these other servers, which act as sources.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-Repl-Inbound-Neighbors             |
| Ldap-Display-Name | msDS-NCReplInboundNeighbors                 |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1705                     |
| System-Id-Guid    | 9edba85a-3e9e-431b-9b1a-a5b6e9eda796        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
