# photo attribute

An object encoded in G3 fax as explained in recommendation T.4, with an ASN.1 wrapper to make it compatible with an X.400 BodyPart as defined in X.420.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | photo                                                 |
| Ldap-Display-Name | photo                                                 |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| System-Id-Guid    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
