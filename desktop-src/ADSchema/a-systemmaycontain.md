# System-May-Contain attribute

The list of optional attributes for a class. The list of attributes can only be modified by the system.



| Entry | Value |
|-------------------|------------------------------------------------------------------------------|
| CN                | System-May-Contain                                                           |
| Ldap-Display-Name | systemMayContain                                                             |
| Size              | \-                                                                           |
| Update Privilege  | Schema administrator                                                         |
| Update Frequency  | When the class is created or a new optional attribute is added to the class. |
| Attribute-Id      | 1.2.840.113556.1.4.196                                                       |
| System-Id-Guid    | bf967a44-0de6-11d0-a285-00aa003049e2                                         |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)              |
