# Last-Set-Time attribute

The last time the secret was changed. This value is stored as a large integer that represents the number of 100-nanosecond intervals since January 1, 1601 (UTC).



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Last-Set-Time                        |
| Ldap-Display-Name | lastSetTime                          |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.53                |
| System-Id-Guid    | bf967998-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
