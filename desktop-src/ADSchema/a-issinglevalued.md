# Is-Single-Valued attribute

If **TRUE**, this attribute can only store one value.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Is-Single-Valued                     |
| Ldap-Display-Name | isSingleValued                       |
| Size              | \-                                   |
| Update Privilege  | Schema administrator                 |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.33                |
| System-Id-Guid    | bf967992-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
