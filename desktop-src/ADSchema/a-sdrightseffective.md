# SD-Rights-Effective attribute

This constructed attribute returns a single **DWORD** value that can have up to three bits set:

-   OWNER\_SECURITY\_INFORMATION
-   DACL\_SECURITY\_INFORMATION
-   SACL\_SECURITY\_INFORMATION

If a bit is set, then the user has write access to the corresponding part of the security descriptor. Owner means both owner and group.
