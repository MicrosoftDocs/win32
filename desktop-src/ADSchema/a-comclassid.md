# COM-ClassID attribute

This attribute stores the list of ClassIDs that are implemented in this application package.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | COM-ClassID                                 |
| Ldap-Display-Name | cOMClassID                                  |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.19                       |
| System-Id-Guid    | bf96793b-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
