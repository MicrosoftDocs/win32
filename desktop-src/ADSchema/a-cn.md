# Common-Name attribute

The name that represents an object. Used to perform searches.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| Ldap-Display-Name | cn                                          |
| Size              | Average size is 64 bytes.                   |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | Only when a new object is created.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-Id-Guid    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
