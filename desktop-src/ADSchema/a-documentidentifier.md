# documentIdentifier attribute

The documentIdentifier attribute type specifies a unique identifier for a document.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | documentIdentifier                          |
| Ldap-Display-Name | documentIdentifier                          |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.11                  |
| System-Id-Guid    | 0b21ce82-ff63-46d9-90fb-c8b9f24e97b9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
