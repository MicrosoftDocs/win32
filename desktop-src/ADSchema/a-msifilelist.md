# Msi-File-List attribute

This attribute contains a list of Microsoft installer files, such as the base MSI file (.msi) and MST transform files (.mst).



|                   |                                             |
|-------------------|---------------------------------------------|
| CN                | Msi-File-List                               |
| Ldap-Display-Name | msiFileList                                 |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.671                      |
| System-Id-Guid    | 7bfdcb7d-4807-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
