# ms-WMI-int8Default attribute

Default value for WMI 64-bit integer parameter objects.

|-------------------|--------------------------------------|
| CN                | ms-WMI-int8Default                   |
| Ldap-Display-Name | msWMI-Int8Default                    |
| Size              | 8 bytes                              |
| Update Privilege  | Group Policy Administrator           |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1632              |
| System-Id-Guid    | f4d8085a-8c5b-4785-959b-dc585566e445 |
