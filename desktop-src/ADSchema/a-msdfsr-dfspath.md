# ms-DFSR-DfsPath attribute

Contains the full path of the associated Distributed File System (DFS) link for DFS Replication service support.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DFSR-DfsPath                             |
| Ldap-Display-Name | msDFSR-DfsPath                              |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.21                  |
| System-Id-Guid    | 2cc903e2-398c-443b-ac86-ff6b01eac7ba        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
