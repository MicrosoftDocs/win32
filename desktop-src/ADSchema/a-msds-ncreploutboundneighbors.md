# ms-DS-NC-Repl-Outbound-Neighbors attribute

Replication partners for this partition. This server sends replication data to these other servers, which act as destinations. This server will notify these other servers when new data is available.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-Repl-Outbound-Neighbors            |
| Ldap-Display-Name | msDS-NCReplOutboundNeighbors                |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1706                     |
| System-Id-Guid    | 855f2ef5-a1c5-4cc4-ba6d-32522848b61f        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
