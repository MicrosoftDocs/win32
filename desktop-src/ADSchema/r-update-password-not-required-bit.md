# Update-Password-Not-Required-Bit extended right

Extended control access right that allows a user to enable or disable the "password not required" setting for user objects.



|              |                                      |
|--------------|--------------------------------------|
| CN           | Update-Password-Not-Required-Bit     |
| Display-Name | Update Password Not Required Bit     |
| Rights-GUID  | 280f369c-67c7-438e-ae98-1d46f3c6f541 |
