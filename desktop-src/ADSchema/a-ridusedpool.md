# RID-Used-Pool attribute

The RID Pools that have been used by a DC.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | RID-Used-Pool                        |
| Ldap-Display-Name | rIDUsedPool                          |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | Whenever a RID pool is emptied.      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| System-Id-Guid    | 6617188b-8f3c-11d0-afda-00c04fd930c9 |
| Syntax            | [**Interval**](s-interval.md)       |
