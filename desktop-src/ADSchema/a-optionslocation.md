# Options-Location attribute

For DHCP, the options location contains the DN for alternate sites that contain the options information.



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | Options-Location                     |
| Ldap-Display-Name | optionsLocation                      |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.713               |
| System-Id-Guid    | 963d274e-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
