# msSFU-30-Domains attribute

Contains the list of UNIX NIS domains that have all been migrated to the same AD NIS domain.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | msSFU-30-Domains                     |
| Ldap-Display-Name | msSFU30Domains                       |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.340          |
| System-Id-Guid    | 93095ed3-6f30-4bdd-b734-65d569f5f7c9 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
