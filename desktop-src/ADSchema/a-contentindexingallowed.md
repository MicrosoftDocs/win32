# Content-Indexing-Allowed attribute

Indicates whether the Volume object can be content indexed.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Content-Indexing-Allowed             |
| Ldap-Display-Name | contentIndexingAllowed               |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.24                |
| System-Id-Guid    | bf967943-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
