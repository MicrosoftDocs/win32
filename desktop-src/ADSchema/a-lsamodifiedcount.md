# LSA-Modified-Count attribute

The **LSA-Modified-Count** attribute is used to support replication to Windows NT 4.0 domains.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | LSA-Modified-Count                   |
| Ldap-Display-Name | lSAModifiedCount                     |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.67                |
| System-Id-Guid    | bf9679ae-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
