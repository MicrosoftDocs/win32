# Domain-Policy-Reference attribute

The Distinguished Name of a domain policy object that a policy object copies from.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Domain-Policy-Reference                 |
| Ldap-Display-Name | domainPolicyReference                   |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.422                  |
| System-Id-Guid    | 80a67e2a-9f22-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
