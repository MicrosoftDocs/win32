# Dns-Record attribute

Used to store binary DNS resource records on DNS objects.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Dns-Record                                            |
| Ldap-Display-Name | dnsRecord                                             |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | Whenever DNS resource records change.                 |
| Attribute-Id      | 1.2.840.113556.1.4.382                                |
| System-Id-Guid    | e0fa1e69-9b45-11d0-afdd-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
