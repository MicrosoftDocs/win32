# ms-TAPI-Unique-Identifier attribute

Provides the name of a TAPI multicast conference. It is the naming attribute of the Rt-Conference class.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------|
| CN                | ms-TAPI-Unique-Identifier                                             |
| Ldap-Display-Name | msTAPI-uid                                                            |
| Size              | Can be an arbitrary string, typically under 100 characters in length. |
| Update Privilege  | No special privileges required.                                       |
| Update Frequency  | Set once at the time of creating the Rt-Conference object.            |
| Attribute-Id      | 1.2.840.113556.1.4.1698                                               |
| System-Id-Guid    | 70a4e7ea-b3b9-4643-8918-e6dd2471bfd4                                  |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                           |
