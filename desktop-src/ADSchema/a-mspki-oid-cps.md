# ms-PKI-OID-CPS attribute

The CPS (Certificate Policy Statement) for the enterprise issuer policy OID.



| Entry | Value |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | ms-PKI-OID-CPS                                                                             |
| Ldap-Display-Name | msPKI-OID-CPS                                                                              |
| Size              | \-                                                                                         |
| Update Privilege  | Enterprise administrator                                                                   |
| Update Frequency  | Whenever a new template is created, or the attributes of an existing templates are edited. |
