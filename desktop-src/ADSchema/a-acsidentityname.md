# ACS-Identity-Name attribute

This attribute contains the DN of a user or OU and is the identity of a user or a group of users to which this QoS policy applies.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ACS-Identity-Name                           |
| Ldap-Display-Name | aCSIdentityName                             |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.784                      |
| System-Id-Guid    | dab029b6-ddf7-11d1-90a5-00c04fd91ab1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
