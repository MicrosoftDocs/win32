# Nt-Pwd-History attribute

The password history of the user in Windows NT one-way format (OWF). Windows 2000 uses the Windows NT OWF.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Nt-Pwd-History                                        |
| Ldap-Display-Name | ntPwdHistory                                          |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | Each time the user changes passwords.                 |
| Attribute-Id      | 1.2.840.113556.1.4.94                                 |
| System-Id-Guid    | bf9679e2-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
