# bootFile attribute

Contains the name of the boot image for UNIX identity management support.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | bootFile                             |
| Ldap-Display-Name | bootFile                             |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.24                     |
| System-Id-Guid    | e3f3cb4e-0f20-42eb-9703-d2ff26e52667 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
