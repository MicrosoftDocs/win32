# Flat-Name attribute

For Windows NT domains, the flat name is the NetBIOS name. For links with non Windows NT domains, the flat name is the identifying name of that domain, or it is **NULL**.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Flat-Name                                   |
| Ldap-Display-Name | flatName                                    |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.511                      |
| System-Id-Guid    | b7b13117-b82e-11d0-afee-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
