# WWW-Home-Page attribute

A web page that is the primary landing page of a website.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-Home-Page                                                               |
| Ldap-Display-Name | wWWHomePage                                                                 |
| Size              | \-                                                                          |
| Update Privilege  | Domain administrator or account owner.                                      |
| Update Frequency  | When the user's record is created and whenever the webpage needs to change. |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| System-Id-Guid    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
