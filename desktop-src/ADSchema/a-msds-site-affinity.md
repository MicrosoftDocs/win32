# ms-DS-Site-Affinity attribute

The **ms-DS-Site-Affinity** attribute is used by the Security Accounts Manager for group expansion during token evaluation.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | ms-DS-Site-Affinity                                                                     |
| Ldap-Display-Name | msDS-Site-Affinity                                                                      |
| Size              | Multiple-valued binary BLOB, where each value is sizeof(GUID) + sizeof(LARGE\_INTEGER). |
| Update Privilege  | \-                                                                                      |
| Update Frequency  | By default once every three months.                                                     |
| Attribute-Id      | 1.2.840.113556.1.4.1443                                                                 |
| System-Id-Guid    | c17c5602-bcb7-46f0-9656-6370ca884b72                                                    |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                                   |
