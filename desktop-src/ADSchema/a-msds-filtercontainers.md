# ms-DS-Filter-Containers attribute

A multiple-valued string that contains the names of classes used to determine which container types should be shown by the Active Directory Users and Computers snap-in when filtering.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Filter-Containers                     |
| Ldap-Display-Name | msDS-FilterContainers                       |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | When the list of container types changes.   |
| Attribute-Id      | 1.2.840.113556.1.4.1703                     |
| System-Id-Guid    | fb00dcdf-ac37-483a-9c12-ac53a6603033        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
