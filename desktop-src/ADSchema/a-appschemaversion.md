# App-Schema-Version attribute

This attribute stores the schema version of the class store. It is used to provide correct behavior across schema changes.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | App-Schema-Version                   |
| Ldap-Display-Name | appSchemaVersion                     |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.848               |
| System-Id-Guid    | 96a7dd65-9118-11d1-aebc-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
