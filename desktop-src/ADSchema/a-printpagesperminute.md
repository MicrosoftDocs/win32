# Print-Pages-Per-Minute attribute

Driver-supplied print rate in pages per minute.

|-------------------|--------------------------------------|
| CN                | Print-Pages-Per-Minute               |
| Ldap-Display-Name | printPagesPerMinute                  |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
