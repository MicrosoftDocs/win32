# Package-Type attribute

This attribute describes the type of installation required for an application package, for example, MSI, EXE, CAB.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Package-Type                         |
| Ldap-Display-Name | packageType                          |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.324               |
