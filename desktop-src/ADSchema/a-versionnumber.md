# Version-Number attribute

A general purpose version number.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Version-Number                       |
| Ldap-Display-Name | versionNumber                        |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.141               |
| System-Id-Guid    | bf967a76-0de6-11d0-a285-00aa003049e2 |
