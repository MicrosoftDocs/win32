# ipServicePort attribute

Contains the section of the services map that specifies the port on which the UNIX service is available.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ipServicePort                        |
| Ldap-Display-Name | ipServicePort                        |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.15                     |
| System-Id-Guid    | ff2daebf-f463-495a-8405-3e483641eaa2 |
