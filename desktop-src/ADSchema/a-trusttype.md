# Trust-Type attribute

The type of trust, for example, Windows NT or MIT.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Trust-Type                           |
| Ldap-Display-Name | trustType                            |
| Size              | 4 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | When a new trust is created.         |
| Attribute-Id      | 1.2.840.113556.1.4.136               |
| System-Id-Guid    | bf967a60-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
