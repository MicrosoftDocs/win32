# Home-Drive attribute

Specifies the drive letter to which to map the UNC path specified by [**homeDirectory**](a-homedirectory.md). The drive letter must be specified in the form *DriveLetter***:** where *DriveLetter* is the letter of the drive to map. The *DriveLetter* must be a single, uppercase letter and the colon (:) is required.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Home-Drive                                                                        |
| Ldap-Display-Name | homeDrive                                                                         |
| Size              | 2 bytes                                                                           |
| Update Privilege  | The domain administrator sets this value.                                         |
| Update Frequency  | When the record of a user is created and whenever the home drive needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.45                                                             |
| System-Id-Guid    | bf967986-0de6-11d0-a285-00aa003049e2                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                       |
