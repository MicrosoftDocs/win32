# Phone-Ip-Primary attribute

The TCP/IP address for the phone. Used by Telephony.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Phone-Ip-Primary                                                                 |
| Ldap-Display-Name | ipPhone                                                                          |
| Size              | \-                                                                               |
| Update Privilege  | Domain administrator or account owner.                                           |
| Update Frequency  | When the user's record is created and whenever the phone number needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.721                                                           |
| System-Id-Guid    | 4d146e4a-48d4-11d1-a9c3-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
