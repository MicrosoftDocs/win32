# NT-Mixed-Domain attribute

Indicates that the domain is in native mode or mixed mode. This attribute is found in the domainDNS (head) object for the domain.



| Entry | Value |
|-------------------|---------------------------------------|
| CN                | NT-Mixed-Domain                       |
| Ldap-Display-Name | nTMixedDomain                         |
| Size              | 4 bytes. Mixed mode 1, native mode 0. |
| Update Privilege  | \-                                    |
| Update Frequency  | \-                                    |
| Attribute-Id      | 1.2.840.113556.1.4.357                |
| System-Id-Guid    | 3e97891f-8c01-11d0-afda-00c04fd930c9  |
| Syntax            | [**Enumeration**](s-enumeration.md)  |
