# Server-Role attribute

For compatibility with pre-Windows 2000 Server servers. A computer running Windows NT Server can be a standalone server, a primary domain controller (PDC), or a backup domain controller (BDC).



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | Server-Role                          |
| Ldap-Display-Name | serverRole                           |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | 4 bytes                              |
| Attribute-Id      | 1.2.840.113556.1.4.157               |
| System-Id-Guid    | bf967a33-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
