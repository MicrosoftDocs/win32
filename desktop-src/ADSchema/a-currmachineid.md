# Curr-Machine-Id attribute

The ID of the machine where a Link-Track-Vol-Entry object is located.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Curr-Machine-Id                                       |
| Ldap-Display-Name | currMachineId                                         |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.337                                |
| System-Id-Guid    | 1f0075fe-7e40-11d0-afd6-00c04fd930c9                  |
