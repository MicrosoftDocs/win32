# COM-Typelib-Id attribute

This attribute stores the list of type library IDs contained in this application package.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-Typelib-Id                                                                   |
| Ldap-Display-Name | cOMTypelibId                                                                     |
| Size              | \-                                                                               |
| Update Privilege  | Anyone may update this object based on the security of the object being created. |
| Update Frequency  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.254                                                           |
| System-Id-Guid    | 281416de-1968-11d0-a28f-00aa003049e2                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
