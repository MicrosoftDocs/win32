# ms-DS-Entry-Time-To-Die attribute

Holds the absolute expiration time of a dynamic object in the directory.



| Entry | Value |
|-------------------|---------------------------------------------------------------|
| CN                | ms-DS-Entry-Time-To-Die                                       |
| Ldap-Display-Name | msDS-Entry-Time-To-Die                                        |
| Size              | \-                                                            |
| Update Privilege  | This value is set by the system.                              |
| Update Frequency  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-Id-Guid    | e1e9bad7-c6dd-4101-a843-794cec85b038                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
