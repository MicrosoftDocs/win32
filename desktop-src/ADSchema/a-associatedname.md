# associatedName attribute

The associatedName attribute type specifies an entry in the organizational DIT that is associated with a DNS domain.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | associatedName                          |
| Ldap-Display-Name | associatedName                          |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.38              |
| System-Id-Guid    | f7fbfc45-85ab-42a4-a435-780e62f7858b    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
