# Super-Scopes attribute

This attribute is used to group together all the different scopes used in the DHCP class into a single entity.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Super-Scopes                         |
| Ldap-Display-Name | superScopes                          |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.710               |
| System-Id-Guid    | 963d274b-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
