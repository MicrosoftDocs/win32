# FRS-Root-Security attribute

The security descriptor of replica set root for file replication.

|-------------------|-----------------------------------------------------|
| CN                | FRS-Root-Security                                   |
| Ldap-Display-Name | fRSRootSecurity                                     |
| Size              | \-                                                  |
| Update Privilege  | \-                                                  |
| Update Frequency  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| System-Id-Guid    | 5245801f-ca6a-11d0-afff-0000f80367c1                |
| Syntax            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
