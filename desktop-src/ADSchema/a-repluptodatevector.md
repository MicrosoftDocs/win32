# Repl-UpToDate-Vector attribute

Tracks internal replication state information for an entire NC. Information here can be extracted in public form through the API DsReplicaGetInfo(). Present on all NC root objects.



| Entry | Value |
|-------------------|--------------------------------------------------------------------------------|
| CN                | Repl-UpToDate-Vector                                                           |
| Ldap-Display-Name | replUpToDateVector                                                             |
| Size              | Length is proportional to the number of replicas (past and present) of the NC. |
| Update Privilege  | This value is set by the system.                                               |
| Update Frequency  | Changes in response to completed inbound replication cycles.                   |
| Attribute-Id      | 1.2.840.113556.1.4.4                                                           |
| System-Id-Guid    | bf967a16-0de6-11d0-a285-00aa003049e2                                           |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                          |
