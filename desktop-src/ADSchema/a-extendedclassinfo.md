# Extended-Class-Info attribute

A multi-valued property that contains strings that represent additional information for each class. Each value contains the governsID, lDAPDisplayName, and schemaIDGUID of the class.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Extended-Class-Info                         |
| Ldap-Display-Name | extendedClassInfo                           |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.908                      |
| System-Id-Guid    | 9a7ad948-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
