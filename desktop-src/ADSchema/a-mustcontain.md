# Must-Contain attribute

The list of mandatory attributes for a class. These attributes must be specified when an instance of the class is created.



| Entry | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | Must-Contain                                                    |
| Ldap-Display-Name | mustContain                                                     |
| Size              | \-                                                              |
| Update Privilege  | Schema administrator                                            |
| Update Frequency  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.24                                           |
| System-Id-Guid    | bf9679d3-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
