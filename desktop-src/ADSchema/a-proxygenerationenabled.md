# Proxy-Generation-Enabled attribute

**TRUE** if proxy generation is enabled.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Proxy-Generation-Enabled             |
| Ldap-Display-Name | proxyGenerationEnabled               |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.523               |
| System-Id-Guid    | 5fd424d6-1262-11d0-a060-00aa006c33ed |
| Syntax            | [**Boolean**](s-boolean.md)         |
