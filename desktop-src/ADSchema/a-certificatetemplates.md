# Certificate-Templates attribute

Contains information for a certificate issued by a Certificate Server.

|-------------------|---------------------------------------------|
| CN                | Certificate-Templates                       |
| Ldap-Display-Name | certificateTemplates                        |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.823                      |
| System-Id-Guid    | 2a39c5b1-8960-11d1-aebc-0000f80367c1        |
