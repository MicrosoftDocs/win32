# ms-DS-Preferred-GC-Site attribute

The **ms-DS-Preferred-GC-Site** attribute is used by the Security Accounts Manager for group expansion during token evaluation.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Preferred-GC-Site                 |
| Ldap-Display-Name | msDS-Preferred-GC-Site                  |
| Size              | \-                                      |
| Update Privilege  | Domain administrator                    |
| Update Frequency  | At the administrator's discretion.      |
| Attribute-Id      | 1.2.840.113556.1.4.1444                 |
| System-Id-Guid    | d921b50a-0ab2-42cd-87f6-09cf83a91854    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
