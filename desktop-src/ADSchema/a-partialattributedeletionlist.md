# Partial-Attribute-Deletion-List attribute

Tracks the internal replication state of partial replicas (that is, on GCs). Attribute of the partial replica NC object. Used when the GC is in the process of removing attributes from the objects in its partial replica NCs.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Partial-Attribute-Deletion-List                       |
| Ldap-Display-Name | partialAttributeDeletionList                          |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | During replication                                    |
| Attribute-Id      | 1.2.840.113556.1.4.663                                |
| System-Id-Guid    | 28630ec0-41d5-11d1-a9c1-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
