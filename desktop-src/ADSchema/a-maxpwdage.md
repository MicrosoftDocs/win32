# Max-Pwd-Age attribute

The maximum amount of time, in 100-nanosecond intervals, a password is valid. This value is stored as a large integer that represents the number of 100-nanosecond intervals from the time the password was set before the password expires.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Max-Pwd-Age                          |
| Ldap-Display-Name | maxPwdAge                            |
| Size              | \-                                   |
| Update Privilege  | Domain administrator                 |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.74                |
| System-Id-Guid    | bf9679bb-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
