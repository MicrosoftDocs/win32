# ms-DS-Has-Master-NCs attribute

Contains the naming contexts held by a domain controller. This attribute deprecates has-Master-NCs.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Has-Master-NCs                    |
| Ldap-Display-Name | msDS-hasMasterNCs                       |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system         |
| Update Frequency  | Each time a new NC is created.          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| System-Id-Guid    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
