# Phone-Mobile-Primary attribute

The primary mobile phone number.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Phone-Mobile-Primary                                                             |
| Ldap-Display-Name | mobile                                                                           |
| Size              | \-                                                                               |
| Update Privilege  | Domain administrator or account owner.                                           |
| Update Frequency  | When the user's record is created and whenever the phone number needs to change. |
| Attribute-Id      | 0.9.2342.19200300.100.1.41                                                       |
| System-Id-Guid    | f0f8ffa3-1191-11d0-a060-00aa006c33ed                                             |
