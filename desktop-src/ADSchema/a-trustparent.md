# Trust-Parent attribute

The parent in KERBEROS trust hierarchy.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Trust-Parent                            |
| Ldap-Display-Name | trustParent                             |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | When a child domain is created.         |
| Attribute-Id      | 1.2.840.113556.1.4.471                  |
| System-Id-Guid    | b000ea7a-a086-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
