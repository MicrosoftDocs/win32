# Lock-Out-Observation-Window attribute

The range of time, in 100-nanosecond intervals, in which the system increments the incorrect logon count.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Lock-Out-Observation-Window          |
| Ldap-Display-Name | lockOutObservationWindow             |
| Size              | \-                                   |
| Update Privilege  | Domain administrator                 |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.61                |
| System-Id-Guid    | bf9679a4-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
