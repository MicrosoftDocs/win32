# Knowledge-Information attribute

Specifies a human-readable accumulated description of knowledge mastered by a specific Directory System Agent.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Knowledge-Information                       |
| Ldap-Display-Name | knowledgeInformation                        |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 2.5.4.2                                     |
| System-Id-Guid    | 1677581f-47f3-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
