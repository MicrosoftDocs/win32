# Inter-Site-Topology-Renew attribute

This class indicates how often the intersite topology generator updates the keep-alive message that is sent to domain controllers contained in the same site.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Inter-Site-Topology-Renew            |
| Ldap-Display-Name | interSiteTopologyRenew               |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1247              |
| System-Id-Guid    | b7c69e5f-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
