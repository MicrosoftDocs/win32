# ms-DS-Az-Generate-Audits attribute

A Boolean field that indicates whether runtime audits need to be turned on (include audits for access checks, and so on).



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Az-Generate-Audits                |
| Ldap-Display-Name | msDS-AzGenerateAudits                   |
| Size              | \-                                      |
| Update Privilege  | AzRoles admin                           |
| Update Frequency  | During initialization or policy change. |
| Attribute-Id      | 1.2.840.113556.1.4.1805                 |
| System-Id-Guid    | f90abab0-186c-4418-bb85-88447c87222a    |
| Syntax            | [**Boolean**](s-boolean.md)            |
