# Phone-Office-Other attribute

A list of alternate office phone numbers.



|                   |                                                                                 |
|-------------------|---------------------------------------------------------------------------------|
| CN                | Phone-Office-Other                                                              |
| Ldap-Display-Name | otherTelephone                                                                  |
| Size              | \-                                                                              |
| Update Privilege  | Domain administrator or account owner.                                          |
| Update Frequency  | When the users record is created and whenever the phone number needs to change. |
| Attribute-Id      | 1.2.840.113556.1.2.18                                                           |
| System-Id-Guid    | f0f8ffa5-1191-11d0-a060-00aa006c33ed                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                     |
