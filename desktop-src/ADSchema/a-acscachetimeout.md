# ACS-Cache-Timeout attribute

The amount of time before the expiration of ACS objects that are cached by the ACS service.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ACS-Cache-Timeout                    |
| Ldap-Display-Name | aCSCacheTimeout                      |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.779               |
| System-Id-Guid    | 1cb355a1-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
