# MS-SQL-LastUpdatedDate attribute

This value represents the time that SQL last updated Active Directory.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastUpdatedDate                      |
| Ldap-Display-Name | mS-SQL-LastUpdatedDate                      |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | At system startup or when AD is updated.    |
| Attribute-Id      | 1.2.840.113556.1.4.1381                     |
| System-Id-Guid    | 9fcc43d4-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
