# ms-DS-Default-Quota attribute

The default quota that will apply to a security principal that creates an object in the NC if no quota specification exists that covers the security principal.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ms-DS-Default-Quota                  |
| Ldap-Display-Name | msDS-DefaultQuota                    |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-Id-Guid    | 6818f726-674b-441b-8a3a-f40596374cea |
| Syntax            | [**Enumeration**](s-enumeration.md) |
