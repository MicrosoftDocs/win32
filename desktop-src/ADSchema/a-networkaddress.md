# Network-Address attribute

The TCP/IP address for a network segment. Also called the subnet address.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Network-Address                             |
| Ldap-Display-Name | networkAddress                              |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.459                      |
| System-Id-Guid    | bf9679d9-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
