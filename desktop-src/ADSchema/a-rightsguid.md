# Rights-Guid attribute

The GUID used to represent an extended right within an access control entry.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Rights-Guid                                 |
| Ldap-Display-Name | rightsGuid                                  |
| Size              | 16 bytes                                    |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | When a new extended right is created.       |
| Attribute-Id      | 1.2.840.113556.1.4.340                      |
| System-Id-Guid    | 8297931c-86d3-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
