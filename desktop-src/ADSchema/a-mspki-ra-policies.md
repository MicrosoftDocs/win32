# ms-PKI-RA-Policies attribute

Contains the list of required policy OIDs from registration authorities who sign the enrollment request.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-Policies                                                                                |
| Ldap-Display-Name | msPKI-RA-Policies                                                                                 |
| Size              | 64 bytes times the number of entries.                                                             |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1438                                                                           |
| System-Id-Guid    | d546ae22-0951-4d47-817e-1c9f96faad46                                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                       |
