# Template-Roots attribute

This attribute is used on the Exchange config container to indicate where the template containers are stored. This information is used by the Active Directory MAPI provider.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Template-Roots                          |
| Ldap-Display-Name | templateRoots                           |
| Size              | \-                                      |
| Update Privilege  | This is used by the system.             |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1346                 |
| System-Id-Guid    | ed9de9a0-7041-11d2-9905-0000f87a57d4    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
