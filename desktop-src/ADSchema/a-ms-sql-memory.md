# MS-SQL-Memory attribute

The amount of memory on the computer.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Memory                        |
| Ldap-Display-Name | mS-SQL-Memory                        |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | At system startup.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1367              |
| System-Id-Guid    | 5b5d448c-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Interval**](s-interval.md)       |
