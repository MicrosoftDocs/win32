# DNS-Tombstoned attribute

True if this object has been tombstoned. This attribute exists to make searching for tombstoned records easier and faster. Tombstoned objects are objects that have been deleted but not yet removed from the directory.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | DNS-Tombstoned                       |
| Ldap-Display-Name | dNSTombstoned                        |
| Size              | 4 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | Whenever an object is deleted.       |
| Attribute-Id      | 1.2.840.113556.1.4.1414              |
| System-Id-Guid    | d5eb2eb7-be4e-463b-a214-634a44d7392e |
| Syntax            | [**Boolean**](s-boolean.md)         |
