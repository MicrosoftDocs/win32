# Pending-CA-Certificates attribute

The certificates that are about to become effective for this certification authority.

|-------------------|-------------------------------------------------------|
| CN                | Pending-CA-Certificates                               |
| Ldap-Display-Name | pendingCACertificates                                 |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.693                                |
