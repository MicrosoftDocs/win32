# ms-DS-Non-Members attribute

This attribute serves the same purpose as the Non-Security-Member attribute but with scoping rules applied.

| Ldap-Display-Name | msDS-NonMembers                         |
| Size              | \-                                      |
| Update Privilege  | AzRoles Admin                           |
