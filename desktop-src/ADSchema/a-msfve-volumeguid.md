# ms-FVE-VolumeGuid attribute

Contains the GUID associated with a BitLocker-supported disk volume.

| CN                | ms-FVE-VolumeGuid                                     |
| Ldap-Display-Name | msFVE-VolumeGuid                                      |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1998                               |
