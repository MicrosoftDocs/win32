# Site-Link-List attribute

List of site links that are associated with this bridge.



| Entry | Value |
|-------------------|--------------------------------------------------------------|
| CN                | Site-Link-List                                               |
| Ldap-Display-Name | siteLinkList                                                 |
| Size              | \-                                                           |
| Update Privilege  | Domain administrator                                         |
| Update Frequency  | Whenever a site link is added to or removed from the bridge. |
| Attribute-Id      | 1.2.840.113556.1.4.822                                       |
| System-Id-Guid    | d50c2cdd-8951-11d1-aebc-0000f80367c1                         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                      |
