# ms-DS-Az-Class-ID attribute

A class ID required by the AzRoles UI on the AzApplication object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Az-Class-ID                           |
| Ldap-Display-Name | msDS-AzClassId                              |
| Size              | 36 characters                               |
| Update Privilege  | AzRoles admin                               |
| Update Frequency  | During initialization or policy change.     |
| Attribute-Id      | 1.2.840.113556.1.4.1816                     |
| System-Id-Guid    | 013a7277-5c2d-49ef-a7de-b765b36a3f6f        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
