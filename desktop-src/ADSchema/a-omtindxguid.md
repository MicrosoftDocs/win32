# OMT-Indx-Guid attribute

The index identifier for a Link-Track-Object-Move table entry.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | OMT-Indx-Guid                                         |
| Ldap-Display-Name | oMTIndxGuid                                           |
| Size              | 16 bytes                                              |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | Whenever the link for a file changes.                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| System-Id-Guid    | 1f0075fa-7e40-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
