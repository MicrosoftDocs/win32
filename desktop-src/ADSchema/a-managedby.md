# Managed-By attribute

The distinguished name of the user that is assigned to manage this object.



| Entry | Value |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Managed-By                                                                          |
| Ldap-Display-Name | managedBy                                                                           |
| Size              | \-                                                                                  |
| Update Privilege  | Domain administrator or account owner.                                              |
| Update Frequency  | When the user's record is created and whenever the managed objects needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.653                                                              |
| System-Id-Guid    | 0296c120-40da-11d1-a9c0-0000f80367c1                                                |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                                             |
