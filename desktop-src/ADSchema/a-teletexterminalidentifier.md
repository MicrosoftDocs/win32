# Teletex-Terminal-Identifier attribute

Specifies the Teletex terminal identifier and, optionally, parameters, for a teletex terminal associated with an object.

| CN                | Teletex-Terminal-Identifier                           |
| Ldap-Display-Name | teletexTerminalIdentifier                             |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
