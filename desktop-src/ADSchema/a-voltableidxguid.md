# Vol-Table-Idx-GUID attribute

The index identifier for a Link-Track-Volume table entry.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Vol-Table-Idx-GUID                                    |
| Ldap-Display-Name | volTableIdxGUID                                       |
| Size              | 16 bytes                                              |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | Whenever a new link to a file is created.             |
| Attribute-Id      | 1.2.840.113556.1.4.334                                |
| System-Id-Guid    | 1f0075fb-7e40-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
