# Structural-Object-Class attribute

This constructed attribute stores a list of classes contained in a class hierarchy, including abstract classes. This list does contain dynamically linked auxiliary classes.



| Entry | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | Structural-Object-Class                                         |
| Ldap-Display-Name | structuralObjectClass                                           |
| Size              | \-                                                              |
| Update Privilege  | This value is set by the system.                                |
| Update Frequency  | At class creation.                                              |
| Attribute-Id      | 2.5.21.9                                                        |
| System-Id-Guid    | 3860949f-f6a8-4b38-9950-81ecb6bc2982                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
