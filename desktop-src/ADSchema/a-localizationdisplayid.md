# Localization-Display-Id attribute

Used to index into the file Extrts.mc to get the localized displayName for the objects for UI purposes.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Localization-Display-Id              |
| Ldap-Display-Name | localizationDisplayId                |
| Size              | 4 bytes                              |
| Update Privilege  | Schema administrator                 |
| Update Frequency  | An ID should never be changed.       |
| Attribute-Id      | 1.2.840.113556.1.4.1353              |
| System-Id-Guid    | a746f0d1-78d0-11d2-9916-0000f87a57d4 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
