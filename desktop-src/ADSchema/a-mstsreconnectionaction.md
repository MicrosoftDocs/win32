# ms-TS-Reconnection-Action attribute

Terminal Services session Reconnection Action specifies whether to allow reconnection to a disconnected Terminal Services session from any client computer.

| CN                | ms-TS-Reconnection-Action            |
| Ldap-Display-Name | msTSReconnectionAction               |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
