# Shell-Property-Pages attribute

The order number and GUID of property pages for managing Active Directory objects. These property pages can be accessed from the Windows shell. For more information see the document, Extending the User Interface for Directory Objects.



| Entry | Value |
|-------------------|------------------------------------------------|
| CN                | Shell-Property-Pages                           |
| Ldap-Display-Name | shellPropertyPages                             |
| Size              | \-                                             |
| Update Privilege  | Domain administrator or application developer. |
| Update Frequency  | Whenever a property page is added or removed.  |
| Attribute-Id      | 1.2.840.113556.1.4.563                         |
| System-Id-Guid    | 52458039-ca6a-11d0-afff-0000f80367c1           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
