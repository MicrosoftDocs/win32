# ms-DS-Az-LDAP-Query attribute

A string that defines the LDAP query (max length 4096) which determines the membership of a user object to the group.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Az-LDAP-Query                         |
| Ldap-Display-Name | msDS-AzLDAPQuery                            |
| Size              | 4096 characters                             |
| Update Privilege  | AzRoles admin                               |
| Update Frequency  | During initialization or policy change.     |
| Attribute-Id      | 1.2.840.113556.1.4.1792                     |
| System-Id-Guid    | 5e53368b-fc94-45c8-9d7d-daf31ee7112d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
