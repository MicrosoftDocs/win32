# Link-Track-Secret attribute

This attribute stores a link to a secret key that allows an encrypted file to be translated to plaintext.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Link-Track-Secret                                     |
| Ldap-Display-Name | linkTrackSecret                                       |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.269                                |
| System-Id-Guid    | 2ae80fe2-47b4-11d0-a1a4-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
