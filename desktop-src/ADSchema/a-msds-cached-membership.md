# ms-DS-Cached-Membership attribute

The **ms-DS-Cached-Membership** attribute is used by the Security Accounts Manager for group expansion during token evaluation.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-Cached-Membership                               |
| Ldap-Display-Name | msDS-Cached-Membership                                |
| Size              | Binary BLOB of up to 3 kilobytes.                     |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1441                               |
| System-Id-Guid    | 69cab008-cdd4-4bc9-bab8-0ff37efe1b20                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
