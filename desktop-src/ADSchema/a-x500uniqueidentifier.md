# x500uniqueIdentifier attribute

Used to distinguish between objects when a distinguished name has been reused. This is a different attribute type from both the uid and uniqueIdentifier types.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | x500uniqueIdentifier                                  |
| Ldap-Display-Name | x500uniqueIdentifier                                  |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 2.5.4.45                                              |
| System-Id-Guid    | d07da11f-8a3d-42b6-b0aa-76c962be719a                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
