# Physical-Delivery-Office-Name attribute

Contains the office location in the user's place of business.



| Entry | Value |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Physical-Delivery-Office-Name                                                       |
| Ldap-Display-Name | physicalDeliveryOfficeName                                                          |
| Size              | \-                                                                                  |
| Update Privilege  | Domain administrator or account owner.                                              |
| Update Frequency  | When the user's record is created and whenever the office location needs to change. |
| Attribute-Id      | 2.5.4.19                                                                            |
| System-Id-Guid    | bf9679f7-0de6-11d0-a285-00aa003049e2                                                |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                         |
