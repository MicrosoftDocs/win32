# Schema-Info attribute

An internal binary value used to detect schema changes between DCs and force a schema NC replication cycle before replicating any other NC. Used to resolve ties when the schema FSMO is seized and a change is made on more than one DC.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Schema-Info                                           |
| Ldap-Display-Name | schemaInfo                                            |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1358                               |
| System-Id-Guid    | f9fb64ae-93b4-11d2-9945-0000f87a57d4                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
