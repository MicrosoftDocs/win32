# ms-DS-Az-Script-Timeout attribute

The maximum time, in milliseconds, to wait for a script to finish auditing a specific policy.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Az-Script-Timeout                 |
| Ldap-Display-Name | msDS-AzScriptTimeout                    |
| Size              | \-                                      |
| Update Privilege  | AzRoles admin                           |
| Update Frequency  | During initialization or policy change. |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| System-Id-Guid    | 87d0fb41-2c8b-41f6-b972-11fdfd50d6b0    |
| Syntax            | [**Enumeration**](s-enumeration.md)    |
