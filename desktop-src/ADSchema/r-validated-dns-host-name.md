# Validated-DNS-Host-Name validated writes

Validated write permission to enable setting of a DNS host name attribute that is compliant with the computer name and domain name.



| Entry | Value |
|--------------|--------------------------------------|
| CN           | Validated-DNS-Host-Name              |
| Display-Name | Validated write to DNS host name     |
| Rights-GUID  | 72e39547-7b18-11d1-adef-00c04fd8d5cd |
