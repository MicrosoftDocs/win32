# ms-DS-Trust-Forest-Trust-Info attribute

Contains forest trust information (a binary BLOB) that is used by the system for a Trusted-Domain object.



| Entry | Value |
|-------------------|--------------------------------------------------------------------------------------------------------------------|
| CN                | ms-DS-Trust-Forest-Trust-Info                                                                                      |
| Ldap-Display-Name | msDS-TrustForestTrustInfo                                                                                          |
| Size              | \-                                                                                                                 |
| Update Privilege  | \-                                                                                                                 |
| Update Frequency  | Only when trust relationship between forests changes. This can happen if the topology of a trusted forest changes. |
| Attribute-Id      | 1.2.840.113556.1.4.1702                                                                                            |
| System-Id-Guid    | 29cc866e-49d3-4969-942e-1dbc0925d183                                                                               |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                                                              |
