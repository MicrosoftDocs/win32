# ms-WMI-CreationDate attribute

The date and time stamp of an object instance's creation.



| Entry | Value |
|-------------------|---------------------------------------------------|
| CN                | ms-WMI-CreationDate                               |
| Ldap-Display-Name | msWMI-CreationDate                                |
| Size              | WMI datetime data type representation - 26 chars. |
| Update Privilege  | This value is set by the system.                  |
| Update Frequency  | \-                                                |
| Attribute-Id      | 1.2.840.113556.1.4.1626                           |
| System-Id-Guid    | 748b0a2e-3351-4b3f-b171-2f17414ea779              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)       |
