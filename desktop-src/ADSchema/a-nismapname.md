# nisMapName attribute

Contains the name of the map to which the object belongs.

|-------------------|--------------------------------------|
| CN                | nisMapName                           |
| Ldap-Display-Name | nisMapName                           |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.26                     |
