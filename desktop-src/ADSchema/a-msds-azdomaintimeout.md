# ms-DS-Az-Domain-Timeout attribute

Time, in milliseconds, after a domain is detected to be unreachable and before the DC is tried again.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Az-Domain-Timeout                 |
| Ldap-Display-Name | msDS-AzDomainTimeout                    |
| Size              | \-                                      |
| Update Privilege  | AzRoles admin                           |
| Update Frequency  | During initialization or policy change. |
| Attribute-Id      | 1.2.840.113556.1.4.1795                 |
| System-Id-Guid    | 6448f56a-ca70-4e2e-b0af-d20e4ce653d0    |
| Syntax            | [**Enumeration**](s-enumeration.md)    |
