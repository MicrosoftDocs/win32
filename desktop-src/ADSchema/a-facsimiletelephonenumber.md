# Facsimile-Telephone-Number attribute

Contains telephone number of the user's business fax machine.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Facsimile-Telephone-Number                  |
| Ldap-Display-Name | facsimileTelephoneNumber                    |
| Size              | \-                                          |
| Update Privilege  | Domain administrator or account owner.      |
| Update Frequency  | Whenever the fax number needs to change.    |
| Attribute-Id      | 2.5.4.23                                    |
| System-Id-Guid    | bf967974-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
