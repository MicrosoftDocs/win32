# MS-SQL-AllowSnapshotFilesFTPDownloading attribute

True if the publication allows snapshot files to be downloaded by using FTP.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | MS-SQL-AllowSnapshotFilesFTPDownloading |
| Ldap-Display-Name | mS-SQL-AllowSnapshotFilesFTPDownloading |
| Size              | 4 bytes                                 |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | When replication is setup.              |
| Attribute-Id      | 1.2.840.113556.1.4.1406                 |
| System-Id-Guid    | c49b8be8-d34b-11d2-999a-0000f87a57d4    |
| Syntax            | [**Boolean**](s-boolean.md)            |
