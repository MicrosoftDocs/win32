# Primary-Group-ID attribute

Contains the relative identifier (RID) for the primary group of the user. By default, this is the RID for the Domain Users group.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Primary-Group-ID                     |
| Ldap-Display-Name | primaryGroupID                       |
| Size              | 4 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.98                |
| System-Id-Guid    | bf967a00-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
