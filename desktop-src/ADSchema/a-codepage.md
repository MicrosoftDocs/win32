# Code-Page attribute

Specifies the code page for the user's language of choice. This value is not used by Windows 2000.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Code-Page                                             |
| Ldap-Display-Name | codePage                                              |
| Size              | 4 bytes                                               |
| Update Privilege  | Domain administrator                                  |
| Update Frequency  | When the user desires to change the default language. |
| Attribute-Id      | 1.2.840.113556.1.4.16                                 |
| System-Id-Guid    | bf967938-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Enumeration**](s-enumeration.md)                  |
