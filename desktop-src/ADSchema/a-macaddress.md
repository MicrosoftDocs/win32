# macAddress attribute

Contains the MAC address in maximal, colon-separated hexadecimal notation.



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | macAddress                           |
| Ldap-Display-Name | macAddress                           |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.22                     |
| System-Id-Guid    | e6a522dd-9770-43e1-89de-1de5044328f7 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
