# ms-WMI-SourceOrganization attribute

The business organization that initially created a policy object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-SourceOrganization                   |
| Ldap-Display-Name | msWMI-SourceOrganization                    |
| Size              | Less than fifty characters.                 |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
