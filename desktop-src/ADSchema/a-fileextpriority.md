# File-Ext-Priority attribute

This attribute contains a list of file name extensions in an application package and their associated priorities.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | File-Ext-Priority                           |
| Ldap-Display-Name | fileExtPriority                             |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.816                      |
| System-Id-Guid    | d9e18315-8939-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
