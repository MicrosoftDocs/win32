# MS-DS-Machine-Account-Quota attribute

The number of computer accounts that a user is allowed to create in a domain.



| Entry | Value |
|-------------------|--------------------------------------------------|
| CN                | MS-DS-Machine-Account-Quota                      |
| Ldap-Display-Name | ms-DS-MachineAccountQuota                        |
| Size              | 4 bytes                                          |
| Update Privilege  | Domain administrator                             |
| Update Frequency  | Whenever the quota for a domain needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.1411                          |
| System-Id-Guid    | d064fb68-1480-11d3-91c1-0000f87a57d4             |
| Syntax            | [**Enumeration**](s-enumeration.md)             |
