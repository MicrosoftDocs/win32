# ms-WMI-ScopeGuid attribute

The GUID for the scope in which the associated encoding is located.



| Entry | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-WMI-ScopeGuid                                                |
| Ldap-Display-Name | msWMI-ScopeGuid                                                 |
| Size              | 16 bytes                                                        |
| Update Privilege  | Domain administrator                                            |
| Update Frequency  | Whenever a class containing the attribute is added or modified. |
| Attribute-Id      | 1.2.840.113556.1.4.1686                                         |
| System-Id-Guid    | 87b78d51-405f-4b7f-80ed-2bd28786f48d                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                     |
