# Show-In-Advanced-View-Only attribute

**TRUE** if this attribute is to be visible in the Advanced mode of the UI.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | Show-In-Advanced-View-Only                                                              |
| Ldap-Display-Name | showInAdvancedViewOnly                                                                  |
| Size              | 4 bytes                                                                                 |
| Update Privilege  | Domain administrator                                                                    |
| Update Frequency  | When the object is created or when the viewing aspects of the attribute need to change. |
| Attribute-Id      | 1.2.840.113556.1.2.169                                                                  |
| System-Id-Guid    | bf967984-0de6-11d0-a285-00aa003049e2                                                    |
| Syntax            | [**Boolean**](s-boolean.md)                                                            |
