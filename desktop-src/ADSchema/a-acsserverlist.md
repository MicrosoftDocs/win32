# ACS-Server-List attribute

Contains the DNS names of Windows NT servers that are allowed to run ACS.



|                   |                                             |
|-------------------|---------------------------------------------|
| CN                | ACS-Server-List                             |
| Ldap-Display-Name | aCSServerList                               |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1312                     |
| System-Id-Guid    | 7cbd59a5-3b90-11d2-90cc-00c04fd91ab1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
