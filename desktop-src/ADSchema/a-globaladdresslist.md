# Global-Address-List attribute

This attribute is used on a Microsoft Exchange container to store the distinguished name of a newly created global address list (GAL). This attribute must have an entry before you can enable Messaging Application Programming Interface (MAPI) clients to use a GAL.



|                   |                                         |
|-------------------|-----------------------------------------|
| CN                | Global-Address-List                     |
| Ldap-Display-Name | globalAddressList                       |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1245                 |
| System-Id-Guid    | f754c748-06f4-11d2-aa53-00c04fd7d83a    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
