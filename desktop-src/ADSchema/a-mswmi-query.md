# ms-WMI-Query attribute

A single WQL query.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-Query                                |
| Ldap-Display-Name | msWMI-Query                                 |
| Size              | Less than 250 characters.                   |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1642                     |
| System-Id-Guid    | 65fff93e-35e3-45a3-85ae-876c6718297f        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
