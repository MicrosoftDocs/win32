# NT-Group-Members attribute

This attribute is not used.

The [**member**](a-member.md) attribute is used to contain the members of a group.

|-------------------|-------------------------------------------------------|
| CN                | NT-Group-Members                                      |
| Ldap-Display-Name | nTGroupMembers                                        |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
