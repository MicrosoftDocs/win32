# MS-SQL-UnicodeSortOrder attribute

The Unicode sort order for the current instance of SQL Server.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-UnicodeSortOrder              |
| Ldap-Display-Name | mS-SQL-UnicodeSortOrder              |
| Size              | 4 bytes                              |
| Update Privilege  | Domain administrator                 |
| Update Frequency  | At system startup.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1372              |
| System-Id-Guid    | 72dc918a-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
