# Volume-Count attribute

The tracked volume quota for a given computer.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Volume-Count                         |
| Ldap-Display-Name | volumeCount                          |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
