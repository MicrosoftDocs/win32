# ACS-Non-Reserved-Tx-Limit attribute

The maximum bandwidth a user application can transmit before a reservation is in place.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ACS-Non-Reserved-Tx-Limit            |
| Ldap-Display-Name | aCSNonReservedTxLimit                |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.780               |
| System-Id-Guid    | 1cb355a2-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Interval**](s-interval.md)       |
