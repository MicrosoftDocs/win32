# Retired-Repl-DSA-Signatures attribute

Tracks the past DS replication identities of a given DC.



| Entry | Value |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Retired-Repl-DSA-Signatures                                                         |
| Ldap-Display-Name | retiredReplDSASignatures                                                            |
| Size              | Length is proportional to the number of times the DC has been restored from backup. |
| Update Privilege  | This value is set by the system.                                                    |
| Update Frequency  | \-                                                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.673                                                              |
| System-Id-Guid    | 7bfdcb7f-4807-11d1-a9c3-0000f80367c1                                                |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                               |
