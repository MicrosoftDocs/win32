# Other-Mailbox attribute

Contains other additional mail addresses in a form such as CCMAIL: BruceKeever.



| Entry | Value |
|-------------------|----------------------------------------------------------|
| CN                | Other-Mailbox                                            |
| Ldap-Display-Name | otherMailbox                                             |
| Size              | \-                                                       |
| Update Privilege  | Domain administrator                                     |
| Update Frequency  | Each time the user is issued an additional mail address. |
| Attribute-Id      | 1.2.840.113556.1.4.651                                   |
| System-Id-Guid    | 0296c123-40da-11d1-a9c0-0000f80367c1                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)              |
