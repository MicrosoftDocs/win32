# Default-Group attribute

The group to which this object is assigned when it is created.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Default-Group                           |
| Ldap-Display-Name | defaultGroup                            |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.480                  |
