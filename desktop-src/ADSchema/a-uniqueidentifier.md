# uniqueIdentifier attribute

The uniqueIdentifier attribute type specifies a "unique identifier" for an object represented in the Directory.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | uniqueIdentifier                            |
| Ldap-Display-Name | uniqueIdentifier                            |
| Size              | \-                                          |
| Update Privilege  | \-                                          |

| Attribute-Id      | 0.9.2342.19200300.100.1.44                  |
| System-Id-Guid    | ba0184c7-38c5-4bed-a526-75421470580c        |
