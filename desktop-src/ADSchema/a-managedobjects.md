# Managed-Objects attribute

Contains the list of objects that are managed by the user. The objects listed are those that have the property managedBy property set to this user. Each item in the list is a linked reference to the managed object.



| Entry | Value |
|-------------------|------------------------------------------------------------------------------------|
| CN                | Managed-Objects                                                                    |
| Ldap-Display-Name | managedObjects                                                                     |
| Size              | \-                                                                                 |
| Update Privilege  | This value is set by the system.                                                   |
| Update Frequency  | When the users record is created and whenever the managed objects needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.654                                                             |
| System-Id-Guid    | 0296c124-40da-11d1-a9c0-0000f80367c1                                               |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                                            |
