# ACS-Non-Reserved-Tx-Size attribute

The token bucket size an application can use before a reservation is in place.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ACS-Non-Reserved-Tx-Size             |
| Ldap-Display-Name | aCSNonReservedTxSize                 |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.898               |
| System-Id-Guid    | f072230d-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**Interval**](s-interval.md)       |
