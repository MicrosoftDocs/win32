# ms-WMI-NormalizedClass attribute

The name of a core WMI policy class.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-NormalizedClass                      |
| Ldap-Display-Name | msWMI-NormalizedClass                       |
| Size              | Less than twenty characters.                |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1640                     |
| System-Id-Guid    | eaba628f-eb8e-4fe9-83fc-693be695559b        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
