# ms-Exch-Owner-BL attribute

The backward link to the owner attribute. Contains a list of owners for an object.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | ms-Exch-Owner-BL                        |
| Ldap-Display-Name | ownerBL                                 |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.104                  |
| System-Id-Guid    | bf9679f4-0de6-11d0-a285-00aa003049e2    |
