# MS-SQL-ThirdParty attribute

This attribute indicates whether the publication data is from a data source other than SQL Server. If it is from another source, then it is set to **TRUE**.



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | MS-SQL-ThirdParty                    |
| Ldap-Display-Name | mS-SQL-ThirdParty                    |
| Size              | 4 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | When replication is setup.           |
| Attribute-Id      | 1.2.840.113556.1.4.1407              |
| System-Id-Guid    | c4e311fc-d34b-11d2-999a-0000f87a57d4 |
| Syntax            | [**Boolean**](s-boolean.md)         |
