# Reps-From attribute

Lists the servers from which the directory will accept changes for the defined naming context.

|-------------------|-------------------------------------------------------|
| CN                | Reps-From                                             |
| Ldap-Display-Name | repsFrom                                              |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.91                                 |
| System-Id-Guid    | bf967a1d-0de6-11d0-a285-00aa003049e2                  |
