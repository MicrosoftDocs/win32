# USN-Source attribute

Value of the [**USN-Changed**](a-usnchanged.md) attribute of the object from the remote directory that replicated the change to the local server.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------------------------------|
| CN                | USN-Source                                                                                          |
| Ldap-Display-Name | uSNSource                                                                                           |
| Size              | 8 bytes                                                                                             |
| Update Privilege  | This value is set by the system.                                                                    |
| Update Frequency  | When the object is changed on a remote directory that replicated the change to the local directory. |
| Attribute-Id      | 1.2.840.113556.1.4.896                                                                              |
| System-Id-Guid    | 167758ad-47f3-11d1-a9c3-0000f80367c1                                                                |
| Syntax            | [**Interval**](s-interval.md)                                                                      |
