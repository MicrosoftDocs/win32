# ms-DS-UpdateScript attribute

This is used to hold the script with the domain restructure instructions.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-UpdateScript                          |
| Ldap-Display-Name | msDS-UpdateScript                           |
| Size              | Maximum size 10K bytes.                     |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | Only during domain restructure.             |
| Attribute-Id      | 1.2.840.113556.1.4.1721                     |
| System-Id-Guid    | 146eb639-bb9f-4fc1-a825-e29e00c77920        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
