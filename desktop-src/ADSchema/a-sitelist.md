# Site-List attribute

List of sites connected to this link object.



| Entry | Value |
|-------------------|----------------------------------------------|
| CN                | Site-List                                    |
| Ldap-Display-Name | siteList                                     |
| Size              | \-                                           |
| Update Privilege  | Domain administrator                         |
| Update Frequency  | Whenever a site is added to or removed from. |
| Attribute-Id      | 1.2.840.113556.1.4.821                       |
| System-Id-Guid    | d50c2cdc-8951-11d1-aebc-0000f80367c1         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
