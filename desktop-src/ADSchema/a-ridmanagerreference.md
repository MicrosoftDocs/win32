# RID-Manager-Reference attribute

The Distinguished Name for the RID Manager of an object.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | RID-Manager-Reference                   |
| Ldap-Display-Name | rIDManagerReference                     |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | Whenever the RID Manager changes.       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| System-Id-Guid    | 66171886-8f3c-11d0-afda-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
