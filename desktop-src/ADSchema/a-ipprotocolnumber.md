# ipProtocolNumber attribute

Contains the section of the protocols map that stores the unique number that identifies the protocol.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ipProtocolNumber                     |
| Ldap-Display-Name | ipProtocolNumber                     |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
