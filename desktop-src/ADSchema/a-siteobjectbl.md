# Site-Object-BL attribute

The list of distinguished names for subnets that belong to this site.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Site-Object-BL                          |
| Ldap-Display-Name | siteObjectBL                            |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | Whenever a site is created or deleted.  |
| Attribute-Id      | 1.2.840.113556.1.4.513                  |
| System-Id-Guid    | 3e10944d-c354-11d0-aff8-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
