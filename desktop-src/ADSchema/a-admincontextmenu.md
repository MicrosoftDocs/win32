# Admin-Context-Menu attribute

The order number and GUID of the context menu to be used on administration screens.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Admin-Context-Menu                          |
| Ldap-Display-Name | adminContextMenu                            |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.614                      |
| System-Id-Guid    | 553fd038-f32e-11d0-b0bc-00c04fd8dca6        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
