# ms-DS-Behavior-Version attribute

This attribute is used to track the domain or forest behavior version. It is a monotonically increasing number that is used to enable certain Active Directory features.



| Entry | Value |
|-------------------|------------------------------------------------|
| CN                | ms-DS-Behavior-Version                         |
| Ldap-Display-Name | msDS-Behavior-Version                          |
| Size              | \-                                             |
| Update Privilege  | This value is set by the Domain Administrator. |
| Update Frequency  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1459                        |
| System-Id-Guid    | d31a8757-2447-4545-8081-3bb610cacbf2           |
| Syntax            | [**Enumeration**](s-enumeration.md)           |
