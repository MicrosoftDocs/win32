# ms-WMI-TargetNameSpace attribute

Namespace in which the object is to be created.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-TargetNameSpace                      |
| Ldap-Display-Name | msWMI-TargetNameSpace                       |
| Size              | Less than twenty characters.                |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1646                     |
| System-Id-Guid    | 1c4ab61f-3420-44e5-849d-8b5dbf60feb7        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
