# ACS-Minimum-Latency attribute

The **ACS-Minimum-Latency** attribute is for internal use only. Based on RFC2210.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ACS-Minimum-Latency                  |
| Ldap-Display-Name | aCSMinimumLatency                    |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1316              |
| System-Id-Guid    | 9517fefb-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Interval**](s-interval.md)       |
