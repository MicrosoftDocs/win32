# ms-DS-Settings attribute

Used to store settings for an object. Its use is solely determined by the object's owner. We recommend using it to store name/value pairs. For example, color=blue.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Settings                              |
| Ldap-Display-Name | msDS-Settings                               |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | Each time the settings values change.       |
| Attribute-Id      | 1.2.840.113556.1.4.1697                     |
| System-Id-Guid    | 0e1b47d7-40a3-4b48-8d1b-4cac0c1cdf21        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
