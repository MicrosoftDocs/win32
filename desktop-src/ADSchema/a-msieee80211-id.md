# ms-ieee-80211-ID attribute

An identifier used for a wireless policy object on AD.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | ms-ieee-80211-ID                                                                 |
| Ldap-Display-Name | msieee80211-ID                                                                   |
| Size              | \-                                                                               |
| Update Privilege  | Domain administrator                                                             |
| Update Frequency  | Each time a Domain Admin changes the wireless network policy for a domain or OU. |
| Attribute-Id      | 1.2.840.113556.1.4.1823                                                          |
| System-Id-Guid    | 7f73ef75-14c9-4c23-81de-dd07a06f9e8b                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
