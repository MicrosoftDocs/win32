# Object-Classes attribute

A multiple-valued property that contains strings that represent each class in the schema. Each value contains the governsID, lDAPDisplayName, mustContain, mayContain, and so on.



| Entry | Value |
|-------------------|--------------------------------------------------|
| CN                | Object-Classes                                   |
| Ldap-Display-Name | objectClasses                                    |
| Size              | About 20 bytes on average per class name.        |
| Update Privilege  | The designer of the object would set this value. |
| Update Frequency  | Whenever the design of the class changes.        |
| Attribute-Id      | 2.5.21.6                                         |
| System-Id-Guid    | 9a7ad94b-ca53-11d1-bbd0-0080c76670c0             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)      |
