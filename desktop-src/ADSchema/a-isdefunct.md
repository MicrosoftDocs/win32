# Is-Defunct attribute

If **TRUE**, the class or attribute is no longer usable. Old versions of this object may exist, but new ones cannot be created.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| Ldap-Display-Name | isDefunct                            |
| Size              | \-                                   |
| Update Privilege  | Schema administrator                 |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| System-Id-Guid    | 28630ebe-41d5-11d1-a9c1-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
