# SubSchemaSubEntry attribute

The distinguished name for the location of the subschema object where a class or attribute is defined.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | SubSchemaSubEntry                                                                |
| Ldap-Display-Name | subSchemaSubEntry                                                                |
| Size              | \-                                                                               |
| Update Privilege  | Schema administrator                                                             |
| Update Frequency  | When the Active Directory is loaded or when a new class or attribute is defined. |
| Attribute-Id      | 2.5.18.10                                                                        |
| System-Id-Guid    | 9a7ad94d-ca53-11d1-bbd0-0080c76670c0                                             |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                                          |
