# ms-DS-NC-Repl-Cursors attribute

A list of past and present replication partners, and how current we are with each of them.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-Repl-Cursors                       |
| Ldap-Display-Name | msDS-NCReplCursors                          |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1704                     |
| System-Id-Guid    | 8a167ce4-f9e8-47eb-8d78-f7fe80abb2cc        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
