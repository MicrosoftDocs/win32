# Context-Menu attribute

The order number and GUID of the context menu to be used for an object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Context-Menu                                |
| Ldap-Display-Name | contextMenu                                 |
| Size              | \-                                          |
| Update Privilege  | Schema administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.499                      |
| System-Id-Guid    | 4d8601ee-ac85-11d0-afe3-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
