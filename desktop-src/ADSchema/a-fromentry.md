# From-Entry attribute

This is a constructed attribute that is **TRUE** if the object is writable and **FALSE** if it is read-only, for example, a GC replica instance.



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | From-Entry                           |
| Ldap-Display-Name | fromEntry                            |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.910               |
| System-Id-Guid    | 9a7ad949-ca53-11d1-bbd0-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
