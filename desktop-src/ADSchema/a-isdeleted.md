# Is-Deleted attribute

If **TRUE**, this object has been marked for deletion and cannot be instantiated. After the tombstone period has expired, it will be removed from the system.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Is-Deleted                           |
| Ldap-Display-Name | isDeleted                            |
| Size              | \-                                   |
| Update Privilege  | Domain administrator                 |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.48                |
| System-Id-Guid    | bf96798f-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
