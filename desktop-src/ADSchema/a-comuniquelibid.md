# COM-Unique-LIBID attribute

Single-valued string GUID LIBID for a type library.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | COM-Unique-LIBID                            |
| Ldap-Display-Name | cOMUniqueLIBID                              |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.250                      |
| System-Id-Guid    | 281416da-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
