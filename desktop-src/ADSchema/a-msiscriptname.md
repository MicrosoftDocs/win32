# Msi-Script-Name attribute

The name for the Microsoft installer advertisement script file for this application.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Msi-Script-Name                             |
| Ldap-Display-Name | msiScriptName                               |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.845                      |
| System-Id-Guid    | 96a7dd62-9118-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
