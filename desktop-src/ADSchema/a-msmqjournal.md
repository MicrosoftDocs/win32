# MSMQ-Journal attribute

Indicates whether messages retrieved form the queue should be kept in journal.

|-------------------|--------------------------------------|
| CN                | MSMQ-Journal                         |
| Ldap-Display-Name | mSMQJournal                          |
| Size              | \-                                   |
| Update Privilege  | \-                                   |

| Attribute-Id      | 1.2.840.113556.1.4.918               |

| Syntax            | [**Boolean**](s-boolean.md)         |
