# WWW-Page-Other attribute

A list of alternate webpages.



| Entry | Value |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-Page-Other                                                              |
| Ldap-Display-Name | url                                                                         |
| Size              | \-                                                                          |
| Update Privilege  | Domain administrator or account owner.                                      |
| Update Frequency  | When the user's record is created and whenever the webpage needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.749                                                      |
| System-Id-Guid    | 9a9a0221-4a5b-11d1-a9c3-0000f80367c1                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
