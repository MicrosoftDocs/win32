# ipServiceProtocol attribute

Contains the section of the services map that stores the protocol number for a UNIX service.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ipServiceProtocol                    |
| Ldap-Display-Name | ipServiceProtocol                    |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.16                     |
| System-Id-Guid    | cd96ec0b-1ed6-43b4-b26b-f170b645883f |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
