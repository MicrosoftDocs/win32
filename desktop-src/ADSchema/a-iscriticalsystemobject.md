# Is-Critical-System-Object attribute

If **TRUE**, the object hosting this attribute must be replicated during installation of a new replica.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Is-Critical-System-Object            |
| Ldap-Display-Name | isCriticalSystemObject               |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.868               |
| System-Id-Guid    | 00fbf30d-91fe-11d1-aebc-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
