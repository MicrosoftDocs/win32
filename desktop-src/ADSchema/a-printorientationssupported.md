# Print-Orientations-Supported attribute

The page rotation for landscape printing.



| Entry | Value |
|-------------------|-------------------------------------------------------------|
| CN                | Print-Orientations-Supported                                |
| Ldap-Display-Name | printOrientationsSupported                                  |
| Size              | 4 bytes. Possible values: 0, 90, 270, and 0 = no landscape. |
| Update Privilege  | \-                                                          |
| Update Frequency  | \-                                                          |
| Attribute-Id      | 1.2.840.113556.1.4.240                                      |
| System-Id-Guid    | 281416d0-1968-11d0-a28f-00aa003049e2                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                 |
