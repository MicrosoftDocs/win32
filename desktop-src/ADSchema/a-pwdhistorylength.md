# Pwd-History-Length attribute

The number of old passwords to save.



| Entry | Value |
|-------------------|------------------------------------------|
| CN                | Pwd-History-Length                       |
| Ldap-Display-Name | pwdHistoryLength                         |
| Size              | 4 bytes                                  |
| Update Privilege  | Domain administrator                     |
| Update Frequency  | When the account policy needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.95                    |
| System-Id-Guid    | bf967a09-0de6-11d0-a285-00aa003049e2     |
| Syntax            | [**Enumeration**](s-enumeration.md)     |
