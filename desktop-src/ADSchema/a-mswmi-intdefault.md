# ms-WMI-intDefault attribute

The default value for WMI 32-bit integer parameter objects.

|-------------------|--------------------------------------|
| CN                | ms-WMI-intDefault                    |
| Ldap-Display-Name | msWMI-IntDefault                     |
| Size              | 4 bytes                              |
| Update Privilege  | Group Policy Administrator           |
