# Obj-Dist-Name attribute

Same as the Distinguished Name for an object. Used by Exchange.



|                   |                                         |
|-------------------|-----------------------------------------|
| CN                | Obj-Dist-Name                           |
| Ldap-Display-Name | distinguishedName                       |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | Whenever an object is created or moved. |
| Attribute-Id      | 2.5.4.49                                |
| System-Id-Guid    | bf9679e4-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
