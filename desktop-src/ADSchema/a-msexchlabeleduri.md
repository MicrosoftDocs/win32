# ms-Exch-LabeledURI attribute

An Exchange URI followed by a label. The label is used to describe the resource to which the URI points and is intended as a human-readable name.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-Exch-LabeledURI                          |
| Ldap-Display-Name | msExchLabeledURI                            |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.593                      |
| System-Id-Guid    | 16775820-47f3-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
