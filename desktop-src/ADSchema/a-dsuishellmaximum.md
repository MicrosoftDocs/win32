# DS-UI-Shell-Maximum attribute

This is the default maximum number of objects to be shown in a container by the shell UI.

|-------------------|--------------------------------------|
| CN                | DS-UI-Shell-Maximum                  |
| Ldap-Display-Name | dSUIShellMaximum                     |
| Size              | \-                                   |
| Update Privilege  | Domain Administrator                 |
