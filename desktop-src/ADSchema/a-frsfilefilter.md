# FRS-File-Filter attribute

A list of file name extensions excluded from file replication.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | FRS-File-Filter                             |
| Ldap-Display-Name | fRSFileFilter                               |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | When replication is setup.                  |
| Attribute-Id      | 1.2.840.113556.1.4.483                      |
| System-Id-Guid    | 1be8f170-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
