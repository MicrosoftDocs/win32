# Phone-Home-Other attribute

A list of alternate home phone numbers.



|                   |                                                                                  |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Phone-Home-Other                                                                 |
| Ldap-Display-Name | otherHomePhone                                                                   |
| Size              | \-                                                                               |
| Update Privilege  | Domain administrator or account owner.                                           |
| Update Frequency  | When the user's record is created and whenever the phone number needs to change. |
| Attribute-Id      | 1.2.840.113556.1.2.277                                                           |
| System-Id-Guid    | f0f8ffa2-1191-11d0-a060-00aa006c33ed                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
