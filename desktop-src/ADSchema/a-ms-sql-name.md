# MS-SQL-Name attribute

The name of a SQL Server instance. The default is MSSQLSERVER.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Name                                 |
| Ldap-Display-Name | mS-SQL-Name                                 |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | At system setup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-Id-Guid    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
