# Link-ID attribute

An integer that indicates that the attribute is a linked attribute. An even integer is a forward link and an odd integer is a backward link.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Link-ID                              |
| Ldap-Display-Name | linkID                               |
| Size              | \-                                   |
| Update Privilege  | Schema administrator                 |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-Id-Guid    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
