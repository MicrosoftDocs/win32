# Template-Roots2 attribute

This attribute is used on the Exchange config container to indicate where the template containers are stored. This information is used by the Active Directory MAPI provider



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Template-Roots2                         |
| Ldap-Display-Name | templateRoots2                          |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2048                 |
| System-Id-Guid    | b1cba91a-0682-4362-a659-153e201ef069    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
