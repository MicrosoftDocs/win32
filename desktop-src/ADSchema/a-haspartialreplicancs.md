# Has-Partial-Replica-NCs attribute

Sibling to Has-Master-NCs. Has-Partial-Replica-NCs reflects the distinguished name for all other-domain NCs that have been replicated into a global catalog.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Has-Partial-Replica-NCs                 |
| Ldap-Display-Name | hasPartialReplicaNCs                    |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.15                   |
| System-Id-Guid    | bf967981-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
