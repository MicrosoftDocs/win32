# ms-DS-ReplicationEpoch attribute

This is used to hold the epoch under which all the DCs are replicating. An epoch is the period of time in which a domain has a specific name. A new epoch starts when a domain name change occurs.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ms-DS-ReplicationEpoch               |
| Ldap-Display-Name | msDS-ReplicationEpoch                |
| Size              | \-                                   |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | Only during domain restructure.      |
| Attribute-Id      | 1.2.840.113556.1.4.1720              |
| System-Id-Guid    | 08e3aa79-eb1c-45b5-af7b-8f94246c8e41 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
