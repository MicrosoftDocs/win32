# Object-Category attribute

An object class name used to group objects of this or derived classes.



| Entry | Value |
|-------------------|--------------------------------------------------|
| CN                | Object-Category                                  |
| Ldap-Display-Name | objectCategory                                   |
| Size              | About 20 bytes on average.                       |
| Update Privilege  | The designer of the object would set this value. |
| Update Frequency  | This value should never change.                  |
| Attribute-Id      | 1.2.840.113556.1.4.782                           |
| System-Id-Guid    | 26d97369-6070-11d1-a9c6-0000f80367c1             |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
