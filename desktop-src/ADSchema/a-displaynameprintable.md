# Display-Name-Printable attribute

The printable display name for an object. The printable display name is usually the combination of the user's first name, middle initial, and last name.



| Entry | Value |
|-------------------|----------------------------------------|
| CN                | Display-Name-Printable                 |
| Ldap-Display-Name | displayNamePrintable                   |
| Size              | \-                                     |
| Update Privilege  | Domain administrator or account owner. |
| Update Frequency  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-Id-Guid    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Syntax            | [**String(IA5)**](s-string-ia5.md)    |
