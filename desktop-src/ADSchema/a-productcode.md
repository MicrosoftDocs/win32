# Product-Code attribute

This attribute contains a unique identifier for an application for a particular product release, represented as a string GUID, for example "{12345678-1234-1234-1234-123456789012}". Letters used in this GUID must be uppercase. This ID must vary for different versions and languages.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Product-Code                                          |
| Ldap-Display-Name | productCode                                           |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.818                                |
| System-Id-Guid    | d9e18317-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
