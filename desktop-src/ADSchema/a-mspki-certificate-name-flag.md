# ms-PKI-Certificate-Name-Flag attribute

Contains the flags related to constructing the subject name in an issued certificate.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Certificate-Name-Flag                                                                      |
| Ldap-Display-Name | msPKI-Certificate-Name-Flag                                                                       |
| Size              | 4 bytes                                                                                           |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1432                                                                           |
| System-Id-Guid    | ea1dddc4-60ff-416e-8cc0-17cee534bce7                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
