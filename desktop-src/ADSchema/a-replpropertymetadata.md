# Repl-Property-Meta-Data attribute

Tracks internal replication state information for DS objects. Information here can be extracted in public form through the public API DsReplicaGetInfo(). Present on all DS objects.



| Entry | Value |
|-------------------|------------------------------------------------------------------------------|
| CN                | Repl-Property-Meta-Data                                                      |
| Ldap-Display-Name | replPropertyMetaData                                                         |
| Size              | Length is proportional to the number of replicable attributes on the object. |
| Update Privilege  | This value is set by the system.                                             |
| Update Frequency  | Changes in response to any replicating changes in the object.                |
| Attribute-Id      | 1.2.840.113556.1.4.3                                                         |
| System-Id-Guid    | 281416c0-1968-11d0-a28f-00aa003049e2                                         |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                        |
