# Help-File-Name attribute

This attribute was used for Exchange 4.0. It contained the name that should be used for the file when the provider downloaded help data to a client computer. It is not used for any other versions of Exchange.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Help-File-Name                              |
| Ldap-Display-Name | helpFileName                                |
| Size              | \-                                          |
| Update Privilege  | This is used by the system.                 |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.327                      |
| System-Id-Guid    | 5fd424a9-1262-11d0-a060-00aa006c33ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
