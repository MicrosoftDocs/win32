# MS-SQL-Alias attribute

An alias used to connect to the database. Default is set to Alias.

| CN                | MS-SQL-Alias                                |
| Ldap-Display-Name | mS-SQL-Alias                                |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1395                     |
| System-Id-Guid    | e0c6baae-ccee-11d2-9993-0000f87a57d4        |
