# MSMQ-Ds-Services attribute

Indicates whether the MSMQ installed on this computer provides MSMQ DS services.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | MSMQ-Ds-Services                     |
| Ldap-Display-Name | mSMQDsServices                       |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1228              |
| System-Id-Guid    | 2df90d78-009f-11d2-aa4c-00c04fd7d83a |
| Syntax            | [**Boolean**](s-boolean.md)         |
