# Extended-Chars-Allowed attribute

Indicates whether extended characters are allowed in the value of this attribute. Only applies to IA5, Numeric, Printable, and Teletex string attributes.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Extended-Chars-Allowed               |
| Ldap-Display-Name | extendedCharsAllowed                 |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.380               |
| System-Id-Guid    | bf967966-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
