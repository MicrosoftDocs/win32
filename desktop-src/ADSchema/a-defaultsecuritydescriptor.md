# Default-Security-Descriptor attribute

The security descriptor to be assigned to the object when it is first created.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Default-Security-Descriptor                 |
| Ldap-Display-Name | defaultSecurityDescriptor                   |
| Size              | \-                                          |
| Update Privilege  | Schema administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.224                      |
| System-Id-Guid    | 807a6d30-1669-11d0-a064-00aa006c33ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
