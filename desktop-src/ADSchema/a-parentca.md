# Parent-CA attribute

The distinguished name of a certification authority (CA) object for a parent CA.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Parent-CA                               |
| Ldap-Display-Name | parentCA                                |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.557                  |
| System-Id-Guid    | 5245801b-ca6a-11d0-afff-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
