# ms-PKI-Enrollment-Flag attribute

Contains the enrollment related flags.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Enrollment-Flag                                                                            |
| Ldap-Display-Name | msPKI-Enrollment-Flag                                                                             |
| Size              | 4 bytes                                                                                           |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1430                                                                           |
| System-Id-Guid    | d15ef7d8-f226-46db-ae79-b34e560bd12c                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
