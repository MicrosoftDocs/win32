# ms-PKI-RA-Application-Policies attribute

The required RA application policy OID in the counter signatures of the certificate request.



| Entry | Value |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-Application-Policies                                                             |
| Ldap-Display-Name | msPKI-RA-Application-Policies                                                              |
| Size              | \-                                                                                         |
| Update Privilege  | Enterprise administrator                                                                   |
| Update Frequency  | Whenever a new template is created, or the attributes of an existing templates are edited. |
| Attribute-Id      | 1.2.840.113556.1.4.1675                                                                    |
| System-Id-Guid    | 3c91fbbf-4773-4ccd-a87b-85d53e7bcf6a                                                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                |
