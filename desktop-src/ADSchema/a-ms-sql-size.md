# MS-SQL-Size attribute

The size, in kilobytes, of the database.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Size                          |
| Ldap-Display-Name | mS-SQL-Size                          |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | When the database changes size.      |
| Attribute-Id      | 1.2.840.113556.1.4.1396              |
| System-Id-Guid    | e9098084-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Interval**](s-interval.md)       |
