# ms-DS-Az-Application-Name attribute

A string that uniquely identifies an application object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Az-Application-Name                   |
| Ldap-Display-Name | msDS-AzApplicationName                      |
| Size              | 128 characters                              |
| Update Privilege  | AzRoles admin                               |
| Update Frequency  | During initialization or policy change.     |
| Attribute-Id      | 1.2.840.113556.1.4.1798                     |
| System-Id-Guid    | db5b0728-6208-4876-83b7-95d3e5695275        |
