# Object-Guid attribute

The unique identifier for an object.



| Entry | Value |
|-------------------|---------------------------------------------------------------------|
| CN                | Object-Guid                                                         |
| Ldap-Display-Name | objectGUID                                                          |
| Size              | 16 bytes                                                            |
| Update Privilege  | This value is set by the system.                                    |
| Update Frequency  | This value is set when the object is created and cannot be changed. |
| Attribute-Id      | 1.2.840.113556.1.4.2                                                |
| System-Id-Guid    | bf9679e7-0de6-11d0-a285-00aa003049e2                                |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)               |
