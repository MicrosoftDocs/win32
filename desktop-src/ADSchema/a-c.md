# Country-Name attribute

The country/region in the address of the user. The country/region is represented as a 2-character code based on ISO-3166.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Country-Name                                |
| Ldap-Display-Name | c                                           |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 2.5.4.6                                     |
| System-Id-Guid    | bf967945-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
