# MS-SQL-Version attribute

The version for the current instance of SQL Server.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Version                              |
| Ldap-Display-Name | mS-SQL-Version                              |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | At system setup.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1388                     |
| System-Id-Guid    | c07cc1d0-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
