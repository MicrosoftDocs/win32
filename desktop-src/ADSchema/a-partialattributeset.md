# Partial-Attribute-Set attribute

Tracks the internal replication state of partial replicas (that is, on GCs). Attribute of the partial replica NC object. Defines the set of attributes present on a particular partial replica NC.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Partial-Attribute-Set                                 |
| Ldap-Display-Name | partialAttributeSet                                   |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | During replication                                    |
| Attribute-Id      | 1.2.840.113556.1.4.640                                |
| System-Id-Guid    | 19405b9e-3cfa-11d1-a9c0-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
