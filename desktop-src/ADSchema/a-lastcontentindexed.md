# Last-Content-Indexed attribute

The time this volume was last content indexed.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Last-Content-Indexed                 |
| Ldap-Display-Name | lastContentIndexed                   |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.50                |
| System-Id-Guid    | bf967995-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
