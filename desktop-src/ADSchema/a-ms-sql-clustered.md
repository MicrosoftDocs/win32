# MS-SQL-Clustered attribute

True if the server is clustered.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Clustered                     |
| Ldap-Display-Name | mS-SQL-Clustered                     |
| Size              | 4 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | At system startup.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1373              |
| System-Id-Guid    | 7778bd90-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Boolean**](s-boolean.md)         |
