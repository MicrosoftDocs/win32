# ms-WMI-PropertyName attribute

The target policy object name for a parameter object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-PropertyName                         |
| Ldap-Display-Name | msWMI-PropertyName                          |
| Size              | Less than twenty characters.                |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1641                     |
| System-Id-Guid    | ab920883-e7f8-4d72-b4a0-c0449897509d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
