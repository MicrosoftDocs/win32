# ms-WMI-Name attribute

The display name for top-level policy objects. Used in the Global Catalog.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-Name                                 |
| Ldap-Display-Name | msWMI-Name                                  |
| Size              | \-                                          |
| Update Privilege  | Group Policy Administrator                  |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1639                     |
| System-Id-Guid    | c6c8ace5-7e81-42af-ad72-77412c5941c4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
