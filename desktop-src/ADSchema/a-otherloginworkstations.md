# Other-Login-Workstations attribute

Non Windows NT or LAN Manager workstations from which a user can log on.

> [!Note]  
> Active Directory does not use or populate this field.

 



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Other-Login-Workstations                    |
| Ldap-Display-Name | otherLoginWorkstations                      |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.91                       |
| System-Id-Guid    | bf9679f1-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
