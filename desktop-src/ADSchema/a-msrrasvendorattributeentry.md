# ms-RRAS-Vendor-Attribute-Entry attribute

A string that enables vendors to add router attributes to the DS for use in queries, in the form AttributeName:vendorID:AttributeType.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-RRAS-Vendor-Attribute-Entry              |
| Ldap-Display-Name | msRRASVendorAttributeEntry                  |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.883                      |
| System-Id-Guid    | f39b98ac-938d-11d1-aebd-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
