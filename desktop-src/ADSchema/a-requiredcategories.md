# Required-Categories attribute

This attribute contains a list of component category IDs an object (such as an application) requires to run.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Required-Categories                                   |
| Ldap-Display-Name | requiredCategories                                    |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.321                                |
| System-Id-Guid    | 7d6c0e93-7e20-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
