# ms-PKI-RA-Signature attribute

Specifies the number of enrollment registration authority signatures that are required in an enrollment request.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-Signature                                                                               |
| Ldap-Display-Name | msPKI-RA-Signature                                                                                |
| Size              | 4 bytes                                                                                           |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| System-Id-Guid    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
