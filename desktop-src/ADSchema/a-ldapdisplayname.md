# LDAP-Display-Name attribute

The name used by LDAP clients, such as the ADSI LDAP provider, to read and write the attribute by using the LDAP protocol.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | LDAP-Display-Name                           |
| Ldap-Display-Name | lDAPDisplayName                             |
| Size              | \-                                          |
| Update Privilege  | Schema administrator                        |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.460                      |
| System-Id-Guid    | bf96799a-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
