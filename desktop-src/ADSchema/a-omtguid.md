# OMT-Guid attribute

The unique identifier for a Link-Track-Object-Move table entry.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | OMT-Guid                                              |
| Ldap-Display-Name | oMTGuid                                               |
| Size              | 16 bytes                                              |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | Whenever the link for a file changes.                 |
| Attribute-Id      | 1.2.840.113556.1.4.505                                |
| System-Id-Guid    | ddac0cf3-af8f-11d0-afeb-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
