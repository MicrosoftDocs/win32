# rpc-Ns-Annotation attribute

A string that describs a given RPC Profile element.

| CN                | rpc-Ns-Annotation                           |
| Ldap-Display-Name | rpcNsAnnotation                             |
| Size              | \-                                          |
| Update Privilege  | \-                                          |

| Attribute-Id      | 1.2.840.113556.1.4.366                      |
