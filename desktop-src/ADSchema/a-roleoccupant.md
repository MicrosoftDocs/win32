# Role-Occupant attribute

The distinguished name of an object that fulfills an organizational role.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Role-Occupant                           |
| Ldap-Display-Name | roleOccupant                            |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 2.5.4.33                                |
| System-Id-Guid    | a8df7465-c5ea-11d1-bbcb-0080c76670c0    |
