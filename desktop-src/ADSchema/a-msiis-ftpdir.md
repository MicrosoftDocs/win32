# ms-IIS-FTP-Dir attribute

This attribute determines the user home directory relative to the file server share. It is used in conjunction with ms-IIS-FTP-Root to determine the FTP user home directory.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------|
| CN                | ms-IIS-FTP-Dir                                                                                                                  |
| Ldap-Display-Name | msIIS-FTPDir                                                                                                                    |
| Size              | 30-50 characters (15-25 Unicode characters for each property)                                                                   |
| Update Privilege  | Domain administrator & local system                                                                                             |
| Update Frequency  | This property is set when the administrator creates the website and sets FTP isolation. It's rarely going to change after that. |
| Attribute-Id      | 1.2.840.113556.1.4.1786                                                                                                         |
| System-Id-Guid    | 8a5c99e9-2230-46eb-b8e8-e59d712eb9ee                                                                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                                                     |
