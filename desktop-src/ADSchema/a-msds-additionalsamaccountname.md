# ms-DS-Additional-Sam-Account-Name attribute

This attribute is used to store the SAM account names that correspond to the DNS host names in ms-DS-Additional-Dns-Host-Name.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Additional-Sam-Account-Name           |
| Ldap-Display-Name | msDS-AdditionalSamAccountName               |
| Size              | Less than 20 bytes.                         |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | When the DC is renamed.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1718                     |
| System-Id-Guid    | 975571df-a4d5-429a-9f59-cdc6581d91e6        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
