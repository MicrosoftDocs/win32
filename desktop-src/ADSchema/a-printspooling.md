# Print-Spooling attribute

A string that represents the type of printer spooling.



| Entry | Value |
|-------------------|----------------------------------------------------------------------|
| CN                | Print-Spooling                                                       |
| Ldap-Display-Name | printSpooling                                                        |
| Size              | Possible values: PrintDirect, PrintWhileSpooling, PrintAfterSpooled. |
| Update Privilege  | \-                                                                   |
| Update Frequency  | \-                                                                   |
| Attribute-Id      | 1.2.840.113556.1.4.274                                               |
| System-Id-Guid    | ba305f6c-47e3-11d0-a1a6-00c04fd930c9                                 |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                          |
