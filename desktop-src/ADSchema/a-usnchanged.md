# USN-Changed attribute

The update sequence number (USN) assigned by the local directory for the latest change, including creation. See also , [**USN-Created**](a-usncreated.md).



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | USN-Changed                          |
| Ldap-Display-Name | uSNChanged                           |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | When the object is changed.          |
| Attribute-Id      | 1.2.840.113556.1.2.120               |
| System-Id-Guid    | bf967a6f-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Interval**](s-interval.md)       |
