# Script-Path attribute

This attribute specifies the path for the user's logon script. The string can be null.



|                   |                                                                        |
|-------------------|------------------------------------------------------------------------|
| CN                | Script-Path                                                            |
| Ldap-Display-Name | scriptPath                                                             |
| Size              | \-                                                                     |
| Update Privilege  | Domain administrator or account owner.                                 |
| Update Frequency  | When the user record is created and whenever the path needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.62                                                  |
| System-Id-Guid    | bf9679a8-0de6-11d0-a285-00aa003049e2                                   |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                            |
