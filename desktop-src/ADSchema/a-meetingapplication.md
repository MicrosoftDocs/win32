# meetingApplication attribute

This attribute contains the name of the application that will be used for an online meeting, such as Microsoft NetMeeting, Windows Media Services, or Microsoft Exchange Conferencing.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | meetingApplication                          |
| Ldap-Display-Name | meetingApplication                          |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.573                      |
| System-Id-Guid    | 11b6cc83-48c4-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
