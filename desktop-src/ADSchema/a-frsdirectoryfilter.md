# FRS-Directory-Filter attribute

The list of directories excluded from file replication, for example, temp, or obj.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | FRS-Directory-Filter                        |
| Ldap-Display-Name | fRSDirectoryFilter                          |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.484                      |
| System-Id-Guid    | 1be8f171-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
