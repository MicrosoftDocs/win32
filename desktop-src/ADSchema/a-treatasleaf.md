# Treat-As-Leaf attribute

Display-specifiers with this flag set to **TRUE** force the related class to be displayed as a leaf class even if it has children.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Treat-As-Leaf                        |
| Ldap-Display-Name | treatAsLeaf                          |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.806               |
| System-Id-Guid    | 8fd044e3-771f-11d1-aeae-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
