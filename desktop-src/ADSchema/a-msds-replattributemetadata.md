# ms-DS-Repl-Attribute-Meta-Data attribute

A list of metadata for each replicated attribute. The metadata indicates who changed the attribute last.

| CN                | ms-DS-Repl-Attribute-Meta-Data              |
| Ldap-Display-Name | msDS-ReplAttributeMetaData                  |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
