# Revision attribute

The revision level for a security descriptor or other change. Only used in the sam-server and ds-ui-settings objects.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Revision                             |
| Ldap-Display-Name | revision                             |
| Size              | 4 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.145               |
| System-Id-Guid    | bf967a21-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
