# DN-Reference-Update attribute

If an object is renamed, this attribute is used to track all of the previous and current names that have been assigned to an object so that linked objects can still find it.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | DN-Reference-Update                     |
| Ldap-Display-Name | dNReferenceUpdate                       |
| Size              | \-                                      |
| Update Privilege  | This is set by the system.              |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1242                 |
| System-Id-Guid    | 2df90d86-009f-11d2-aa4c-00c04fd7d83a    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
