# ms-PKI-Certificate-Policy attribute

Contains the list of policy OIDs and their optional CSPs in the issued certificate.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Certificate-Policy                                                                         |
| Ldap-Display-Name | msPKI-Certificate-Policy                                                                          |
| Size              | 64 bytes times the number of entries.                                                             |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1439                                                                           |
| System-Id-Guid    | 38942346-cc5b-424b-a7d8-6ffd12029c5f                                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                       |
