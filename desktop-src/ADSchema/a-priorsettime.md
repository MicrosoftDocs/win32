# Prior-Set-Time attribute

The previous time set for a secret.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Prior-Set-Time                       |
| Ldap-Display-Name | priorSetTime                         |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |

| Attribute-Id      | 1.2.840.113556.1.4.99                |
