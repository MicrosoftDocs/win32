# Time-Vol-Change attribute

This attribute indicates the last time that a file in the remote storage volume was changed.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | Time-Vol-Change                      |
| Ldap-Display-Name | timeVolChange                        |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.502               |
| System-Id-Guid    | ddac0cf0-af8f-11d0-afeb-00c04fd930c9 |
| Syntax            | [**Interval**](s-interval.md)       |
