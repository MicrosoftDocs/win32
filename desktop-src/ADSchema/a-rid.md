# Rid attribute

The relative Identifier of an object.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Rid                                         |
| Ldap-Display-Name | rid                                         |
| Size              | 4 bytes                                     |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | When an object that needs a RID is created. |
| Attribute-Id      | 1.2.840.113556.1.4.153                      |
| System-Id-Guid    | bf967a22-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**Enumeration**](s-enumeration.md)        |
