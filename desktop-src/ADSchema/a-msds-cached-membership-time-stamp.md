# ms-DS-Cached-Membership-Time-Stamp attribute

The **ms-DS-Cached-Membership-Time-Stamp** attribute is used by the Security Accounts Manager for group expansion during token evaluation.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ms-DS-Cached-Membership-Time-Stamp   |
| Ldap-Display-Name | msDS-Cached-Membership-Time-Stamp    |
| Size              | 8 bytes                              |
| Update Privilege  | This value is set by the system.     |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1442              |
| System-Id-Guid    | 3566bf1f-beee-4dcb-8abe-ef89fcfec6c1 |
| Syntax            | [**Interval**](s-interval.md)       |
