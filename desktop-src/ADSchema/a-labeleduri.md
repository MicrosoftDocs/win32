# labeledURI attribute

A Uniform Resource Identifier followed by a label. The label is used to describe the resource to which the URI points, and is intended as a display name that is human-readable.



|                   |                                             |
|-------------------|---------------------------------------------|
| CN                | labeledURI                                  |
| Ldap-Display-Name | labeledURI                                  |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.3.6.1.4.1.250.1.57                        |
| System-Id-Guid    | c569bb46-c680-44bc-a273-e6c227d71b45        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
