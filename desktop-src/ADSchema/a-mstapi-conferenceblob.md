# ms-TAPI-Conference-Blob attribute

A binary BLOB of data that describes various properties of a TAPI multicast conference. Its format and content are determined by the value of the attribute Protocol-Id in the same object. Typically, the data in this BLOB conforms to RFC2327.



| Entry | Value |
|-------------------|--------------------------------------------------------------------------------------------------------------|
| CN                | ms-TAPI-Conference-Blob                                                                                      |
| Ldap-Display-Name | msTAPI-ConferenceBlob                                                                                        |
| Size              | Can be of arbitrary length.                                                                                  |
| Update Privilege  | No special privileges required.                                                                              |
| Update Frequency  | Can change at the discretion of a TAPI conference owner when some data about the conference needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.1700                                                                                      |
| System-Id-Guid    | 4cc4601e-7201-4141-abc8-3e529ae88863                                                                         |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                                                        |
