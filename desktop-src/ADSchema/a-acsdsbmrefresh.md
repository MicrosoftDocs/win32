# ACS-DSBM-Refresh attribute

This attribute contains the interval timer value that determines when the Designated Subnet Bandwidth Manager (DSBM) sends out a refresh message (I\_AM\_DSBM) to all of the Subnet Bandwidth Managers in a domain.



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-Refresh                     |
| Ldap-Display-Name | aCSDSBMRefresh                       |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.777               |
| System-Id-Guid    | 1cb3559f-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Enumeration**](s-enumeration.md) |
