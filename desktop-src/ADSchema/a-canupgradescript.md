# Can-Upgrade-Script attribute

This attribute stores the list of application packages that can be upgraded by or which can upgrade this application package.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | Can-Upgrade-Script                          |
| Ldap-Display-Name | canUpgradeScript                            |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.815                      |
| System-Id-Guid    | d9e18314-8939-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
