# MS-SQL-LastBackupDate attribute

The last date that the database was backed up.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastBackupDate                       |
| Ldap-Display-Name | mS-SQL-LastBackupDate                       |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | When the database is backed up.             |
| Attribute-Id      | 1.2.840.113556.1.4.1398                     |
| System-Id-Guid    | f2b6abca-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
