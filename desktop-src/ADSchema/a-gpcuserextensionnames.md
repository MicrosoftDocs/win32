# GPC-User-Extension-Names attribute

Used by the Group Policy Object for user policies.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-User-Extension-Names                                                        |
| Ldap-Display-Name | gPCUserExtensionNames                                                           |
| Size              | Depends on the number of client-side extensions that have a policy in this GPO. |
| Update Privilege  | Domain or policy administrator.                                                 |
| Update Frequency  | Whenever a GPO is updated through the GPE.                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1349                                                         |
| System-Id-Guid    | 42a75fc6-783f-11d2-9916-0000f87a57d4                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                     |
