# Profile-Path attribute

Specifies a path to the user's profile. This value can be a null string, a local absolute path, or a UNC path.



| Entry | Value |
|-------------------|--------------------------------------------------------------------------|
| CN                | Profile-Path                                                             |
| Ldap-Display-Name | profilePath                                                              |
| Size              | \-                                                                       |
| Update Privilege  | Domain administrator or account owner.                                   |
| Update Frequency  | When the user's record is created and whenever the path needs to change. |
| Attribute-Id      | 1.2.840.113556.1.4.139                                                   |
| System-Id-Guid    | bf967a05-0de6-11d0-a285-00aa003049e2                                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                              |
