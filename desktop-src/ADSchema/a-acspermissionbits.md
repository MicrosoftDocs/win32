# ACS-Permission-Bits attribute

Allows multicast flow origination for a given user.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ACS-Permission-Bits                  |
| Ldap-Display-Name | aCSPermissionBits                    |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.765               |
| System-Id-Guid    | 7f561282-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**Interval**](s-interval.md)       |
