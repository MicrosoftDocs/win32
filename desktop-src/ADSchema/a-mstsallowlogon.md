# ms-TS-Allow-Logon attribute

Specifies whether the user is allowed to log on to the Terminal Server. The value is 1 if logon is allowed, and 0 if logon is not allowed.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | ms-TS-Allow-Logon                    |
| Ldap-Display-Name | msTSAllowLogon                       |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1979              |
| System-Id-Guid    | 3a0cd464-bc54-40e7-93ae-a646a6ecc4b4 |
| Syntax            | [**Boolean**](s-boolean.md)         |
