# MS-DS-Consistency-Guid attribute

This attribute is used to check consistency between the directory and another object, database, or application, by comparing GUIDs.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Consistency-Guid                                |
| Ldap-Display-Name | mS-DS-ConsistencyGuid                                 |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1360                               |
| System-Id-Guid    | 23773dc2-b63a-11d2-90e1-00c04fd91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
