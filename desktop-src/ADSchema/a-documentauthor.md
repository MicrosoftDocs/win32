# documentAuthor attribute

The documentAuthor attribute type specifies the distinguished name of the author of a document.

|-------------------|-----------------------------------------|
| CN                | documentAuthor                          |
| Ldap-Display-Name | documentAuthor                          |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
