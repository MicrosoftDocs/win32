# MS-SQL-Database attribute

The name of the SQL Server database involved in replication.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Database                             |
| Ldap-Display-Name | mS-SQL-Database                             |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | When replication is setup.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1393                     |
| System-Id-Guid    | d5a0dbdc-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
