# DS-Execute-Intentions-Script extended right

Control access right, which should be granted to the partitions container, that allows the Rendom.exe or prepare operation to be used in a domain rename. This control access right also appears as an audit-only right when the Rendom.exe or execute step operation is performed.



| Entry | Value |
|--------------|--------------------------------------|
| CN           | DS-Execute-Intentions-Script         |
| Display-Name | Execute Forest Update Script         |
| Rights-GUID  | 2f16c4a5-b98e-432c-952a-cb388ba33f2e |
