# ms-ds-Schema-Extensions attribute

A binary BLOB used to store information about extensions to schema objects.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | ms-ds-Schema-Extensions                               |
| Ldap-Display-Name | msDs-Schema-Extensions                                |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | When the schema is extended.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1440                               |
| System-Id-Guid    | b39a61be-ed07-4cab-9a4a-4963ed0141e1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
