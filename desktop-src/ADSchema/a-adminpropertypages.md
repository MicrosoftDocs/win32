# Admin-Property-Pages attribute

The order number and GUID of the property pages for an object to be displayed on Active Directory administration screens. For more information see the document, Extending the User Interface for Directory Objects.



| Entry | Value |
|-------------------|------------------------------------------------|
| CN                | Admin-Property-Pages                           |
| Ldap-Display-Name | adminPropertyPages                             |
| Size              | \-                                             |
| Update Privilege  | Domain administrator or application developer. |
| Update Frequency  | Whenever a property page is added or removed.  |
| Attribute-Id      | 1.2.840.113556.1.4.562                         |
| System-Id-Guid    | 52458038-ca6a-11d0-afff-0000f80367c1           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
