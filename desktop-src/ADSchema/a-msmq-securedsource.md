# MSMQ-Secured-Source attribute

This is part of an MSMQ object, it is set by calling API to MQCreateQueue or MQSetProperties. It controls whether MSMQ accepts messages only from a secured source (for example, https) for this queue.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | MSMQ-Secured-Source                  |
| Ldap-Display-Name | MSMQ-SecuredSource                   |
| Size              | 4 bytes                              |
| Update Privilege  | The queue owner.                     |
| Update Frequency  | When a queue is created.             |
| Attribute-Id      | 1.2.840.113556.1.4.1713              |
| System-Id-Guid    | 8bf0221b-7a06-4d63-91f0-1499941813d3 |
| Syntax            | [**Boolean**](s-boolean.md)         |
