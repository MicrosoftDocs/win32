# DS-Replication-Monitor-Topology extended right

Extended control access right that allows the reading of replication monitoring data, such as replication status and object metadata.



| Entry | Value |
|--------------|--------------------------------------|
| CN           | DS-Replication-Monitor-Topology      |
| Display-Name | Monitor Active Directory Replication |
| Rights-GUID  | f98340fb-7c5b-4cdb-a00b-2ebdfa115a96 |
