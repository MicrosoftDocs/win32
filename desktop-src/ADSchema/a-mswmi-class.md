# ms-WMI-Class attribute

The name of a WMI Class object in an associated encoding (for example, Win32\_ComputerSystem). The encoding can be either an instance or class object.



| Entry | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-WMI-Class                                                    |
| Ldap-Display-Name | msWMI-Class                                                     |
| Size              | \-                                                              |
| Update Privilege  | Domain administrator                                            |
| Update Frequency  | Whenever a class containing the attribute is added or modified. |
| Attribute-Id      | 1.2.840.113556.1.4.1676                                         |
| System-Id-Guid    | 90c1925f-4a24-4b07-b202-be32eb3c8b74                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                     |
