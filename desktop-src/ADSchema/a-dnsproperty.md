# DNS-Property attribute

Used to store binary settings (properties) on DNS zone objects.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| Ldap-Display-Name | dNSProperty                                           |
| Size              | \-                                                    |
| Update Privilege  | This value is set by the system.                      |
| Update Frequency  | Whenever DNS zone records change.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| System-Id-Guid    | 675a15fe-3b70-11d2-90cc-00c04fd91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
