# ANR attribute

Ambiguous name resolution attribute to be used when choosing between objects.



| Entry | Value |
|-------------------|-------------------------------------------------------------------|
| CN                | ANR                                                               |
| Ldap-Display-Name | aNR                                                               |
| Size              | \-                                                                |
| Update Privilege  | Schema administrator                                              |
| Update Frequency  | When an attribute needs to be added or removed from the ANR list. |
| Attribute-Id      | 1.2.840.113556.1.4.1208                                           |
| System-Id-Guid    | 45b01500-c419-11d1-bbc9-0080c76670c0                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                       |
