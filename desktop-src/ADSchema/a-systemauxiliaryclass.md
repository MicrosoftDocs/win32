# System-Auxiliary-Class attribute

A list of auxiliary classes that cannot be modified by the user.



| Entry | Value |
|-------------------|--------------------------------------------------------------------|
| CN                | System-Auxiliary-Class                                             |
| Ldap-Display-Name | systemAuxiliaryClass                                               |
| Size              | \-                                                                 |
| Update Privilege  | Schema administrator                                               |
| Update Frequency  | When the class is created or a new auxiliary class is added to it. |
| Attribute-Id      | 1.2.840.113556.1.4.198                                             |
| System-Id-Guid    | bf967a43-0de6-11d0-a285-00aa003049e2                               |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)    |
