# netboot-SCP-BL attribute

A list of service connection points that reference this NetBoot server.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | netboot-SCP-BL                          |
| Ldap-Display-Name | netbootSCPBL                            |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.864                  |
| System-Id-Guid    | 07383082-91df-11d1-aebc-0000f80367c1    |
