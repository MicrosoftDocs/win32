# MS-SQL-Description attribute

The description of the SQL Server database.



| Entry | Value |
|-------------------|--------------------------------------------------|
| CN                | MS-SQL-Description                               |
| Ldap-Display-Name | mS-SQL-Description                               |
| Size              | \-                                               |
| Update Privilege  | Domain administrator                             |
| Update Frequency  | When the user decides to change the description. |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| System-Id-Guid    | 8386603c-ccef-11d2-9993-0000f87a57d4             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)      |
