# documentPublisher attribute

The documentPublisher attribute is the person or organization that published a document.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | documentPublisher                           |
| Ldap-Display-Name | documentPublisher                           |
| Size              | \-                                          |
| Update Privilege  | \-                                          |

| Attribute-Id      | 0.9.2342.19200300.100.1.56                  |
