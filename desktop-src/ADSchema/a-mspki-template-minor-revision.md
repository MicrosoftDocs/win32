# ms-PKI-Template-Minor-Revision attribute

Keeps track of attributes in the class changing. However, this will not trigger auto-enrollment.



|                   |                                                                                                   |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Template-Minor-Revision                                                                    |
| Ldap-Display-Name | msPKI-Template-Minor-Revision                                                                     |
| Size              | 4 bytes                                                                                           |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| System-Id-Guid    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
