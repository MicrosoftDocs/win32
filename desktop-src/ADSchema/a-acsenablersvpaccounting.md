# ACS-Enable-RSVP-Accounting attribute

True if RSVP accounting is enabled.

| CN                | ACS-Enable-RSVP-Accounting           |
| Ldap-Display-Name | aCSEnableRSVPAccounting              |
| Size              | 4 bytes                              |
| Update Privilege  | \-                                   |
