# ms-WMI-ID attribute

A unique ID for an object instance. Indexed in the Global Catalog.



|                   |                                             |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-ID                                   |
| Ldap-Display-Name | msWMI-ID                                    |
| Size              | \-                                          |
| Update Privilege  | This value is set by the system.            |
| Update Frequency  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1627                     |
| System-Id-Guid    | 9339a803-94b8-47f7-9123-a853b9ff7e45        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
