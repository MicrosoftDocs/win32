# MS-SQL-Publisher attribute

The name of the publisher database for replication.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Publisher                            |
| Ldap-Display-Name | mS-SQL-Publisher                            |
| Size              | \-                                          |
| Update Privilege  | Domain administrator                        |
| Update Frequency  | When replication is setup.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| System-Id-Guid    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
