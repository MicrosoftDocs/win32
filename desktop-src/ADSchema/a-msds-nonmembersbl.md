# ms-DS-Non-Members-BL attribute

Backward link from non-member group or user to Az groups that link to it (same functionality as Non-Security-Member-BL).



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Non-Members-BL                    |
| Ldap-Display-Name | msDS-NonMembersBL                       |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
| Update Frequency  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1794                 |
| System-Id-Guid    | 2a8c68fc-3a7a-4e87-8720-fe77c51cbe74    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
