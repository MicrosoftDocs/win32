# DS-UI-Admin-Maximum attribute

This is the default maximum number of objects to be shown in a container by the admin UI.



| Entry | Value |
|-------------------|--------------------------------------|
| CN                | DS-UI-Admin-Maximum                  |
| Ldap-Display-Name | dSUIAdminMaximum                     |
| Size              | \-                                   |
| Update Privilege  | Domain Administrator                 |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1344              |
