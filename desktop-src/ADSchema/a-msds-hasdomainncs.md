# ms-DS-Has-Domain-NCs attribute

Directory service replication information that details the domain naming contexts present on a particular server.



| Entry | Value |
|-------------------|-----------------------------------------------------|
| CN                | ms-DS-Has-Domain-NCs                                |
| Ldap-Display-Name | msDS-HasDomainNCs                                   |
| Size              | \-                                                  |
| Update Privilege  | This value is set by the system                     |
| Update Frequency  | It is set by DCPromo and never modified thereafter. |
| Attribute-Id      | 1.2.840.113556.1.4.1820                             |
| System-Id-Guid    | 6f17e347-a842-4498-b8b3-15e007da4fed                |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)             |
