# ms-TS-Initial-Program attribute

Terminal Services Session Initial Program specifies the path and file name of the application that the user wants to start automatically when the user logs on to the Terminal Server.



| Entry | Value |
|-------------------|---------------------------------------------|
| CN                | ms-TS-Initial-Program                       |
| Ldap-Display-Name | msTSInitialProgram                          |
| Size              | \-                                          |
| Update Privilege  | \-                                          |
| Update Frequency  | \-                                          |

| System-Id-Guid    | 9201ac6f-1d69-4dfb-802e-d95510109599        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
