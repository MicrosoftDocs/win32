# Reps-To attribute

Lists the servers that the directory will notify of changes and servers to which the directory will send changes on Request for the defined naming context.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Reps-To                                               |
| Ldap-Display-Name | repsTo                                                |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.83                                 |
| System-Id-Guid    | bf967a1e-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
