# Upgrade-Product-Code attribute

This attribute contains the product code of other packages, such as applications, that can be upgraded by this package, or that can upgrade this package.



| Entry | Value |
|-------------------|-------------------------------------------------------|
| CN                | Upgrade-Product-Code                                  |
| Ldap-Display-Name | upgradeProductCode                                    |
| Size              | \-                                                    |
| Update Privilege  | \-                                                    |
| Update Frequency  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| System-Id-Guid    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
