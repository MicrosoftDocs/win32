# ms-PKI-Private-Key-Flag attribute

Contains the private key related flags.



| Entry | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Private-Key-Flag                                                                           |
| Ldap-Display-Name | msPKI-Private-Key-Flag                                                                            |
| Size              | 4 bytes                                                                                           |
| Update Privilege  | Domain administrator                                                                              |
| Update Frequency  | When the certificate template (ms-PKI-Certificate-Template) object is edited, created, or cloned. |
| Attribute-Id      | 1.2.840.113556.1.4.1431                                                                           |
| System-Id-Guid    | bab04ac2-0435-4709-9307-28380e7c7001                                                              |
| Syntax            | [**Enumeration**](s-enumeration.md)                                                              |
