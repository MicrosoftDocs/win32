# Owner attribute

The distinguished name of an object that has ownership of an object.



| Entry | Value |
|-------------------|-----------------------------------------|
| CN                | Owner                                   |
| Ldap-Display-Name | owner                                   |
| Size              | \-                                      |
| Update Privilege  | \-                                      |
| Update Frequency  | \-                                      |
| Attribute-Id      | 2.5.4.32                                |
| System-Id-Guid    | bf9679f3-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
