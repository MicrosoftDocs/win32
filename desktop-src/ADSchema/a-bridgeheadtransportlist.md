# Bridgehead-Transport-List attribute

|-------------------|-----------------------------------------|
| CN                | Bridgehead-Transport-List               |
| Ldap-Display-Name | bridgeheadTransportList                 |
| Size              | \-                                      |
| Update Privilege  | This value is set by the system.        |
