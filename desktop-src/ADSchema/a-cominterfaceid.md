# COM-InterfaceID attribute

This attribute stores the list of interfaces that are implemented in this application package.



| Entry | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-InterfaceID                                                                  |
| Ldap-Display-Name | cOMInterfaceID                                                                   |
| Size              | \-                                                                               |
| Update Privilege  | Anyone may update this object based on the security of the object being created. |
| Update Frequency  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.20                                                            |
| System-Id-Guid    | bf96793c-0de6-11d0-a285-00aa003049e2                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
