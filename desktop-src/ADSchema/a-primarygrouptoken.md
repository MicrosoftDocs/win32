# Primary-Group-Token attribute

A computed attribute that is used in retrieving the membership list of a group, such as Domain Users. The complete membership of such groups is not stored explicitly for scaling reasons.



| Entry | Value |
|-------------------|--------------------------------------------|
| CN                | Primary-Group-Token                        |
| Ldap-Display-Name | primaryGroupToken                          |
| Size              | 4 bytes                                    |
| Update Privilege  | This value is set by the system.           |
| Update Frequency  | Whenever an objects primary group changes. |
| Attribute-Id      | 1.2.840.113556.1.4.1412                    |
| System-Id-Guid    | c0ed8738-7efd-4481-84d9-66d2db8be369       |
| Syntax            | [**Enumeration**](s-enumeration.md)       |
