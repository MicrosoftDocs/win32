# HCRYPTPROV\_LEGACY

The **HCRYPTPROV\_LEGACY** data type is used to replace the [**HCRYPTPROV**](hcryptprov.md) data type where the **HCRYPTPROV** data type is no longer used. Function parameters and data structures that are of type **HCRYPTPROV\_LEGACY** should be set to **NULL**.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
