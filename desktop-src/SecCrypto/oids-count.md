# OIDs.Count property

\[The **Count** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**OidCollection Class**](/dotnet/api/system.security.cryptography.oidcollection) in the [**System.Security.Cryptography**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true) namespace.\]

The **Count** property retrieves the number of OIDs in the collection.

## Syntax


```VB
OIDs.Count As Long
```



## Property value

Number of [**OID**](oid.md) objects in the collection.
