# PrivateKey.ContainerName property

\[The **ContainerName** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**X509Certificate2.PrivateKey Property**](/dotnet/api/system.security.cryptography.x509certificates.x509certificate2.privatekey) in the [**System.Security.Cryptography.X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor) namespace.\]

The **ContainerName** property retrieves the private key container name. This is the default property.

## Syntax


```VB
PrivateKey.ContainerName As String
```



## Property value

A string that contains the private key container name.
