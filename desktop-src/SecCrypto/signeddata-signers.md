# SignedData.Signers property

\[The **Signers** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**SignedCms Class**](/dotnet/api/system.security.cryptography.pkcs.signedcms?view=dotnet-plat-ext-3.1&preserve-view=true) in the [**System.Security.Cryptography.Pkcs**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) namespace.\]

The **Signers** property retrieves the signature creators of the data.

## Syntax


```VB
SignedData.Signers As Signers
```



## Property value

The [**Signers**](signers.md) collection that represents the signature creators of the data.
