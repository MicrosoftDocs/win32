# Extension.OID property

\[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows Server 2008, Windows Vista, and Windows XP. Instead, use the [**X509Extension Class**](https://msdn.microsoft.com/library/x5x51x86(v=VS.100).aspx) in the [**System.Security.Cryptography.X509Certificates**](https://msdn.microsoft.com/library/73091bzx(v=VS.71).aspx) namespace.\]

The **OID** property retrieves the object identifier for the extension. This is the default property.

## Syntax


```VB
Extension.OID As OID
```



## Property value

An [**OID**](oid.md) object that represents the object identifier of the extension.
