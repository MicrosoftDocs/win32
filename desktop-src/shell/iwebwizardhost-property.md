# WebWizardHost.Property property

Sets or retrieves a property's current value.

This property is read/write.

## Syntax


```JScript
Property = WebWizardHost.Property(bstrPropertyName)
WebWizardHost.Property = Property
```



## Property value

A variable of type [**BSTR**](/previous-versions/windows/desktop/automat/bstr) that specifies or receives the property.
