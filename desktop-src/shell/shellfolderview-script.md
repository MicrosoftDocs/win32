# ShellFolderView.Script property

\[This property is not supported in Windows XP or later.\]

Contains the scripting object for the view.

This property is read-only.

## Syntax


```JScript
objScript = ShellFolderView.Script
```



## Property value

A variable of type [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) that receives the scripting object.
