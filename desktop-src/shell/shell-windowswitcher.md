# Shell::WindowSwitcher method

Displays your open windows in a 3D stack that you can flip through.

## Syntax


```JScript
.WindowSwitcher()
```


```VB

Sub WindowSwitcher()
```





## Parameters

This method has no parameters.

## Return value

### JScript

This method does not return a value.

### VB

This method does not return a value.
