# Authorization Interfaces

The following interfaces are used with authorization applications.

## In this section



| Topic                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAzApplication**](iazapplication.md)<br/>                   | Defines an installed instance of an application. An [**IAzApplication**](iazapplication.md) object is created when an application is installed.<br/>                                                                                                                                                                                                                                                                                                                                           |
| [**IAzApplication2**](iazapplication2.md)<br/>                 | Inherits from the [**IAzApplication**](iazapplication.md) interface and implements additional methods to initialize [**IAzClientContext2**](iazclientcontext2.md) objects.<br/>                                                                                                                                                                                                                                                                                                               |
| [**IAzApplication3**](iazapplication3.md)<br/>                 | Provides methods to manage [**IAzRoleAssignment**](iazroleassignment.md), [**IAzRoleDefinition**](iazroledefinition.md), and [**IAzScope2**](iazscope2.md) objects.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**IAzApplications**](iazapplications.md)<br/>                 | Represents a collection of [**IAzApplication**](iazapplication.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IAzApplicationGroup**](iazapplicationgroup.md)<br/>         | Defines a collection of principals.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IAzApplicationGroup2**](iazapplicationgroup2.md)<br/>       | Extends the [**IAzApplicationGroup**](iazapplicationgroup.md) interface by adding support for the **BizRule** group type.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzApplicationGroups**](iazapplicationgroups.md)<br/>       | Represents a collection of [**IAzApplicationGroup**](iazapplicationgroup.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IAzAuthorizationStore**](azauthorizationstore.md)<br/>      | Defines the container that is the root of the authorization policy store.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IAzAuthorizationStore2**](iazauthorizationstore2.md)<br/>   | Inherits from the [**AzAuthorizationStore**](azauthorizationstore.md) object and implements methods to create and open [**IAzApplication2**](iazapplication2.md) objects.<br/>                                                                                                                                                                                                                                                                                                                |
| [**IAzAuthorizationStore3**](iazauthorizationstore3.md)<br/>   | Extends the [**IAzAuthorizationStore2**](iazauthorizationstore2.md) interface with methods that manage business rule (BizRule) support and caching.<br/>                                                                                                                                                                                                                                                                                                                                       |
| [**IAzBizRuleContext**](azbizrulecontext.md)<br/>              | Contains information about a Business Rule (BizRule) operation.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IAzBizRuleInterfaces**](iazbizruleinterfaces.md)<br/>       | Provides methods and properties used to manage a list of [**IDispatch**](ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interfaces that can be called by business rule (BizRule) scripts.<br/>                                                                                                                                                                                                                                                                                                           |
| [**IAzBizRuleParameters**](iazbizruleparameters.md)<br/>       | Provides methods and properties used to manage a list of parameters that can be passed to business rule (BizRule) scripts.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzClientContext**](iazclientcontext.md)<br/>               | Maintains the state that describes a particular client.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**IAzClientContext2**](iazclientcontext2.md)<br/>             | Inherits from the [**IAzClientContext**](iazclientcontext.md) interface and implements new methods that manipulate the client context.<br/>                                                                                                                                                                                                                                                                                                                                                    |
| [**IAzClientContext3**](iazclientcontext3.md)<br/>             | Extends the [**IAzClientContext2**](iazclientcontext2.md) interface.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzNameResolver**](iaznameresolver.md)<br/>                 | Translates [security identifiers](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-identifier-gly) (SIDs) into principal display names.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**IAzObjectPicker**](iazobjectpicker.md)<br/>                 | Displays a dialog box that allows users to select one or more principals from a list.<br/>                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzOperation**](iazoperation.md)<br/>                       | Defines a low-level operation supported by an application.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzOperation2**](iazoperation2.md)<br/>                     | Extends the [**IAzOperation**](iazoperation.md) with a method that returns the role assignments associated with the operation.<br/>                                                                                                                                                                                                                                                                                                                                                            |
| [**IAzOperations**](iazoperations.md)<br/>                     | Represents a collection of [**IAzOperation**](iazoperation.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IAzPrincipalLocator**](iazprincipallocator.md)<br/>         | Locates and chooses Active Directory Application Mode (ADAM) principals in Authorization Manager.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IAzRole**](iazrole.md)<br/>                                 | Defines the set of operations that can be performed by a set of users within a scope.<br/>                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzRoleAssignment**](iazroleassignment.md)<br/>             | Represents a role to which users and groups can be assigned.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IAzRoleAssignments**](iazroleassignments.md)<br/>           | Represents a collection of [**IAzRoleAssignment**](iazroleassignment.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IAzRoleDefinition**](iazroledefinition.md)<br/>             | Represents one or more [**IAzRoleDefinition**](iazroledefinition.md), [**IAzTask**](iaztask.md), and [**IAzOperation**](iazoperation.md) objects that specify a set of operations.<br/>                                                                                                                                                                                                                                                                                                      |
| [**IAzRoleDefinitions**](iazroledefinitions.md)<br/>           | Represents a collection of [**IAzRoleDefinition**](iazroledefinition.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IAzRoles**](iazroles.md)<br/>                               | Represents a collection of [**IAzRole**](iazrole.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IAzScope**](iazscope.md)<br/>                               | Defines a logical container of resources to which the application manages access.<br/>                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IAzScope2**](iazscope2.md)<br/>                             | Extends the [**IAzScope**](iazscope.md) interface to manage [**IAzRoleAssignment**](iazroleassignment.md) and [**IAzRoleDefinition**](iazroledefinition.md) objects.<br/>                                                                                                                                                                                                                                                                                                                    |
| [**IAzScopes**](iazscopes.md)<br/>                             | Represents a collection of [**IAzScope**](iazscope.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IAzTask**](iaztask.md)<br/>                                 | Describes a set of operations.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IAzTask2**](iaztask2.md)<br/>                               | Extends the [**IAzTask**](iaztask.md) interface with a method that returns the role assignments associated with the task.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzTasks**](iaztasks.md)<br/>                               | Represents a collection of [**IAzTask**](iaztask.md) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IeAxiService**](ieaxiservice.md)<br/>                       | Initializes a system service object to install an ActiveX object when the current user does not have permission to install the object.<br/>                                                                                                                                                                                                                                                                                                                                                     |
| [**IeAxiServiceCallback**](ieaxiservicecallback.md)<br/>       | Called by the [**IeAxiSystemInstaller**](ieaxisysteminstaller.md) interface to verify that an ActiveX object can be installed.<br/>                                                                                                                                                                                                                                                                                                                                                            |
| [**IeAxiSystemInstaller**](ieaxisysteminstaller.md)<br/>       | Installs an ActiveX object.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IEffectivePermission**](ieffectivepermission.md)<br/>       | Provides a means to determine effective permission for a security principal on an object.<br/>                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IEffectivePermission2**](ieffectivepermission2.md)<br/>     | Provides a way to determine effective permission for a security principal on an object.<br/>                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISecurityInformation**](isecurityinformation.md)<br/>       | Enables the access control editor to communicate with the caller of the [**CreateSecurityPage**](createsecuritypage.md) and [**EditSecurity**](editsecurity.md) functions.<br/>                                                                                                                                                                                                                                                                                                               |
| [**ISecurityInformation2**](isecurityinformation2.md)<br/>     | Enables the access control editor to obtain information from the client that is not provided by the [**ISecurityInformation**](isecurityinformation.md) interface.<br/>                                                                                                                                                                                                                                                                                                                        |
| [**ISecurityInformation3**](isecurityinformation3.md)<br/>     | Provides methods necessary for displaying an elevated access control editor when a user clicks the **Edit** button on an access control editor page that displays an image of a shield on that **Edit** button.<br/>                                                                                                                                                                                                                                                                            |
| [**ISecurityInformation4**](isecurityinformation4.md)<br/>     | Enables the access control editor (ACE) to obtain the share's security descriptor to initialize the share page.<br/>                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISecurityObjectTypeInfo**](isecurityobjecttypeinfo.md)<br/> | Provides a means of determining the source of inherited [access control entries](https://msdn.microsoft.com/library/windows/desktop/ms721532#-security-access-control-entry-gly) (ACEs) in [discretionary access control lists](https://msdn.microsoft.com/library/windows/desktop/ms721573#-security-discretionary-access-control-list-gly) (DACLs) and [system access control lists](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-system-access-control-list-gly) (SACLs).<br/> |



 

Authorization interfaces are categorized according to usage as follows:

-   [Access Control Editor Interfaces](#access-control-editor-interfaces)
-   [ActiveX Installer Interfaces](#activex-installer-interfaces)
-   [Authorization Manager Interfaces](#authorization-manager-interfaces)
