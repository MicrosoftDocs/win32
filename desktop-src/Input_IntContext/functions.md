# Interaction Context Functions

The topics in this section provide the reference specifications for [Interaction Context](interaction-context-portal.md) functions.

## In this section



| Topic                                                                                                             | Description                                                                                                                                                                                                                                                                                                                                                       |
|-------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddPointerInteractionContext**](addpointerinteractioncontext.md)<br/>                                   | Include the specified pointer in the set of pointers processed by the [Interaction Context](interaction-context-portal.md) object. <br/>                                                                                                                                                                                                                   |
| [**BufferPointerPacketsInteractionContext**](bufferpointerpacketsinteractioncontext.md)<br/>               | Adds the history for a single input pointer to the buffer of the [Interaction Context](interaction-context-portal.md) object.<br/>                                                                                                                                                                                                                         |
| [**CreateInteractionContext**](createinteractioncontext.md)<br/>                                           | Creates and initializes an [Interaction Context](interaction-context-portal.md) object.<br/>                                                                                                                                                                                                                                                               |
| [**DestroyInteractionContext**](destroyinteractioncontext.md)<br/>                                         | Destroys the specified [Interaction Context](interaction-context-portal.md) object.<br/>                                                                                                                                                                                                                                                                   |
| [**GetCrossSlideParameterInteractionContext**](getcrossslideparameterinteractioncontext.md)<br/>           | Gets the cross-slide interaction behavior. <br/>                                                                                                                                                                                                                                                                                                            |
| [**GetInertiaParameterInteractionContext**](getinertiaparameterinteractioncontext.md)<br/>                 | Gets the inertia behavior of a manipulation (translation, rotation, scaling). <br/>                                                                                                                                                                                                                                                                         |
| [**GetInteractionConfigurationInteractionContext**](getinteractionconfigurationinteractioncontext.md)<br/> | Gets interaction configuration state for the [Interaction Context](interaction-context-portal.md) object.<br/>                                                                                                                                                                                                                                             |
| [**GetMouseWheelParameterInteractionContext**](getmousewheelparameterinteractioncontext.md)<br/>           | Gets the mouse wheel state for the [Interaction Context](interaction-context-portal.md) object. <br/>                                                                                                                                                                                                                                                      |
| [**GetPropertyInteractionContext**](getpropertyinteractioncontext.md)<br/>                                 | Gets [Interaction Context](interaction-context-portal.md) object properties.<br/>                                                                                                                                                                                                                                                                          |
| [**GetStateInteractionContext**](getstateinteractioncontext.md)<br/>                                       | Gets current [Interaction Context](interaction-context-portal.md) state and the time when the context will return to idle state. <br/>                                                                                                                                                                                                                     |
| [**ProcessBufferedPacketsInteractionContext**](processbufferedpacketsinteractioncontext.md)<br/>           | Process buffered packets at the end of a pointer input frame.<br/>                                                                                                                                                                                                                                                                                          |
| [**ProcessInertiaInteractionContext**](processinertiainteractioncontext.md)<br/>                           | Sends timer input to the [Interaction Context](interaction-context-portal.md) object for inertia processing.<br/>                                                                                                                                                                                                                                          |
| [**ProcessPointerFramesInteractionContext**](processpointerframesinteractioncontext.md)<br/>               | Processes a set of pointer input frames.<br/>                                                                                                                                                                                                                                                                                                               |
| [**RegisterOutputCallbackInteractionContext**](registeroutputcallbackinteractioncontext.md)<br/>           | Registers a callback to receive interaction events from an [Interaction Context](interaction-context-portal.md) object.<br/>                                                                                                                                                                                                                               |
| [**RemovePointerInteractionContext**](removepointerinteractioncontext.md)<br/>                             | Remove the specified pointer from the set of pointers processed by the [Interaction Context](interaction-context-portal.md) object. <br/>                                                                                                                                                                                                                  |
| [**ResetInteractionContext**](resetinteractioncontext.md)<br/>                                             | Resets the [**interaction state**](interaction-state.md), interaction configuration settings, and all parameters to their initial state. Current interactions are cancelled without notifications. <br/> [Interaction Context](interaction-context-portal.md) must be reconfigured before next use.<br/>                                            |
| [**SetCrossSlideParametersInteractionContext**](setcrossslideparametersinteractioncontext.md)<br/>         | Configures the cross-slide interaction. <br/>                                                                                                                                                                                                                                                                                                               |
| [**SetInertiaParameterInteractionContext**](setinertiaparameterinteractioncontext.md)<br/>                 | Configures the inertia behavior of a manipulation (translation, rotation, scaling) after the contact is lifted. <br/>                                                                                                                                                                                                                                       |
| [**SetInteractionConfigurationInteractionContext**](setinteractionconfigurationinteractioncontext.md)<br/> | Configures the [Interaction Context](interaction-context-portal.md) object to process the specified manipulations.<br/>                                                                                                                                                                                                                                    |
| [**SetMouseWheelParameterInteractionContext**](setmousewheelparameterinteractioncontext.md)<br/>           | Sets the parameter values for mouse wheel input. <br/>                                                                                                                                                                                                                                                                                                      |
| [**SetPivotInteractionContext**](setpivotinteractioncontext.md)<br/>                                       | Sets the center point, and the pivot radius from the center point, for a rotation manipulation using a single input pointer. <br/>                                                                                                                                                                                                                          |
| [**SetPropertyInteractionContext**](setpropertyinteractioncontext.md)<br/>                                 | Sets [Interaction Context](interaction-context-portal.md) object properties.<br/>                                                                                                                                                                                                                                                                          |
| [**StopInteractionContext**](stopinteractioncontext.md)<br/>                                               | Sets the [**interaction state**](interaction-state.md) to INTERACTION\_STATE\_IDLE and leaves all interaction configuration settings and parameters intact. Current interactions are cancelled and notifications sent as required.<br/> [Interaction Context](interaction-context-portal.md) does not have to be reconfigured before next use.<br/> |
