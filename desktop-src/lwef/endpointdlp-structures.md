# Endpoint data loss prevention structures

Structures used by the endpoint data loss prevention feature.

| Structures | Description |
|------------|-------------|
| [AuthGroupOverrideRequest](endpointdlp-authgroupoverriderequest.md) | Structure containing the parameters for requesting authorized group overrides for an enforcement operation. |
| [DLP_APP_OP_ENLIGHTENED_LEVEL](endpointdlp-dlp_app_op_enlightened_level.md) | Associates an endpoint Data Loss Prevention (DLP) action with an enforcement level. |
| [DLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md)                       | Specifies information about a document that is associated with an endpoint DLP operation. |
| [DLP_PRINT_INFO](endpointdlp-dlp_print_info.md)                             | Specifies information about a document that is associated with an endpoint DLP print operation. |
| [DlpRequest](endpointdlp-dlprequest.md)                                     | Specifies a request for Endpoint DLP to handle. |
| [DlpSerializedBuffer](endpointdlp-dlpserializedbuffer.md)                   | Specifies a dynamically allocated string from an EndpointDlp API. |
| [GetFileApplicationAccessRequest](endpointdlp-getfileapplicationaccessrequest.md) | Structure containing the parameters for requesting a file's application access. |
| [GetFileCloudApplicationPolicyRequest](endpointdlp-getfilecloudapplicationpolicyrequest.md) | Structure containing the parameters for requesting a file's cloud application policy. |
| [GetNotificationSettingRequest](endpointdlp-getnotificationsettingrequest.md) | Structure containing the parameters getting notification settings for an enforcement action. |
| [SendFileAccessEventRequest](endpointdlp-sendfileaccesseventrequest.md) | Structure containing the parameters for sending file access events. |
| [SendOperationEnforcementEventRequest](endpointdlp-sendoperationenforcementeventrequest.md) | Structure containing the parameters for sending operation enforcement events. |
