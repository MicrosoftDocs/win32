# Visible Property (CommandsWindow Object)

\[Microsoft Agent is deprecated as of Windows 7, and may be unavailable in subsequent versions of Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Description**
</dt> <dd>

Returns or sets whether the Voice Commands Window is visible (open).

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax** 
</dt> <dd>

*agent***.CommandsWindow.Visible** \[ = *boolean*\]



| Part      | Description                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | A Boolean expression specifying whether the Voice Commands Window is visible.<br/> **True** The window is visible.<br/> **False** The window is hidden (closed).<br/> |



 

</dd> </dl>

## Remarks

The server displays the window based on the location and size set by the user.

> [!Note]  
> The user can override this property.
