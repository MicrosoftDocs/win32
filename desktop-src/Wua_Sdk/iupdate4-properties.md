# IUpdate4 Properties

The [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4) interface defines the following property.



| Property                            | Description                                                            |
|-------------------------------------|------------------------------------------------------------------------|
| [**PerUser**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser) | Gets a Boolean value that indicates whether this is a per-user update. |



 

For information about the members inherited by this interface, see the following interfaces:

-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
