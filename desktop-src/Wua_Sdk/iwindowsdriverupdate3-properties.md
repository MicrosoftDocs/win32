# IWindowsDriverUpdate3 Properties

The **IWindowsDriverUpdate3** interface defines the following property.



| Property                                               | Description                                                                                                             |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate3-get_browseonly) | Gets a Boolean value that indicates whether an update can be discovered only by browsing through the available updates. |



 

For information about the members inherited by this interface, see the following interface.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
