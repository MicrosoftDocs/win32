# IInstallationProgressChangedCallback Methods

The [**IInstallationProgressChangedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallback) interface defines the following method.



| Method                                                        | Description                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Invoke**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallback-invoke) | Handles the notification of the change of progress of an asynchronous installation or uninstallation that was initiated by a call to the [**IUpdateInstaller.BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall) or [**IUpdateInstaller.BeginUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) method. |
