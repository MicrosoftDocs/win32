# IWebProxy Methods

The [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) interface defines the following methods.



| Method                                                                         | Description                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**PromptForCredentials**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Prompts the user for the password to use for proxy authentication.  |
| [**PromptForCredentialsFromHwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Prompts the user for the password to use for proxy authentication.  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Sets the password to submit to the proxy server for authentication. |
