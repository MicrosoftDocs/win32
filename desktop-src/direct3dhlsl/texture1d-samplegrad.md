# Texture1D::SampleGrad methods

Samples a [**Texture1D**](sm5-object-texture1d.md) using a gradient to influence the way the sample location is calculated.

### Overload list



| Method                                                                                                       | Description                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad(S,float,float,float,int)**](dx-graphics-hlsl-to-samplegrad.md)                                | Samples a texture, using a gradient to influence the way the sample location is calculated.<br/>                                                                                                             |
| [**SampleGrad(S,float,float,float,int,float)**](t1d-samplegrad-s-float-float-float-int-float-.md)           | Samples a texture, using a gradient to influence the way the sample location is calculated, with an optional value to clamp sample level-of-detail (LOD) values to.<br/>                                     |
| [**SampleGrad(S,float,float,float,int,float,uint)**](t1d-samplegrad-s-float-float-float-int-float-uint-.md) | Samples a texture, using a gradient to influence the way the sample location is calculated, with an optional value to clamp sample level-of-detail (LOD) values to. Returns status about the operation.<br/> |
