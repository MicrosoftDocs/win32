# Texture1D::Sample methods

Samples a [**Texture1D**](sm5-object-texture1d.md).

### Overload list



| Method                                                                       | Description                                                                                                                                  |
|:-----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sample(S,float,int)**](dx-graphics-hlsl-to-sample.md)                    | Samples a texture.<br/>                                                                                                                |
| [**Sample(S,float,int,float)**](t1d-sample-s-float-int-float-.md)           | Samples a texture with an optional value to clamp sample level-of-detail (LOD) values to.<br/>                                         |
| [**Sample(S,float,int,float,uint)**](t1d-sample-s-float-int-float-uint-.md) | Samples a texture with an optional value to clamp sample level-of-detail (LOD) values to, and returns status about the operation.<br/> |
