# D3DX\_SRGB\_to\_FLOAT function

Converts an SRGB value to FLOAT.

## Syntax

``` syntax
FLOAT D3DX_SRGB_to_FLOAT(
   UINT val
);
```

## Parameters

<dl> <dt>

*val* 
</dt> <dd>

SRGB value to convert.

</dd> </dl>

## Return value

The converted SRGB value.
