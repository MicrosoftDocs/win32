# Texture2DArray::SampleCmpLevelZero methods

Samples a [**Texture2DArray**](sm5-object-texture2darray.md) on mipmap level 0 only, using a comparison value to reject samples.

### Overload list



| Method                                                                                                    | Description                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero(S,float,float,int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Samples a texture on mipmap level 0 only and compares the result to a comparison value.<br/>                                     |
| [**SampleCmpLevelZero(S,float,float,int,uint)**](t2darray-samplecmplevelzero-s-float-float-int-uint-.md) | Samples a texture on mipmap level 0 only and compares the result to a comparison value. Returns status about the operation.<br/> |
