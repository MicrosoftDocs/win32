# dcl\_hs\_join\_phase\_instance\_count (sm5 - asm)

Declare the join phase instance count in a hull shader join phase.



| dcl\_hs\_join\_phase\_instance\_count {1... max 32-bit UINT} |
|--------------------------------------------------------------|



 



| Item                                                   | Description                           |
|--------------------------------------------------------|---------------------------------------|
| <span id="N"></span><span id="n"></span>*N*<br/> | \[in\] The instance count.<br/> |



 

## Remarks

This instruction applies to the following shader stages:



| Vertex | Hull | Domain | Geometry | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
|        | X    |        |          |       |         |
