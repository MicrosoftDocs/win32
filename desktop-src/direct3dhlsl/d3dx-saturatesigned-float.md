# D3DX\_SaturateSigned\_FLOAT function

Retrieves a signed saturated value from the given FLOAT.

## Syntax

``` syntax
 D3DX_SaturateSigned_FLOAT(
   FLOAT _V
);
```

## Parameters

<dl> <dt>

*\_V* 
</dt> <dd>

The value to saturate.

</dd> </dl>

## Return value

The signed saturated value.
