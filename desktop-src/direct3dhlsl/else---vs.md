# else - vs

Start of an else block.

## Syntax



| else |
|------|



 

## Remarks



| Vertex shader versions | 1\_1 | 2\_0 | 2\_x | 2\_sw | 3\_0 | 3\_sw |
|------------------------|------|------|------|-------|------|-------|
| else                   |      | x    | x    | x     | x    | x     |



 

If the condition in the corresponding [if bool - vs](if-bool---vs.md) statement is true, the code enclosed by the if bool - vs statement and the matching [endif](endif---vs.md) is run. Otherwise, the code enclosed by the else...endif statements is run.
