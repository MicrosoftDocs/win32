# else - ps

Start of an else block.

## Syntax



| else |
|------|



 

## Remarks



| Pixel shader versions | 1\_1 | 1\_2 | 1\_3 | 1\_4 | 2\_0 | 2\_x | 2\_sw | 3\_0 | 3\_sw |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| else                  |      |      |      |      |      | x    | x     | x    | x     |



 

If the condition in the corresponding [if](if-bool---ps.md) statement is true, the code enclosed by the if statement and the matching else is run.
