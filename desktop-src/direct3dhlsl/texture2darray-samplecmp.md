# Texture2DArray::SampleCmp methods

Samples a [**Texture2DArray**](sm5-object-texture2darray.md), using a comparison value to reject samples.

### Overload list



| Method                                                                                              | Description                                                                                                                                                                           |
|:----------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp(S,float,float,int)**](dx-graphics-hlsl-to-samplecmp.md)                               | Samples a texture, using a comparison value to reject samples.<br/>                                                                                                             |
| [**SampleCmp(S,float,float,int,float)**](t2darray-samplecmp-s-float-float-int-float-.md)           | Samples a texture, using a comparison value to reject samples, with an optional value to clamp sample level-of-detail (LOD) values to.<br/>                                     |
| [**SampleCmp(S,float,float,int,float,uint)**](t2darray-samplecmp-s-float-float-int-float-uint-.md) | Samples a texture, using a comparison value to reject samples, with an optional value to clamp sample level-of-detail (LOD) values to. Returns status about the operation.<br/> |
