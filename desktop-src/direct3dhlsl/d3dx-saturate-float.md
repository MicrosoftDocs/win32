# D3DX\_Saturate\_FLOAT function

Retrieves the saturated value of the given FLOAT.

## Syntax

``` syntax
FLOAT D3DX_Saturate_FLOAT(
   FLOAT _V
);
```

## Parameters

<dl> <dt>

*\_V* 
</dt> <dd>

The value to saturate.

</dd> </dl>

## Return value

The saturated value.
