# domain

Defines the patch type used in the HS.


```
domain(X)      
```



## Remarks

X is either **tri**, **quad**, or **isoline**.

This attribute is supported in the following types of shaders:



| Vertex | Hull | Domain | Geometry | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
|        | x    | x      |          |       |         |
