# D3DX\_Truncate\_FLOAT function

Truncates a FLOAT value.

## Syntax

``` syntax
FLOAT D3DX_Truncate_FLOAT(
   FLOAT _V
);
```

## Parameters

<dl> <dt>

*\_V* 
</dt> <dd>

The value to truncate.

</dd> </dl>

## Return value

The truncated value.
