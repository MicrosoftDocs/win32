# Music Properties

Windows Portable Devices supports the following music properties.



| Property               | VarType        | Description                                |
|------------------------|----------------|--------------------------------------------|
| **WPD\_MUSIC\_ALBUM**  | **VT\_LPWSTR** | The album the track belongs to.            |
| **WPD\_MUSIC\_LYRICS** | **VT\_LPWSTR** | The lyrics for the music.                  |
| **WPD\_MUSIC\_MOOD**   | **VT\_LPWSTR** | A string describing the mood of the music. |
| **WPD\_MUSIC\_TRACK**  | **VT\_UI4**    | The recording's track number on the album. |
