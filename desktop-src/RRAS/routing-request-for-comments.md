# Routing Request for Comments

Requests for Comments (RFCs) are an ongoing series of documents that discuss the Internet and Internet-related technologies. Particular RFCs are identified by number. The following RFCs are referenced in the Remote Access Service (RAS) section of the Platform Software Development Kit (SDK). For more information about RFCs and how to locate copies of a particular RFC, see [http://www.rfc-editor.org/rfc.html](Http://go.microsoft.com/fwlink/p/?linkid=84570).

-   [RFC 791](Http://go.microsoft.com/fwlink/p/?linkid=84067) – Internet Protocol, DARPA Internet Program Protocol Specification, September 1981
-   [RFC 2863](Http://go.microsoft.com/fwlink/p/?linkid=84054) – The Interfaces Group MIB, June 2000
-   [RFC 1354](Http://go.microsoft.com/fwlink/p/?linkid=84028) – IP Forwarding Table MIB, July, 1992
