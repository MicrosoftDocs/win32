# ReloadZone method of the MicrosoftDNS\_Zone class

The **ReloadZone** method reloads the DNS Zone from its database.

## Syntax


```mof
void ReloadZone();
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
