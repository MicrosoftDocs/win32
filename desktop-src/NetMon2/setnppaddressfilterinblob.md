# SetNPPAddressFilterInBlob function

The **SetNPPAddressFilterInBlob** function sets the given address filter in the BLOB.

## Syntax


```C++
DWORD SetNPPAddressFilterInBlob(
  _In_ HBLOB          hBlob,
  _In_ LPADDRESSTABLE pAddressTable
);
```



## Parameters

<dl> <dt>

*hBlob* \[in\]
</dt> <dd>

Handle to a BLOB.

</dd> <dt>

*pAddressTable* \[in\]
</dt> <dd>

Pointer to an [**ADDRESSTABLE**](addresstable.md) structure that defines the user-allocated address table.

</dd> </dl>

## Return value

If the function is successful, the return value is NMERR\_SUCCESS.

If the function is unsuccessful, the return value is a NMERR value that indicates the error.
