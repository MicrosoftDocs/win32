# SetNetworkInfoInBlob function

The **SetNetworkInfoInBlob** function fills in the **NETWORKINFO** structure in the BLOB.

## Syntax


```C++
DWORD SetNetworkInfoInBlob(
  _In_ HBLOB         hBlob,
  _In_ LPNETWORKINFO lpNetworkInfo
);
```



## Parameters

<dl> <dt>

*hBlob* \[in\]
</dt> <dd>

Handle to a BLOB.

</dd> <dt>

*lpNetworkInfo* \[in\]
</dt> <dd>

Pointer to the user-allocated [NETWORKINFO](networkinfo.md) structure that the function fills in.

</dd> </dl>

## Return value

If the function is successful, the return value is NMERR\_SUCCESS.

If the function is unsuccessful, the return value is a NMERR value that indicates the error.
