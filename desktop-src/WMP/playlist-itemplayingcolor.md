# PLAYLIST.itemPlayingColor

The **itemPlayingColor** attribute specifies or retrieves the highlight color that indicates the currently playing item in the playlist.

``` syntax
        elementID.itemPlayingColor
```

## Possible Values

This attribute is a read/write **String** containing any Microsoft Internet Explorer color value. It has a default value of "\#00FF00".

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
