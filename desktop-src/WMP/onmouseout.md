# onmouseout

The **onmouseout** event handler handles an event that occurs when the user moves the pointer off the element.

``` syntax
onmouseout
```

## Remarks

This event handler is not applicable to the **PLAYLIST** element.

## Requirements



| Requirement | Value |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player version 70 or later<br/> |
