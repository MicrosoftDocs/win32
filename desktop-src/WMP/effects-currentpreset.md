# EFFECTS.currentPreset

The **currentPreset** attribute specifies or retrieves the current preset of the current visualization.

``` syntax
        elementID.currentPreset
```

## Possible Values

This attribute is a read/write **Number** (**long**) indicating the index of the preset. The indexes begin with zero, which is also the default value.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
