# VIEW.ontimer

The **ontimer** event handles timer events.

``` syntax
ontimer
```

## Remarks

The **timerInterval** attribute specifies the interval at which timer events will be fired.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
