# EQUALIZERSETTINGS.splineTension

The **splineTension** attribute specifies or retrieves the spline tension for the equalizer control.

``` syntax
        elementID.splineTension
```

## Possible Values

This attribute is a read/write **Number** (**float**) ranging from 0.0 (no tension) to 10.0 (full tension) with a default value of 3.0.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
