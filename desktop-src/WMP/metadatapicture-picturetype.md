# MetadataPicture.pictureType

The **pictureType** property retrieves the picture type of the metadata image.

## Syntax

*player*.*currentMedia*.**getItemInfoByType**( *name*, *language*, *index*).**pictureType**

## Possible Values

This property is a read-only **String**.

## Remarks

To retrieve the value of this property, read access to the library is required. For more information, see [Library Access](library-access.md).

**Windows Media Player 10 Mobile:** This method is not supported.
