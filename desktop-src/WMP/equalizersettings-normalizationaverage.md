# EQUALIZERSETTINGS.normalizationAverage

The **normalizationAverage** attribute retrieves the average normalization value.

``` syntax
        elementID.normalizationAverage
```

## Possible Values

This attribute is a read-only **Number** (**float**).

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
