# EQUALIZERSETTINGS.currentPreset

The **currentPreset** attribute specifies or retrieves the index of the current preset.

``` syntax
        elementID.currentPreset
```

## Possible Values

This attribute is a read/write **Number** (**long**).

## Remarks

If this attribute is not specified, the previous value will be retained.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
