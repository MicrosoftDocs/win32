# THEME.authorVersion

The **authorVersion** attribute specifies or retrieves the version number of the skin as assigned by the author.

``` syntax
theme.authorVersion
```

## Possible Values

This attribute is a read-only **String** with no default value.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
