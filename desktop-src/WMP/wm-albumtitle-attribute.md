# WM/AlbumTitle Attribute

The **WM/AlbumTitle** attribute is the title of the album on which the content was originally released.

## Remarks

This attribute is stored in both the library (or cache) and the digital media file.

**Album** is an alias for this attribute.

The Windows Media Format SDK constant for this attribute is g\_wszWMAlbumTitle.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
