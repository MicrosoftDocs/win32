# PLAYLIST.moveSelectedDown

The **moveSelectedDown** method moves the selected item down one position in the list.

``` syntax
        elementID.moveSelectedDown()
```

## Parameters

This method has no parameters.

## Return Value

This method does not return a value.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
