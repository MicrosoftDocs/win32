# EQUALIZERSETTINGS.crossFadeWindow

The **crossFadeWindow** attribute specifies or retrieves the amount of cross-fade overlap in milliseconds.

``` syntax
        elementID.crossFadeWindow
```

## Possible Values

This attribute is a read/write **Number** (**long**) ranging from 0 to 10,000 with a default value of 250.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
