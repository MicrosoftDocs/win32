# PartOfSet Attribute

The **PartOfSet** attribute is the part number and the total number of parts of the set to which the item belongs.

## Applies To

-   [Audio Items](audio-item-attributes.md)

## Remarks

This attribute is stored in the library.

To determine whether you can change the value of this attribute, use the [Media.isReadOnlyItem](media-isreadonlyitem.md) method.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------|
| Version<br/> | Windows Media Player 11<br/> |
