# VIEW.onerror

The **onerror** event handles an error event if **Settings.enableErrorDialogs** is set to false.

``` syntax
onerror
```

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
