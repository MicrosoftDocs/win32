# BUTTONELEMENT.index

The **index** attribute retrieves the index of the **BUTTONELEMENT** within the **BUTTONGROUP**.

``` syntax
        elementID.index
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
