# IWMPMetadataPicture::Description property

The `Description` property gets the description of the image represented by the metadata attribute.

## Syntax


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## Property value

A **System.String** that is the description of the image.

## Remarks

Before using this property, you must have read access to the library. For more information, see [Library Access](library-access.md).
