# event.y

The **y** attribute retrieves the y-coordinate of the mouse pointer with respect to the application window.

``` syntax
event.y
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
