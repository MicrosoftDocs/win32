# Windows Media Player Mobile Skins

Windows Media Player Mobile software includes a default skin that consists of various buttons and sliders that you can use to play, rewind, or stop digital media. A skin is a customizable user interface that provides a unique appearance and functionality for Windows Media Player Mobile.

The following sections describe how to create skins for Windows Media Player Mobile.

|---------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [About Skins](about-skins-mobile.md) | Discusses the concepts needed to understand skin architecture and the rules you must follow to create a skin. |
| [Skin Guide](skin-guide.md)          | Shows how to create a simple skin.                                                                            |
| [Skin Reference](skin-reference.md)  | Contains detailed references for all the parts of a skin.                                                     |
