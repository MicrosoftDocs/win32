# THEME.copyright

The **copyright** attribute specifies or retrieves the copyright string for the skin.

``` syntax
theme.copyright
```

## Possible Values

This attribute is a read/write **String** with no default value.

## Requirements



|                    |                                                      |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
