# event.keyCode

The **keyCode** attribute retrieves the ASCII key code if a key was pressed when the event occurred.

``` syntax
event.keyCode
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
