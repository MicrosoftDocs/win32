# VIEW.onclose

The **onclose** event handler handles an event that occurs when the **VIEW** is about to be closed.

``` syntax
onclose
```

## Requirements



|                    |                                                      |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
