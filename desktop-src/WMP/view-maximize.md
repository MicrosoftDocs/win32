# VIEW.maximize

The **maximize** method maximizes the **VIEW**.

``` syntax
        elementID.maximize()
```

## Parameters

This method has no parameters.

## Return Value

This method does not return a value.

## Examples


```JScript
<THEME>
  <VIEW id="View1">
    <BUTTON id="Open" Image="Open.png" 
        onclick="jscript:View1.maximize()">
    </BUTTON>
  </VIEW>
</THEME>
```
