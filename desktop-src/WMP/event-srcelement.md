# event.srcElement

The **srcElement** attribute retrieves the element that fired the event.

``` syntax
event.srcElement
```

## Possible Values

This attribute is a read-only object.

## Requirements



|                    |                                                      |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
