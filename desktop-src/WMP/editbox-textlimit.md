# EDITBOX.textLimit

The **textLimit** attribute specifies or retrieves the maximum number of characters that the user can type in the edit box control.

``` syntax
        elementID.textLimit
```

## Possible Values

This attribute is a read/write **Number** (**long**) with a default value of 0, which indicates no limit.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
