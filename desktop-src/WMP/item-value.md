# ITEM.value

The **value** attribute specifies or retrieves the text that displays for a list box or pop-up item.

``` syntax
        elementID.value
```

## Possible Values

This attribute is a read/write **String**.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
