# RETURNBUTTON

This is a predefined **BUTTON** with the following default values.

``` syntax
onclick="jscript:view.returnToMediaCenter();"
upToolTip="Return to Full Mode"
```

## Remarks

This creates a **BUTTON** control that returns the Player to the media center. The ToolTips are localized. All properties of this **BUTTON** can be overridden by explicitly specifying them.

## Requirements



| Requirement | Value |
|--------------------|----------------------------------------------|
| Version<br/> | Windows Media Player 7.0 or later<br/> |
