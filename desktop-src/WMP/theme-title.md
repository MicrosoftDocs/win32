# THEME.title

The **title** attribute specifies or retrieves the title of the skin.

``` syntax
theme.title
```

## Possible Values

This attribute is a read/write **String** with no default value.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
