# EFFECTS.effectCanGoFullScreen

The **effectCanGoFullScreen** attribute retrieves a value indicating whether the current visualization can be displayed full-screen.

``` syntax
        elementID.effectCanGoFullScreen
```

## Possible Values

This attribute is a read-only **Boolean**.



| Value | Description                                    |
|-------|------------------------------------------------|
| true  | Visualization can be displayed full-screen.    |
| false | Visualization cannot be displayed full-screen. |



 

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
