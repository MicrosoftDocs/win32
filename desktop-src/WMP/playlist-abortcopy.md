# PLAYLIST.abortCopy

The **abortCopy** method cancels a copy operation.

``` syntax
        elementID.abortCopy()
```

## Parameters

This method has no parameters.

## Return Value

This method does not return a value.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
