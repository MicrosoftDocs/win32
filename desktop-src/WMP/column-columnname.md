# COLUMN.columnName

The **columnName** attribute specifies or retrieves a column name in the **PLAYLIST** control.

``` syntax
        elementID.columnName
```

## Possible Values

This attribute is a read/write **String**.

## Remarks

The column name is the friendly name that appears in the header of the column.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
