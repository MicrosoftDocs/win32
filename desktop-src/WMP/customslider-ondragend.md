# CUSTOMSLIDER.onDragEnd

The **onDragEnd** event handler handles an event that occurs when the left mouse button is released after a dragging operation.

``` syntax
onDragEnd
```

## Requirements



|                    |                                                      |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
