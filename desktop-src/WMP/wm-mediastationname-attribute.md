# WM/MediaStationName Attribute

The **WM/StationName** attribute specifies the station name on which a given show was first broadcast.

## Applies To

-   [Video Items](video-item-attributes.md)

## Remarks

The Windows Media Format SDK constant for this attribute is g\_wszWMMediaStationName.

To determine whether you can change the value of this attribute, use the [Media.isReadOnlyItem](media-isreadonlyitem.md) method.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------|
| Version<br/> | Windows Media Player 10 or later<br/> |
