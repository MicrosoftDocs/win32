# IWMPPlaylistArray::count property

The **count** property gets the number of playlists in the playlist array.

## Syntax


```CSharp
public System.Int32 count {get; set;}
```


```VB

Public ReadOnly Property count As System.Int32
```





## Property value

A **System.Int32** that is the number of playlists in the array.

## Remarks

Before using this property, you must have read access to the library. For more information, see [Library Access](library-access.md).
