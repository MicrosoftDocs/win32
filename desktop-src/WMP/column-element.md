# COLUMN Element

The **COLUMN** element represents a column within a playlist control. **COLUMN** elements are always children of **PLAYLIST** elements.

The **COLUMN** element supports the following attributes.



| Attribute                                       | Description                                                        |
|-------------------------------------------------|--------------------------------------------------------------------|
| [columnName](column-columnname.md)             | Specifies or retrieves a column name in the **PLAYLIST** control.  |
| [columnID](column-columnid.md)                 | Specifies or retrieves a column ID in the **PLAYLIST** control.    |
| [columnResizeMode](column-columnresizemode.md) | Specifies or retrieves the resize mode for this column.            |
| [columnWidth](column-columnwidth.md)           | Specifies or retrieves a column width in the **PLAYLIST** control. |
