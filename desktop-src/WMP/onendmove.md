# onendmove

The **onendmove** event handler handles an event that occurs when an element completes a moveTo operation.

``` syntax
onendmove
```

## Requirements



| Requirement | Value |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player version 70 or later<br/> |
