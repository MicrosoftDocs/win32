# LISTBOX.dismiss

The **dismiss** method hides the control.

``` syntax
        elementID.dismiss()
```

## Parameters

This method has no parameters.

## Return Value

This method does not return a value.

## Remarks

This method is primarily used with the **POPUP** element, which represents a list box control that is displayed only when needed.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
