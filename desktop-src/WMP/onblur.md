# onblur

The **onblur** event handler handles an event that occurs when the element loses the keyboard focus.

``` syntax
onblur
```

## Remarks

The **onblur** event handler is not applicable to the **EFFECTS** element or the **POPUP** element.

## Requirements



| Requirement | Value |
|--------------------|-----------------------------------------------------|
| Version<br/> | Windows Media Player version 70 or later<br/> |
