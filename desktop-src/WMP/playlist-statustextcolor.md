# PLAYLIST.statusTextColor

The **statusTextColor** attribute specifies or retrieves a value indicating the color of the status text.

``` syntax
        elementID.statusTextColor
```

## Possible Values

This attribute is a read/write **String** representing a color. The default value is "none".

|--------------------|------------------------------------|
| Version<br/> | Windows Media Player 10<br/> |
