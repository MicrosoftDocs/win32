# CUSTOMSLIDER.onDragBegin

The **onDragBegin** event handler handles an event that occurs when the user clicks and holds the left mouse button down and begins to drag the mouse.

``` syntax
onDragBegin
```

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
