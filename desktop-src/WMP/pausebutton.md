# PAUSEBUTTON

This is a predefined **BUTTON** with the following default values.

``` syntax
onclick="jscript:player.controls.pause()" 
upToolTip="Pause"
cursor="system"
enabled="wmpenabled:player.controls.pause"
```

## Remarks

All properties of this **BUTTON** can be overridden by explicitly specifying them.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
