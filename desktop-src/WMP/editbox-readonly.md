# EDITBOX.readOnly

The **readOnly** attribute specifies or retrieves a value indicating whether text in the edit box control is read-only or can be edited.

``` syntax
        elementID.readOnly
```

## Possible Values

This attribute is a read/write **Boolean** with a default value of false.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
