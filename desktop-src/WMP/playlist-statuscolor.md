# PLAYLIST.statusColor

The **statusColor** attribute specifies or retrieves the color of the status line in the **PLAYLIST** element.

``` syntax
        elementID.statusColor
```

## Possible Values

This attribute is a read/write **String** containing any Microsoft Internet Explorer color value. It has a default value equal to the value of the **backgroundColor** attribute.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
