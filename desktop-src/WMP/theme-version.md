# THEME.version

The **version** attribute specifies or retrieves the Windows Media Player version number for which the skin was authored.

``` syntax
theme.version
```

## Possible Values

This attribute is a **Number** (**float**). Specifying a value for this attribute at design time or run time has no effect. The value is always 1.0.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
