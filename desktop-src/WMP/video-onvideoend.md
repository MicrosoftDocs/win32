# VIDEO.onvideoend

The **onvideoend** event handler handles an event that occurs when the video stops rendering and is unloaded.

``` syntax

onvideoend 
```

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
