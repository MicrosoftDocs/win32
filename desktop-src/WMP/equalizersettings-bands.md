# EQUALIZERSETTINGS.bands

The **bands** attribute retrieves the number of frequency bands supported.

``` syntax
        elementID.bands
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



|                    |                                                      |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
