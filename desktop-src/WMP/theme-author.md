# THEME.author

The **author** attribute specifies or retrieves the name of the author of the skin.

``` syntax
theme.author
```

## Possible Values

This attribute is a read/write **String** with no default value.

## Requirements



|                    |                                                      |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
