# IWMPDVD::resume method

The **resume** method returns to playback mode from menu mode, resuming at the same position as when the menu was invoked.

## Syntax


```CSharp
public void resume();
```


```VB

Public Sub resume()
Implements IWMPDVD.resume
```





## Parameters

This method has no parameters.

## Return value

This method does not return a value.
