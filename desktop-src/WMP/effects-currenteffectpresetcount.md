# EFFECTS.currentEffectPresetCount

The **currentEffectPresetCount** attribute retrieves number of available presets for the current visualization.

``` syntax
        elementID.currentEffectPresetCount
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
