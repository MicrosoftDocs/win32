# Player.dvd

The **dvd** property retrieves the DVD object.

## Syntax

*player* .**dvd**

## Possible Values

This property is a read-only DVD object.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later.<br/>                              |
| DLL<br/>     | <dl> <dt>Wmp.dll</dt> </dl> |
