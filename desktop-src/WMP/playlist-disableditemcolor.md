# PLAYLIST.disabledItemColor

The **disabledItemColor** attribute specifies or retrieves the color of a disabled CD track or of online content when offline.

``` syntax
        elementID.disabledItemColor
```

## Possible Values

This attribute is a read/write **String** containing any Microsoft Internet Explorer color value. It has a default value of "graytext".

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
