# PLAYLIST.checkboxesVisible

The **checkboxesVisible** attribute specifies or retrieves a value indicating whether check boxes are visible.

``` syntax
        elementID.checkboxesVisible
```

## Possible Values

This attribute is a read/write **Boolean**.



| Value | Description                           |
|-------|---------------------------------------|
| true  | Check boxes are visible.              |
| false | Default. Check boxes are not visible. |



 

## Remarks

Check boxes appear in the far left column.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
