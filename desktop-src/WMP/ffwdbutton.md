# FFWDBUTTON

This is a predefined **BUTTON** with the following default values.


```C++
onclick="jscript:player.controls.fastForward()"
upToolTip="Fast Forward"
cursor="system"
enabled="wmpenabled:player.controls.fastForward"

```



## Remarks

All properties of this **BUTTON** can be overridden by explicitly specifying them.
