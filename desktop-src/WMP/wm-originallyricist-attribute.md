# WM/OriginalLyricist Attribute

The **WM/Lyricist** attribute is the name of the person who wrote the original lyrics.

## Applies To

-   [Music Files](music-file-attributes.md)

## Remarks

This attribute is stored only in a music file that is not in the library.

The Windows Media Format SDK constant for this attribute is g\_wszWMOriginalLyricist.

To determine whether you can change the value of this attribute, use the [Media.isReadOnlyItem](media-isreadonlyitem.md) method.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
