# Player.error

The **error** property retrieves the Error object.

## Syntax

*player* .**error**

## Possible Values

This property is a read-only Error object.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later.<br/>                              |
