# LISTBOX.foregroundColor

The **foregroundColor** attribute specifies or retrieves the text color in the list box control.

``` syntax
        elementID.foregroundColor
```

## Possible Values

This attribute is a read/write **String** containing any Microsoft Internet Explorer color value. It has a default value equal to the Windows text color.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
