# VIEW.maxWidth

The **maxWidth** attribute specifies or retrieves the maximum width in pixels of the **VIEW** when resizing.

``` syntax
        elementID.maxWidth
```

## Possible Values

This attribute is a read/write **Number** (**long**) with a value of zero or greater. It has a default value of zero, which means there is no restriction on the maximum width of the **VIEW**.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
