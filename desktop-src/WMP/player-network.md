# Player.network

The **network** property retrieves the Network object.

## Syntax

*player* .**network**

## Possible Values

This property is a read-only Network object.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later.<br/>                              |
