# EFFECTS.settings

The **settings** method displays the attribute page for the current visualization, if present.

``` syntax
        elementID.settings()
```

## Parameters

This method has no parameters.

## Return Value

This method does not return a value.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
