# WM/SubscriptionContentID Attribute

The **WM/SubscriptionContentID** attribute is the subscription content identifier.

## Applies To

-   [Audio Items](audio-item-attributes.md)
-   [Commonly Used Windows Media File Attributes](commonly-used-windows-media-file-attributes.md)
-   [Video Items](video-item-attributes.md)

## Remarks

This attribute is stored in both the library and the digital media file.

**SubscriptionContentID** is an alias for this attribute.

The Windows Media Format SDK constant for this attribute is g\_wszWMSubscriptionContentID.

To determine whether you can change the value of this attribute, use the [Media.isReadOnlyItem](media-isreadonlyitem.md) method.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------|
| Version<br/> | Windows Media Player 9 Series or later<br/> |
