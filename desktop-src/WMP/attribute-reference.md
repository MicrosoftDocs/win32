# Attribute Reference

The Windows Media Player Attribute Reference documents metadata attributes that are of interest to Windows Media Player SDK developers. The following table lists these attributes in various categories.



| Category                                                                                       | Description                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Alphabetical Reference](alphabetical-attribute-reference.md)                                 | Provides an alphabetical list of predefined attributes.                                                                  |
| [Audio Items](audio-item-attributes.md)                                                       | Lists attributes that are available for an audio item in the library.                                                    |
| [CD Playlists](cd-playlist-attributes.md)                                                     | Lists attributes that are available for a CD.                                                                            |
| [CD Tracks](cd-track-attributes.md)                                                           | Lists attributes that are available for a CD track.                                                                      |
| [Commonly Used Windows Media File Attributes](commonly-used-windows-media-file-attributes.md) | Lists attributes that are available for a digital media file.                                                            |
| [DVDs](dvd-attributes.md)                                                                     | Lists attributes that are available for a DVD.                                                                           |
| [Music Files](music-file-attributes.md)                                                       | Lists attributes that are available for a music file that is not in the library.                                         |
| [Other Items](other-item-attributes.md)                                                       | Lists attributes that are available for items in the library that are not audio, video, playlist, photo, or radio items. |
| [Photo Items](photo-item-attributes.md)                                                       | Lists attributes that are available for a photo item in the library.                                                     |
| [Playlists](playlist-attributes-ref.md)                                                       | Lists attributes that are available for playlists retrieved as media items from the library.                             |
| [Radio Items](radio-item-attributes.md)                                                       | Lists attributes that are available for a radio item in the library.                                                     |
| [Video Items](video-item-attributes.md)                                                       | Lists attributes that are available for a video item in the library.                                                     |



 

For more info about the metadata attributes, see the [Windows Media Metadata Usage Guidelines](/previous-versions/ms867702(v=msdn.10)).
