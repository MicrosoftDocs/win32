# LISTBOX.appendItem

The **appendItem** method inserts new text at the end of the list box control.

``` syntax
        elementID.appendItem(newVal)
```

## Parameters

<dl> <dt>

<span id="newVal"></span><span id="newval"></span><span id="NEWVAL"></span>*newVal*
</dt> <dd>

**String** containing the text to be inserted.

</dd> </dl>

## Return Value

This method does not return a value.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
