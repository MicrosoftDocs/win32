# event.x

The **x** attribute retrieves the x-coordinate of the mouse pointer with respect to the application window.

``` syntax
event.x
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
