# LISTBOX.deleteAll

The **deleteAll** method deletes all items from the list box control.

``` syntax
        elementID.deleteAll()
```

## Parameters

This method has no parameters.

## Return Value

This method does not return a value.

## Requirements



| Requirement | Value |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
