# Player.mediaCollection

The **mediaCollection** property retrieves the **MediaCollection** object.

## Syntax

*player* .**mediaCollection**

## Possible Values

This property is a read-only **MediaCollection** object.

## Remarks

To retrieve the value of this property, read access to the library is required. For more information, see [Library Access](library-access.md).

## Requirements
