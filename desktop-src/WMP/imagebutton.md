# IMAGEBUTTON

This is a predefined **BUTTON** with the following default value.

``` syntax
cursor="Hand"
```

## Remarks

All buttons can contain images. This element is useful for code clarification purposes to indicate a button that is used primarily as an image container.

All properties of this **BUTTON** can be overridden by explicitly specifying them.

## Requirements



| Requirement | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
