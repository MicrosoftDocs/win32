# LOCALE\_REPLACEMENT

**Windows Vista and later:** Enumerate [replacement locales](custom-locales.md). This constant is valid input only for [**EnumLocalesProcEx**](https://msdn.microsoft.com/en-us/library/Dd317823(v=VS.85).aspx).
