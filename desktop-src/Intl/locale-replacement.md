# LOCALE\_REPLACEMENT

**Windows Vista and later:** Enumerate [replacement locales](custom-locales.md). This constant is valid input only for [**EnumLocalesProcEx**](/windows/win32/api/winnls/nc-winnls-locale_enumprocex).
