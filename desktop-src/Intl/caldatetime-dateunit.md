# CALDATETIME\_DATEUNIT enumeration

Deprecated. Specifies the date units for adjusting the [**CALDATETIME**](caldatetime.md) structure.

## Syntax


```C++
enum CALDATETIME_DATEUNIT {
  EraUnit, 
  YearUnit, 
  MonthUnit, 
  WeekUnit, 
  DayUnit, 
  HourUnit, 
  MinuteUnit, 
  SecondUnit, 
  TickUnit 

};
```
