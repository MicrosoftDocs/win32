# LOCALE\_ILDATE

Long date format-ordering specifier. It is preferred for your application to use the [LOCALE\_SLONGDATE](locale-slongdate.md) constant instead of LOCALE\_ILDATE.



| Value | Meaning        |
|-------|----------------|
| 0     | Month-Day-Year |
| 1     | Day-Month-Year |
| 2     | Year-Month-Day |
