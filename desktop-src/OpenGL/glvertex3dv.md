# glVertex3dv function

Specifies a vertex.

## Syntax


```C++
void WINAPI glVertex3dv(
   const GLdouble *v
);
```



## Parameters

<dl> <dt>

*v* 
</dt> <dd>

A pointer to an array of three elements. The elements are the x, y, and z coordinates of a vertex.

</dd> </dl>

## Return value

This function does not return a value.
