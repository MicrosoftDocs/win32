# delete urlacl

Deletes a reserved URL.

``` syntax
delete urlacl [url=]string
 
```

## Parameters

__\[url=\]__*string*

## Examples

**delete urlacl url=https://+:80/MyUri**

**delete urlacl url=https://www.contoso.com:80/MyUri**
