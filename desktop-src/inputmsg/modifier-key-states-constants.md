# Modifier Key State

Indicates which keyboard modifier keys were pressed at the time input was being generated.

</dt> <dd> <dl> <dt>

0x0004
</dt> <dt>



A Shift key was pressed.


</dt> </dl> </dd> <dt>

<span id="POINTER_MOD_CTRL"></span><span id="pointer_mod_ctrl"></span>**POINTER_MOD_CTRL**

</dt> <dt>

</dt> </dl> </dd> </dl>
