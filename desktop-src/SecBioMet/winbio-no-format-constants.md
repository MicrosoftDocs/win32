# WINBIO\_NO\_FORMAT Constants

If the members of a [**WINBIO\_REGISTERED\_FORMAT**](winbio-registered-format.md) structure cannot be zero, you can use the following constants to check for errors.



| Constant/value                                                                                                                                                                                                                                                                   | Description                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO\_NO\_FORMAT\_OWNER\_AVAILABLE**</dt> <dt>0</dt> </dl> | No IBIA (International Biometric Industry Association) assigned owner value is available.<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO\_NO\_FORMAT\_TYPE\_AVAILABLE**</dt> <dt>0</dt> </dl>    | No owner assigned format type is available.<br/>                                               |
