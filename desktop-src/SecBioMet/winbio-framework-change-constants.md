# WINBIO\_FRAMEWORK\_CHANGE Constants

The following constants can be used in the [**WinBioAsyncMonitorFrameworkChanges**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncmonitorframeworkchanges) function to identify the type of change that occurred in the framework.

<dl> <dt>

<span id="WINBIO_FRAMEWORK_CHANGE_UNIT"></span><span id="winbio_framework_change_unit"></span>**WINBIO\_FRAMEWORK\_CHANGE\_UNIT**
</dt> <dd> <dl> <dt>

0x00000001
</dt> <dt>



A biometric unit was attached to or detached from the computer.


</dt> </dl> </dd> </dl>
