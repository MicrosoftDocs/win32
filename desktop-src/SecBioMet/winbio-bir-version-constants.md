# WINBIO\_BIR\_VERSION Constants

The following flags are used by the **HeaderVersion** and **PatronHeaderVersion** members of the [**WINBIO\_BIR\_HEADER**](winbio-bir-header.md) structure to specify the version.



| Constant/value                                                                                                                                                                                                                                                         | Description                                     |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------|
| <span id="WINBIO_CBEFF_HEADER_VERSION"></span><span id="winbio_cbeff_header_version"></span><dl> <dt>**WINBIO\_CBEFF\_HEADER\_VERSION**</dt> <dt>0x11</dt> </dl>    | Specifies the header version.<br/>        |
| <span id="WINBIO_PATRON_HEADER_VERSION"></span><span id="winbio_patron_header_version"></span><dl> <dt>**WINBIO\_PATRON\_HEADER\_VERSION**</dt> <dt>0x11</dt> </dl> | Specifies the patron header version.<br/> |
