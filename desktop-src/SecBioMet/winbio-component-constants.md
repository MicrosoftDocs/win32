# WINBIO\_COMPONENT Constants

The following constants can be used when calling [**WinBioControlUnit**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunit) or [**WinBioControlUnitPrivileged**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunitprivileged) to specify the type of adapter being used.



| Constant                                                                                                                                                                                        | Description                             |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------|
| <span id="WINBIO_COMPONENT_SENSOR"></span><span id="winbio_component_sensor"></span><dl> <dt>**WINBIO\_COMPONENT\_SENSOR**</dt> </dl>    | Specifies a sensor adapter.<br/>  |
| <span id="WINBIO_COMPONENT_ENGINE"></span><span id="winbio_component_engine"></span><dl> <dt>**WINBIO\_COMPONENT\_ENGINE**</dt> </dl>    | Specifies an engine adapter.<br/> |
| <span id="WINBIO_COMPONENT_STORAGE"></span><span id="winbio_component_storage"></span><dl> <dt>**WINBIO\_COMPONENT\_STORAGE**</dt> </dl> | Specifies a storage adapter.<br/> |
