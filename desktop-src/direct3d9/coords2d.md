# Coords2d

Defines a two dimensional vector used to define a mesh's (u, v) texture coordinates.

``` syntax
template Coords2d
{
    < F6F23F44-7686-11cf-8F52-0040333594A3 >
    float u;
    float v;
} 
```

-   u - u coordinate value.
-   v - v coordinate value.
