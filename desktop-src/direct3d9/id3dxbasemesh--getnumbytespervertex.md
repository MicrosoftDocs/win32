# ID3DXBaseMesh::GetNumBytesPerVertex method

Gets the number of bytes per vertex.




## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](../winprog/windows-data-types.md)**

Returns the number of bytes per vertex.
