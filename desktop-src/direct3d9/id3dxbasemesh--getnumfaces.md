# ID3DXBaseMesh::GetNumFaces method

Retrieves the number of faces in the mesh.

## Syntax


```C++
DWORD GetNumFaces();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](../winprog/windows-data-types.md)**

Returns the number of faces in the mesh.
