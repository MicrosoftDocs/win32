# ID3DXEffect::GetCurrentTechnique method

Gets the current technique.

## Syntax


```C++
D3DXHANDLE GetCurrentTechnique();
```



## Parameters

This method has no parameters.

## Return value

Type: **[D3DXHANDLE](dx9-graphics-reference-effects-constants.md)**

A unique identifier to the current technique. See [Handles (Direct3D 9)](handles.md).
