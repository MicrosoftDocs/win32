# ID3DXPatchMesh::GetNumPatches method

Gets the number of patches in the mesh.

## Syntax


```C++
DWORD GetNumPatches();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](https://msdn.microsoft.com/en-us/library/Aa383751(v=VS.85).aspx)**

The number of patches.
