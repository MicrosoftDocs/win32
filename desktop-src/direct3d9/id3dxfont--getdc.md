# ID3DXFont::GetDC method

Returns a handle to a display device context (DC) that has the font set.

## Syntax


```C++
HDC GetDC();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**HDC**](../winprog/windows-data-types.md)**

Handle to a display DC.
