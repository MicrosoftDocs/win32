# ID3DXLine::GetWidth method

Gets the thickness of the line.

## Syntax


```C++
FLOAT GetWidth();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**FLOAT**](https://msdn.microsoft.com/en-us/library/Aa383751(v=VS.85).aspx)**

The line thickness.
