# D3DADAPTER\_DEFAULT

Specifies the primary display adapter.



| \#define            | Value | Description                                  |
|---------------------|-------|----------------------------------------------|
| D3DADAPTER\_DEFAULT | 0     | Used to specify the primary display adapter. |



 

## Constant Information



|   Requirement            | Value      |
|--------------------------|------------|
| Header                   | d3d9.h     |
| Minimum operating system | Windows 98 |
