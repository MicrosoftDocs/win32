# ID3DXPatchMesh::GetNumVertices method

Gets the number of vertices in the mesh.

## Syntax


```C++
DWORD GetNumVertices();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](../winprog/windows-data-types.md)**

The number of vertices.
