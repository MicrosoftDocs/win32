# X (Security Glossary)

[A](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F [G](g-gly.md) [H](h-gly.md) [I](i-gly.md) J [K](k-gly.md) [L](l-gly.md) [M](m-gly.md) [N](n-gly.md) [O](o-gly.md) [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) [U](u-gly.md) [V](v-gly.md) [W](w-gly.md) X Y Z

<dl> <dt>

<span id="_security_x.200_gly"></span><span id="_SECURITY_X.200_GLY"></span>**X.200**
</dt> <dd>

An Open Systems Interconnect (OSI) standard.

</dd> <dt>

<span id="_security_x.208_gly"></span><span id="_SECURITY_X.208_GLY"></span>**X.208**
</dt> <dd>

The ITU standard defining ASN.1.

</dd> <dt>

<span id="_security_x.209_gly"></span><span id="_SECURITY_X.209_GLY"></span>**X.209**
</dt> <dd>

The ITU standard that defines BER encodings for ASN.1 objects.

See also [*Basic Encoding Rules*](b-gly.md).

</dd> <dt>

<span id="_security_x.400_gly"></span><span id="_SECURITY_X.400_GLY"></span>**X.400**
</dt> <dd>

The ITU standard that defines message tokens.

</dd> <dt>

<span id="_security_x.500_gly"></span><span id="_SECURITY_X.500_GLY"></span>**X.500**
</dt> <dd>

The ITU standard that defines the structure of a global dictionary.

</dd> <dt>

<span id="_security_x.509_gly"></span><span id="_SECURITY_X.509_GLY"></span>**X.509**
</dt> <dd>

An internationally recognized standard for certificates that defines their required parts.

</dd> <dt>

<span id="_security_x509_asn_encoding_gly"></span><span id="_SECURITY_X509_ASN_ENCODING_GLY"></span>**X509\_ASN\_ENCODING**
</dt> <dd>

Encoding type that specifies certificate encoding. Certificate encoding types are stored in the low-order word of a **DWORD** (value is: 0x00000001). This encoding type is functionally the same as CRYPT\_ASN\_ENCODING.

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**XOR**
</dt> <dd>

Exclusive **OR**. A Boolean operation that yields true only if one of its operands are true and the other is false. If both operands are the same (either true or false), the operation yields false.

</dd> </dl>
