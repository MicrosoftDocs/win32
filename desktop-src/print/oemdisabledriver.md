# OEMDisableDriver function

## Syntax


```C++
VOID APIENTRY OEMDisableDriver(
   VOID 
);
```



## Parameters

This method has no parameters.

## Return value

This function does not return a value.

## 

This function is obsolete for Windows XP and later. It is supported only for earlier plug-ins. Use [**IPrintOemPS::DisableDriver**](iprintoemps-disabledriver.md) (Pscript) or [**IPrintOemUni::DisableDriver**](iprintoemuni-disabledriver.md) (Unidrv).
