# OEMTTDownloadMethod function

## Syntax


```C++
DWORD APIENTRY OEMTTDownloadMethod(
   PDEVOBJ     pdevobj,
   PUNIFONTOBJ pUFObj
);
```



## Parameters

<dl> <dt>

*pdevobj* 
</dt> <dd></dd> <dt>

*pUFObj* 
</dt> <dd></dd> </dl>

## 

This function is obsolete for Windows XP and later. It is supported only for earlier Unidrv plug-ins. Use [**IPrintOemUni::TTDownloadMethod**](iprintoemuni-ttdownloadmethod.md).
