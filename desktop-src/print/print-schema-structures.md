# Print Schema Enumerations

This section provides information about the enumerations that enable the new print driver in Windows 8 (the V4 print driver) to configure schema elements.

These enumerations provide the mechanism for setting feature selection modes and for configuring the constraint types.

Here is a list of the enumerations:

<dl>

[**PRINTSCHEMACONSTRAINEDSETTING**](printschemaconstrainedsetting.md)  
[**PRINTSCHEMASELECTIONTYPE**](printschemaselectiontype.md)  
</dl>
