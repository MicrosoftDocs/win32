# UI\_ANIMATION\_SECONDS\_INFINITE

Indicates that there are no scheduled events.



| Constant/value                                                                                                                                                                                                                                                            | Description                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------|
| <span id="UI_ANIMATION_SECONDS_INFINITE"></span><span id="ui_animation_seconds_infinite"></span><dl> <dt>**UI\_ANIMATION\_SECONDS\_INFINITE**</dt> <dt>-1.0</dt> </dl> | UI\_ANIMATION\_SECONDS<br/> |



## Remarks

This value can be returned in a call to the [**IUIAnimationManager2::EstimateNextEventTime**](/windows/desktop/api/UIAnimation/nf-uianimation-iuianimationmanager2-estimatenexteventtime) method.
