Blocking state.

## Syntax


```C++
BLOCK_STATE m_BlockState;
```



## Remarks

The following states are defined:

-   NOT\_BLOCKED
-   PENDING
-   BLOCKED

Before accessing this variable, hold the [**CDynamicOutputPin::m\_BlockStateLock**](cdynamicoutputpin-m-blockstatelock.md) critical section.
