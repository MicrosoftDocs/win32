# AVEncMPACodingMode property

Specifies the MPEG-1/2 audio encoding mode. This property applies to MPEG audio encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPACodingMode**

## Property value

The value of this property is a member of the [**eAVEncMPACodingMode**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpacodingmode) enumeration.
