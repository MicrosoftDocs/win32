# AVEncDDLoRoCenterMixLvl\_x10 property

Specifies the level shift that is applied to the center channel for Lo/Ro downmixing. This property applies to Dolby Digital audio encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncDDLoRoCenterMixLvl\_x10**

## Property value

The value is specified as decibels (dB) x 10.
