# Mapping DirectX Video Acceleration to IAMVideoAccelerator

The following section is primarily relevant for software decoder developers. It provides detailed information on how to use the [**IAMVideoAccelerator**](/windows/win32/videoacc/nn-videoacc-iamvideoaccelerator?branch=master) Interface when communicating through the [Overlay Mixer](overlay-mixer-filter.md) or Video Mixing Renderer filter with a DirectX VA-enabled hardware device.
