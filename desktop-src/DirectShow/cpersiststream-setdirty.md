Changes the dirty flag for the current stream.

## Syntax


```C++
HRESULT SetDirty(
   BOOL fDirty
);
```



## Parameters

<dl> <dt>

*fDirty* 
</dt> <dd>

New dirty flag for this stream. **TRUE** means that the data has not been saved.

</dd> </dl>

## Return value

Returns an **HRESULT** value.
