# CBaseList.AddTailI method

The `AddTailI` method adds an item to the end of the list.

## Syntax


```C++
POSITION AddTailI(
   void *pObj
);
```



## Parameters

<dl> <dt>

*pObj* 
</dt> <dd>

Pointer to the item.

</dd> </dl>

## Return value

Returns a POSITION value for the new tail position.

## Remarks

If the method fails, it returns **NULL**.
