# CBaseWindow.GetMemoryHDC method

The `GetMemoryHDC` method retrieves a handle to the memory device context (DC).

## Syntax


```C++
virtual HDC GetMemoryHDC();
```



## Parameters

This method has no parameters.

## Return value

Returns a handle to the memory DC.
