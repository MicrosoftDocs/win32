# CGenericList.GetHead method

The `GetHead` method retrieves the item at the head of the list.

## Syntax


```C++
OBJECT* GetHead();
```



## Parameters

This method has no parameters.

## Return value

Returns a pointer to an object of type **OBJECT** (the template type).
