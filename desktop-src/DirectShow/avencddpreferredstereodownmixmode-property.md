# AVEncDDPreferredStereoDownMixMode property

Specifies the preferred stereo downmix mode for a Dolby Digital audio stream. This property applies to Dolby Digital audio encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncDDPreferredStereoDownMixMode**

## Property value

The value of this property is a member of the [**eAVEncDDPreferredStereoDownMixMode**](/windows/win32/codecapi/ne-codecapi-eavencddpreferredstereodownmixmode?branch=master) enumeration.

## Remarks

This property is read/write.
