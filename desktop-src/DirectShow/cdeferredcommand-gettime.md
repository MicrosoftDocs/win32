# CDeferredCommand.GetTime method

The `GetTime` method retrieves the time at which the method will be run.

## Syntax


```C++
CRefTime GetTime();
```



## Parameters

This method has no parameters.

## Return value

Returns a [**CRefTime**](creftime.md) object containing a reference time.
