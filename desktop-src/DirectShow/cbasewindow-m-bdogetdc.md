# CBaseWindow::m\_bDoGetDC member

Flag that specifies whether to retrieve the device context (DC). If **TRUE**, the object retrieves a handle to the window's DC and creates a compatible memory DC.

## Syntax


```C++
BOOL m_bDoGetDC;
```
