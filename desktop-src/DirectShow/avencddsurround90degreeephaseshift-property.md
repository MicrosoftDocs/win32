# AVEncDDSurround90DegreeePhaseShift property

Specifies whether a 90-degree phase shift is applied to the Surround channels in a Dolby Digital audio stream. This property applies to Dolby Digital audio encoders.

This property is read/write.

## Data type

**VARIANT\_BOOL** (**VT\_BOOL**)

## Property GUID

**CODECAPI\_AVEncDDSurround90DegreeePhaseShift**
