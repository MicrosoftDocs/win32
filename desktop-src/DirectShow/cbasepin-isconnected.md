# CBasePin.IsConnected method

The `IsConnected` method determines whether the pin is connected to another pin.

## Syntax


```C++
BOOL IsConnected();
```



## Parameters

This method has no parameters.

## Return value

Returns **TRUE** if the pin is connected. Otherwise, returns **FALSE**.
