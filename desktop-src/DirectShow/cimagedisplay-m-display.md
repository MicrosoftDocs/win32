# CImageDisplay::m\_Display member

The `m_Display` member is a [**VIDEOINFO**](/previous-versions/windows/desktop/api/amvideo/ns-amvideo-videoinfo) structure that describes the current display format.

## Syntax


```C++
VIDEOINFO  m_Display;
```
