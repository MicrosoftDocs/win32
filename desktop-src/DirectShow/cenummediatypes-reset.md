# CEnumMediaTypes.Reset method

The `Reset` method resets the enumeration sequence to the beginning. This method implements the [**IEnumMediaTypes::Reset**](/windows/desktop/api/Strmif/nf-strmif-ienummediatypes-reset) method.

## Syntax


```C++
HRESULT Reset();
```



## Parameters

This method has no parameters.

## Return value

Returns S\_OK.
