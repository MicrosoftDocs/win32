# CAMThread.ThreadExists method

The `ThreadExists` method queries whether the thread exists.

## Syntax


```C++
BOOL ThreadExists();
```



## Parameters

This method has no parameters.

## Return value

Returns **TRUE** if the thread exists, or **FALSE** if the thread does not exist.
