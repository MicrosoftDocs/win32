The `AddHead` method adds a list to the front of the list.

## Syntax


```C++
BOOL AddHead(
   CGenericList<OBJECT> *pList
);
```



## Parameters

<dl> <dt>

*pList* 
</dt> <dd>

Pointer to the list of items to add.

</dd> </dl>

## Return value

Returns **TRUE** if successful, or **FALSE** otherwise.
