# CBaseWindow.GetWindowWidth method

The `GetWindowWidth` method retrieves the current width of the window.

## Syntax


```C++
LONG GetWindowWidth();
```



## Parameters

This method has no parameters.

## Return value

Returns the width of the window, in pixels.
