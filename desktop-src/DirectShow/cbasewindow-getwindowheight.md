# CBaseWindow.GetWindowHeight method

The `GetWindowHeight` method retrieves the current height of the window.

## Syntax


```C++
LONG GetWindowHeight();
```



## Parameters

This method has no parameters.

## Return value

Returns the height of the window, in pixels.
