# AVEncMPVIntraVLCTable property

Specifies which variable-length coding (VLC) table to use for entropy coding. This property applies to MPEG video encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPVIntraVLCTable**

## Property value

The value of this property is a member of the [**eAVEncMPVIntraVLCTable**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvintravlctable) enumeration.
