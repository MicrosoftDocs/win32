# AVEncVideoInputColorLighting property

Specifies the intended lighting conditions for viewing the input video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoInputColorLighting**

## Property value

The value of this property is a member of the [**eAVEncVideoColorLighting**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting) enumeration.
