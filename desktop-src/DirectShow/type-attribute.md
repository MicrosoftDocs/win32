# type Attribute

> [!Note]  
> \[Deprecated. This API may be removed from future releases of Windows.\]

 

The `type` attribute specifies the media type of a group.

## Possible Values

The value must be "audio" or "video".

## Applies To

[**group**](group-element.md)
