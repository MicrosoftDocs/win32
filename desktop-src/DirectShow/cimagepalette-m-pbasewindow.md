# CImagePalette::m\_pBaseWindow member

The `m_pBaseWindow` member variable is a pointer to the [**CBaseWindow**](cbasewindow.md) object that manages the window.

## Syntax


```C++
CBaseWindow *m_pBaseWindow;
```
