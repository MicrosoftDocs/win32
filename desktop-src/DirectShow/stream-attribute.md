# stream Attribute

> [!Note]  
> \[Deprecated. This API may be removed from future releases of Windows.\]

 

The `stream` attribute specifies which stream to use in a multi-stream source file.

## Possible Values

Must be an integer value.

[**clip**](clip-element.md)
