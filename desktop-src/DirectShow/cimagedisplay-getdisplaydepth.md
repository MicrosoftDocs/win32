# CImageDisplay.GetDisplayDepth method

The `GetDisplayDepth` method retrieves the bit depth of the current display mode.

## Syntax


```C++
WORD GetDisplayDepth();
```



## Parameters

This method has no parameters.

## Return value

Returns the number of bits per pixel.
