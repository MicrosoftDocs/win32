# CBaseObject.ObjectsActive method

The `ObjectsActive` method retrieves a process-wide count of active objects.

## Syntax


```C++
static LONG ObjectsActive();
```



## Parameters

This method has no parameters.

## Return value

Returns the number of active objects.
