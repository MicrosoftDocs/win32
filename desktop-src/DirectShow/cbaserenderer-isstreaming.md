# CBaseRenderer.IsStreaming method

The `IsStreaming` method queries whether the filter is streaming data.

## Syntax


```C++
BOOL IsStreaming();
```



## Parameters

This method has no parameters.

## Return value

Returns the [**CBaseRenderer::m\_bStreaming**](cbaserenderer-m-bstreaming.md) flag.
