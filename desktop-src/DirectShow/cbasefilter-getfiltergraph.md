The `GetFilterGraph` method retrieves a pointer to the filter graph manager.

## Syntax


```C++
IFilterGraph* GetFilterGraph();
```



## Parameters

This method has no parameters.

## Return value

Returns the value of [**CBaseFilter::m\_pGraph**](cbasefilter-m-pgraph.md).
