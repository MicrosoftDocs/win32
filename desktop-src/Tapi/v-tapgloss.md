<span id="tapi2.vcomm_tapgloss"></span><span id="TAPI2.VCOMM_TAPGLOSS"></span>**VCOMM**
</dt> <dd>

A communications device driver within TAPI. VCOMM can assign protected mode to TAPI services and allows Windows applications and drivers to use ports and modems. See [*UniModem*](u-tapgloss.md).

</dd> <dt>

<span id="tapi2.voice_signal_tapgloss"></span><span id="TAPI2.VOICE_SIGNAL_TAPGLOSS"></span>**voice signal**
</dt> <dd>

Considered to be any signal that can travel over a 3.1 kHz-bandwidth channel.
