# LINEQOSREQUESTTYPE\_ Constants

These constants are used by a TSP to identify the type of a QoS (Quality of Service) request being made. Currently, only a service level request is supported. See [LINEQOSSERVICELEVEL\_ Constants](lineqosservicelevel--constants.md).

<dl> <dt>

<span id="LINEQOSREQUESTTYPE_SERVICELEVEL"></span><span id="lineqosrequesttype_servicelevel"></span>**LINEQOSREQUESTTYPE\_SERVICELEVEL**
</dt> <dd> <dl> <dt>

 0x00000001
</dt> <dt>



The QoS request being made is for a particular service level.


</dt> </dl> </dd> </dl>
