# Core Tablet PC Enumerations

This section contains the core enumerations for Tablet PC.

## In This Section



| Enumeration                                                                                  | Description                                                                                                                                                                                                                                                                               |

| [**InkApplicationGesture**](inkapplicationgesture.md)                                       | Specifies the interest in a set of application-specific gesture.<br/>                                                                                                                                                                                                               |

| [**InkCursorButtonState**](inkcursorbuttonstate.md)                                         | Specifies the state of a cursor button.<br/>                                                                                                                                                                                                                                        |

| [**InkRasterOperation**](inkrasteroperation.md)                                             | Defines values for performing raster operations on drawn ink.<br/>                                                                                                                                                                                                                  |

You also use this enumeration to determine which attributes to use when you search for an installed recognizer.<br/>                                                                                                                      |
