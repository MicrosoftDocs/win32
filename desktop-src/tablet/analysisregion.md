# AnalysisRegion class

Implements the [**IAnalysisRegion**](ianalysisregion.md) interface.

## Remarks

This class implements the [**IAnalysisRegion**](ianalysisregion.md) COM interface, and is equivalent to System.Windows.Ink.AnalysisCore.AnalysisRegionBase in the .NET Framework.
