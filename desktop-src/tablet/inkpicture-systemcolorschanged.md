# InkPicture.SystemColorsChanged event

Occurs after the system colors change.

## Syntax


```C++
void SystemColorsChanged();
```



## Parameters

This event has no parameters.

## Return value

This event does not return a value.

## Remarks

This event method is defined in the **\_IInkPictureEvents** interface. The **\_IInkPictureEvents** interface implements the [**IDispatch**](https://msdn.microsoft.com/en-us/library/ms221608(v=VS.71).aspx) interface with an identifier of DISPID\_IPESystemColorsChanged.
