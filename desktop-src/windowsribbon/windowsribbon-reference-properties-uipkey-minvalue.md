# UI\_PKEY\_MinValue

Identifies the UI\_PKEY\_MinValue property.

```
propertyDescription
   name = UI_PKEY_MinValue
   shellPKey = UI_PKEY_MinValue
   formatID = 00000204-7363-696e-8441798acf5aebb7
   propID = 204
   typeInfo
      type = VT_DECIMAL
```

## Remarks

UI\_PKEY\_MinValue is used by an application to query the minimum value of the [Spinner](windowsribbon-controls-spinner.md) control.
