# UI\_PKEY\_MaxValue

Identifies the UI\_PKEY\_MaxValue property.

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## Remarks

UI\_PKEY\_MaxValue is used by an application to query the maximum value of the [Spinner](windowsribbon-controls-spinner.md) control.
