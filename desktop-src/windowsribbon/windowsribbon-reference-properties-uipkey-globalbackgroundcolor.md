# UI\_PKEY\_GlobalBackgroundColor

Identifies the UI\_PKEY\_GlobalBackgroundColor property.

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## Remarks

UI\_PKEY\_GlobalBackgroundColor is used by an application to query the global background color for a customized ribbon.

**Introduced in Windows 8:** [UI\_PKEY\_GlobalHighlightColor](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) cannot be set independently of UI\_PKEY\_GlobalBackgroundColor.
