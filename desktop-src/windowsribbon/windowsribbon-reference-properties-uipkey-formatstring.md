# UI\_PKEY\_FormatString

Identifies the UI\_PKEY\_FormatString property.

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## Remarks

UI\_PKEY\_FormatString is used by an application to query the format (or unit of measurement) for the value of the [Spinner](windowsribbon-controls-spinner.md) control.
