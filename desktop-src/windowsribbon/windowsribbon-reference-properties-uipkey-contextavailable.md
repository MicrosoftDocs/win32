# UI\_PKEY\_ContextAvailable

Identifies the UI\_PKEY\_ContextAvailable property.

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## Remarks

UI\_PKEY\_ContextAvailable is used by an application to query the availability of a [Tab Group](windowsribbon-controls-tabgroup.md) control based on the current application context.
