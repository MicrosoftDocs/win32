# UI\_PKEY\_DecimalPlaces

Identifies the UI\_PKEY\_DecimalPlaces property.

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## Remarks

UI\_PKEY\_DecimalPlaces is used by an application to query the number of digits shown after the decimal point in the edit field of the [Spinner](windowsribbon-controls-spinner.md) control.
