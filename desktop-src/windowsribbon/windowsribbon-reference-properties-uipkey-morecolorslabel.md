# UI\_PKEY\_MoreColorsLabel

Identifies the UI\_PKEY\_MoreColorsLabel property.

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## Remarks

UI\_PKEY\_MoreColorsLabel is used by an application to query the value of the label associated with the **More colors** button of a [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

UI\_PKEY\_MoreColorsLabel is only valid for a [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) where `ThemeColors` or `StandardColors` is specified as the value of the **ColorTemplate** attribute.
