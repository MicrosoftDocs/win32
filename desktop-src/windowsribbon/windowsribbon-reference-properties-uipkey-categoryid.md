# UI\_PKEY\_CategoryId

Identifies the UI\_PKEY\_CategoryId property.

```
propertyDescription
   name = UI_PKEY_CategoryId
   shellPKey = UI_PKEY_CategoryId
   formatID = 00000103-7363-696e-8441798acf5aebb7
   propID = 103
   typeInfo
      type = VT_UI4
```

## Remarks

UI\_PKEY\_CategoryId is used by an application to query the IDs of the categories that are used to group related items in a gallery control.

The property value is the zero-based index of an item in a category collection.

The following screen shot illustrates the use of categories in a [**SplitButton**](windowsribbon-element-splitbutton.md) menu where **Selection shapes** has an ID of 0 and **Selection options** has an ID of 1.
