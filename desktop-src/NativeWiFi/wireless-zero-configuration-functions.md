# Wireless Zero Configuration Functions

\[The Wireless Zero Configuration programming interface no longer supported as of Windows Vista and Windows Server 2008. Instead, use the [Native Wifi API](native-wifi-reference.md), which provides similar functionality. For more information, see [About the Native Wifi API](about-the-native-wifi-api.md).\]

The following functions enable software developers to use the functionality provided by the Wireless Zero Configuration module.

-   [**WZCEapolGetInterfaceParams**](wzceapolgetinterfaceparams.md)
-   [**WZCEnumInterfaces**](wzcenuminterfaces.md)
-   [**WZCQueryInterface**](wzcqueryinterface.md)
-   [**WZCRefreshInterface**](wzcrefreshinterface.md)
