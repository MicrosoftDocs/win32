# name (WLANPolicy) Element

The name (WLANPolicy) element contains the name of a wireless LAN policy.

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

The **name** element is defined by the [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md) element.

## Remarks

Names are not case-sensitive.
