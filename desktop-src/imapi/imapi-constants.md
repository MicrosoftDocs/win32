# IMAPI Constants

IMAPI defines the following constants.



| Constant                                                                                                                                                                                                                          | Description                                                                              |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------|
| <span id="IMAPI_SECTOR_SIZE"></span><span id="imapi_sector_size"></span><dl> <dt>**IMAPI\_SECTOR\_SIZE**</dt> </dl>                                                        | Number of bytes in a sector.<br/>                                                  |
