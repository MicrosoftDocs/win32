# WER Functions

The following functions are used in Windows Error Reporting. For examples that use these functions, see samples in the Windows SDK under WinBase\\WindowsErrorReporting and WinBase\\Debug\\FailFast.

## In this section



| Topic                                                                                                               | Description                                                                                                                                                                                                                                                                                                                        |
|---------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddERExcludedApplication**](adderexcludedapplication.md)<br/>                                             | Excludes the specified application from error reporting.<br/>                                                                                                                                                                                                                                                                |
| [*PFN\_WER\_RUNTIME\_EXCEPTION\_EVENT*](outofprocessexceptioneventcallback.md)<br/>                          | WER calls this function to determine whether the exception handler is claiming the crash.<br/>                                                                                                                                                                                                                               |
| [*PFN\_WER\_RUNTIME\_EXCEPTION\_DEBUGGER\_LAUNCH*](outofprocessexceptioneventdebuggerlaunchcallback.md)<br/> | WER calls this function to let you customize the debugger launch options and launch string.<br/>                                                                                                                                                                                                                             |
| [*PFN\_WER\_RUNTIME\_EXCEPTION\_EVENT\_SIGNATURE*](outofprocessexceptioneventsignaturecallback.md)<br/>      | WER can call this function multiple times to get the report parameters that uniquely describe the problem.<br/>                                                                                                                                                                                                              |
| [**ReportFault**](reportfault.md)<br/>                                                                       | Enables an application that performs its own exception handling to report faults to Microsoft.<br/>                                                                                                                                                                                                                          |
| [**WerAddExcludedApplication**](weraddexcludedapplication.md)<br/>                                           | Adds the specified application to the list of applications that are to be excluded from error reporting.<br/>                                                                                                                                                                                                                |
| [**WerFreeString**](werfreestring.md)<br/>                                                                   | Frees up the memory used to store a report key string. This should be called after each successive call to [**WerStoreGetFirstReportKey**](werstoregetfirstreportkey.md) or [**WerStoreGetNextReportKey**](werstoregetnextreportkey.md), once the particular report key string has been used and is no longer needed.<br/> |
| [**WerGetFlags**](wergetflags.md)<br/>                                                                       | Retrieves the fault reporting settings for the specified process.<br/>                                                                                                                                                                                                                                                       |
| [**WerRegisterAdditionalProcess**](werregisteradditionalprocess.md)<br/>                                     | Registers a process to be included in the error report along with the main application process. Optionally specifies a thread within that registered process to get additional data from.<br/>                                                                                                                               |
| [**WerRegisterAppLocalDump**](werregisterapplocaldump.md)<br/>                                               | Registers a path relative to the local app store for the calling application where Windows Error Reporting (WER) should save a copy of the diagnostic memory dump that WER collects when one of the processes for the application stops responding.<br/>                                                                     |
| [**WerRegisterCustomMetadata**](werregistercustommetadata.md)<br/>                                           | Registers app-specific metadata to be collected (in the form of key/value strings) when WER creates an error report.<br/>                                                                                                                                                                                                    |
| [**WerRegisterExcludedMemoryBlock**](werregisterexcludedmemoryblock.md)<br/>                                 | Marks a memory block (that is normally included by default in error reports) to be excluded from the error report.<br/>                                                                                                                                                                                                      |
| [**WerRegisterFile**](werregisterfile.md)<br/>                                                               | Registers a file to be collected when WER creates an error report.<br/>                                                                                                                                                                                                                                                      |
| [**WerRegisterMemoryBlock**](werregistermemoryblock.md)<br/>                                                 | Registers a memory block to be collected when WER creates an error report.<br/>                                                                                                                                                                                                                                              |
| [**WerRegisterRuntimeExceptionModule**](werregisterruntimeexceptionmodule.md)<br/>                           | Registers a custom runtime exception handler that is used to provide custom error reporting for crashes.<br/>                                                                                                                                                                                                                |
| [**WerRemoveExcludedApplication**](werremoveexcludedapplication.md)<br/>                                     | Removes the specified application from the list of applications that are to be excluded from error reporting.<br/>                                                                                                                                                                                                           |
| [**WerReportAddDump**](werreportadddump.md)<br/>                                                             | Adds a dump of the specified type to the specified report.<br/>                                                                                                                                                                                                                                                              |
| [**WerReportAddFile**](werreportaddfile.md)<br/>                                                             | Adds a file to the specified report.<br/>                                                                                                                                                                                                                                                                                    |
| [**WerReportCloseHandle**](werreportclosehandle.md)<br/>                                                     | Closes the specified report.<br/>                                                                                                                                                                                                                                                                                            |
| [**WerReportCreate**](werreportcreate.md)<br/>                                                               | Creates a problem report that describes an application event.<br/>                                                                                                                                                                                                                                                           |
| [**WerReportHang**](werreporthang.md)<br/>                                                                   | Initiates "no response" reporting on the specified window.<br/>                                                                                                                                                                                                                                                              |
| [**WerReportSetParameter**](werreportsetparameter.md)<br/>                                                   | Sets the parameters that uniquely identify an event for the specified report.<br/>                                                                                                                                                                                                                                           |
| [**WerReportSetUIOption**](werreportsetuioption.md)<br/>                                                     | Sets the user interface options for the specified report.<br/>                                                                                                                                                                                                                                                               |
| [**WerReportSubmit**](werreportsubmit.md)<br/>                                                               | Submits the specified report.<br/>                                                                                                                                                                                                                                                                                           |
| [**WerSetFlags**](wersetflags.md)<br/>                                                                       | Sets the fault reporting settings for the current process.<br/>                                                                                                                                                                                                                                                              |
| [**WerStoreClose**](werstoreclose.md)<br/>                                                                   | Closes the collection of stored reports.<br/>                                                                                                                                                                                                                                                                                |
| [**WerStoreGetFirstReportKey**](werstoregetfirstreportkey.md)<br/>                                           | Gets a reference to the first report in the report store.<br/>                                                                                                                                                                                                                                                               |
| [**WerStoreGetNextReportKey**](werstoregetnextreportkey.md)<br/>                                             | Gets a reference to the next report in the error report store.<br/>                                                                                                                                                                                                                                                          |
| [**WerStoreOpen**](werstoreopen.md)<br/>                                                                     | Opens the collection of stored error reports.<br/>                                                                                                                                                                                                                                                                           |
| [**WerStoreQueryReportMetadataV2**](werstorequeryreportmetadatav2.md)<br/>                                   | Retrieves metadata about a report in the store.<br/>                                                                                                                                                                                                                                                                         |
| [**WerUnregisterAdditionalProcess**](werunregisteradditionalprocess.md)<br/>                                 | Removes a process from the list of additional processes to be included in the error report.<br/>                                                                                                                                                                                                                             |
| [**WerUnregisterAppLocalDump**](werunregisterapplocaldump.md)<br/>                                           | Cancels the registration that was made by calling the [**WerRegisterAppLocalDump**](werregisterapplocaldump.md) function to specify that Windows Error Reporting (WER) should save a copy of the diagnostic memory dump that WER collects when one of the processes for the application stops responding.<br/>              |
| [**WerUnregisterCustomMetadata**](werunregistercustommetadata.md)<br/>                                       | Removes an item of app-specific metadata being collected during error reporting for the application.<br/>                                                                                                                                                                                                                    |
| [**WerUnregisterExcludedMemoryBlock**](werunregisterexcludedmemoryblock.md)<br/>                             | Removes a memory block that was previously marked as excluded (it will again be included in error reports).<br/>                                                                                                                                                                                                             |
| [**WerUnregisterFile**](werunregisterfile.md)<br/>                                                           | Removes a file from the list of files to be added to reports generated for the current process.<br/>                                                                                                                                                                                                                         |
| [**WerUnregisterMemoryBlock**](werunregistermemoryblock.md)<br/>                                             | Removes a memory block from the list of data to be collected during error reporting for the application.<br/>                                                                                                                                                                                                                |
| [**WerUnregisterRuntimeExceptionModule**](werunregisterruntimeexceptionmodule.md)<br/>                       | Removes the registration of your WER exception handler.<br/>                                                                                                                                                                                                                                                                 |
