# MF\_PD\_TOTAL\_FILE\_SIZE attribute

Specifies the total size of the source file, in bytes. This attribute applies to presentation descriptors. A media source can optionally set this attribute.

## Data type

**UINT64**

## Remarks

The GUID constant for this attribute is exported from mfuuid.lib.
