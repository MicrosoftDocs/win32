# MF\_MT\_AUDIO\_FLOAT\_SAMPLES\_PER\_SECOND attribute

Number of audio samples per second in an audio media type.

## Data type

**double**

## Remarks

The GUID constant for this attribute is exported from mfuuid.lib.
