# MFPKEY\_PASSESUSED Property

Specifies the number of passes that the encoder will use to encode the content. Read-write.

## Constant for IPropertyBag

g\_wszWMVCPassesUsed

## Data Type

VT\_I4

## Default Value

1

## Remarks

This value cannot exceed the value of [MFPKEY\_PASSESRECOMMENDED](mfpkey-passesrecommendedproperty.md).
