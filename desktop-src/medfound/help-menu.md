# Help Menu

The **Help** menu provides the following commands:



| Command      | Description                                                                      |
|--------------|----------------------------------------------------------------------------------|
| **Help...**  | Opens TopoEdit help on the MSDN website.                                         |
| **About...** | Opens the **About** dialog box that displays version information about TopoEdit. |
