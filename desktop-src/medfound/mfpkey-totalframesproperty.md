# MFPKEY\_TOTALFRAMES Property

Specifies the total number of video frames passed to the encoder during the endocing process.

## Constant for IPropertyBag

g\_wszWMVCTotalFrames

## Data Type

VT\_I4

## Remarks

You can get this value after you are finished passing samples.
