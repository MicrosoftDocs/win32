# MFPKEY\_DYN\_BANDTRUNC\_QFLOOR Property

Specifies the quality at which maximum coded bandwidth is allowed. At quality better than or equal to the value of this property, maximum coded bandwidth is allowed.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](https://msdn.microsoft.com/en-us/library/Bb761474(v=VS.85).aspx).

## Data Type

**VT\_R4**
