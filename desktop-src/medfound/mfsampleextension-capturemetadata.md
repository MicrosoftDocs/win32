# MFSampleExtension\_CaptureMetadata attribute

The [**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes) store for all the metadata related to the capture pipeline.

## Data type

**[**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes)** stored as **IUnknown** 

## Remarks

This attribute can be present on any [**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample).

This attribute is read-only.
