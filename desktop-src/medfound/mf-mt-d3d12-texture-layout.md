# MF\_MT\_D3D12\_TEXTURE\_LAYOUT attribute

Indicates the texture layout options that were used to create the associated Direct3D resources.

## Data type

**UINT32**

## Remarks

The value of this attribute corresponds to a member of the [D3D12_TEXTURE_LAYOUT](/windows/win32/api/d3d12/ne-d3d12-d3d12_texture_layout) enumeration.
