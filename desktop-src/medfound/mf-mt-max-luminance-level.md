# MF\_MT\_MAX\_LUMINANCE\_LEVEL attribute

Specifies the maximum luminance level of the content in Nits. This attribute has the same semantics as **MaxCLL** as defined in the CEA-861.3 standard.

## Data type

**UINT32**
