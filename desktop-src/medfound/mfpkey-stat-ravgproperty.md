# MFPKEY\_STAT\_RAVG Property

Specifies the average bit rate, in bits per second, of an encoded stream. Read-only.

## Constant for IPropertyBag

**g\_wszWMVCAvgBitrate**

## Data Type

**VT\_I4**

## Remarks

To determine the average bit rate of an encoded stream, read this property at the end of encoding.
