# MF\_DISABLE\_LOCALLY\_REGISTERED\_PLUGINS attribute

Specifies if locally registered plugins are disabled.

## Data type

**UINT32**

## Remarks

Set this attribute to **TRUE** to prevent the MF Topology Loader from inserting locally registered transforms.
