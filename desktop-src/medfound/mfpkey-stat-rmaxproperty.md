# MFPKEY\_STAT\_RMAX Property

Specifies the maximum bit rate, in bits per second, of an encoded stream. Read-only.

## Constant for IPropertyBag

**g\_wszWMVCMaxBitrate**

## Data Type

**VT\_I4**

## Remarks

To determine the maximum bit rate of an encoded stream, read this property at the end of encoding.
