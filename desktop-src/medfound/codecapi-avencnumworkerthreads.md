# CODECAPI\_AVEncNumWorkerThreads property

Sets the number of worker threads used by a video encoder.

## Data type

**ULONG** (VT\_UI4)

## Property GUID

**CODECAPI\_AVEncNumWorkerThreads**

## Remarks

If the value is zero, the encoder selects the number of threads.
