# CODECAPI\_AVDecSoftwareDynamicFormatChange property

Sets whether to use software dynamic format change for internal resizing.

## Data type

**BOOL**

## Property GUID

**CODECAPI\_AVDecSoftwareDynamicFormatChange**

## Property value

If set to **TRUE**, software dynamic format change is used for internal resizing.
