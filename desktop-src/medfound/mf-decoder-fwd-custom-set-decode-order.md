# MF\_DECODER\_FWD\_CUSTOM\_SEI\_DECODE\_ORDER attribute

Specifies that the SEI unit type to forward on output samples of the decoder shall be sent out in decode order (i.e. ahead of time). This is required for downstream apps to process the SEI in advance of receiving the frame it is meant to be attached to.

## Data type

**UINT32**
