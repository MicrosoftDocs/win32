# MF\_MT\_D3D\_RESOURCE\_VERSION attribute

Specifies the Direct3D version of the resources stored in the data stream associated with the media type. 

## Data type

**UINT32**

## Remarks

The value of this attribute corresponds to a member of the [MF_MT_D3D_RESOURCE_VERSION_ENUM](/windows/win32/api/mfd3d12/ne-mfd3d12-mf_mt_d3d_resource_version_enum) enumeration.
