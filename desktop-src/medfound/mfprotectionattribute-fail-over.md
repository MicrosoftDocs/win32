# MFPROTECTIONATTRIBUTE\_FAIL\_OVER attribute

Indicates whether the protection fails over to this if the best effort fails. This attribute can be used with [**IMFOutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) objects.

## Data type

**BOOL** stored as **UINT32**

## Remarks

This attribute can be used with [**IMFOutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) objects.
