# MF\_MT\_D3D12\_CPU\_READBACK attribute

Indicates whether CPU access is required for the associated Direct3D resources. 

## Data type

**UINT32**

## Remarks

The default value of this attribute is 0 (FALSE).

|-------------------------------------|------------------------------------------------------------------------------------|
