# MFPKEY\_STAT\_BMAX Property

Specifies the maximum buffer window, in milliseconds, of an encoded stream. Read-only.

## Constant for IPropertyBag

**g\_wszWMVCMaxBitrate**

## Data Type

**VT\_I4**

## Remarks

To determine the maximum buffer windows of an encoded stream, read this property at the end of encoding.
