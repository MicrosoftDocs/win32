# MF\_MSE\_BUFFERLIST\_CALLBACK attribute

Contains a pointer to the application's callback interface for the [**IMFBufferListNotify**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfbufferlistnotify).

## Data type

**IMFBufferListNotify\*** stored as **IUnknown\***
