# MF\_SA\_REQUIRED\_SAMPLE\_COUNT\_PROGRESSIVE attribute

Indicates the number of samples that a Microsoft Media Foundation transform (MFT) requires to be allocated for progressive content.

## Data type

**UINT32**

## Remarks

This value is used if the next node downstream has an [**IMFVideoSampleAllocator**](/windows/desktop/api/mfidl/nn-mfidl-imfvideosampleallocator).
