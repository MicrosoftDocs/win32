# IMFMediaSourceExtension::GetDuration method

Gets the duration of the media source in 100-nanosecond units.

## Syntax


```C++
double GetDuration();
```



## Parameters

This method has no parameters.

## Return value

The duration of the media source in 100-nanosecond units.
