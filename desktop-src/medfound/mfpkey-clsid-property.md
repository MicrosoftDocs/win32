# MFPKEY\_CLSID property

Contains the CLSID for a Media Foundation transform (MFT).



Data type

PROPVARIANT type (vt)

PROPVARIANT member

**GUID** (**CLSID**\*)

VT\_CLSID

**puuid**



## Remarks

This property is optional and is informational only. To get this property, query the MFT for the **IPropertyStore** interface.
