# MFPKEY\_WMAVOICE\_ENC\_BufferWindow Property

Specifies the buffer window, in milliseconds, to be used for the voice codec.

## Constant for IPropertyBag

g\_wszWMACVoiceBuffer

## Data Type

VT\_I4

## Default Value

1500 for 4, 5, 8, 12, and 16 Kbps

2177 for 10 and 20 Kbps
