# MFSampleExtension\_Encryption\_HardwareProtection attribute

Specifies whether a media sample is hardware protected.

## Data type

**BOOL** stored as **UINT32**

## Remarks

A value of 0 indicates that the sample is not hardware protected. A non-zero value indicates the sample is hardware protected.
