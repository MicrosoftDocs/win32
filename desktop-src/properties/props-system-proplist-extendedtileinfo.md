# System.PropList.ExtendedTileInfo

The list of properties to show in the listview on extended tiles. Register under the regvalue of "ExtendedTileInfo".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.ExtendedTileInfo
   shellPKey = PKEY_PropList_ExtendedTileInfo
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
