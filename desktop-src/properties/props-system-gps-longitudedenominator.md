# System.GPS.LongitudeDenominator

The denominator of PKEY\_GPS\_Longitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.LongitudeDenominator
   shellPKey = PKEY_GPS_LongitudeDenominator
   formatID = BE6E176C-4534-4D2C-ACE5-31DEDAC1606B
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt32
```

## Remarks

PKEY values are defined in Propkey.h.
