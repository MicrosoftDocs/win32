# System.Communication.PolicyTag

This property identifies the retention policy applied to the item.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.PolicyTag
   shellPKey = PKEY_Communication_PolicyTag
   formatID = EC0B4191-AB0B-4C66-90B6-C6637CDEBBAB
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
