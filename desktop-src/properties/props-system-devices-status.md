# System.Devices.Status

Array of device status strings

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Status
   shellPKey = PKEY_Devices_Status
   formatID = D08DD4C0-3A9E-462E-8290-7B636B2576B9
   propID = 259
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

PKEY values are defined in Propkey.h.
