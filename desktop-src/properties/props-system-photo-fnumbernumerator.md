# System.Photo.FNumberNumerator

The numerator of PKEY\_Photo\_FNumber.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FNumberNumerator
   shellPKey = PKEY_Photo_FNumberNumerator
   formatID = 1B97738A-FDFC-462F-9D93-1957E08BE90C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
