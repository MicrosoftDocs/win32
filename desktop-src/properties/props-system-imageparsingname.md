# System.ImageParsingName

Image parsing name.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.ImageParsingName
   shellPKey = PKEY_ImageParsingName
   formatID = D7750EE0-C6A4-48EC-B53E-B87B52E6D073
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## Windows 8, Windows 7

```
propertyDescription
   name = System.ImageParsingName
   shellPKey = PKEY_ImageParsingName
   formatID = D7750EE0-C6A4-48EC-B53E-B87B52E6D073
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## Remarks

PKEY values are defined in Propkey.h.
