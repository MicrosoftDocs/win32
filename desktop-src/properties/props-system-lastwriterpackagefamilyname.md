# System.LastWriterPackageFamilyName

Mark of the app container. The package family name of the last app to edit the file's contents.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.LastWriterPackageFamilyName
   shellPKey = PKEY_LastWriterPackageFamilyName
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
