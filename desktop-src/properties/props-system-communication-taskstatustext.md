# System.Communication.TaskStatusText

The user-friendly form of [System.Communication.TaskStatus](./props-system-communication-taskstatus.md). This value is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.TaskStatusText
   shellPKey = PKEY_Communication_TaskStatusText
   formatID = A6744477-C237-475B-A075-54F34498292A
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
