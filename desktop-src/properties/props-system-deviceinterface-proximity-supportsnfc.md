# System.DeviceInterface.Proximity.SupportsNfc

Indicates whether the device supports NFC communications (NDEF).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.DeviceInterface.Proximity.SupportsNfc
   shellPKey = PKEY_DeviceInterface_Proximity_SupportsNfc
   formatID = FB3842CD-9E2A-4F83-8FCC-4B0761139AE9
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
