# System.ParsingName

The Shell namespace name of an item relative to a parent folder.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParsingName
   shellPKey = PKEY_ParsingName
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 24
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.

This value can be passed to the [**ParseDisplayName**](/windows/win32/api/shobjidl_core/nf-shobjidl_core-ishellfolder-parsedisplayname) method of the [**IShellFolder**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellfolder) object that represents the parent Shell folder.
