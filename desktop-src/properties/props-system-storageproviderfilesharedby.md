# System.StorageProviderFileSharedBy

The display name of the user who last shared the file or folder, specified by the storage provider.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileSharedBy
   shellPKey = PKEY_StorageProviderFileSharedBy
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 15
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
