# System.PercentFull

The amount of space filled, as a percentage.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PercentFull
   shellPKey = PKEY_PercentFull
   formatID = 9B174B35-40FF-11D2-A27E-00C04FC30871
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.

The valid range for this property is 0 through 100.
