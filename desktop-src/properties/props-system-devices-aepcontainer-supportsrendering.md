# System.Devices.AepContainer.SupportsRendering

Indicates if the Device Association Endpoint Container Supports Rendering

## Windows 10, version 1703, Windows 10, version 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsRendering
   shellPKey = PKEY_Devices_AepContainer_SupportsRendering
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
