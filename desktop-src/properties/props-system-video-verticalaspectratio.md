# System.Video.VerticalAspectRatio

Indicates the horizontal portion of the pixel aspect ratio. The Y portion of XX:YY. For example, 11 is the Y portion of 10:11 .

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Video.VerticalAspectRatio
   shellPKey = PKEY_Video_VerticalAspectRatio
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 45
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## Remarks

PKEY values are defined in Propkey.h.
