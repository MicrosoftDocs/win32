# System.GPS.TrackDenominator

The denominator of PKEY\_GPS\_Track.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.TrackDenominator
   shellPKey = PKEY_GPS_TrackDenominator
   formatID = C8D1920C-01F6-40C0-AC86-2F3A4AD00770
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
