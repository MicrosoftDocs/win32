# System.Photo.SharpnessText

The user-friendly form of [System.Photo.Sharpness](./props-system-photo-sharpness.md). Not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.SharpnessText
   shellPKey = PKEY_Photo_SharpnessText
   formatID = 51EC3F47-DD50-421D-8769-334F50424B1E
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
