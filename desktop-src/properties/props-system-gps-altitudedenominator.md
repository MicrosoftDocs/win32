# System.GPS.AltitudeDenominator

The denominator of PKEY\_GPS\_Altitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.AltitudeDenominator
   shellPKey = PKEY_GPS_AltitudeDenominator
   formatID = 78342DCB-E358-4145-AE9A-6BFE4E0F9F51
   propID = 100
   SearchInfo
```
