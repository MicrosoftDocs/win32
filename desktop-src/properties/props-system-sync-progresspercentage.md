# System.Sync.ProgressPercentage

An integer value between 0 and 100 that represents the percentage completed.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Sync.ProgressPercentage
   shellPKey = PKEY_Sync_ProgressPercentage
   formatID = 7BD5533E-AF15-44DB-B8C8-BD6624E1D032
   propID = 23
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```

## Remarks

PKEY values are defined in Propkey.h.
