# System.StorageProviderFullyQualifiedId

This property represents the entire fully-qualified provider identifier.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFullyQualifiedId
   shellPKey = PKEY_StorageProviderFullyQualifiedId
   formatID = FCEFF153-E839-4CF3-A9E7-EA22832094B8
   propID = 119
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.

The format is "[Storage Provider ID]![Windows SID]![Account ID]".
