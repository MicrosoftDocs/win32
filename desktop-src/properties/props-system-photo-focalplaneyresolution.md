# System.Photo.FocalPlaneYResolution

Indicates the number of pixels in the image height (Y direction) per FocalPlaneResolutionUnit on the camera focal plane. Calculated from PKEY\_Photo\_FocalPlaneYResolutionNumerator and PKEY\_Photo\_FocalPlaneYResolutionDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalPlaneYResolution
   shellPKey = PKEY_Photo_FocalPlaneYResolution
   formatID = 4FFFE4D0-914F-4AC4-8D6F-C9C61DE169B1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
