# System.Null

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Null
   shellPKey = PKEY_Null
   formatID = 00000000-0000-0000-0000-000000000000
   propID = 0
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Null
      IsInnate = true
```
