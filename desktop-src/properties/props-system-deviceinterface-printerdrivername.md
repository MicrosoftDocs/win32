# System.DeviceInterface.PrinterDriverName

The name of the printer driver file.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DeviceInterface.PrinterDriverName
   shellPKey = PKEY_DeviceInterface_PrinterDriverName
   formatID = AFC47170-14F5-498C-8F30-B0D19BE449C6
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
