# System.GPS.ImgDirection

Indicates the direction of the image when it was captured. Calculated from PKEY\_GPS\_ImgDirectionNumerator and PKEY\_GPS\_ImgDirectionDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.ImgDirection
   shellPKey = PKEY_GPS_ImgDirection
   formatID = 16473C91-D017-4ED9-BA4D-B6BAA55DBCF8
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
