# System.Photo.ProgramModeText

The user-friendly form of [System.Photo.ProgramMode](./props-system-photo-programmode.md). Not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ProgramModeText
   shellPKey = PKEY_Photo_ProgramModeText
   formatID = 7FE3AA27-2648-42F3-89B0-454E5CB150C3
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
