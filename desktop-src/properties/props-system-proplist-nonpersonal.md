# System.PropList.NonPersonal

The list of properties considered 'non-personal'. The system will leave these properties untouched when directed to remove all non-personal properties from a given file. Register under the regvalue of "NonPersonal".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.NonPersonal
   shellPKey = PKEY_PropList_NonPersonal
   formatID = 49D1091F-082E-493F-B23F-D2308AA9668C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
