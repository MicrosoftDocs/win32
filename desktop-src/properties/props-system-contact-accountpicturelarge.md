# System.Contact.AccountPictureLarge

This is a stream containing the data necessary to render a contact's large account picture.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Contact.AccountPictureLarge
   shellPKey = PKEY_Contact_AccountPictureLarge
   formatID = 0B8BB018-2725-4B44-92BA-7933AEB2DDE7
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Stream
```

## Remarks

PKEY values are defined in Propkey.h.
