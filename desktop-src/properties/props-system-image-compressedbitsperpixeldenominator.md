# System.Image.CompressedBitsPerPixelDenominator

The denominator of PKEY\_Image\_CompressedBitsPerPixel.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.CompressedBitsPerPixelDenominator
   shellPKey = PKEY_Image_CompressedBitsPerPixelDenominator
   formatID = 1F8844E1-24AD-4508-9DFD-5326A415CE02
   propID = 100
   SearchInfo
      InInvertedIndex = false

      type = UInt32
```
