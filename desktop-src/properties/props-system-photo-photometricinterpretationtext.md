# System.Photo.PhotometricInterpretationText

The user-friendly form of [System.Photo.PhotometricInterpretation](./props-system-photo-photometricinterpretation.md). Not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.PhotometricInterpretationText
   shellPKey = PKEY_Photo_PhotometricInterpretationText
   formatID = 821437D6-9EAB-4765-A589-3B1CBBD22A61
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
