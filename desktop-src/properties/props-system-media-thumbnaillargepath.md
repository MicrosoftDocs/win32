# System.Media.ThumbnailLargePath

Filesystem path to the large thumbnail representation of the media item.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Media.ThumbnailLargePath
   shellPKey = PKEY_Media_ThumbnailLargePath
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 47
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
