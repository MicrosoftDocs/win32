# System.FlagColorText

The user-friendly form of [System.FlagColor](./props-system-flagcolor.md). This value is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.FlagColorText
   shellPKey = PKEY_FlagColorText
   formatID = 45EAE747-8E2A-40AE-8CBF-CA52ABA6152A
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
