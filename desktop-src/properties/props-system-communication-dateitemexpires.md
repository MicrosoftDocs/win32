# System.Communication.DateItemExpires

The date the item expires due to the retention policy.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.DateItemExpires
   shellPKey = PKEY_Communication_DateItemExpires
   formatID = 428040AC-A177-4C8A-9760-F6F761227F9A
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

PKEY values are defined in Propkey.h.
