# System.DeviceInterface.Hid.UsageId

HID device Usage Id.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Hid.UsageId
   shellPKey = PKEY_DeviceInterface_Hid_UsageId
   formatID = CBF38310-4A17-4310-A1EB-247F0B67593B
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

PKEY values are defined in Propkey.h.
