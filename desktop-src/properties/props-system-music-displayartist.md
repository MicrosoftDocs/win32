# System.Music.DisplayArtist

This property returns the best representation of the album artist for a specific music file based upon [System.Music.AlbumArtist](./props-system-music-albumartist.md), [System.Music.Artist](./props-system-music-artist.md), and [System.Music.IsCompilation](./props-system-music-iscompilation.md) information.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Music.DisplayArtist
   shellPKey = PKEY_Music_DisplayArtist
   formatID = FD122953-FA93-4EF7-92C3-04C946B2F7C8
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
