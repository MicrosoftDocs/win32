# System.Security.EncryptionOwnersDisplay

File ownership

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511

```
propertyDescription
   name = System.Security.EncryptionOwnersDisplay
   shellPKey = PKEY_Security_EncryptionOwnersDisplay
   formatID = DE621B8F-E125-43A3-A32D-5665446D632A
   propID = 25
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## Remarks

PKEY values are defined in Propkey.h.
