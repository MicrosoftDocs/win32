# System.IsShared

Indicates whether the item is shared. This checks only the non-inherited ACLs.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.IsShared
   shellPKey = PKEY_IsShared
   formatID = EF884C5B-2BFE-41BB-AAE5-76EEDF4F9902
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.IsShared
   shellPKey = PKEY_IsShared
   formatID = EF884C5B-2BFE-41BB-AAE5-76EEDF4F9902
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
