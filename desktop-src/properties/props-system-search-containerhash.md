# System.Search.ContainerHash

The hash code used to identify attachments to be deleted based on a common container url.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.ContainerHash
   shellPKey = PKEY_Search_ContainerHash
   formatID = BCEEE283-35DF-4D53-826A-F36A3EEFC6BE
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
