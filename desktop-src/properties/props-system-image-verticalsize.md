# System.Image.VerticalSize

The vertical size of the image, in pixels.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Image.VerticalSize
   shellPKey = PKEY_Image_VerticalSize
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.Image.VerticalSize
   shellPKey = PKEY_Image_VerticalSize
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 4
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

PKEY values are defined in Propkey.h.
