# System.StorageProviderFileChecksum

The checksum computed by the storage provider for the file. Files with the same checksum value will have the same contents.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.StorageProviderFileChecksum
   shellPKey = PKEY_StorageProviderFileChecksum
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
