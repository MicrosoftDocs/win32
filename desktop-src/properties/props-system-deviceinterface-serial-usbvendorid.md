# System.DeviceInterface.Serial.UsbVendorId

Serial device USB Vendor Id.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.DeviceInterface.Serial.UsbVendorId
   shellPKey = PKEY_DeviceInterface_Serial_UsbVendorId
   formatID = 4C6BF15C-4C03-4AAC-91F5-64C0F852BCF4
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
