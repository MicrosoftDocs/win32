# System.RecordedTV.StationCallSign

Any recorded station call signs. For example, "TOONP".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.RecordedTV.StationCallSign
   shellPKey = PKEY_RecordedTV_StationCallSign
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
   propID = 5
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
