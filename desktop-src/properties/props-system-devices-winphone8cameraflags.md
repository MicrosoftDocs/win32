# System.Devices.WinPhone8CameraFlags

Flags for a WP8 camera device.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WinPhone8CameraFlags
   shellPKey = PKEY_Devices_WinPhone8CameraFlags
   formatID = B7B4D61C-5A64-4187-A52E-B1539F359099
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
