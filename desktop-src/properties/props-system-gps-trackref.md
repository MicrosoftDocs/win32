# System.GPS.TrackRef

Indicates reference for the direction of the GPS receiver movement (for example, true direction, magnetic direction).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.TrackRef
   shellPKey = PKEY_GPS_TrackRef
   formatID = 35DBE6FE-44C3-4400-AAAE-D2C799C407E8
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
