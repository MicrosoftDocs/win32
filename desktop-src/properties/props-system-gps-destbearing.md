# System.GPS.DestBearing

Indicates the bearing to the destination point. Calculated from PKEY\_GPS\_DestBearingNumerator and PKEY\_GPS\_DestBearingDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestBearing
   shellPKey = PKEY_GPS_DestBearing
   formatID = C66D4B3C-E888-47CC-B99F-9DCA3EE34DEA
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
