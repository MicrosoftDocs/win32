# System.Calendar.IsRecurring

Indicates if the event will recur.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Calendar.IsRecurring
   shellPKey = PKEY_Calendar_IsRecurring
   formatID = 315B9C8D-80A9-4EF9-AE16-8E746DA51D70
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## Windows Vista

```
propertyDescription
   name = System.Calendar.IsRecurring
   shellPKey = PKEY_Calendar_IsRecurring
   formatID = 315B9C8D-80A9-4EF9-AE16-8E746DA51D70
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
