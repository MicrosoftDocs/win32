# System.Devices.Notifications.LowBattery

Device low battery notification.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Notifications.LowBattery
   shellPKey = PKEY_Devices_Notifications_LowBattery
   formatID = C4C07F2B-8524-4E66-AE3A-A6235F103BEB
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = LowBatteryNotification
            minValue = 0
            setValue = 0
            text = Charge your device or it may disconnect.
```
