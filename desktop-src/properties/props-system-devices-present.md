# System.Devices.Present

Device is present.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Present
   shellPKey = PKEY_Devices_Present
   formatID = 540B947E-8B40-45BC-A8A2-6A0B894CBDA2
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

PKEY values are defined in Propkey.h.
