# System.Devices.ServiceId

Identifier of the device service.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.ServiceId
   shellPKey = PKEY_Devices_ServiceId
   formatID = 656A3BB3-ECC0-43FD-8477-4AE0404A96CD
   propID = 16385
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
