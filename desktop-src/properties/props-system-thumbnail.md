# System.Thumbnail

Represents the thumbnail in VT\_CF format.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.Thumbnail
   shellPKey = PKEY_Thumbnail
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Clipboard
      IsInnate = false
```

## Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Thumbnail
   shellPKey = PKEY_Thumbnail
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Clipboard
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
