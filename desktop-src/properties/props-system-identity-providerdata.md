# System.Identity.ProviderData

The provider's custom data for an identity.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.ProviderData
   shellPKey = PKEY_Identity_ProviderData
   formatID = A8A74B92-361B-4E9A-B722-7C4A7330A312
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
