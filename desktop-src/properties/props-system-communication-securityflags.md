# System.Communication.SecurityFlags

Security flags associated with the item to indicate whether the item is encrypted, signed, or DRM enabled.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.SecurityFlags
   shellPKey = PKEY_Communication_SecurityFlags
   formatID = 8619A4B6-9F4D-4429-8C0F-B996CA59E335
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
```

## Remarks

PKEY values are defined in Propkey.h.
