# System.Photo.FlashText

The user-friendly form of [System.Photo.Flash](./props-system-photo-flash.md). Not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashText
   shellPKey = PKEY_Photo_FlashText
   formatID = 6B8B68F6-200B-47EA-8D25-D8050F57339F
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
