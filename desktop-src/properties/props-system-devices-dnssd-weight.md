# System.Devices.Dnssd.Weight

SRV record weight field. Not usually used.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.Weight
   shellPKey = PKEY_Devices_Dnssd_Weight
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
