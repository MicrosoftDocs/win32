# System.DeviceInterface.Bluetooth.ModelNumber

Bluetooth device model number.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.ModelNumber
   shellPKey = PKEY_DeviceInterface_Bluetooth_ModelNumber
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```
