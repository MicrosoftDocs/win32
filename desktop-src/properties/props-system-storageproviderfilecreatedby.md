# System.StorageProviderFileCreatedBy

The display name of the user who created the file or folder specified by the storage provider.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileCreatedBy
   shellPKey = PKEY_StorageProviderFileCreatedBy
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
