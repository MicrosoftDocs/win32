# System.Devices.AepService.Bluetooth.ServiceGuid

Bluetooth service GUID.

## Windows 10, version 1703, Windows 10, version 1607

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.ServiceGuid
   shellPKey = PKEY_Devices_AepService_Bluetooth_ServiceGuid
   formatID = A399AAC7-C265-474E-B073-FFCE57721716
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
