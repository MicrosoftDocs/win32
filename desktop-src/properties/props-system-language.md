# System.Language

The primary language of the file, particularly if that file is a document.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.Language
   shellPKey = PKEY_Language
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 28
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Language
   shellPKey = PKEY_Language
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 28
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
