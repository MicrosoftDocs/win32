# System.Contact.PagerTelephone

Pager telephone number of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.PagerTelephone
   shellPKey = PKEY_Contact_PagerTelephone
   formatID = D6304E01-F8F5-4F45-8B15-D024A6296789
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
