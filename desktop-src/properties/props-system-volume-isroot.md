# System.Volume.IsRoot

A Boolean value stating whether a volume is a root volume.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Volume.IsRoot
   shellPKey = PKEY_Volume_IsRoot
   formatID = 9B174B35-40FF-11D2-A27E-00C04FC30871
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
