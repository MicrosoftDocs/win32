# System.GPS.LatitudeDenominator

The denominator of PKEY\_GPS\_Latitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.LatitudeDenominator
   shellPKey = PKEY_GPS_LatitudeDenominator
   formatID = 16E634EE-2BFF-497B-BD8A-4341AD39EEB9
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt32
```

PKEY values are defined in Propkey.h.
