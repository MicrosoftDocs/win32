# System.Devices.ClassGuid

Device Class Guid.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.ClassGuid
   shellPKey = PKEY_Devices_ClassGuid
   formatID = A45C254E-DF1C-4EFD-8020-67D146A850E0
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```
