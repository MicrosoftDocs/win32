# System.StorageProviderFileModifiedBy

The display name of the user who last modified the file or folder specified by the storage provider.

## Windows 11, version 22H2

```
propertyDescription
   name = System.StorageProviderFileModifiedBy
   shellPKey = PKEY_StorageProviderFileModifiedBy
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
