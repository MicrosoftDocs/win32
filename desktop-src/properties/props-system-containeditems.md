# System.ContainedItems

A list of the type of content in the item. This value is represented as a vector array of GUIDs, where each GUID represents a certain type, such as URLs or attachments.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ContainedItems
   shellPKey = PKEY_ContainedItems
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 29
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue Guid
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
