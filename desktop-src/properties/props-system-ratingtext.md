# System.RatingText

The user-friendly form of [System.Rating](./props-system-rating.md). This value is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.RatingText
   shellPKey = PKEY_RatingText
   formatID = 90197CA7-FD8F-4E8C-9DA3-B57E1E609295
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
