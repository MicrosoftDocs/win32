# System.Contact.AssistantName

Contact's assistant name.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.AssistantName
   shellPKey = PKEY_Contact_AssistantName
   formatID = CD102C9C-5540-4A88-A6F6-64E4981C8CD1
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
