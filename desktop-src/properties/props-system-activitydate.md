# System.ActivityDate

The date and time of when the activity described in [System.ActivityInfo](./props-system-activityinfo.md) took place for this item.

## Windows 11, version 24H2

```
propertyDescription
   name = System.ActivityDate
   shellPKey = PKEY_ActivityDate
   formatID = 30C8EEF4-A832-41E2-AB32-E3C3CA28FD29
   propID = 23
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = DateTime
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.

If [System.ActivityInfo](./props-system-activityinfo.md) is VT_EMPTY, then this property should be too.
