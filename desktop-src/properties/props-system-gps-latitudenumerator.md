# System.GPS.LatitudeNumerator

The numerator of PKEY\_GPS\_Latitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.LatitudeNumerator
   shellPKey = PKEY_GPS_LatitudeNumerator
   formatID = 7DDAAAD1-CCC8-41AE-B750-B2CB8031AEA2
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt32
```
