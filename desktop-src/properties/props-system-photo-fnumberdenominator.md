# System.Photo.FNumberDenominator

The denominator of PKEY\_Photo\_FNumber.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FNumberDenominator
   shellPKey = PKEY_Photo_FNumberDenominator
   formatID = E92A2496-223B-4463-A4E3-30EABBA79D80
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
