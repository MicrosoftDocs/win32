# System.Devices.WiFiDirect.IsConnected

Indicates Wi-Fi Direct Device's Connectivity State

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsConnected
   shellPKey = PKEY_Devices_WiFiDirect_IsConnected
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
