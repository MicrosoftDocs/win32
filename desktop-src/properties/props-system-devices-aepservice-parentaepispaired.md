# System.Devices.AepService.ParentAepIsPaired

Whether the parent Device Association Endpoint is paired with the system or not

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepService.ParentAepIsPaired
   shellPKey = PKEY_Devices_AepService_ParentAepIsPaired
   formatID = C9C141A9-1B4C-4F17-A9D1-F298538CADB8
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
