# System.DeviceInterface.WinUsb.DeviceInterfaceClasses

WinUSB device interface GUID(s) used to open a handle to the device.

```
propertyDescription
   name = System.DeviceInterface.WinUsb.DeviceInterfaceClasses
   shellPKey = PKEY_DeviceInterface_WinUsb_DeviceInterfaceClasses
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
