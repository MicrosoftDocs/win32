# System.Devices.Dnssd.FullName

Complete DNS-SD service instance name, including instance, service, and domain.

```

   name = System.Devices.Dnssd.FullName
   shellPKey = PKEY_Devices_Dnssd_FullName

   propID = 5
   SearchInfo
      InInvertedIndex = false
```
