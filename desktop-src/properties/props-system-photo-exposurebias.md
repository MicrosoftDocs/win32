# System.Photo.ExposureBias

The amount of exposure bias used in the photo, as read from the camera. This property is calculated from [System.Photo.ExposureBiasNumerator](https://msdn.microsoft.com/library/Bb760426(v=VS.85).aspx) and [System.Photo.ExposureBiasDenominator](https://msdn.microsoft.com/library/Bb760424(v=VS.85).aspx). The unit is the APEX value. Ordinarily it ranges from -99.99 through 99.99.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureBias
   shellPKey = PKEY_Photo_ExposureBias
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 37380
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
