# System.Contact.FirstName

Indicates the first name of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.FirstName
   shellPKey = PKEY_Contact_FirstName
   formatID = 14977844-6B49-4AAD-A714-A4513BF60460
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
