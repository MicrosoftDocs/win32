# System.VolumeId

The GUID of the NTFS Volume.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.VolumeId
   shellPKey = PKEY_VolumeId
   formatID = 446D16B1-8DAD-4870-A748-402EA43D788C
   propID = 104
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Guid
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
