# System.Message.ProofInProgress

Identifies whether the message junk e-mail proofing is still in progress.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Message.ProofInProgress
   shellPKey = PKEY_Message_ProofInProgress
   formatID = 9098F33C-9A7D-48A8-8DE5-2E1227A64E91
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
