# System.Calendar.RequiredAttendeeNames

Names of all the required attendees.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.RequiredAttendeeNames
   shellPKey = PKEY_Calendar_RequiredAttendeeNames
   formatID = B33AF30B-F552-4584-936C-CB93E5CDA29F
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

PKEY values are defined in Propkey.h.
