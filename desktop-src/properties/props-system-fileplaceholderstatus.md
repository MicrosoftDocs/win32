# System.FilePlaceholderStatus

Contains the placeholder file's status flags.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.FilePlaceholderStatus
   shellPKey = PKEY_FilePlaceholderStatus
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
