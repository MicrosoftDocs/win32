# System.StorageProviderFileRemoteUri

The storage provider's remote Uri for this file.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileRemoteUri
   shellPKey = PKEY_StorageProviderFileRemoteUri
   formatID = FCEFF153-E839-4CF3-A9E7-EA22832094B8
   propID = 112
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
