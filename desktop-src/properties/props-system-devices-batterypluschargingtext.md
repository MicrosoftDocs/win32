# System.Devices.BatteryPlusChargingText

The remaining battery life of the device, and the device's charging state as a string.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.BatteryPlusChargingText
   shellPKey = PKEY_Devices_BatteryPlusChargingText
   formatID = 49CD1F76-5626-4B17-A4E8-18B4AA1A2213
   propID = 23
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
