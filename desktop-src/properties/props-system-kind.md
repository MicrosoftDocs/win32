## Remarks

PKEY values are defined in Propkey.h.

Extensions are mapped to Kind values in the registry at:

```
HKEY_LOCAL_MACHINE
   Software
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  KindMap
```

The list of Kind values is not extensible.
