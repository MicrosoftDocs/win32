# System.Search.EntryID

The entry ID for an item within a given catalog in the Windows Search Index.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.EntryID
   shellPKey = PKEY_Search_EntryID
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## Remarks

[System.Search.EntryID](https://msdn.microsoft.com/library/Bb760166(v=VS.85).aspx) is used in the SQL that is generated to query the index. This value is not considered unique over time because it may be recycled.

PKEY values are defined in Propkey.h.
