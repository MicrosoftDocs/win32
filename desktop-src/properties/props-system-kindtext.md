# System.KindText

The user-friendly form of [System.Kind](https://msdn.microsoft.com/library/Bb787521(v=VS.85).aspx). This value is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.KindText
   shellPKey = PKEY_KindText
   formatID = F04BEF95-C585-4197-A2B7-DF46FDC9EE6D
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
