# System.DeviceInterface.Hid.IsReadOnly

Indicates if a HID device is a Read-Only device.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Hid.IsReadOnly
   shellPKey = PKEY_DeviceInterface_Hid_IsReadOnly
   formatID = CBF38310-4A17-4310-A1EB-247F0B67593B
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
