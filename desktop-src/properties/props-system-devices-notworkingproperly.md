# System.Devices.NotWorkingProperly

If VARIANT\_TRUE, the device is not working properly.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.NotWorkingProperly
   shellPKey = PKEY_Devices_NotWorkingProperly
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 83
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enum
            name = NotWorkingProperly
            value = #TRUE#
            text = Needs troubleshooting
            defineToken = ISNOTWORKINGPROPERLY_NOTWORKINGPROPERLY
```
