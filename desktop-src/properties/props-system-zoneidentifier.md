# System.ZoneIdentifier

Mark of the Web zone, as URLZONE enumeration value.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.ZoneIdentifier
   shellPKey = PKEY_ZoneIdentifier
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
