# System.Supplemental.ResourceId

Contains the identifier for the item on the remote sync service. Used for comparing a file on the system to ones that are available in the cloud.

## Windows 10, version 1703

```
propertyDescription
   name = System.Supplemental.ResourceId
   shellPKey = PKEY_Supplemental_ResourceId
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
