# System.StorageProviderFileDateShared

The most recent date and time the file or folder was shared by any user, specified by the storage provider.

## Windows 11, version 24H2

```
propertyDescription
   name = System.StorageProviderFileDateShared
   shellPKey = PKEY_StorageProviderFileDateShared
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = DateTime
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
