# System.Devices.Aep.DeviceAddress

Address based on the protocol of the Device Association Endpoint. IP Address for an IP device, Bluetooth address for Bluetooth device, etc.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Aep.DeviceAddress
   shellPKey = PKEY_Devices_Aep_DeviceAddress
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
