# System.LibraryLocationsCount

Indicates the number of library locations.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.LibraryLocationsCount
   shellPKey = PKEY_LibraryLocationsCount
   formatID = 908696C7-8F87-44F2-80ED-A8C1C6894575
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
