# System.Devices.WiFiDirect.Services

Indicates services supported by the Wi-Fi Direct Device

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.Services
   shellPKey = PKEY_Devices_WiFiDirect_Services
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
