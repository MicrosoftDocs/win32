# System.StorageProviderUserId

The unique identifier for the user who is authenticated with this storage provider.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderUserId
   shellPKey = PKEY_StorageProviderUserId
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
