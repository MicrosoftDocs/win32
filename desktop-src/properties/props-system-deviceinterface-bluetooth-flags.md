# System.DeviceInterface.Bluetooth.Flags

Bluetooth device flags.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.Flags
   shellPKey = PKEY_DeviceInterface_Bluetooth_Flags
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```
