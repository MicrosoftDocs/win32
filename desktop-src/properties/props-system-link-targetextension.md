# System.Link.TargetExtension

The file extension of the link target. See [System.FileExtension](./props-system-fileextension.md)

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Link.TargetExtension
   shellPKey = PKEY_Link_TargetExtension
   formatID = 7A7D76F4-B630-4BD7-95FF-37CC51A975C9
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## Remarks

PKEY values are defined in Propkey.h.
