# System.Photo.FlashEnergy

Indicates the strobe energy at the time the image was captured, measured in Beam Candle Power Seconds. Calculated from PKEY\_Photo\_FlashEnergyNumerator and PKEY\_Photo\_FlashEnergyDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashEnergy
   shellPKey = PKEY_Photo_FlashEnergy
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 41483
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
