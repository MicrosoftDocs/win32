# System.Devices.AepContainer.SupportsInformation

Indicates if the Device Association Endpoint Container Supports Information

## Windows 10, version 1703, Windows 10, version 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsInformation
   shellPKey = PKEY_Devices_AepContainer_SupportsInformation
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
