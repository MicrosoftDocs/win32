# System.Devices.Category

Singular form of device category.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Category
   shellPKey = PKEY_Devices_Category
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 91
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Windows 7

```
propertyDescription
   name = System.Devices.Category
   shellPKey = PKEY_Devices_Category_Desc_Singular
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 91
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
