# System.Category

Deprecated. The category that can be assigned to an item such as a document or file. This property is inherited from OLE document properties and is deprecated for Windows Vista. Keywords should be used instead. Older code treats this property as VT\_LPSTR.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Category
   shellPKey = PKEY_Category
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## Remarks

PKEY values are defined in Propkey.h.
