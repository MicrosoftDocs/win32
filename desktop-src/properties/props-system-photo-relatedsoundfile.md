# System.Photo.RelatedSoundFile

The file name of a sound annotation file associated with the photo.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.RelatedSoundFile
   shellPKey = PKEY_Photo_RelatedSoundFile
   formatID = 318A6B45-087F-4DC2-B8CC-05359551FC9E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
