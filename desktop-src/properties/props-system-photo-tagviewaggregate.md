# System.Photo.TagViewAggregate

A read-only aggregation of tag-like properties for use in building views.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Photo.TagViewAggregate
   shellPKey = PKEY_Photo_TagViewAggregate
   formatID = B812F15D-C2D8-4BBF-BACD-79744346113F
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
