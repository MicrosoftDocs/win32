# System.Devices.Aep.IsPaired

Whether the Device Association Endpoint is paired with the system or not

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Aep.IsPaired
   shellPKey = PKEY_Devices_Aep_IsPaired
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 16
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
