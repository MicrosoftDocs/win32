# System.FolderKind

This property represents the types of content stored in this folder specified by the storage provider.Each folder type must be one of the known value specified by System.Kind definitionSystem.FolderKind is a readonly property, it should only be updated by the storage provider.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.FolderKind
   shellPKey = PKEY_FolderKind
   formatID = FCEFF153-E839-4CF3-A9E7-EA22832094B8
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
