# System.Contact.Label

Contact's calendar event label.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.Label
   shellPKey = PKEY_Contact_Label
   formatID = 97B0AD89-DF49-49CC-834E-660974FD755B
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
