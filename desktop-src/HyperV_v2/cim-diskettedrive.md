# CIM_DisketteDrive class (Hyper-V management)

Represents a diskette drive.

## Syntax

``` syntax
[Abstract, Version("2.6.0"), UMLPackagePath("CIM::Device::StorageDevices"), AMENDMENT]
class CIM_DisketteDrive : CIM_MediaAccessDevice
{
};
```

## Members

The **CIM\_DisketteDrive** class does not define any members.
