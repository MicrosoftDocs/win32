# CIM_DiskDrive class (Hyper-V management)

Represents a disk drive.

## Syntax

``` syntax
[Abstract, Version("2.6.0"), UMLPackagePath("CIM::Device::StorageDevices"), AMENDMENT]
class CIM_DiskDrive : CIM_MediaAccessDevice
{
};
```

## Members

The **CIM\_DiskDrive** class does not define any members.
