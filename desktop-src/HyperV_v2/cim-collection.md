# CIM\_Collection class

A superclass for classes that represent a collection of [**CIM\_ManagedElement**](cim-managedelement.md) objects and the subclasses of managed elements.

## Syntax

``` syntax
[Abstract, Version("2.6.0"), UMLPackagePath("CIM::Core::Collection"), AMENDMENT]
class CIM_Collection : CIM_ManagedElement
{
};
```

## Members

The **CIM\_Collection** class does not define any members.
