# StopService method of the Msvm\_SecurityService class

Stops the service.

## Syntax


```mof
uint32 StopService();
```



## Parameters

This method has no parameters.

## Return value

On success, returns a 0; otherwise, returns an error.

<dl> <dt>

**Completed with No Error** (0)
</dt> <dt>

**Not supported** (1)
</dt> </dl>
