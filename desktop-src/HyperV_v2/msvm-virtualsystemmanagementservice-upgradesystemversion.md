# UpgradeSystemVersion method of the Msvm\_VirtualSystemManagementService class

Upgrades the virtual system.

When applied to the system settings of a "current" virtual system configuration

## Syntax


```mof
uint32 UpgradeSystemVersion(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 UpgradeSettingData,
  [out] CIM_ConcreteJob    REF Job
);
```



## Parameters

<dl> <dt>

*ComputerSystem* \[in\]
</dt> <dd>

A reference to a [**CIM\_ComputerSystem**](cim-computersystem.md) that represents the virtual computer system to be upgraded.

</dd> <dt>

*UpgradeSettingData* \[in\]
</dt> <dd>

The upgrade setting data.

</dd> <dt>

*Job* \[out\]
</dt> <dd>

If the operation is performed asynchronously, this method will return 4096, and this parameter will contain a reference to an object derived from [**CIM\_ConcreteJob**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>

## Return value

On success, returns a 0 or 4096; otherwise, returns an error.

</dt> <dt>

**Incompatible Parameters** (6)
</dt> <dt>

</dt> <dt>

</dt> <dt>

**Vendor Specific** (32768..65535)
</dt> </dl>
