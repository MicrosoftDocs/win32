# StopService method of the Msvm\_ReplicationService class

Stops the service.

## Syntax


```mof
uint32 StopService();
```



## Parameters

This method has no parameters.

## Return value

The method returns one of the following values.

<dl> <dt>

**Completed with No Error** (0)
</dt> <dt>

**Not supported** (1)
</dt> </dl>
