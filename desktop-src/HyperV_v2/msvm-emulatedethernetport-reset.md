# Reset method of the Msvm\_EmulatedEthernetPort class

Resets the virtual device.

## Syntax


```mof
uint32 Reset();
```



## Parameters

This method has no parameters.

## Return value

This method returns one of the following values:

<dl> <dt>

**Completed with No Error** (0)
</dt> <dt>

**Not supported** (1)
</dt> </dl>
