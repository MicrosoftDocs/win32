# CIM_Display class (Hyper-V management)

A superclass for classes that represent display devices.

## Syntax

``` syntax
[Abstract, Version("2.6.0"), UMLPackagePath("CIM::Device::UserDevices")]
class CIM_Display : CIM_UserDevice
{
};
```

## Members

The **CIM\_Display** class does not define any members.
