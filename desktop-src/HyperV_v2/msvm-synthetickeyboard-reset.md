# Reset method of the Msvm\_SyntheticKeyboard class

Resets the virtual keyboard.

## Syntax


```mof
uint32 Reset();
```



## Parameters

This method has no parameters.

## Return value

On success, returns 0; otherwise, returns an error.

<dl> <dt>

**Completed with No Error** (0)
</dt> <dt>

**Not supported** (1)
</dt> </dl>
