# Reset method of the Msvm\_Ps2Mouse class

Requests a reset.

## Syntax


```mof
uint32 Reset();
```



## Parameters

This method has no parameters.

## Return value

This method returns one of the following values:

<dl> <dt>

**Completed with No Error** (0)
</dt> <dt>

**Not supported** (1)
