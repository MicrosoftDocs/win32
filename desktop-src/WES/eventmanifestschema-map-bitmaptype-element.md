# map (BitMapType) Element

Defines the mapping between a bit value and a string value.

``` syntax
<xs:element name="map"
    type="BitMapValueType"
 />
```

The **map** element is defined by the [**BitMapType**](eventmanifestschema-bitmaptype-complextype.md) complex type.
