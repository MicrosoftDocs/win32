# maps (ProviderType) Element

Defines a list of name/value pairs that you can reference in the template section of the manifest.

``` syntax
<xs:element name="maps"
    type="MapType"
 />
```

The **maps** element is defined by the [**ProviderType**](eventmanifestschema-providertype-complextype.md) complex type.
