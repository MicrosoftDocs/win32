# importChannel (ChannelListType) Element

Identifies a channel that has been defined by another provider.

``` syntax
<xs:element name="importChannel"
    type="ImportChannelType"
 />
```

The **importChannel** element is defined by the [**ChannelListType**](eventmanifestschema-channellisttype-complextype.md) complex type.
