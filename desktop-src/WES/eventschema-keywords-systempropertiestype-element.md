# Keywords (SystemPropertiesType) Element

A bitmask of the keywords defined in the event.

``` syntax
<xs:element name="Keywords"
    type="HexInt64Type"
 />
```

The **Keywords** element is defined by the [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md) complex type.
