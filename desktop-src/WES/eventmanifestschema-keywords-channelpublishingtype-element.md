# keywords (ChannelPublishingType) Element

A bitmask that determines the category of events that are written to the channel.

``` syntax
<xs:element name="keywords"
    type="UInt64Type"
 />
```

The **keywords** element is defined by the [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) complex type.
