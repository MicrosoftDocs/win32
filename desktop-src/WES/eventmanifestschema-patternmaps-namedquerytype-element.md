# patternMaps (NamedQueryType) Element

Defines a list of regular expression pairs that are used to alter the message string.

``` syntax
<xs:element name="patternMaps"
    type="PatternMapListType"
 />
```

The **patternMaps** element is defined by the [**NamedQueryType**](eventmanifestschema-namedquerytype-complextype.md) complex type.
