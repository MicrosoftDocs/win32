# ProcessingErrorData (EventType) Element

Contains details of the error that occurred while trying to render the event.

``` syntax
<xs:element name="ProcessingErrorData"
    type="ProcessingErrorDataType"
 />
```

The **ProcessingErrorData** element is defined by the [**EventType**](eventschema-eventtype-complextype.md) complex type.
