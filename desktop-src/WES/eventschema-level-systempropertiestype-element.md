# Level (SystemPropertiesType) Element

Contains the severity level of the event.

``` syntax
<xs:element name="Level"
    type="unsignedByte"
 />
```

The **Level** element is defined by the [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md) complex type.
