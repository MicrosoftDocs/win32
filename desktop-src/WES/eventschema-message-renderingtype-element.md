# Message (RenderingInfoType) Element

Contains the event message that is rendered for the event.

``` syntax
<xs:element name="Message"
    type="string"
 />
```

The **Message** element is defined by the [**RenderingInfoType**](eventschema-renderingtype-complextype.md) complex type.
