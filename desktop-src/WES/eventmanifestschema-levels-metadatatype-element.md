# levels (MetadataType) Element

Defines a list of levels that specify the severity of an event.

``` syntax
<xs:element name="levels"
    type="LevelListType"
 />
```

The **levels** element is defined by the [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) complex type.
