# Component (DebugDataType) Element

The name of the component that logged the trace message.

``` syntax
<xs:element name="Component"
    type="string"
 />
```

The **Component** element is defined by the [**DebugDataType**](eventschema-debugdatatype-complextype.md) complex type.
