# opcodes (ProviderType) Element

Defines a list of opcodes that you can use to group events within a task.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

The **opcodes** element is defined by the [**ProviderType**](eventmanifestschema-providertype-complextype.md) complex type.
