# valueMap (MapType) Element

Defines a list of name/value pairs that map integer values and string values.

``` syntax
<xs:element name="valueMap"
    type="ValueMapType"
 />
```

The **valueMap** element is defined by the [**MapType**](eventmanifestschema-maptype-complextype.md) complex type.
