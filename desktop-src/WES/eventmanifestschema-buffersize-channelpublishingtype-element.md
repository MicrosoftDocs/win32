# bufferSize (ChannelPublishingType) Element

Contains the amount of memory, in kilobytes, allocated for the session buffers.

``` syntax
<xs:element name="bufferSize"
    type="unsignedInt"
 />
```

The **bufferSize** element is defined by the [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) complex type.

## Requirements



| Requirement | Value |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
