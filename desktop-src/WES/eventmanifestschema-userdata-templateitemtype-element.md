# UserData (TemplateItemType) Element

An XML fragment that is used to render the event data.

``` syntax
<xs:element name="UserData"
    type="XmlType"
 />
```

The **UserData** element is defined by the [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md) complex type.
