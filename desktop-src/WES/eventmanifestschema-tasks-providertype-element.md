# tasks (ProviderType) Element

Defines a list of tasks that a provider can use to group events.

``` syntax
<xs:element name="tasks"
    type="TaskListType"
 />
```

The **tasks** element is defined by the [**ProviderType**](eventmanifestschema-providertype-complextype.md) complex type.
