# level (LevelListType) Element

Defines a severity value that determines the verbosity of events during logging.

``` syntax
<xs:element name="level"
    type="LevelType"
 />
```

The **level** element is defined by the [**LevelListType**](eventmanifestschema-levellisttype-complextype.md) complex type.
