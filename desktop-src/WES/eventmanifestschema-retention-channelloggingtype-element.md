# retention (ChannelLoggingType) Element

Determines whether the log file is a sequential or circular log file.

``` syntax
<xs:element name="retention"
    type="boolean"
 />
```

The **retention** element is defined by the [**ChannelLoggingType**](eventmanifestschema-channelloggingtype-complextype.md) complex type.

|-------------------------------------|------------------------------------------------------|
