# DebugData (EventType) Element

Contains the data that can be logged for Windows software trace preprocessor (WPP) events.

``` syntax
<xs:element name="DebugData"
    type="DebugDataType"
 />
```

The **DebugData** element is defined by the [**EventType**](eventschema-eventtype-complextype.md) complex type.

## Requirements



| Requirement | Value |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
