# event (DefinitionType) Element

Defines an event that your provider can log.

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

The **event** element is defined by the [**DefinitionType**](eventmanifestschema-definitiontype-complextype.md) complex type.
