# instrumentation (instrumentationManifest) Element

Defines the types of instrumentation that the manifest contains (for example, event or counter instrumentation).

``` syntax
<xs:element name="instrumentation"
    type="InstrumentationType"
 />
```

The **instrumentation** element is defined by the [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md) element.
