# SubComponent (DebugDataType) Element

The name of the subcomponent that logged the trace message.

``` syntax
<xs:element name="SubComponent"
    type="string"
 />
```

The **SubComponent** element is defined by the [**DebugDataType**](eventschema-debugdatatype-complextype.md) complex type.
