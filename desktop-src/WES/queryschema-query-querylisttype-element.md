# Query (QueryListType) Element

Defines a set of selectors and suppressors that are used to include events in or exclude events from the result set.

``` syntax
<xs:element name="Query"
    type="QueryType"
 />
```

The **Query** element is defined by the [**QueryListType**](queryschema-querylisttype-complextype.md) complex type.
