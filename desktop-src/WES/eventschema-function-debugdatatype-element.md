# Function (DebugDataType) Element

The name of the function that logged the trace message.

``` syntax
<xs:element name="Function"
    type="string"
    minOccurs="0"
 />
```

The **Function** element is defined by the [**DebugDataType**](eventschema-debugdatatype-complextype.md) complex type.
