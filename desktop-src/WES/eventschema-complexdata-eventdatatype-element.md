# ComplexData (EventDataType) Element

A structure that is defined in the template for the event.

``` syntax
<xs:element name="ComplexData"
    type="ComplexDataType"
 />
```

The **ComplexData** element is defined by the [**EventDataType**](eventschema-eventdatatype-complextype.md) complex type.
