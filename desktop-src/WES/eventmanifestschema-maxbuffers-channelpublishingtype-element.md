# maxBuffers (ChannelPublishingType) Element

The maximum number of buffers to allocate for the session.

``` syntax

    type="unsignedInt"
 />
```

The **maxBuffers** element is defined by the [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) complex type.
