# outType (InputType) Element

A list of output data types that determine how the data is rendered.

``` syntax
<xs:element name="outType"
    type="OutputType"
 />
```

The **outType** element is defined by the [**InputType**](eventmanifestschema-inputtype-complextype.md) complex type.
