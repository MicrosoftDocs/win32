# SequenceNumber (DebugDataType) Element

The local or global sequence number of the trace message.

``` syntax
<xs:element name="SequenceNumber"
    type="unsignedInt"
    minOccurs="0"
 />
```

The **SequenceNumber** element is defined by the [**DebugDataType**](eventschema-debugdatatype-complextype.md) complex type.
