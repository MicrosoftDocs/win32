# RenderingInfo (EventType) Element

Contains the rendered message strings for the event (includes the event's message string and the message strings for any of the event's properties such as level, task, and opcode).

``` syntax
<xs:element name="RenderingInfo"
    type="RenderingType"
 />
```

The **RenderingInfo** element is defined by the [**EventType**](eventschema-eventtype-complextype.md) complex type.
