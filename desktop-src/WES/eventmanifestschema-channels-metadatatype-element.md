# channels (MetadataType) Element

Defines a list of channels to which providers can log events.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

The **channels** element is defined by the [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) complex type.
