# xmlTypes (TypeListType) Element

Defines a list of output types that the service uses to determine how to render an input data type.

``` syntax
<xs:element name="xmlTypes"
    type="XmlTypeListType"
 />
```

The **xmlTypes** element is defined by the [**TypeListType**](eventmanifestschema-typelisttype-complextype.md) complex type.
