# opcodes (TaskType) Element

Defines a list of task-specific opcodes.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

The **opcodes** element is defined by the [**TaskType**](eventmanifestschema-tasktype-complextype.md) complex type.
