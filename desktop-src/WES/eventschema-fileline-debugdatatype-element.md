# FileLine (DebugDataType) Element

The name of the source file and the line within the source file that logged the trace message.

``` syntax
<xs:element name="FileLine"
    type="string"
 />
```

The **FileLine** element is defined by the [**DebugDataType**](eventschema-debugdatatype-complextype.md) complex type.

|-------------------------------------|------------------------------------------------------|
