# level (ChannelPublishingType) Element

The severity level of the events to write to the channel.

``` syntax
<xs:element name="level"
    type="UInt8Type"
 />
```

The **level** element is defined by the [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) complex type.
