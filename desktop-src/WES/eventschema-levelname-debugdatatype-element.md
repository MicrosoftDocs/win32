# LevelName (DebugDataType) Element

The level value passed to the provider when it was enabled.

``` syntax
<xs:element name="LevelName"
    type="string"
 />
```

The **LevelName** element is defined by the [**DebugDataType**](eventschema-debugdatatype-complextype.md) complex type.
