# Version (SystemPropertiesType) Element

Contains the version number of the event's definition.

``` syntax
<xs:element name="Version"
    type="unsignedByte"
 />
```

The **Version** element is defined by the [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md) complex type.

|-------------------------------------|------------------------------------------------------|
