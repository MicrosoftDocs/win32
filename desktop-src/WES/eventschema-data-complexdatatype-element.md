# Data (ComplexDataType) Element

The list of data items in the structure. The list of items are in the same order as defined in the template.

``` syntax
<xs:element name="Data"
    type="DataType"
 />
```

The **Data** element is defined by the [**ComplexDataType**](eventschema-complexdatatype-complextype.md) complex type.
