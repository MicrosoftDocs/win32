# BinaryEventData (EventType) Element

Contains the event data as a binary blob.

``` syntax
<xs:element name="BinaryEventData"
    type="hexBinary"
 />
```

The **BinaryEventData** element is defined by the [**EventType**](eventschema-eventtype-complextype.md) complex type.
