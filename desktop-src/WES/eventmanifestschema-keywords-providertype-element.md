# keywords (ProviderType) Element

Defines a list of keywords that categorize events.

``` syntax
<xs:element name="keywords"
    type="KeywordListType"
 />
```

The **keywords** element is defined by the [**ProviderType**](eventmanifestschema-providertype-complextype.md) complex type.
