# data (StructDefinitionType) Element

Defines a data item that you want to include in the structure.

``` syntax
<xs:element name="data"
    type="DataDefinitionType"
 />
```

The **data** element is defined by the [**StructDefinitionType**](eventmanifestschema-structdefinitiontype-complextype.md) complex type.
