# namedQueries (MetadataType) Element

Not used. Defines a list of named queries that query the event message string for a value and perform a specified action if found.

``` syntax
<xs:element name="namedQueries"
    type="NamedQueryType"
 />
```

The **namedQueries** element is defined by the [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) complex type.
