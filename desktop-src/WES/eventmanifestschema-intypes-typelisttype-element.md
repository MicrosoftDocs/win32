# inTypes (TypeListType) Element

Contains a list of input type definitions.

``` syntax
<xs:element name="inTypes"
    type="InputTypeListType"
 />
```

The **inTypes** element is defined by the [**TypeListType**](eventmanifestschema-typelisttype-complextype.md) complex type.
