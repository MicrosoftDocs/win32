# keywords (MetadataType) Element

Defines a list of keywords that categorize events.

``` syntax
<xs:element name="keywords"
    type="KeywordListType"
 />
```

The **keywords** element is defined by the [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) complex type.
