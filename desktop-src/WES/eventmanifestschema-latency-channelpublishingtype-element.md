# latency (ChannelPublishingType) Element

The time to wait before flushing the buffers, in milliseconds.

``` syntax
<xs:element name="latency"
    type="unsignedInt"
 />
```

The **latency** element is defined by the [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) complex type.
