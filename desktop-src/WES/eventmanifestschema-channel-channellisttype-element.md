# channel (ChannelListType) Element

Defines a channel to which events can be logged.

``` syntax
<xs:element name="channel"
    type="ChannelType"
 />
```

The **channel** element is defined by the [**ChannelListType**](eventmanifestschema-channellisttype-complextype.md) complex type.
