# FlagName (DebugDataType) Element

The flag value passed to the provider when it was enabled.

``` syntax
<xs:element name="FlagName"
    type="string"
 />
```

The **FlagName** element is defined by the [**DebugDataType**](eventschema-debugdatatype-complextype.md) complex type.
